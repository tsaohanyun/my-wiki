# C13 烟草行业方案集

> 项目：**售前方案集** | 分类：**C13-烟草行业** | 文件数：3
> 交叉引用：[[方案集/方案集知识库-总览]] | [[方案集/B01-MES制造执行系统|B01-MES制造执行系统]] | [[方案集/D07-GE方案集]]

## 概述

本分类汇集了烟草行业智能制造和MES系统相关的售前方案。烟草行业是中国制造业中信息化和自动化水平较高的行业之一，具有生产工艺复杂（制丝+卷接+包装）、质量管控严格、集团化管理等特点。方案聚焦于GE Proficy平台在烟草行业的MES应用和六西格玛过程能力测评。

## 1. 烟草制造工艺

### 卷烟生产流程

```
烟叶原料 → 打叶复烤 → 制丝（叶片/叶丝/梗丝） → 卷接 → 包装 → 成品入库
              ↑              ↑                      ↑
           原料预处理    核心工艺环节           包装环节
```

### 制丝工艺（核心环节）

制丝是卷烟生产最关键的工艺环节，决定了产品的品质和风格：

1. **叶片处理**：回潮→切丝→烘丝→冷却→加香→贮丝
2. **叶丝处理**：膨胀→切丝→烘丝→冷却→加香→贮丝
3. **梗丝处理**：洗梗→压梗→切丝→烘丝→冷却→贮丝
4. **分组加工**：根据品牌配方，对不同组分分别处理后掺配

### 工艺特点

- **全配方管理**：每个品牌有独立配方，精确控制各组分比例
- **分组加工**：不同烟叶组分分别处理，实现产品差异化
- **批次追溯**：从烟叶投料到成品出库的完整追溯链
- **精准控制**：温度、湿度、流量等工艺参数严格受控

## GE Proficy MES解决方案

### "三三"MES架构

GE针对烟草行业提出的独特的"三三"MES解决方案框架：

| 原则 | 含义 | 价值 |
|------|------|------|
| 三模统一 | 现场设备模型、生产执行跟踪模型、计划调度模型一致 | 打破传统金字塔模式，调度指令直达机台 |
| 三级协调 | 计划阶段→生产前→生产时的物料管理协调 | 物料及时准确供应，保证连续生产 |
| 三级联动 | 工单→设备控制→物料出库联动执行 | 实现一键式生产触发 |

### 核心功能模块

1. **系统管理**：用户权限、角色管理、系统配置
2. **生产管理**：工单下达、生产排程、执行跟踪
3. **工艺管理**：配方管理、工艺参数下发、分组加工控制
4. **质量管理**：在线检测、过程能力测评、质量追溯
5. **设备管理**：设备状态监控、维保管理
6. **系统建模**：产线建模、设备建模、物料建模
7. **报表系统**：生产报表、质量报表、KPI看板

### 常德卷烟厂案例

**项目概况**
- 客户：常德卷烟厂（湖南中烟）
- 产品线：芙蓉王制丝专线
- 产能：6000公斤制丝生产线
- 特点：国内领先的分组加工工艺
- MES平台：GE Proficy（iFix + Historian + Plant Application）
- 投运时间：2009年8月试运行，2009年10月量产

**系统架构**
- 现场采集：iFix双机冗余方式
- 实时数据库：Historian独立服务器
- MES主服务器：Stratus容错服务器
- 网络：控制网（ProfiNet千兆冗余环网）+ 管理网（双网架构）
- 对外发布：通过防火墙提供Web监控画面和报表

**实施要点**
- 计划调度层与MES执行跟踪层的物料需求协同
- 工单信息下达与设备厂商联调（工单状态字、控制通道号）
- 工艺参数随工单一同下达至生产现场
- 实现真正意义上的"管控一体化"

**运行效果**
- 系统运行稳定，客户高度评价
- 实现烟草加工管控一体化目标
- 为常德烟厂"数字烟草"平台奠定基础

## 2. 六西格玛过程能力测评

### 青岛卷烟厂案例

**项目概况**
- 客户：青岛卷烟厂（山东中烟）
- 目标：对制丝、卷包过程能力进行客观测评
- 平台：GE Proficy Plant Application + Historian

**系统架构**

| 组件 | 功能 |
|------|------|
| Historian实时历史数据库 | 高速采集现场数据，螺旋门压缩算法 |
| 西格玛应用服务器 | 搭建测评模型，维护工艺标准，计算查询 |
| 客户端 | 人工录入、合成数据、西格玛查询 |
| 接口机 | 与其他系统数据交互 |

**核心功能**
- 基础数据管理
- 测评模型管理
- 数据预处理
- 工艺标准管理
- 制丝车间西格玛合成模型
- 卷包计算模型
- 公式库管理
- 结果可视化展示

**应用效果**
- 制丝生产线特殊工序、关键工序得到有效控制
- 重点数据采集点总体合格率提升
- 卷接包生产线设备运行稳定
- 烟支物理指标控制显著改善

### 六西格玛测评方法论

```
定义工序 → 选择参数 → 设定标准 → 采集数据 → 计算西格玛水平
    ↓                                              ↓
统一评价指标                              发现质量短板 → 资源优化
```

- 统一测评工序、参数和评价指标，使各地工厂过程能力客观可比
- 通过西格玛水平对比，找到企业质量短板
- 根据短板进行资源分配，提高整体西格玛水平

## 烟草行业MES特点总结

### 与其他行业MES差异

| 维度 | 烟草行业 | 一般制造业 |
|------|----------|------------|
| 配方管理 | 全配方+分组加工 | BOM+配方 |
| 质量管控 | 六西格玛过程能力测评 | SPC统计过程控制 |
| 生产模式 | 连续+离散混合 | 以离散为主 |
| 合规要求 | 专卖体制+国家局标准 | 行业标准 |
| 集团管理 | 中烟公司统一管理 | 企业独立管理 |
| 工艺核心 | 制丝工艺（温湿度精准控制） | 各行业不同 |

### 数字化发展方向

- **智能在线评估**：在线实时过程能力评价
- **数字烟草平台**：全厂数字化集成
- **AI辅助品控**：基于机器学习的质量预测
- **柔性制造**：适应小批量多品种市场需求
