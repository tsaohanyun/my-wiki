## 概述

本方案集共收录 **16 份**钢铁冶金行业数字化转型方案，覆盖炼铁炼钢、连铸、轧钢、粉末冶金等核心工艺环节。内容涵盖钢铁生产工艺流程优化、MES系统建设、质量管控体系、能源管理、供应链协同、工业互联网平台等方向，为钢铁企业从原料投入到成品出库的全流程智能制造提供方案参考。

## 1. 钢铁生产工艺流程总览

**来源文件**：C06_01（钢铁厂生产工艺流程介绍PPT）

### 1.1 钢铁厂整体生产流程

现代化钢铁厂的生产涵盖从原料到成品的完整工艺链：

```
采矿 → 烧结/球团/焦化 → 炼铁 → 炼钢连铸 → 轧钢 → 成品
```

典型钢材产品分类：
- **线、棒材**
- **型材**（管、型、丝）
- **板材**：中厚板 → 热轧薄板 → 冷轧薄板 → 热镀锌板 → 彩涂板 → 汽车家电板 / 电工用板

### 1.2 炼铁工艺

**高炉工艺原理**：从炉顶装入烧结矿、燃料（焦炭）、溶剂（石灰石等），从炉腹风口向炉内吹送热风（900～1200℃）。燃料中的碳与热风中的氧发生燃烧反应，产生炙热的还原性煤气。煤气上升，炉料下降，经过加热和还原反应，铁矿中的氧被夺取，生成海绵铁，再溶化渗碳生成生铁。

**高炉七大系统**：
1. **高炉本体**：炉基、炉壳、炉衬、冷却设备、炉体框架；分为炉喉、炉身、炉腰、炉腹、炉缸五段
2. **上料系统**：炉料料仓 → 筛分 → 称量 → 料罐 → 皮带机
3. **装料系统**：受料斗、布料装置、探尺、均压放散阀；包括钟式炉顶和无料钟炉顶
4. **送风系统**：鼓风机、热风炉（内燃式/外燃式/顶燃式），风温最高可达1300℃
5. **煤气除尘系统**：上升管、下降管、重力除尘器、洗涤塔、文氏管、脱水器
6. **渣铁处理系统**：出铁场、渣铁沟、泥炮、开口机、炉前天车、水渣池
7. **喷吹系统**：煤粉制备、储煤罐、喷吹罐、混合器、喷枪

### 1.3 炼钢工艺

**钢与生铁的区别**：碳含量小于2.11%为钢（熔点1450-1500℃），生铁熔点1100-1200℃。碳含量增加，强度硬度增加，塑性和冲击韧性降低。

**炼钢基本任务**："四脱"（脱碳、脱氧、脱磷、脱硫），"二去"（去气、去夹杂），"二调整"（成分、温度）。

**氧气转炉种类**：
- 氧气顶吹转炉
- 氧气底吹转炉
- 氧气侧吹转炉
- 氧气顶底复合转炉

**转炉炼钢特点**：生产率高（冶炼时间20分钟以内）；质量好（有害成分<80ppm）；冶炼成本低；原材料适应性强。

### 1.4 连铸工艺

**技术演进**：
- 1947-1974年：转炉大型化，以模铸-初轧为核心
- 1974-1989年：全连铸工艺，以连铸机为核心
- 1989年至今：连铸-连轧工艺，薄板坯连铸连轧为代表，钢厂向紧凑化发展

**连铸工艺优势**：
- 提高综合成材率：连铸95-96%（vs 模铸84-88%）
- 降低能耗：每吨钢坯节能约130kg标准煤
- 产品均一性好，质量好
- 易于实现机械化和自动化

**连铸机机型**：立式、立弯式、多点弯曲立弯式、弧形、椭圆型（超低头）连铸机

**主要设备**：钢包及支撑装置、中间包、结晶器及振动装置、二次冷却区、铸坯导向和拉坯矫直机、切割装置

## 2. 钢铁行业数字化转型

**来源文件**：C06_03（2021年中国钢铁行业数字化转型短报告）、C06_09（钢铁行业数字化：智能制造重塑竞争力）

### 2.1 行业现状与痛点

中国是全球最大粗钢生产国，2020年粗钢年产量10.5亿吨，占全球产量约56.3%。钢材产量从2015年的11.2亿吨增长至2020年的13.2亿吨，CAGR为3.3%。

**三大核心痛点**：
1. **产能过剩**：2013-2018年间生产线过度投资导致产能集中释放，缺乏标准供需信息配置
2. **生产流程复杂**：炼铁、炼钢、轧钢三环节涉及多个生产系统和控制层级，流程复杂、体系庞大
3. **缺乏数据标准**：设备种类和应用场景繁多，各类工业环境及设备数据格式差异大，企业间数据难以兼容

2020年中国黑色金属冶炼企业数量已减少至5,000多家，传统技术主导领域亏损企业逐步增长。

### 2.2 数字化转型三大方向

1. **现代化数据资源体系建设**：主要技术为大数据、人工智能
2. **生产体系数字化改造**：主要技术为工业机器人、5G+
3. **生态体系数字化建设**：主要技术为物联网、5G建设网络基础设施

**数字化转型目标**：钢铁生产流程化、营销布局优化、成本可控化

### 2.3 数字化赋能钢铁价值链

| 环节 | 数字化应用 | 技术手段 |
|------|-----------|----------|
| **冶炼** | 铁水中硅含量预测、稳定炉况 | AI、大数据分析 |
| **监测** | 产钢设备状态检测、预警 | 传感器、AA分析技术 |
| **浇钢** | 无人浇钢、钢包智能追踪 | 传感器、自动化技术 |
| **能效** | 能源效率分析、优化 | 智能诊断、大数据 |
| **管理** | 各阶层管理信息互通、质量漏洞根源分析 | 数字化管理系统 |
| **销售** | 客户自助服务平台、线上+线下渠道 | 互联网平台 |
| **采购** | 采购效率提升、供应商协同 | 数字化工具、物联网 |

### 2.4 企业数字化层次升级需求

| 层次 | 升级内容 |
|------|----------|
| **企业层** | 多业务协同系统、ERP-MES-PCS、数据分析与运营 |
| **控制层** | 流程建模优化、生产自动化、料场无人化控制 |
| **设备层** | 检测监测智能化、机器人放料搬运、手持终端升级 |
| **网络层** | 全供应链协同、供应商协同、扩大IoT覆盖范围 |

### 2.5 行业数字化发展成果

- **十一五**：基础自动化全行业普及，重点钢铁企业全面实施MES
- **十二五**：关键工艺流程数控化率超65%，ERP装备率超70%；宝钢热连轧智能车间、鞍钢冶金数字矿山等示范项目
- **十三五**：全面推进智能制造，钢铁行业智能制造就绪率达7.1%（高于行业平均值）
- **2025目标**：关键工序数控化率达80%，生产设备数字化率达55%，打造50个以上智能工厂

## 3. 冶金行业MES系统方案

### 3.1 冶金行业MES需求分析

**来源文件**：C06_04（Proficy 冶金行业 MES 系统解决方案）

#### 3.1.1 行业流程特点

冶金行业以钢铁行业为例，是制造业中**流程最长**的行业，信息化最复杂。ERP系统在大型钢铁企业已基本普及，涵盖了采购、销售及财务等领域。

但MES（制造执行系统）在钢铁行业的成功实施案例较少。MES之难在于：
- 炼制过程中含碳量等指标不能完全稳定控制
- 炉内精炼、连铸等后续工序存在随时改判和动态调整的复杂性
- 上联ERP接收生产计划，下联PCS收集生产实绩，中间的协调难度大

#### 3.1.2 MES系统管理目标

- 实现生产过程组织、管理和决策的最优化
- 实现生产过程的快速反应与敏捷、精确制造
- 实现工厂自动化连续化均衡生产
- 实现产品生产交货期的准确预估
- 实现产品质量以及生产过程的可追溯性
- 实现产量、消耗、质量、设备状况等的全面动态可视和可控
- 建立预警指标，提供调度方案供决策

#### 3.1.3 MES系统功能要求

**生产配置管理**：工艺路径配置，适应不同产品的生产要求

**生产计划管理**：接受ERP主生产计划或手工输入，根据设备、原材料情况进行计划分解，支持优先级调整、计划插入和删除

**生产物料管理**：记录整个生产过程中批次原料使用情况、在制品、半成品、成品、废次品情况；物料成本分析；主辅料质量追溯

**生产过程监控**：实时监控现场生产状况、设备运行、质量状况、消耗状况，异常报警

**生产性能分析**：停机检测、故障原因分析、事件记录、废品分析、OEE（整体设备效率）计算

**质量管理与质量追踪**：质量标准管理、质量数据采集、在线质量控制、质量统计分析、质量追踪；通过批次编号进行全方位跟踪，对质量召回具有重大意义

**企业生产报表分析**：工艺质量统计分析（工序能力指数、质量检测统计）、设备统计分析（利用率、故障统计）、生产统计分析（产量、在制品、废品、投入产出）、消耗统计分析

**信息集成**：内部子系统集成、与ERP集成、与生产自动化系统集成

### 3.2 ISA 95标准与MES规范

**来源文件**：C06_04（续）

ISA 95是国际组织对MES（即ERP与生产低层自动化系统之间的管理和信息交换部分）的标准定义。

**ISA 95标准组成部分**：
1. **ISA-95.00.01**：模型和术语（2000年通过）
2. **ISA-95.00.02**：数据结构和属性（2001年通过）
3. **ISA-95.00.03**：制造业运作模型（2005年通过）
4. **ISA-95.00.04**：制造运作管理对象模型（草案）
5. **ISA-95.00.05**：商业系统与制造系统交易（草案）

**四种信息流**：
1. **产品定义信息**：描述如何制造一个产品（配方、生产指令等）
2. **生产能力信息**：说明需要的和可获得的生产资源的容量和能力
3. **生产计划信息**：说明何时何地生产何物及需要何种资源
4. **生产性能信息**：说明生产了什么、消耗了什么资源

**九大对象模型**：人力资源模型、设备资源模型、材料资源模型、过程段模型、过程段能力模型、生产能力模型、产品定义模型、生产计划模型、生产性能模型

### 3.4 钢铁企业MES生产计划管理

**来源文件**：C06_15（钢铁企业基于MES的生产计划管理设计）

钢铁行业生产过程兼具**离散和连续**双重特点，生产线批量小、产品品种多、生产过程复杂。

**核心问题**：
1. 传统ERP基于MRP分解，使用BOM（物料清单），但钢铁企业靠检索工艺文件进行工艺设计，用不到BOM，传统ERP难以发挥作用
2. ERP与PCS之间出现信息"断层"——计划与过程控制间无法有效衔接
3. 生产计划的制定长期依赖市场，控制系统与信息管理系统各成体系

**基于MES的生产计划管理方案**：
- **三层生产计划模型**：基于约束理论的一体化计划排程原理
- **关键技术**：基于动态倒序Petri网的调度排程技术、多模态甘特图技术、钢铁企业关联多层次冶金规范编码技术
- **一体化计划编制**：实现物料实时跟踪和合同动态控制跟踪，贯穿PDCA循环管理思想
- **系统架构**：采用J2EE多层体系结构，SOA架构设计

**生产模式转变**：从面向库存生产（MTS）到面向订单生产（MTO），客户需求呈现多品种、小批量特点。

## 4. PSImetals冶金行业MES方案

### 4.1 方案定位

PSImetals定位于冶金行业核心业务流程的最佳解决方案，覆盖**生产和物流**领域，是全球及本地的领跑者。

**关键竞争力**：
- 冶金具体业务咨询
- 业务计划、生产计划、生产执行
- 物流管理、能源管理

### 4.2 覆盖的冶金工序

| 工序 | 覆盖范围 |
|------|----------|
| 烧结、炼焦、炼铁 | 钢铁板材、不锈钢/特钢、长材、铝/铜、钢管 |

| 碳钢板材热轧 | 全球广泛客户基础 |
| 碳钢板材冷轧和精整 | 覆盖主要产线 |
| 中厚板 | 支持定制化需求 |
| 长材和钢管 | 无缝覆盖 |
| 铝业和铜业 | 有色金属生产管理 |

### 4.3 端到端流程支持

从原料到定制产品，从订单进入到发货的全流程集成优化：

**计划系统**：
- 需求和销售计划 → 交期承诺 → 主能力计划 → 订单排程 → 物料计划 → 产线序列 → 制造执行
- 数学算法优化+人工智慧结合，提供考虑各种约束的全流程详细生产计划

**质量管理**：
- 高度灵活的闭环：质量目标定义 → 实现质量要求 → 分析原因 → 目标调整
- 工艺路径、原料规格、目标值、检验规则的实时评估
- 返工/降级管理、数据挖掘
- 与SAP QM标准接口集成

**生产订单生命周期管理**：
- 创建（从销售订单初始化或ERP导入）→ 跟踪/订单执行 → 关闭
- 返工订单管理、物料挂单/订单合并

**库存和物料管理**：
- 从进厂物料到出厂发货的完全追踪（单件次、批次和捆包）
- 可配置库区布局、库存位置跟踪和追溯
- 支持手持设备数据收集、盘库
- 与SAP R/3-MM同步、与仓储运输自动化系统标准接口

**能源管理**：
- 最小化能源成本
- 不同事件周期内的消耗预测
- 合同约定消耗峰值违反预测
- 能源消耗高度透明

标准KPI仪表盘：

- KPI网状图展示

### 5.1 智能制造定义

在生产制造过程中，在变化的输入和输出条件下，能够**代替人做决策**的制造过程。应用最新信息和通信技术，使工厂生产环境更有效、可靠、安全和环境友好，同时降低生产成本，提高产品质量。

### 5.2 典型智能制造应用场景

- **无人天车**：自动定位、取放物料
- **在线监控**：实时状态监控与预警
- **机器人&RGV**：自动化物料搬运
- **在线仿真**：虚拟工厂与实际生产的互动

### 5.3 智能制造组件架构

| 维度 | 内容 |
|------|------|
| 智能设计 | 产品研发数字化 |
| 智能计划 | 基于有限产能的计划排产 |
| 智能生产 | 工艺质量管控、生产模式优化 |
| 智能物流 | 物料自动调度 |
| 智能服务 | 远程运维、客户定制 |
| 智能决策 | 经营决策支持 |

- **基于有限产能的计划排产**：考虑设备能力、物料可用性、工艺约束

- **工艺质量管控模式**：一体化管控，从原料到成品的全过程质量控制

在没有控制系统的环境中，使用**条码/RFID**等技术手段进行现场控制，实现物料跟踪和过程记录。

### 5.6 "黄金批次"架构

基于工业大数据平台下的"黄金批次"理念——通过大数据分析找到最优生产批次的工艺参数组合，实现质量的持续提升。

## 6. 钢铁企业SRM采购管理系统

**来源文件**：C06_08（实施服务工作说明书——西城钢铁SRM项目）

### 6.1 项目背景

钢铁企业需要实现供应商管理全面数字化应用，建立采购与供应商之间的高效协作平台。

### 6.2 SRM系统目标

| 目标 | 说明 |
|------|------|
| **降成本** | 更多寻源手段及各环节业务效率提升，降低直接/间接成本 |
| **提效率** | 建立统一集成的供应商管理系统，提升内外部效率 |
| **保质量** | 建立供应商全生命周期管理体系，优胜劣汰 |
| **控风险** | 通过IT手段规范采购流程，全程可视化、可追溯，大数据风控 |

### 6.3 SRM系统功能模块

| 模块 | 功能说明 |
|------|----------|
| **供应商门户** | 供应商注册登录、企业公告、待办事项、图形报表 |

| **询报价管理** | 线上询价/报价，支持多轮次在线议价，突破传统电话/邮件/传真模式 |
| **招投标管理** | 公开招标/邀请招标/在线开标/专家评分/决标全流程 |

- **ERP→SRM**：基础资料（物料、单位）、入库单/退库单、付款信息、开票信息、库存信息
- **SRM→ERP**：供应商准入信息、采购订单信息、供应商开票申请信息

采用六阶段实施方法：项目定义 → 蓝图设计 → 系统实现 → 最终准备 → 上线切换 → 上线支持（一个月）

## 7. 粉末冶金产业链与市场

**来源文件**：C06_11（粉末冶金产业链全景图）、C06_12（粉末冶金行业市场和应用调研报告）

### 7.1 粉末冶金技术概述

粉末冶金技术：以金属粉末或非金属粉末或其混合物为原料，经过成形和烧结，制造金属材料、复合材料及各种类型制品的工艺技术。

**技术优势**：
- 能生产用普通熔炼方法无法生产的特殊结构和性能的材料和制品（如硬质合金）
- 某些材料性能优于普通熔炼法（如粉末高速钢）
- 致密度可控，晶粒细小、显微组织均匀、无成分偏析
- 近净成形，原材料利用率>95%，能耗最低
- 少无切削，切削加工仅40-50%
- 材料组元可控，利于制备复合材料

**材料利用率对比**：

| 工艺 | 材料利用率 | 零件能耗(MJ/kg) |
|------|-----------|-----------------|
| 粉末冶金 | 95% | 28.5 |
| 铸造 | 90% | 38 |
| 冷成形 | 85% | 41 |
| 锻造 | 80% | 49 |
| 机械加工 | 50% | 82 |

### 7.2 产业链全景

**上游——原料**：
- 金属粉末制造（铁粉、铜粉、铝粉、镍粉、钛粉等）
- 制粉设备
- 脱脂设备、压制设备、烧结设备、模具

**中游——粉末冶金工艺制造商**：
- 压坯 → 脱脂烧结 → 变形 → 热处理 → 机加工/涂层 → 制成品

**下游——应用领域**：
- 汽车零部件（最大应用领域）
- 航空航天（制动用粉末冶金材料）
- 电子器械
- 刀具
- 医疗器械
- 机械制造

### 7.3 全球市场分析

全球粉末冶金行业按地区分布：北美约50%、亚太约26%、欧洲约24%。

2013-2016年全球粉末冶金生产总值年均增长率>3%，预计2020年达到4%。

粉末冶金广泛应用于：采矿（硬质合金）、机械加工（硬质合金/陶瓷刀具）、汽车制造（机械零件/多孔轴承）、造船（摩擦材料）、电机制造（多孔轴承）、电子工业（电触头/软磁材料）、化工（过滤器/防腐零件）、军工（穿甲弹头/军械零件）、航空（粉末超合金/纤维强化材料）等领域。

## 9. 钢铁智能工厂及产业互联网

**来源文件**：C06_16（钢铁智能工厂及产业互联网）

### 9.1 智能工厂概念理解

**智能工厂定义**：以数字化工厂为基础，集成了制造技术、新的信息技术和智能科学技术，是信息化同工业相结合的发展新阶段，是信息化的高级形态。

**信息时代的三大技术概念**：
1. **互联网**：开放、公正、参与——带来"共享"思维
2. **大数据**：有产生背景数据，可检索和可关联——数据分析、数据挖掘
3. **云计算**：硬件资源虚拟化后按需分配——降低IT专业要求，按使用付费

**智能工厂的目标**：定制、高效、优质、低耗、绿色
- **定制**：满足互联网下用户需求的刚性需求
- **高效/优质/低耗/绿色**：企业的根本需求

**实现手段**：感知、互连、控制、执行、协同、分析、预测、决策

### 9.2 智能工厂与产业互联网

**核心关系**：
- **智能工厂**解决产业互联网满足用户定制化需求的**制造问题**
- **产业互联网**解决智能工厂与用户和供应商的**嵌入问题**
- 两者是**共生关系**，任何一方脱节都会造成严重伤害

**消费互联网 vs 产业互联网**：
- 消费互联网是"+互联网"——交易的新形态，不改变管理架构和商业模式
- 产业互联网是"互联网+"——产业新生态，对经营模式、管理模式做实质性改变

### 9.3 钢铁智能工厂架构

**一个核心和一个基础**：
- 生产核心：满足用户需求定制化生产（定制、优质）
- 成本核心：完成上述生产的成本控制（高效、低耗、绿色）

**架构层次**：
```
L5层：决策支持（数据挖掘、商务智能）
L4层：经营管理（ERP、SCM、SRM、CRM等）
L3层：生产控制（LES、MES、设备、能源、生产管控）
L2层：过程级控制（PCS、检化验、计量）
L1层：基础自动化级控制（PLC、DCS、传感器、调节器）
L0层：设备层（电机、高低压设备、变压器、液压系统）
```

**订单闭环控制系统**：将用户定制化需求 → 生产和成本预报 → 订单控制系统 → 产品反馈 → 成本反馈的完整闭环

**虚拟仿真**：企业是一个多并行的企业，一个真实企业和多个虚拟企业并行存在，生产的全过程在虚拟企业内部瞬间完成后再到真实企业实际生产。

### 9.4 钢铁企业基础与问题

**已有基础**：
- 生产线自动化基本完备，实现自动控制、远程控制、无人值守
- 过程控制系统普遍采用，部分实现"一键式"生产
- MES基本普及，生产按计划/订单执行
- 比较完整的ERP系统
- 电子商务、质量预报开始建设

**存在问题**：
- 自动化、信息化"最后一公里"问题
- 个别数据采集问题影响整个数据分析
- 引进系统出问题导致整个过程控制无法实现
- 信息化分期建设形成信息孤岛
- 以业务为核心，没有数据中心，数据交换爆炸式增长
