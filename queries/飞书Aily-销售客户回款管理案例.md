# 飞书 Aily 销售 / 客户 / 回款管理案例

> 前置阅读：[[concepts/飞书Aily-自定义智能体配置|自定义智能体配置指南]]

## 案例一：销售管理智能助手

### 需求分析

| 角色 | 核心需求 |
|------|---------|
| 销售代表 | 查询个人业绩、客户跟进状态、目标完成率 |
| 销售主管 | 团队业绩排行、漏斗分析、异常预警 |
| 销售总监 | 全局趋势分析、区域对比、预测模型 |

### System Prompt 核心技能

1. **销售数据查询**：按时间/人员/区域/产品线筛选，通过多维表格 MCP 查询
2. **业绩趋势分析**：同比/环比增长率、目标完成率、漏斗转化率
3. **报告生成**：日报（商机/签约/回款）、周报（排名/进展）、月报（趋势/建议）
4. **异常预警**：连续3天无新商机、重点客户>7天未跟进、目标完成率低于时间进度20%

### 多维表格字段

**商机表**：商机名称/客户名称/商机金额/阶段(初步接触→需求确认→方案报价→商务谈判→赢单/输单)/负责人/预计签约日期/创建时间/最后跟进时间

**回款表**：合同编号/客户名称/回款金额/回款日期/回款状态(已到账/待确认/逾期)/负责人

### 模拟对话

```
👤 销售主管：帮我看看华东区本月的销售情况

🤖 销售管理助手：
📊 华东区 2026年6月 销售概况
┌──────────┬──────────┐
│ 新增商机   │ 47 个     │
│ 商机总额   │ 386.50万  │
│ 签约金额   │ 152.30万  │
│ 目标完成率  │ 61.2%    │
└──────────┴──────────┘
✅ 整体进度超前。⚠️ 张三已连续5天无新商机，建议关注。
```

## 案例二：客户管理智能助手

### 需求分析

| 功能模块 | 具体需求 |
|---------|---------|
| 客户信息查询 | 按名称/行业/区域/等级快速检索 |
| 客户画像分析 | 价值/活跃/风险维度 + 偏好标签 |
| 跟进提醒 | 超期未跟进客户自动提醒 |
| 客户分级 | 基于交易额(40%)/合作年限(20%)/回款及时性(20%)/互动频率(20%) |
| 流失预警 | >30天未跟进且无进行中商机 |

### 多维表格字段

**客户主表**：客户名称/客户编码/行业/区域/客户等级(S/A/B/C)/联系人/联系电话/累计交易额/最近交易日期/最近跟进日期/负责人/客户状态(活跃/沉默/流失/冻结)/合作起始日

**客户跟进记录表**：客户名称(关联)/跟进日期/跟进方式(电话/拜访/微信/邮件/会议)/跟进内容/下次跟进日期/跟进人

### 模拟对话

```
👤 销售代表：帮我查一下"星辰科技"这家客户的情况

🤖 客户管理助手：
📋 客户档案：星辰科技
• 行业：制造业 | 区域：华东 | 等级：A级 | 合作3年
• 累计交易额：287.50万 | 合同6份
🏷 画像：高价值 · 活跃 · 安全 | 标签：品质优先 · 长期合作
✅ 跟进正常（2026-06-10 拜访，讨论Q3采购计划）
💡 建议：Q3采购窗口临近，建议提前准备报价方案。
```

## 案例三：回款管理智能助手

### 需求分析

| 功能模块 | 具体需求 |
|---------|---------|
| 应收账款查询 | 按客户/销售/时间查询未回款明细 |
| 逾期预警 | 🟡1-30天→提醒销售 / 🟠31-90天→通知主管 / 🔴91-180天→上报管理层 / ⚫180+→法务介入 |
| 账龄分析 | 未到期/1-30/31-60/61-90/91-180/180+天分段统计 |
| 催收策略 | S/A级+轻度→友善提醒 / B/C级+轻度→正式催收函 / 中度→电话+上门 / 重度→暂停新订单+法务函 |

### 多维表格字段

**合同台账表**：合同编号/客户名称/合同金额/签约日期/付款条款/已回款总额(公式)/未回款余额(公式)/负责销售/合同状态

**回款记录表**：回款编号/合同编号(关联)/客户名称/计划回款日期/计划回款金额/实际回款日期/实际回款金额/回款状态/逾期天数(公式)/负责销售

### 模拟对话

```
👤 财务经理：帮我看看目前的应收账款情况

🤖 回款管理助手：
📊 应收账款概览（截至 2026-06-16）
• 应收总额：1,286.50万 | 已回款：945.20万 | 未回款：341.30万
【账龄分析】
• 未到期：186.00万(54.5%) | 1-30天：68.30万(20.0%)
• 31-90天：52.00万(15.2%) | 91-180天：25.00万(7.3%) | 180+：10.00万(2.9%)
🚨 重点：XX建设集团(10万,逾期215天)→建议法务评估
```

## 案例四：销售-客户-回款一体化管理

将三个智能体整合为一个综合智能体，实现跨模块联动查询。

### 架构

```
销售经营管理助手（主智能体）
├── 销售管理子技能
├── 客户管理子技能
└── 回款管理子技能
    ↓
MCP 工具层（多维表格 MCP）
    ↓
知识空间（销售制度 + 客户政策 + 财务制度）
```

### 智能路由规则

- "业绩/目标/商机/漏斗" → 销售管理技能
- "客户/跟进/画像/等级" → 客户管理技能
- "回款/应收/逾期/账龄" → 回款管理技能
- 综合性问题 → 联动查询三个模块

### 跨模块联动查询示例

```
👤 销售主管：给我一份"星辰科技"的全景分析

🤖 销售经营管理助手：
📋 客户全景报告：星辰科技

📌 一、客户基本信息
• 行业：制造业 | 区域：华东 | 等级：A级 | 合作3年
• 标签：品质优先 · 长期合作 · 年底集中采购

📌 二、销售情况
• 累计签约：6份，总额287.50万
• 在执行商机：2个（Q3设备采购45万 + 年度维保23万）
• 本年签约：120万（目标200万，完成60%）

📌 三、回款情况
• 应收：22.50万（剩余两期）| 逾期：无 ✅
• 回款信用：⭐⭐⭐⭐⭐（历史零逾期）

📌 四、综合评估
🟢 整体健康度：优
⚠️ Q3设备采购商机停留报价阶段10天，建议加速推进
💡 本周安排拜访，推动进入商务谈判
```
