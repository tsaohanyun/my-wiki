# 天然气处理工艺流程

> 天然气处理的完整工艺路线，从井口原料气到合格产品气。

## 总体流程

```
原料气 → 段塞流捕集器 → 入口分离器 → 胺法脱硫/脱碳 → TEG脱水 → 深冷NGL回收 → 产品气外输
              ↓               ↓              ↓             ↓            ↓
          凝液去储罐      含硫富液→再生  废气去焚烧炉   NGL分馏塔→LPG/轻油
```

## 1. 入口分离

### 目的
脱除原料气中携带的游离水、凝析油和固体颗粒。

### 设备
- **段塞流捕集器（Slug Catcher）**：处理气田来气中的段塞流（气液交替团块）
- **三相分离器**：将天然气、凝析油、水三相分离

### 操作条件
- 压力：通常 4-8 MPa
- 温度：常温（10-40°C）

## 2. 脱硫与脱碳（天然气净化）

### 2.1 胺法脱硫/脱碳

最广泛使用的化学吸收法。

**原理：**
弱碱性胺溶液在低温高压下与H₂S、CO₂发生化学反应生成胺盐；在高温低压下反应逆向进行，胺液再生，释放酸气。

**常用胺液：**

| 胺液类型 | 全称 | 特点 |
|---------|------|------|
| MEA | 一乙醇胺 | 反应快，选择性低，腐蚀性强 |
| DEA | 二乙醇胺 | 适用于含COS/CS₂的原料气 |
| MDEA | 甲基二乙醇胺 | 选择性好，能耗低，主流选择 |
| 配方胺 | MDEA+活化剂 | 兼顾脱硫脱碳，深度脱除 |

**典型流程：**

```
原料气 → 吸收塔(逆流接触贫胺液) → 净化气(去脱水单元)
                ↓
           富胺液
                ↓
           闪蒸罐(释放溶解烃)
                ↓
           贫富液换热器 → 再生塔(蒸汽汽提) → 贫胺液(循环回吸收塔)
                                    ↓
                               酸气(去硫磺回收)
```

**操作参数：**
- 吸收压力：4-8 MPa
- 吸收温度：35-45°C
- 再生温度：120-125°C
- 贫胺液H₂S负荷：< 0.005 mol/mol

### 2.2 物理溶剂法

- **Selexol工艺**（聚乙二醇二甲醚）：适用于高压高CO₂工况
- **Rectisol工艺**（低温甲醇洗）：深冷甲醇吸收，适用于煤制气

### 2.3 膜分离法

利用聚合物膜对CO₂和H₂S的选择性渗透进行分离。适用于高CO₂含量（>10%）工况的粗脱，常与胺法组合使用。

### 2.4 干法脱硫

- **氧化铁脱硫剂**：适用于低含硫（< 100 ppm）的精细脱硫
- **分子筛脱硫**：同时脱水和脱硫
- **LO-CAT工艺**：液相氧化还原，将H₂S直接氧化为元素硫

## 3. 脱水

### 3.1 三甘醇（TEG）脱水

最主流的溶剂吸收脱水法。

**流程：**
```
湿天然气 → 吸收塔(逆流接触贫TEG) → 干气(水露点降至-10°C以下)
                ↓
           富TEG(含水)
                ↓
           闪蒸罐 → 过滤 → 贫富换热 → 再生塔(200°C,常压) → 贫TEG循环
```

**关键参数：**
- TEG循环量：2-5 L/kg H₂O
- 再生温度：195-200°C
- 贫TEG浓度：≥ 98.5%（常压再生）或 ≥ 99.5%（汽提再生）
- 干气水露点降：40-60°C

### 3.2 分子筛脱水

用于需要深度脱水的场景（LNG原料气、深冷NGL回收）。

**流程：**
湿气经两个分子筛床切换运行——一个吸附，另一个再生。再生使用260-300°C热干气。

**脱水深度：**
- 水露点可降至 -80°C 以下
- 出口水含量 < 0.1 ppmv

### 3.3 其他方法

- **氯化钙脱水**：简易撬装，适用于小气量
- **膜分离脱水**：新兴技术，设备紧凑

## 4. NGL回收（天然气凝液回收）

### 目的
回收乙烷(C₂)、丙烷(C₃)、丁烷(C₄)、戊烷及以上(C₅⁺)等重组分，用作化工原料（乙烯裂解）或燃料（LPG）。

### 4.1 浅冷工艺

通过制冷将天然气冷至 -20°C 至 -30°C，回收C₃⁺组分。适用于以回收LPG为目标的工况。

### 4.2 深冷工艺（膨胀机制冷）

利用透平膨胀机将高压原料气膨胀至低压，产生深度制冷（-70°C 至 -100°C），实现C₂⁺高收率回收。

**典型流程（ISS - Industry Standard Single Stage）：**
```
干燥原料气 → 冷箱预冷 → 低温分离器 → 气相 → 膨胀机 → 脱甲烷塔顶部回流
                                    → 液相 → 脱甲烷塔进料
```

**关键设备：**
- 透平膨胀机（Turbo-expander）
- 冷箱（Brazed Aluminum Heat Exchanger, BAHE）
- 脱甲烷塔（Demethanizer）

**回收效率：**
| 组分 | 回收率 |
|------|--------|
| 乙烷(C₂) | 85-95% |
| 丙烷(C₃) | 95-99% |
| 丁烷+(C₄⁺) | ~100% |

### 4.3 NGL分馏

回收后的NGL混合物进入分馏序列：
```
NGL混合物 → 脱甲烷塔 → 脱乙烷塔 → 脱丙烷塔 → 脱丁烷塔
               ↓           ↓          ↓          ↓
             甲烷尾气    乙烷产品   丙烷产品   C₅⁺/轻油
                                      ↓
                                  丁烷(LPG)
```

## 5. 硫磺回收

脱硫过程产生的含H₂S酸气必须进行处理。

### Claus工艺

```
H₂S + ½O₂ → SO₂ + H₂O  (热反应段, 1100-1400°C)
2H₂S + SO₂ → 3S + 2H₂O  (催化反应段, 200-350°C)
```

**硫回收率：** 两级Claus可达94-96%，三级可达97-98%

### 尾气处理（TGTU）

- **SCOT工艺**（Shell Claus Off-gas Treating）：将尾气中残留硫化物加氢还原为H₂S，再经胺液吸收返回Claus单元。总硫回收率可达99.8%+
- **SUPERCLAUS**：选择性氧化残余H₂S为元素硫

## 关键工艺参数汇总

| 指标 | 典型值 |
|------|--------|
| 原料气H₂S脱除率 | >99.9% |
| 产品气H₂S含量 | <6 mg/m³ |
| 产品气总硫 | <60 mg/m³ |
| TEG脱水后水露点 | -10°C ~ -20°C |
| 分子筛脱水后水露点 | <-80°C |
| 深冷NGL乙烷回收率 | 85-95% |
| Claus+SCOT总硫回收率 | >99.8% |
