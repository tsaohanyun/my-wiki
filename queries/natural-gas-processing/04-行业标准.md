## 4. 核心安全规范要点

### 4.1 防火间距（GB 50183）

| 设施 | 与居民区 | 与其他生产设施 |
|------|:-------:|:------------:|
| 天然气处理厂 | ≥ 100m | ≥ 30~60m |
| 高含硫处理厂 | ≥ 300m | 按HAZOP确定 |
| 火炬 | ≥ 100~300m | ≥ 60m |

### 4.2 H₂S安全阈值

| 浓度（ppm） | 危害 |
|:----------:|------|
| 10 | 允许暴露8小时（OSHA PEL） |
| 20 | 中国工作场所最高允许浓度 |
| 100 | 嗅觉丧失（麻痹） |
| 300 | 立即对生命健康有危险（IDLH） |
| 1000+ | 立即死亡 |

### 4.3 SIL定级

天然气处理厂典型安全功能SIL等级：
- 高压保护：SIL 2~3
- 火灾/气体检测与关断：SIL 2
- 储罐超液位保护：SIL 2~3

## 5. 环保排放标准

| 项目 | 排放限值（GB 31571） |
|------|:---:|
| SO₂ | ≤ 100~400 mg/Nm³（视区域） |
| NOx | ≤ 150~200 mg/Nm³ |
| 颗粒物 | ≤ 20~30 mg/Nm³ |
| VOC | ≤ 120 mg/Nm³（非甲烷总烃） |
| 废水COD | ≤ 60 mg/L |
| 废水氨氮 | ≤ 8 mg/L |
