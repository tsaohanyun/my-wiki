# 天然气处理厂 — 质量管理

## 1. 质量管理概述

天然气处理厂的质量管理以"产品达标、过程受控、持续改进"为方针，覆盖从原料气入厂到产品气外输的全过程。核心管理对象包括：

- **气质质量**：天然气组分、热值、水露点、烃露点、H₂S含量等
- **产品质量**：管输天然气、LPG、乙烷、天然汽油、硫磺等
- **过程质量**：工艺参数控制、设备运行质量、操作规范性
- **环境质量**：排放达标、废水处理、噪声控制

## 3. 天然气产品质量标准

### 3.1 中国管输天然气标准

**GB 17820-2018《天然气》**

| 项目 | 一类气 | 二类气 | 三类气 |
|------|--------|--------|--------|
| 高位发热量（MJ/m³） | ≥36.0 | ≥31.4 | ≥31.4 |
| 总硫（以硫计）（mg/m³） | ≤20 | ≤100 | ≤200 |
| H₂S含量（mg/m³） | ≤6 | ≤20 | ≤350 |
| CO₂含量（%） | ≤2.0 | ≤3.0 | — |
| 水露点（℃） | 低于输气管线最低操作温度 | | |

**GB 17820-2018 修订要点**：
- 提高了一类气总硫限值（从≤100降至≤20mg/m³）
- 增加了CO₂含量限值
- 强化了水露点控制要求

### 3.2 LNG原料气标准

| 项目 | 典型要求 |
|------|---------|
| CH₄含量 | ≥85%（摩尔分数） |
| C₂+含量 | 根据液化工艺确定 |
| H₂S | ≤4 ppmv |
| CO₂ | ≤50 ppmv |
| H₂O | ≤0.1 ppmv（-100℃露点） |
| 汞 | ≤0.01 μg/Nm³ |
| 硅氧烷 | ≤0.01 mg/Nm³ |

### 3.3 LPG产品标准

**GB 11174-2011《液化石油气》**

| 项目 | 一级品 | 二级品 | 三级品 |
|------|--------|--------|--------|
| C₃+C₄烯烃含量（%） | ≤2.0 | ≤3.0 | ≤5.0 |
| C₅及以上（%） | ≤1.0 | ≤2.0 | ≤3.0 |
| 蒸发残留物（mL/100mL） | ≤0.05 | ≤0.10 | — |
| 腐蚀（铜片） | ≤1级 | ≤1级 | — |
| H₂S（mg/m³） | ≤5 | ≤10 | ≤20 |

### 3.4 硫磺产品标准

**GB 2449-2022《工业硫磺》**

| 项目 | 优等品 | 一等品 | 合格品 |
|------|--------|--------|--------|
| 硫含量（%） | ≥99.90 | ≥99.50 | ≥99.00 |
| 水分（%） | ≤0.10 | ≤0.50 | ≤1.00 |
| 灰分（%） | ≤0.03 | ≤0.10 | ≤0.20 |
| 酸度（以H₂SO₄计，%） | ≤0.003 | ≤0.005 | ≤0.02 |

## 4. 气质在线监测

### 4.1 在线分析仪表配置

| 监测点 | 分析项目 | 仪表类型 | 采样频率 |
|--------|---------|---------|---------|
| 原料气进站 | 组分（C₁~C₆+、CO₂、H₂S、N₂） | 在线气相色谱（GC） | 5~15分钟 |
| 脱硫后 | H₂S、CO₂ | 红外/紫外分析仪 | 连续 |
| 脱水后 | 水露点 | 镜面露点仪/电容式 | 连续 |
| 产品气外输 | 组分、热值、密度 | 在线GC + 热值仪 | 5~15分钟 |
| NGL产品 | 组分分析 | 在线GC | 每批次 |
| 硫磺回收 | 尾气H₂S/SO₂ | 在线色谱/红外 | 连续 |

### 4.2 在线监测数据管理

```
在线分析仪表 → 4~20mA/Modbus → DCS/PLC → 历史数据库（PHD/IP21）
                                                   ↓
                                         质量管理平台（LIMS）
                                                   ↓
                                         质量趋势分析/报表/报警
```

**关键报警设定**：

| 参数 | 报警级别 | 报警值 | 动作 |
|------|---------|--------|------|
| H₂S含量 | 高高 | ≥20 mg/m³ | 联锁切断外输 |
| H₂S含量 | 高 | ≥10 mg/m³ | 工艺调整 |
| 水露点 | 高 | 高于管输要求 | 加强脱水 |
| 热值 | 低 | <31.4 MJ/m³ | 调质/停输 |
| CO₂含量 | 高 | >3% | 调整脱碳 |

## 6. 质量控制流程

### 6.1 来料质量控制（原料气入厂）

```
原料气进站
  → 取样分析（组分、H₂S、CO₂、水含量）
  → 与上游气田/管道交接气量计量
  → 数据录入LIMS系统
  → 原料气质量判定（合格/异常）
  → 异常情况上报调度和工艺工程师
```

### 6.2 过程质量控制

**工艺参数控制要点**：

| 工序 | 关键质量控制参数 | 控制范围 | 异常处理 |
|------|----------------|---------|---------|
| 脱硫 | 净化气H₂S | ≤6 mg/m³ | 调整胺液循环量/浓度 |
| 脱硫 | 贫液酸气负荷 | <0.5 mol/mol | 检查再生效果 |
| 脱水 | 出塔干气水露点 | 低于管输要求 | 更换分子筛/调整再生 |
| NGL回收 | 膨胀机出口温度 | -80~-95℃ | 调整膨胀比/处理量 |
| 分馏 | 产品纯度 | 按产品规格 | 调整回流比/塔温 |
| 硫磺回收 | 尾气SO₂ | ≤400mg/Nm³ | 调整配风比/催化剂 |

**SPC（统计过程控制）应用**：
- 关键质量参数绘制X-R控制图
- 设定控制限和行动限
- 异常趋势（连续7点上升/下降、超控制限）触发调查
- 定期进行过程能力分析（Cpk≥1.33）

### 6.3 产品质量控制

**产品出厂检验流程**：

```
生产完成
  → 取样（按标准取样点和取样方法）
  → 检验（按国家标准方法全项检验）
  → 出具检验报告
  → 质量判定（合格/不合格）
  → 合格 → 出厂放行 → 打印产品质量证明书
  → 不合格 → 隔离标识 → 不合格品评审 → 处置（返工/降级/报废）
```

## 7. 不合格品管理

### 7.1 不合格品分类

| 等级 | 描述 | 示例 |
|------|------|------|
| A级（严重不合格） | 产品指标严重超标 | H₂S超标、热值严重偏低 |
| B级（一般不合格） | 产品指标轻微超标 | 水露点略高、总硫略高 |
| C级（潜在不合格） | 过程偏差可能影响质量 | 工艺参数异常但产品暂合格 |

### 7.2 不合格品处理流程

```
发现不合格 → 标识隔离 → 填写不合格品报告
  → 质量工程师评审 → 原因分析（5Why/鱼骨图）
  → 处置决定：
    ├─ 返工（重新处理至合格）
    ├─ 降级使用（满足低级别标准）
    ├─ 让步接收（与客户协商）
    └─ 报废（无法使用）
  → 纠正措施实施 → 效果验证 → 关闭
```

### 7.3 纠正与预防措施（CAPA）

**8D问题解决流程**：
1. 建立小组
2. 问题描述
3. 临时遏制措施
4. 根本原因分析
5. 永久纠正措施选择
6. 实施永久纠正措施
7. 预防再发生
8. 恭贺小组

## 8. 计量管理

### 8.1 气量计量

| 计量环节 | 计量方式 | 精度要求 |
|---------|---------|---------|
| 进厂计量 | 超声波流量计/涡轮流量计 | ±0.5% |
| 出厂计量 | 超声波流量计/科里奥利流量计 | ±0.5% |
| 厂内计量 | 孔板流量计/涡街流量计 | ±1.0% |
| 贸易交接 | 超声波流量计（双冗余） | ±0.3% |

### 8.2 产品计量

| 产品 | 计量方式 | 计量设备 |
|------|---------|---------|
| 管输天然气 | 质量流量+组分校正 | 在线GC+流量计 |
| LPG | 体积/质量 | 科里奥利流量计+液位计 |
| 天然汽油 | 体积+密度 | 质量流量计+密度计 |
| 硫磺 | 质量 | 电子皮带秤/汽车衡 |

### 8.3 计量器具检定

| 器具类别 | 检定周期 | 检定标准 |
|---------|---------|---------|
| 流量计 | 1年 | JJG 1030/1033 |
| 温度仪表 | 1年 | JJG 130 |
| 压力仪表 | 半年~1年 | JJG 52 |
| 分析仪器 | 半年 | 相应检定规程 |

## 9. 质量追溯体系

### 9.1 追溯链路

```
产品批次号 → 生产记录（工艺参数、操作记录）
            → 原料来源（气井/管道、进厂气质量）
```

## 11. 特种设备与压力管道质量

### 11.1 特种设备管理

天然气处理厂涉及的主要特种设备：

| 设备类别 | 数量级 | 监管要求 |
|---------|--------|---------|
| 压力容器 | 100~500台 | 定期检验、使用登记 |
| 压力管道 | 10~50km | 定期检验、在线检测 |
| 锅炉 | 2~10台 | 定期检验、运行管理 |
| 安全阀 | 200~1000个 | 定期校验 |
| 起重机械 | 10~50台 | 定期检验 |

### 11.2 检验与合规

- **安装监检**：特种设备安装过程监督检验
- **定期检验**：压力容器一般6年一次全面检验，压力管道3~6年
- **在线检测**：管道腐蚀监测、壁厚检测
- **安全阀校验**：每年至少1次
- **无损检测**：焊缝检测（RT/UT/MT/PT）
