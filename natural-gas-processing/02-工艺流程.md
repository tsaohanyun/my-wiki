# 天然气处理工艺流程

## 1. 总体工艺流程

天然气处理厂的典型全流程如下：

```
原料气进站
  │
  ├─→ 1. 入口分离（气液分离）
  │
  ├─→ 2. 脱硫/脱碳（酸性气体脱除）
  │
  ├─→ 3. 脱水（水分脱除）
  │
  ├─→ 4. 凝液回收（NGL提取）或直接管输
  │
  ├─→ 5. 硫磺回收 + 尾气处理
  │
  └─→ 6. 产品气增压外输
```

## 2. 入口分离（Inlet Separation）

### 2.1 目的
将原料气中的游离水、凝析油、固体杂质（砂粒、铁锈等）与气相分离。

### 2.2 主要设备
- 段塞流捕集器（Slug Catcher）：处理气液两相段塞流
- 三相分离器：同时分离气、油、水三相
- 过滤分离器：精细过滤固体颗粒和液滴

### 2.3 操作参数
- 操作压力：3.0 ~ 12.0 MPa（视气田压力等级）
- 操作温度：10 ~ 60℃

## 3. 脱硫脱碳（Acid Gas Removal，AGR）

脱硫脱碳是移除H₂S和CO₂的工艺。H₂S剧毒且有腐蚀性，CO₂降低热值并引起腐蚀。

### 3.1 化学吸收法（醇胺法）

**原理**：利用弱碱性醇胺溶液与酸性气体发生可逆化学反应。

| 吸收剂 | 特点 | 适用场景 |
|--------|------|----------|
| MEA（一乙醇胺） | 活性高，腐蚀性强，降解严重 | 低压、低酸气浓度 |
| DEA（二乙醇胺） | 中等活性，降解较少 | 中等酸气浓度 |
| MDEA（N-甲基二乙醇胺） | 选择性脱硫，能耗低 | 高碳硫比气体 |
| 配方胺溶液 | 针对特定气体配方 | 各种工况 |

**典型流程**：
```
原料气 → 吸收塔（逆流接触胺液）→ 净化气
胺液（富液）→ 闪蒸罐 → 贫富液换热器 → 再生塔（蒸汽汽提）→ 贫液循环
再生塔顶 → 酸气去硫磺回收
```

### 3.2 物理溶剂法

| 工艺 | 溶剂 | 特点 |
|------|------|------|
| Selexol | 聚乙二醇二甲醚 | 选择性脱硫，适合高压高CO₂ |
| Rectisol | 冷甲醇 | 深度脱除，温度-40~-60℃ |
| Purisol | NMP（N-甲基吡咯烷酮） | 脱硫选择性高 |

### 3.3 膜分离法

利用聚合物膜（醋酸纤维素、聚酰亚胺等）的选择性渗透实现CO₂/CH₄分离。适合高CO₂含量气体预处理。

### 3.4 变压吸附法（PSA）

利用分子筛、活性炭等吸附剂在高压下吸附、低压下脱附的循环操作。

### 3.5 混合工艺

- 膜 + 醇胺：先用膜粗脱CO₂，再用醇胺精脱
- 物理溶剂 + 化学溶剂：Sulfinol工艺（环丁砜 + 胺液）

## 4. 脱水（Dehydration）

### 4.1 目的
脱除天然气中的水蒸气，将水露点降至管输要求以下（通常低于最低环境温度5~10℃），防止：
- 水合物生成堵塞管道
- 液态水引起的腐蚀
- 水分在深冷段结冰

### 4.2 甘醇脱水法（TEG Dehydration）

**原理**：三甘醇（TEG）具有强吸湿性，通过气液接触吸收水分。

**流程**：
```
湿气 → 吸收塔（接触贫TEG）→ 干气
富TEG → 闪蒸罐 → 过滤器 → 贫富换热器 → 再生塔（~200℃常压再生）→ 贫TEG循环
```

**典型指标**：水露点降 40~70℃

### 4.3 分子筛脱水法

**原理**：利用分子筛（3A、4A、5A型）的微孔结构，在高压常温下吸附水分，在高温低压下再生。

**流程**（双塔切换）：
- 吸附周期：8~24小时
- 再生温度：230~290℃
- 水露点可降至 -100℃以下（适用于深冷NGL回收和LNG）

### 4.4 其他脱水方法

| 方法 | 特点 |
|------|------|
| 氯化钙脱水 | 简单便宜，露点降有限 |
| 膜脱水 | 适合海上平台等空间受限场景 |
| 超音速分离（Twister）| 新型技术，无化学药剂 |

## 5. 凝液回收（NGL Recovery）

### 5.1 目的
从天然气中回收乙烷（C₂）、丙烷（C₃）、丁烷（C₄）和天然汽油（C₅⁺），提高资源综合利用率。

### 5.2 浅冷分离工艺

**原理**：通过外部制冷（丙烷制冷）降温至-25~-35℃，冷凝回收C₃⁺组分。

**流程**：
```
干气 → 冷箱预冷 → 丙烷制冷 → 低温分离器 → 冷凝液去分馏
                                          → 气相（产品气）复热外输
```

**回收率**：丙烷回收率 70~85%，乙烷几乎不回收

### 5.3 深冷分离工艺（Cryogenic）

**原理**：通过膨胀机制冷或联合制冷降温至-70~-100℃，回收C₂⁺组分。

**核心流程**：

#### 5.3.1 膨胀机制冷（Expander Plant）
```
干气 → 分子筛深度脱水 → 冷箱预冷 → 膨胀机等熵膨胀（降温降压）
    → 低温分离器（-80~-95℃）→ 气相经冷箱复热 → 增压机 → 外输
                                → 液相 → 脱甲烷塔 → NGL去分馏
```

#### 5.3.2 脱甲烷塔（Demethanizer）
- 操作压力：1.5~3.5 MPa
- 塔顶温度：-85~-95℃
- 塔底温度：10~30℃
- 功能：将甲烷与C₂⁺分离，控制NGL中甲烷/乙烷比

**回收率**：乙烷回收率可达 85~95%+，丙烷回收率 99%+

### 5.4 NGL分馏

深冷分离后得到的混合NGL需要进一步分馏为单一产品：

```
混合NGL → 脱乙烷塔 → 乙烷产品
         塔底 → 脱丙烷塔 → 丙烷（或LPG）
                 塔底 → 脱丁烷塔 → 丁烷（或LPG）
                         塔底 → 天然汽油（C₅⁺）
```

## 6. 硫磺回收与尾气处理（SRU & TGTU）

### 6.1 Claus硫磺回收

脱硫脱碳产生的酸气（H₂S）需要转化为元素硫：

**反应**：
```
H₂S + 3/2 O₂ → SO₂ + H₂O    （热反应段）
2H₂S + SO₂ → 3S + 2H₂O       （催化反应段）
```

**典型硫回收率**：94~97%（二级Claus），可搭配尾气处理达99.8%+

### 6.2 尾气处理工艺

| 工艺 | 技术路线 | 总硫回收率 |
|------|----------|:---------:|
| SCOT | 还原吸收 | >99.8% |
| SuperClaus | 选择氧化 | >99.0% |
| S-Zorb | 低温加氢 | >99.5% |
| LO-CAT | 液相氧化还原 | >99.9% |

## 7. 产品气增压与外输

处理后的天然气经压缩机增压至管输压力（通常6.0~12.0 MPa），通过外输管道送往用户。压缩机类型：
- 离心式压缩机（大流量）
- 往复式压缩机（高压比、中小流量）

## 8. 辅助系统

| 系统 | 功能 |
|------|------|
| 火炬系统 | 紧急放空与安全泄压 |
| 燃料气系统 | 为锅炉、加热炉、燃气轮机提供燃料 |
| 热媒系统 | 为再沸器、再生塔提供热源 |
| 冷却水/空冷系统 | 工艺介质冷却 |
| 仪表风/氮气系统 | 控制阀驱动、吹扫、惰化 |
| 污水处理系统 | 生产废水处理与回注 |
