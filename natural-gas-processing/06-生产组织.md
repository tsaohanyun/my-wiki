# 天然气处理厂 — 生产组织

## 1. 组织架构概述

天然气处理厂的生产组织以"安全稳定运行、高效达标外输"为核心目标，采用分层分级管理模式。典型组织架构包含四个层级：

```
厂级管理层
  ├─ 生产技术部（工艺、设备、仪表）
  ├─ 安全环保部（HSE）
  ├─ 质量管理部（化验、质检）
  ├─ 综合管理部（人事、财务、行政）
  └─ 调度指挥中心
        ├─ 首站/集气站
        ├─ 净化/处理装置
        ├─ NGL回收/分馏装置
        ├─ 硫磺回收装置
        ├─ 公用工程车间
        └─ 外输计量站
```

### 1.1 规模与人员配置

| 处理厂规模 | 日处理能力 | 常驻人员 | 操作班次 |
|-----------|-----------|---------|---------|
| 小型厂 | <100万Nm³/d | 50~100人 | 三班两倒或四班三倒 |
| 中型厂 | 100~500万Nm³/d | 100~200人 | 四班三倒 |
| 大型厂 | 500~2000万Nm³/d | 200~500人 | 四班两倒（含大班） |
| 超大型厂 | >2000万Nm³/d | 500~1000人 | 四班两倒+专业班组 |

### 1.2 中国典型处理厂组织模式

以普光净化厂（120亿m³/年高含硫气田）为例：

- **厂级**：厂长、副厂长（生产/安全/技术）、总工程师
- **车间级**：集气站、净化装置车间、硫磺回收车间、公用工程车间、仪表维修车间
- **班组级**：每车间下设操作班组、巡检班组

## 3. 倒班制度

### 3.1 四班三倒制（最常用）

```
┌──────────────────────────────────────────────┐
│  班组A  │  白班 08:00~16:00  │  休  │  夜班 00:00~08:00  │
│  班组B  │  中班 16:00~00:00  │  白班 08:00~16:00  │  休  │
│  班组C  │  休  │  中班 16:00~00:00  │  白班 08:00~16:00  │
│  班组D  │  夜班 00:00~08:00  │  休  │  中班 16:00~00:00  │
└──────────────────────────────────────────────┘
```

- 每班8小时，三班连续轮转
- 3个班组在岗，1个班组轮休
- 典型周期：白→中→夜→休→白→中→夜→休（每8天一个完整循环）

### 3.2 四班两倒制（大型厂）

- 2个班组在岗（各12小时），2个班组轮休
- 适合自动化程度高、人员编制精简的大型处理厂
- 白班/夜班各12小时，"上二休二"或"上三休三"

### 3.3 交接班管理

交接班是确保生产连续性的关键环节：

**交接班六交清**：
1. **产量交清**：当日处理量、产品产量
2. **质量交清**：产品气质指标、异常情况
3. **设备交清**：运行设备状态、备用设备情况
4. **安全交清**：安全隐患、环保排放
5. **工具交清**：工器具、钥匙、资料
6. **指令交清**：上级指令、注意事项

**交接班流程**：
```
交班准备（交班前30分钟填写交接班记录）
  → 现场交接（双方巡检关键设备和参数）
  → 会议室交接（逐项确认交接内容）
  → 接班确认签字（班长签字确认）
  → 接班后15分钟内完成岗位确认
```

## 4. 生产调度管理

### 4.1 调度层级

| 层级 | 职责 | 响应时间 |
|------|------|---------|
| 公司调度 | 全管网气量平衡、贸易结算 | 小时级 |
| 厂级调度 | 厂内装置协调、开停工指挥 | 分钟级 |
| 车间调度 | 装置内操作协调 | 即时 |
| 现场操作 | 参数调整、设备切换 | 即时 |

### 4.2 日常调度流程

```
08:00  调度晨会：交接班汇总、当日计划确认
08:30  日报编报：处理量、产量、质量、能耗、安全
09:00  随时调度：工艺参数调整、设备切换、异常处置
16:00  中班交接
20:00  晚间巡检确认
00:00  夜班交接
```

### 4.3 异常情况调度

| 异常等级 | 描述 | 调度权限 | 响应要求 |
|---------|------|---------|---------|
| Ⅰ级（特别重大） | 装置紧急停车、人员伤亡 | 公司级 | 5分钟内上报 |
| Ⅱ级（重大） | 装置非计划停车、设备严重故障 | 厂级 | 15分钟内上报 |
| Ⅲ级（较大） | 工艺参数大幅波动、设备一般故障 | 车间级 | 30分钟内上报 |
| Ⅳ级（一般） | 轻微参数偏差、小故障 | 班组级 | 当班处理 |

## 5. 设备管理

### 5.1 设备完整性管理

天然气处理厂关键设备实行全生命周期管理：

| 设备类别 | 关键设备 | 管理重点 |
|---------|---------|---------|
| 动设备 | 压缩机、泵、膨胀机 | 振动监测、润滑管理、密封泄漏 |
| 静设备 | 塔器、容器、换热器 | 腐蚀监测、壁厚检测、压力试验 |
| 管道 | 工艺管道、高压管道 | 腐蚀速率监测、在线检测、检验 |
| 安全附件 | 安全阀、爆破片、联锁 | 定期校验、功能测试 |
| 自控系统 | DCS、SIS、ESD | 系统冗余、联锁测试、定期切换 |

### 5.2 检维修策略

| 策略 | 适用设备 | 周期 |
|------|---------|------|
| 预防性维修（PM） | 泵、压缩机、阀门 | 按运行小时/日历月 |
| 预测性维修（PdM） | 压缩机、大型泵 | 基于状态监测数据 |
| 故障维修（CM） | 一般仪表、小阀门 | 故障后修复 |
| 基于风险的检验（RBI） | 压力容器、管道 | 按风险等级3~10年 |

### 5.3 备件管理

- **关键备件**：压缩机转子、膨胀机叶轮、安全阀 → 常备库存
- **重要备件**：泵机械密封、换热器管束 → 适量库存
- **一般备件**：阀门、垫片、螺栓 → 按需采购

## 6. 安全管理（HSE）

### 6.1 天然气处理厂主要危险源

| 危险源 | 主要风险 | 防控措施 |
|--------|---------|---------|
| H₂S（硫化氢） | 剧毒、窒息、腐蚀 | 固定式/便携式H₂S检测仪、SCBA、风向标 |
| 高压气体（3~12 MPa） | 物理爆炸、气体泄漏 | 压力监控、泄漏检测、防爆设计 |
| 可燃气体（CH₄/C₃/C₄） | 火灾爆炸 | 可燃气体检测、防爆电气、禁火区管理 |
| 高温设备（锅炉/加热炉） | 烫伤、火灾 | 保温、防护罩、高温警示 |
| 转动设备（压缩机/泵） | 机械伤害 | 防护罩、联锁、LOTO |
| 深冷设备（-70~-100℃） | 冻伤、材料脆断 | 低温材料、保温、防冻措施 |

### 6.2 安全管理体系

| 体系 | 内容 | 频次 |
|------|------|------|
| 安全风险分级管控 | 危险源辨识、风险评估、分级管控 | 年度辨识、季度评估 |
| 隐患排查治理 | 日常巡检、专项检查、综合检查 | 日/周/月/季 |
| 作业许可管理 | 动火、受限空间、高处、吊装、临时用电、动土、断路、盲板抽堵 | 作业前审批 |
| 变更管理（MOC） | 工艺变更、设备变更、人员变更、操作变更 | 变更前审批 |
| 事故事件管理 | 未遂事件报告、事故调查分析 | 事件驱动 |
| 承包商管理 | 资质审查、安全培训、作业监督 | 入场前+全程监督 |

### 6.3 应急管理

**应急预案体系**（三级）：

| 级别 | 预案 | 内容 |
|------|------|------|
| 综合预案 | 全厂综合应急预案 | 组织体系、预警机制、应急响应流程 |
| 专项预案 | H₂S泄漏、火灾爆炸、管道泄漏、自然灾害 | 具体事故类型的处置程序 |
| 现场处置方案 | 关键设备/区域的快速处置 | 一装置一方案，操作卡式 |

**应急演练频次**：
- 综合演练：每年1次
- 专项演练：每半年1次
- 现场处置演练：每季度1次
- 岗位级应急演练：每月1次

### 6.4 个人防护装备（PPE）

| PPE类型 | 适用场景 |
|---------|---------|
| 便携式H₂S/CO检测仪 | 全员佩戴（含巡检） |
| 正压式空气呼吸器（SCBA） | H₂S区域应急处置 |
| 防静电工作服 | 防爆区域标准着装 |
| 安全帽、防护眼镜 | 全区域 |
| 防化手套、护目镜 | 化学品操作 |
| 防噪声耳塞/耳罩 | 高噪声区域（>85dB） |
| 安全鞋（防砸防刺） | 全区域 |

## 7. 环保管理

### 7.1 主要排放源与控制

| 排放类型 | 来源 | 控制措施 | 排放标准 |
|---------|------|---------|---------|
| SO₂排放 | 硫磺回收尾气 | SCOT/SuperClaus尾气处理 | ≤400mg/Nm³ |
| NOₓ排放 | 加热炉/锅炉 | 低氮燃烧器/SCR脱硝 | ≤100mg/Nm³ |
| 颗粒物排放 | 硫磺成型/包装 | 除尘系统 | ≤30mg/Nm³ |
| VOCs排放 | 储罐/装卸/工艺放空 | 油气回收/密封/火炬 | 地方标准 |
| 废水排放 | 生产废水/含油污水 | 隔油+气浮+生化处理 | GB 8978 |
| 噪声 | 压缩机/泵/风机 | 减振+隔声+消声 | 厂界达标 |

### 7.2 碳排放管理

- **CO₂排放核算**：燃料燃烧排放 + 工艺排放（脱碳释放CO₂）+ 火炬排放
- **减排路径**：余热回收利用、CCUS（碳捕集利用与封存）、电气化替代
- **碳交易**：纳入全国碳市场重点排放单位管理

## 9. 信息化与数字化

### 9.1 主要信息系统

| 系统 | 功能 |
|------|------|
| DCS（集散控制系统） | 装置过程控制、参数监控 |
| SIS/ESD（安全仪表/紧急停车系统） | 安全联锁、紧急停车 |
| MES（制造执行系统） | 生产计划、物料跟踪、报工统计 |
| EAM（资产管理系统） | 设备台账、检维修计划、备件管理 |
| LIMS（实验室管理系统） | 化验数据管理、质量报告 |
| ERP（企业资源计划） | 财务、采购、销售、人力资源 |
| 视频监控系统 | 关键区域监控、周界安防 |
| 气体检测系统 | H₂S/可燃气体在线监测 |

### 9.2 智能化发展方向

- **APC（先进过程控制）**：多变量预测控制，提高操作稳定性
- **实时优化（RTO）**：基于模型的工艺参数实时优化
- **数字孪生**：虚拟装置仿真、操作培训、预测性维护
- **智能巡检**：机器人巡检、无人机巡检、AI视觉识别
- **大数据分析**：设备故障预警、能耗优化、质量预测
