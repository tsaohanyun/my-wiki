# 天然气处理 — 行业现状与主要企业

## 1. 全球天然气处理行业概况

### 1.1 市场规模

据国际能源署（IEA）和国际气体联盟（IGU）数据，全球天然气年产量约 **4万亿立方米**（2023年），其中相当比例需经处理厂净化。全球天然气处理市场总值约 **200~250亿美元/年**（2023年估），预计2024-2030年CAGR约4~5%。

### 1.2 主要驱动因素

| 因素 | 说明 |
|------|------|
| 天然气产量增长 | 中东、北美、非洲增产 |
| 气质变差 | 高含硫/高含CO₂气田开发比例上升 |
| LNG贸易增长 | 深度净化与液化需求 |
| 环保法规趋严 | SO₂、温室气体排放管控 |
| CCUS联动 | CO₂捕集与天然气处理耦合 |

### 1.3 技术发展趋势

- **高效胺液+物理溶剂混合工艺**：降低能耗30%+
- **膜+胺液混合系统**：适应高CO₂气田
- **模块化/橇装化**：缩短建设周期，适合页岩气
- **数字化/智能化**：APC（先进过程控制）、数字孪生
- **超音速分离（Twister）**：无化学药剂脱水脱重烃
- **CO₂-EOR耦合**：脱碳与提高采收率联动
- **氦气联合回收**：高氦天然气增值利用

## 2. 中国天然气处理行业现状

### 2.1 产能规模

中国天然气产量约 **2300亿m³/年**（2024年），天然气处理总能力超过 **3000亿m³/年**（含进口管道气处理），处理厂数量 **300+座**。

### 2.2 资源分布与处理基地

| 盆地区域 | 代表气田 | 处理厂 | 特点 |
|----------|---------|--------|------|
| **四川盆地** | 普光、元坝、安岳、罗家寨 | 普光净化厂（120亿m³/年）| 高含硫气田集中处理 |
| **塔里木盆地** | 克拉2、迪那2、克深 | 克拉2处理厂 | 高压高产气田 |
| **鄂尔多斯盆地** | 苏里格、靖边、榆林 | 苏里格处理群 | 致密砂岩气，分散处理 |
| **南海** | 荔湾3-1、陵水17-2 | 荔湾深水平台+陆上终端 | 深水天然气 |
| **东海** | 春晓、平湖 | 平湖处理厂 | 海上气田群 |
| **松辽盆地** | 徐深、长深 | 大庆处理厂 | 火山岩气藏、含CO₂ |

### 2.3 页岩气处理

以涪陵页岩气田为代表：
- 产能：100亿m³/年以上
- 特点：分散井口+集中脱水+外输
- 挑战：气质波动、产水量递减管理

### 2.4 煤层气处理

山西沁水盆地、鄂尔多斯东缘：
- 特点：低压、低产、需增压
- 主要采用TEG脱水+浅冷工艺

### 2.5 进口管道气处理

| 管道 | 来源 | 入境处理 |
|------|------|---------|
| 中亚管道（A/B/C/D线）| 土库曼斯坦、乌兹别克斯坦 | 霍尔果斯计量+处理 |
| 中缅管道 | 缅甸 | 瑞丽处理站 |
| 中俄东线 | 俄罗斯 | 黑河首站 |

## 6. 行业挑战与前景

### 6.1 挑战

- 高含硫气田安全风险（普光、元坝、罗家寨）
- CO₂排放约束（碳达峰/碳中和目标）
- 老气田气质劣化（CO₂/H₂S逐年上升）
- 设备腐蚀与完整性管理
- 偏远气田经济性

### 6.2 前景方向

- **数字化**：AI优化运行、预测维护、远程监控
- **CCUS集成**：处理厂碳捕集 + 驱油/封存
- **模块化**：快速建设、灵活扩展
- **氦气+稀有气体联合提取**：战略资源回收
- **绿氢掺混**：天然气与氢能基础设施融合
