# 天然气处理厂 — 概述与定义

## 1. 定义

天然气处理厂（Natural Gas Processing Plant）是将从气井产出的原料天然气（Raw Natural Gas）经过一系列物理和化学处理工艺，使其达到管输标准或液化天然气（LNG）原料标准的工业设施。原料天然气除主要成分甲烷（CH₄）外，通常含有硫化氢（H₂S）、二氧化碳（CO₂）、水蒸气、氮气（N₂）、氦气（He）以及乙烷（C₂H₆）、丙烷（C₃H₈）、丁烷（C₄H₁₀）和天然汽油（C₅⁺）等重烃组分。

## 2. 处理目标

天然气处理厂的核心目标包括：

| 目标 | 说明 |
|------|------|
| **净化** | 脱除H₂S、CO₂等酸性气体，防止管道腐蚀和环境污染 |
| **脱水** | 脱除水分，防止水合物生成和管道堵塞 |
| **凝液回收** | 提取乙烷、丙烷、丁烷等高附加值产品（NGL） |
| **调质** | 调整热值和华白指数，满足管输气质要求 |
| **安全合规** | 确保产品气符合国家标准和安全规范 |

## 3. 分类

### 3.1 按处理规模分类

| 类型 | 日处理能力 | 典型应用 |
|------|-----------|---------|
| 小型处理厂 | < 100万Nm³/d | 边远气田、煤层气 |
| 中型处理厂 | 100万 ~ 500万Nm³/d | 常规气田 |
| 大型处理厂 | 500万 ~ 2000万Nm³/d | 主力气田如普光、克拉2 |
| 超大型处理厂 | > 2000万Nm³/d | 特大型气田群集中处理 |

### 3.2 按工艺路线分类

- **浅冷分离工艺**：以回收C₃⁺为主，制冷温度约-25~-35℃
- **深冷分离工艺**：以回收C₂⁺为主，制冷温度约-70~-100℃
- **物理溶剂工艺**：采用Selexol、Rectisol等物理溶剂脱酸
- **化学溶剂工艺**：采用醇胺（MDEA等）化学吸收脱酸
- **膜分离工艺**：利用聚合物膜选择性渗透分离气体
- **变压吸附工艺（PSA）**：利用吸附剂选择性吸附杂质

### 3.3 按产品方案分类

- **燃料气型**：主要产出管输天然气
- **NGL回收型**：联产乙烷、丙烷、丁烷、天然汽油
- **LNG原料型**：深度处理为液化天然气提供原料
- **氦气回收型**：联产高纯氦气（如四川威远气田）

## 4. 天然气处理在产业链中的位置

```
气井采气 → 集输管网 → [天然气处理厂] → 产品气外输管道
                                      ├→ NGL分馏 → LPG、乙烷等
                                      ├→ LNG液化厂
                                      └→ 硫磺回收 → 工业硫磺
```

天然气处理厂是连接上游气田开发与下游市场供应的核心枢纽，其技术水平直接影响天然气资源的综合利用效率和经济效益。

## 5. 原料气组成典型范围

| 组分 | 体积分数（%） |
|------|:---------:|
| 甲烷 CH₄ | 70 ~ 98 |
| 乙烷 C₂H₆ | 0 ~ 10 |
| 丙烷 C₃H₈ | 0 ~ 5 |
| 丁烷 C₄H₁₀ | 0 ~ 3 |
| 戊烷以上 C₅⁺ | 0 ~ 2 |
| 二氧化碳 CO₂ | 0 ~ 20 |
| 硫化氢 H₂S | 0 ~ 15（高含硫气田可达20%+）|
| 氮气 N₂ | 0 ~ 15 |
| 水蒸气 H₂O | 饱和状态 |

## 6. 产品气质标准（中国GB 17820-2018）

| 指标 | 一类 | 二类 |
|------|------|------|
| 高位发热量（MJ/m³） | ≥ 34.0 | ≥ 31.4 |
| 总硫（mg/m³） | ≤ 20 | ≤ 100 |
| 硫化氢（mg/m³） | ≤ 6 | ≤ 20 |
| 二氧化碳（mol%） | ≤ 3.0 | ≤ 4.0 |
| 水露点（℃） | 在交接压力下，水露点≤输送条件下最低环境温度 | |

## 7. 安全与环保重要性

天然气处理厂涉及高压、易燃、易爆、有毒介质（H₂S），属于重大危险源。安全设计需遵循：
- HAZOP分析（危险与可操作性分析）
- SIL定级（安全完整性等级）
- 紧急停车系统（ESD）
- 火炬与放空系统
- 泄漏检测与报警系统

环境排放管控包括SO₂排放、VOCs逸散、废水处理与回注等。
