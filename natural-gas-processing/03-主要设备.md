# 天然气处理厂 — 主要设备

## 1. 气液分离设备

### 1.1 段塞流捕集器（Slug Catcher）

**功能**：接收和处理集输管道中因地形起伏产生的段塞流（液塞），缓冲气液波动。

**类型**：
| 类型 | 特点 | 适用 |
|------|------|------|
| 容器式 | 立式或卧式容器，占地小 | 中小型气田 |
| 管排式（Finger Type）| 多根平行斜管，储液量大 | 大型气田、段塞量大 |

**设计参数**：设计压力按管输压力，储液容积需满足最大段塞量，通常设置液位联锁。

### 1.2 三相分离器（3-Phase Separator）

**功能**：同时分离气、油（凝析油）、水三相。

**内部构件**：
- 入口分流器/挡板
- 聚结板/波纹板（提高分离效率）
- 除雾器（Wire Mesh / Vane Pack）
- 堰板（控制油水界面）

**操作原理**：利用重力沉降、惯性分离和聚结原理。气相流速控制在足以让100μm以上液滴沉降。

## 2. 吸收/再生塔类设备

### 2.1 胺液吸收塔（Amine Absorber）

**功能**：醇胺溶液与原料气逆流接触，化学吸收H₂S和CO₂。

**结构特点**：
- 填料塔或板式塔（浮阀塔盘为常见）
- 塔径：1~6m（按处理量）
- 操作压力：3.0~12.0 MPa
- 材质：碳钢 + 不锈钢内件（湿酸气区域需耐腐蚀）
- 塔顶设除沫器

**主要设计参数**：
- 气液比：取决于酸气负荷
- 胺液循环量：由酸气量×胺液酸气负荷决定
- 贫液温度：比进口气温高5~10℃（防止烃冷凝）

### 2.2 胺液再生塔（Amine Regenerator/Stripper）

**功能**：将富胺液中的酸气通过蒸汽汽提释放出来，恢复胺液吸收能力。

**结构**：
- 上部：填料段或塔盘（精馏段）
- 下部：再沸器（釜式或热虹吸式）
- 塔底温度：120~128℃（MDEA系统）
- 塔顶冷凝回流

**能耗**：再生热耗是胺法脱硫的主要运行成本，MDEA约占总能耗降低30~40%。

### 2.3 脱甲烷塔（Demethanizer）

**功能**：深冷分离的核心塔，将甲烷与C₂⁺分离。

**操作特点**：
- 低温塔：塔顶-85~-95℃，需低温钢（3.5Ni钢或不锈钢）
- 低压操作：1.5~3.5 MPa
- 塔顶冷凝器：乙烯制冷或膨胀机制冷
- 再沸器：蒸汽或热媒加热

**设计要点**：
- 保冷设计：聚氨酯泡沫或泡沫玻璃
- 低温材料冲击韧性
- 塔盘间距需考虑结霜裕量

## 3. 换热设备

### 3.1 冷箱（Cold Box / Brazed Aluminum Heat Exchanger, BAHE）

**功能**：多股流高效换热，回收冷量，降低制冷负荷。

**结构**：铝制钎焊板翅式，体积小、换热效率高。单台可同时进行5~10股流换热。

**应用**：深冷NGL回收装置的核心设备，实现进料预冷和产品复热。

### 3.2 贫富液换热器

**功能**：将来自再生塔的热贫胺液与来自吸收塔的冷富胺液换热，回收热量。

**类型**：
- 管壳式换热器
- 板式换热器（紧凑高效，近年应用增多）

### 3.3 再沸器（Reboiler）

**功能**：为再生塔/分馏塔提供汽相回流。

**类型**：
| 类型 | 特点 |
|------|------|
| 釜式再沸器（Kettle）| 汽化率高，占地大 |
| 热虹吸式（Thermosiphon）| 自然循环，紧凑 |
| 强制循环式 | 适合黏稠或易结垢介质 |

**加热介质**：蒸汽、导热油（热媒）、或直接火加热。

## 4. 转动设备

### 4.1 膨胀机（Turbo-Expander）

**功能**：高压气体等熵膨胀，同时降温降压并输出轴功。

**应用**：深冷NGL回收装置的核心制冷设备。

**结构**：
- 膨胀端：气体膨胀做功
- 增压端（同轴）：回收膨胀功驱动压缩机
- 转速：10,000~50,000 RPM
- 采用油润滑或磁悬浮轴承

**效率**：等熵效率 85~92%

### 4.2 离心压缩机

**功能**：原料气增压/产品气外输增压。

**驱动方式**：
- 燃气轮机（适合无电力条件）
- 电动机（变频调速，维护简单）
- 蒸汽轮机（有余热蒸汽时）

**特点**：大流量，压比适中，多级压缩带中间冷却。

### 4.3 往复压缩机

**功能**：高压比、中小流量场景，如分子筛再生气的压缩。

**特点**：压比可达3:1以上，排出压力可达数十MPa。

### 4.4 泵类设备

| 类型 | 应用 |
|------|------|
| 多级离心泵 | 胺液循环泵、NGL输送泵 |
| 计量泵 | 消泡剂、缓蚀剂注入 |
| 屏蔽泵/磁力泵 | 轻烃等易泄漏介质 |

## 5. 塔器内件与填料

### 5.1 塔盘类型

| 类型 | 特点 | 应用 |
|------|------|------|
| 浮阀塔盘 | 操作弹性大，效率高 | 胺液吸收/再生 |
| 筛板塔盘 | 结构简单，成本低 | 清洁介质 |
| 泡罩塔盘 | 低气速下不泄漏 | 低负荷工况 |
| 规整填料 | 低压降，高效率 | 脱甲烷塔等低温塔 |

### 5.2 填料

- **散堆填料**：鲍尔环、矩鞍环等
- **规整填料**：Mellapak系列、Gempak系列
- **选择要点**：比表面积、空隙率、压降、润湿性能

## 6. 脱水设备

### 6.1 TEG脱水单元

| 设备 | 功能 |
|------|------|
| TEG吸收塔 | 气液接触脱水（填料/塔盘） |
| TEG闪蒸罐 | 释放溶解烃类 |
| TEG过滤器 | 去除固体颗粒和降解产物 |
| TEG再生塔/再沸器 | 常压加热再生（~200℃） |
| TEG缓冲罐 | 贫/富TEG储存 |

### 6.2 分子筛脱水塔

- **结构**：立式固定床，底部支撑格栅 + 瓷球 + 分子筛床层
- **切换操作**：2塔或3塔切换（吸附→再生→冷却）
- **再生加热炉**：燃气直接加热再生气至230~290℃
- **分子筛寿命**：3~5年（取决于操作条件和再生频次）

## 7. 硫磺回收设备

### 7.1 Claus反应炉/废热锅炉

**功能**：H₂S部分燃烧生成SO₂并回收高温热量产生蒸汽。

**操作温度**：反应炉 1000~1400℃，出口~300℃。

### 7.2 Claus催化反应器

- 2~3级固定床反应器
- 催化剂：活性氧化铝、钛基催化剂
- 操作温度：200~350℃（露点以上）

### 7.3 硫磺冷凝器

- 将气态硫冷凝为液态硫（S₈）
- 液硫封：液硫收集与脱气

### 7.4 尾气处理单元（TGTU）

- SCOT反应器：加氢还原 + 胺液吸收
- 焚烧炉：将残余H₂S焚烧为SO₂排放

## 8. 储运设备

| 设备 | 功能 |
|------|------|
| NGL球罐/卧罐 | 乙烷、丙烷、丁烷低温或压力储存 |
| 液硫储罐 | 液硫储存与成型（硫磺造粒机） |
| 产品气计量撬 | 超声波/孔板流量计 + 在线色谱 |
| 装车撬 | NGL/LPG/硫磺槽车装卸 |

## 9. 安全与控制系统设备

| 系统 | 设备 |
|------|------|
| 火炬系统 | 火炬塔、分液罐、水封罐、点火系统 |
| 紧急停车（ESD） | ESD阀门、PLC/安全PLC |
| 火气系统（F&G） | H₂S/CH₄/火焰探测器、报警器 |
| 泄压系统 | 安全阀、爆破片 |
| 消防系统 | 消防泵、泡沫系统、喷淋冷却 |
