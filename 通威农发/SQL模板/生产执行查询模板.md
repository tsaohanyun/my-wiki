# 生产执行查询模板

## 概述

本模板整理了通威农发MES系统的常用SQL查询，对应技术架构中的MES生产执行模块。

## 常用查询

### 1. 工单状态统计

```sql
-- 统计各状态工单数量
SELECT 
    order_status,
    COUNT(*) as order_count,
    SUM(planned_qty) as total_planned_qty,
    SUM(actual_qty) as total_actual_qty
FROM mes_work_order
WHERE validflag = '1'
    AND create_time >= DATE_SUB(NOW(), INTERVAL 30 DAY)
GROUP BY order_status
ORDER BY order_count DESC;
```

### 2. 生产效率分析

```sql
-- 计算各产线的生产效率
SELECT 
    line_name,
    COUNT(DISTINCT order_id) as order_count,
    SUM(actual_qty) as total_output,
    SUM(TIMESTAMPDIFF(MINUTE, start_time, end_time)) as total_minutes,
    ROUND(SUM(actual_qty) / SUM(TIMESTAMPDIFF(HOUR, start_time, end_time)), 2) as output_per_hour
FROM mes_work_order wo
JOIN mes_production_line pl ON wo.line_id = pl.line_id
WHERE wo.validflag = '1'
    AND wo.order_status = 'COMPLETED'
    AND wo.complete_time >= DATE_SUB(NOW(), INTERVAL 7 DAY)
GROUP BY line_name
ORDER BY output_per_hour DESC;
```

### 3. 配料准确率

```sql
-- 检查配料准确率
SELECT 
    batch_id,
    material_code,
    material_name,
    planned_qty,
    actual_qty,
    ROUND(ABS(actual_qty - planned_qty) / planned_qty * 100, 2) as deviation_pct,
    CASE 
        WHEN ABS(actual_qty - planned_qty) / planned_qty <= 0.01 THEN '合格'
        WHEN ABS(actual_qty - planned_qty) / planned_qty <= 0.05 THEN '警告'
        ELSE '不合格'
    END as quality_status
FROM mes_batch_material
WHERE validflag = '1'
    AND batch_date >= DATE_SUB(NOW(), INTERVAL 7 DAY)
ORDER BY deviation_pct DESC;
```

### 4. 清洗记录查询

```sql
-- 查询清洗记录
SELECT 
    equipment_name,
    cleaning_type,
    start_time,
    end_time,
    TIMESTAMPDIFF(MINUTE, start_time, end_time) as duration_minutes,
    operator_name,
    cleaning_result
FROM mes_cleaning_record cr
JOIN mes_equipment eq ON cr.equipment_id = eq.equipment_id
WHERE cr.validflag = '1'
    AND cr.cleaning_date >= DATE_SUB(NOW(), INTERVAL 30 DAY)
ORDER BY cr.start_time DESC;
```

### 5. 报工统计

```sql
-- 按日报统计报工情况
SELECT 
    DATE(report_time) as report_date,
    COUNT(*) as report_count,
    SUM(report_qty) as total_qty,
    COUNT(DISTINCT operator_id) as operator_count,
    ROUND(SUM(report_qty) / COUNT(DISTINCT operator_id), 2) as avg_qty_per_person
FROM mes_work_report
WHERE validflag = '1'
    AND report_time >= DATE_SUB(NOW(), INTERVAL 30 DAY)
GROUP BY DATE(report_time)
ORDER BY report_date DESC;
```

## 性能优化建议

1. **索引优化**：确保`validflag`、`create_time`、`order_status`等字段有索引
2. **分区表**：对大表按时间分区
3. **查询优化**：避免SELECT *，只查询需要的字段
4. **批量操作**：使用批量插入/更新，减少数据库交互
