# 仓储管理查询模板

## 概述

本模板整理了通威农发WMS系统的常用SQL查询，对应技术架构中的WMS仓储管理模块。

## 常用查询

### 1. 库存概览

```sql
-- 各仓库库存汇总
SELECT 
    warehouse_name,
    COUNT(DISTINCT material_code) as material_count,
    SUM(current_qty) as total_qty,
    SUM(current_qty * unit_price) as total_value
FROM wms_inventory inv
JOIN wms_warehouse wh ON inv.warehouse_id = wh.warehouse_id
WHERE inv.validflag = '1'
GROUP BY warehouse_name
ORDER BY total_value DESC;
```

### 2. 库龄分析

```sql
-- 库龄分析（按天数分组）
SELECT 
    CASE 
        WHEN DATEDIFF(NOW(), inbound_date) <= 30 THEN '0-30天'
        WHEN DATEDIFF(NOW(), inbound_date) <= 60 THEN '31-60天'
        WHEN DATEDIFF(NOW(), inbound_date) <= 90 THEN '61-90天'
        ELSE '90天以上'
    END as age_group,
    COUNT(*) as item_count,
    SUM(current_qty) as total_qty,
    SUM(current_qty * unit_price) as total_value
FROM wms_inventory
WHERE validflag = '1'
GROUP BY age_group
ORDER BY age_group;
```

### 3. 出入库统计

```sql
-- 按日统计出入库数量
SELECT 
    DATE(operation_time) as op_date,
    operation_type,
    COUNT(*) as operation_count,
    SUM(quantity) as total_qty
FROM wms_inventory_transaction
WHERE validflag = '1'
    AND operation_time >= DATE_SUB(NOW(), INTERVAL 30 DAY)
GROUP BY DATE(operation_time), operation_type
ORDER BY op_date DESC, operation_type;
```

### 4. 拣货效率

```sql
-- 拣货效率分析
SELECT 
    picker_name,
    COUNT(*) as pick_count,
    SUM(pick_qty) as total_qty,
    AVG(TIMESTAMPDIFF(MINUTE, pick_start_time, pick_end_time)) as avg_duration,
    ROUND(SUM(pick_qty) / SUM(TIMESTAMPDIFF(HOUR, pick_start_time, pick_end_time)), 2) as qty_per_hour
FROM wms_picking_task
WHERE validflag = '1'
    AND pick_status = 'COMPLETED'
    AND pick_date >= DATE_SUB(NOW(), INTERVAL 7 DAY)
GROUP BY picker_name
ORDER BY qty_per_hour DESC;
```

### 5. 盘点差异

```sql
-- 盘点差异分析
SELECT 
    warehouse_name,
    material_code,
    material_name,
    system_qty,
    actual_qty,
    actual_qty - system_qty as diff_qty,
    ROUND((actual_qty - system_qty) / system_qty * 100, 2) as diff_pct
FROM wms_stocktaking_detail sd
JOIN wms_warehouse wh ON sd.warehouse_id = wh.warehouse_id
WHERE sd.validflag = '1'
    AND sd.stocktaking_id = 'ST202606001'
    AND actual_qty != system_qty
ORDER BY ABS(diff_pct) DESC;
```

## 性能优化建议

1. **索引优化**：确保`validflag`、`warehouse_id`、`material_code`等字段有索引
2. **分区表**：对`wms_inventory_transaction`按月分区
3. **物化视图**：对常用统计查询创建物化视图
4. **缓存策略**：对库存概览等查询使用Redis缓存
