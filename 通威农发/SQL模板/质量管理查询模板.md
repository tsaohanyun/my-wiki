# 质量管理查询模板

## 概述

本模板整理了通威农发QMS系统的常用SQL查询，对应技术架构中的QMS质量管理模块。

## 常用查询

### 1. 来料检验统计

```sql
-- 来料检验合格率统计
SELECT 
    DATE(inspect_date) as inspect_date,
    supplier_name,
    COUNT(*) as inspect_count,
    SUM(CASE WHEN inspect_result = 'PASS' THEN 1 ELSE 0 END) as pass_count,
    ROUND(SUM(CASE WHEN inspect_result = 'PASS' THEN 1 ELSE 0 END) / COUNT(*) * 100, 2) as pass_rate
FROM qms_incoming_inspection qi
JOIN qms_supplier qs ON qi.supplier_id = qs.supplier_id
WHERE qi.validflag = '1'
    AND qi.inspect_date >= DATE_SUB(NOW(), INTERVAL 30 DAY)
GROUP BY DATE(inspect_date), supplier_name
ORDER BY inspect_date DESC, pass_rate ASC;
```

### 2. 过程质量分析

```sql
-- 过程检验缺陷分析
SELECT 
    defect_type,
    defect_name,
    COUNT(*) as defect_count,
    SUM(defect_qty) as total_defect_qty,
    ROUND(SUM(defect_qty) / (SELECT SUM(actual_qty) FROM mes_work_order WHERE validflag = '1' AND create_time >= DATE_SUB(NOW(), INTERVAL 30 DAY)) * 100, 4) as defect_rate
FROM qms_process_inspection
WHERE validflag = '1'
    AND inspect_date >= DATE_SUB(NOW(), INTERVAL 30 DAY)
GROUP BY defect_type, defect_name
ORDER BY defect_count DESC
LIMIT 20;
```

### 3. 成品检验统计

```sql
-- 成品检验合格率趋势
SELECT 
    DATE(inspect_date) as inspect_date,
    product_line,
    COUNT(*) as inspect_count,
    SUM(CASE WHEN inspect_result = 'PASS' THEN 1 ELSE 0 END) as pass_count,
    ROUND(SUM(CASE WHEN inspect_result = 'PASS' THEN 1 ELSE 0 END) / COUNT(*) * 100, 2) as pass_rate
FROM qms_final_inspection
WHERE validflag = '1'
    AND inspect_date >= DATE_SUB(NOW(), INTERVAL 90 DAY)
GROUP BY DATE(inspect_date), product_line
ORDER BY inspect_date DESC;
```

### 4. 质量追溯

```sql
-- 批次追溯查询
SELECT 
    batch_code,
    material_code,
    material_name,
    supplier_name,
    incoming_date,
    inspect_result,
    used_in_orders,
    final_product_batch
FROM qms_batch_traceability
WHERE validflag = '1'
    AND batch_code = 'BATCH202606001';
```

### 5. 不合格品处理

```sql
-- 不合格品处理统计
SELECT 
    disposition_type,
    COUNT(*) as count,
    SUM(quantity) as total_qty,
    AVG(TIMESTAMPDIFF(DAY, discover_date, close_date)) as avg_process_days
FROM qms_nonconforming_product
WHERE validflag = '1'
    AND discover_date >= DATE_SUB(NOW(), INTERVAL 90 DAY)
GROUP BY disposition_type
ORDER BY count DESC;
```

## 质量指标计算

### DPMO（百万机会缺陷数）

```sql
-- 计算DPMO
SELECT 
    product_line,
    SUM(defect_count) as total_defects,
    SUM(inspect_count * opportunity_count) as total_opportunities,
    ROUND(SUM(defect_count) / SUM(inspect_count * opportunity_count) * 1000000, 2) as dpmo
FROM qms_quality_metrics
WHERE validflag = '1'
    AND metric_date >= DATE_SUB(NOW(), INTERVAL 30 DAY)
GROUP BY product_line
ORDER BY dpmo ASC;
```
