## 项目概述

本文档描述EBS（Oracle ERP）与DIM（数字化智能制造平台）之间的系统接口规范，涵盖数据导入、数据查询、实时同步三种接口方式。接口统一通过ESB服务总线调用，使用JSON报文格式传输。

- **导入接口**：DIM → EBS，通过 `CUX_DIM_IFACE_PKG.IFACE_TRS_P` 存储过程处理
- **查询接口**：EBS → DIM，通过 `CUX_DIM_IFACE_PKG.IFACE_GET_P` 存储过程处理
- **实时同步**：双向实时数据推送
