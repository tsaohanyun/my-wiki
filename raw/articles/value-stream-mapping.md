# VSM 价值流图析培训材料

## 课程目的
1. 转换思维：望远镜、站在山顶、总经理的思维
2. 学会绘制现状价值流图、布局图、制程山积图，将八大浪费具体化
3. 基于公司规划目标，根据精益原则绘制未来制程山积图、价值流图、布局图
4. 根据现状与未来的差异进行方针策略展开，制定一年详细计划

## 大纲
一、价值流相关概念
二、绘制价值流图

## 核心概念
- 价值流（VSM）：从原材料到成品交付给客户的全过程，包括增值和非增值活动
- 增值比：增值时间 / 总时间（案例：去医院买药，增值比仅2%）
- 三大系统：物流改善、情报流改善、效率改善
- 八大浪费：等待、搬运、不良、加工、管理...

## 改善框架
- 现状价值流图 → 现状山积图 → 现状布局图
- 公司3-5年规划 → 未来山积图 → 未来价值流图 → 未来布局图
- 差异(GAP)分析 → 方针策略展开 → 年份计划
