## 一、概述：什么是 SIE-IIDP？

SIE-IIDP 是一个**企业级工业互联网数据平台**，为制造业提供 MES（制造执行系统）功能。其前端设计风格是一种典型的**中国企业级后台管理系统风格**，以强品牌色、高数据密度、功能主义为核心特征。

技术栈基于：
- **Vue.js 2.x + Element UI** 底层框架
- **SMI Design** 自研设计系统/组件库
- **Tech Framework** 自研微前端架构
- 微前端内部应用体系（base-inner、workFlow-inner、user-inner、permission-inner）

## 三、排版系统

### 3.1 根字号缩放策略

核心特征：**html 根元素 font-size: 100px**

这是一个极具特色的做法：
- `1rem = 100px`
- 所有尺寸以 rem 为单位，形成直观的映射关系
- `0.14rem = 14px`、`0.32rem = 32px`、`0.10rem = 10px`

### 3.2 字号层级

| 用途 | 大小 |
|------|------|
| 表格内容 | 0.14rem (14px) |
| 按钮文字 | 0.14rem (14px) |
| 表单标签 | 0.14rem (14px) |
| 对话框标题 | 12-14px |
| 小号文字 | 12px |

### 3.3 字体族

- 使用系统默认字体，无自定义Web字体
- 中文优先，完整CJK支持
- 全部元素 `font: inherit; font-size: 100%`（reset）
- 行高：`line-height: 1`（紧凑型）

## 五、组件设计语言

### 5.1 侧边栏 (Sidebar)
- **背景色**：`#E60012`（#E60012）
- **激活项背景**：`#BF0016`（深红）
- **图标**：20×20 SVG，白色填充，圆形徽章风格
- **图标激活色**：`#0878CB`（蓝色）
- **非激活透明度**：1（完全可见，不降低透明度）
- **菜单文字**：白色

### 5.2 数据表格 (Table)
- **表头背景**：`#f2f3f5`（浅灰）
- **已查看行**：`#EDF1F7`（浅蓝灰）
- **选中行**：`#53aefc` 边框 + 半透明背景
- **切换视图按钮**：`#677177`
- **搜索按钮**：`var(--button-primary-bg, #677177)`
- **重置按钮**：白色背景 + `#677177` 边框
- 基于 Element UI `el-table` + 自定义 `vxe-table`

### 5.3 标签页 (Tabs)
- **激活态**：底部 `#E60012` 下划线 + `#E60012` 文字色
- **悬停态**：`rgba(70, 144, 199, 1)` 文字色
- 基于 Element UI `el-tabs`

### 5.4 对话框 (Dialog)
- **标题栏背景**：`#E60012`
- **标题栏文字**：白色
- **确认按钮**：`#E60012`（悬停：`#f22731`）
- **圆角**：5px（较大，是少数圆角明显的组件）
- 基于 Element UI `el-dialog`

### 5.5 按钮 (Button)
- **主按钮**：`--button-primary-bg`（默认 `#AD053D` 或 `#677177`）
- **圆角**：**2px**（极小，近乎直角！这是关键特征）
- **内边距**：`0 16px`
- **文字按钮**：干净简洁
- 工具栏按钮组：紧凑排列
- 基于 Element UI `el-button` + 自定义 `smi-btn`

### 5.6 表单 (Form)
- 基于 Element UI `el-form` / `el-form-item`
- 表单项间距：`margin-bottom: 0.12rem`
- 支持多种数据源配置表单（SQL、API、WebSocket、Kafka、RabbitMQ、Excel、OPC UA、视频流）
- 紧凑布局，标签左对齐

## 八、总体设计哲学

### 8.1 设计理念关键词
- **企业优先**：一切为数据展示和操作效率服务
- **品牌驱动**：#E60012贯穿始终，强识别性
- **功能主义**：极低圆角（2px）、最少阴影、无渐变
- **高密度**：表格多列、表单多字段、信息紧凑
- **结构化**：侧边栏 → 面包屑 → 标签页 → 内容区，清晰的层级

### 8.2 与其他设计风格对比

| 维度 | SIE-IIDP | Apple Design | Ant Design | Material Design |
|------|----------|-------------|-----------|-----------------|
| 信息密度 | 极高 | 低 | 中高 | 中 |
| 圆角 | 2px (极小) | 大圆角 | 2-6px | 4-8px |
| 阴影 | 极少 | 极轻 | 中等 | 多层次 |
| 主色调 | 红色 | 中性灰白 | 蓝色 | 蓝色 |
| 动效 | 极少 | 无 | 丰富 | 丰富 |
| 留白 | 紧凑 | 大量 | 适中 | 适中 |
| 目标用户 | 工厂操作员 | 消费者 | 企业用户 | 混合 |
| 设计哲学 | 功能主义 | 极简优雅 | 企业效率 | 材料隐喻 |

## 九、技术实现特征

### 9.1 微前端架构
- 内部应用（inner-app）体系实现模块化加载
- 每个业务模块独立打包部署
- 共享基础依赖（iidp.dep.js）

### 9.2 组件库分层
```
smi-design ← 设计Token和基础样式
 ├── smi-base ← 基础组件
 ├── smi-ida ← 核心架构组件
 ├── smi-ida-two/three ← 扩展组件
 ├── smi-ida-scene/newScene ← 场景/页面组件
 ├── smi-ida-chart ← 图表组件
 └── smi-ida-report ← 报表组件
```

### 9.3 样式架构
- CSS重置（Eric Meyer风格）
- CSS变量实现主题定制
- 组件级样式隔离（scoped styles with unique IDs）
- rem基准字号 100px 实现统一缩放

## 十一、总结

SIE-IIDP 设计风格是一种**典型的中国大型制造企业后台管理系统风格**。其核心特征可概括为：

1. **#E60012** (#E60012) 作为品牌主色，深度嵌入每个UI组件
2. **100px根字号** rem缩放体系，形成独特的尺寸规范
3. **2px极小圆角** 的按钮和输入框，体现功能主义
4. **白色圆形徽章SVG图标**，在红色侧边栏上形成强烈对比
5. **高密度数据展示**，面向工厂操作场景优化
6. **Element UI + SMI Design** 双层组件体系
7. **微前端模块化**架构，支持大规模企业应用

这是一种**高度实用主义**的设计风格，美学让位于功能效率，品牌色贯穿始终形成统一识别，是工业互联网领域的一个典型参考案例。
