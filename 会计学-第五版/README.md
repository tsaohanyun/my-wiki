```
第1章（概念入门）→ 第2章（理论基础）→ 第3章（财务报表）
                                        ↓
第4章（会计循环）→ 第5章（收入与资产）→ 后续章节...
```

## 各章关联关系

- **第1章** 为全书导论，建立会计整体认知
- **第2章** 提供理论框架（假设、原则、质量要求）
- **第3章** 介绍财务报表（会计信息的最终载体）
- **第4章** 讲解会计循环（从交易到报表的处理过程）
- **第5章** 深入收入确认和货币性资产（第3章报表项目的具体处理）
