# 第6章 存货与销售成本

## 章节概述

本章系统介绍了存货的确认、计量与报告方法。存货是企业重要的流动资产，其计价直接影响资产负债表中的存货金额和利润表中的销售成本金额。本章涵盖存货的种类与范围界定、定期盘存制与永续盘存制、存货取得成本计量、发出存货的计价方法（个别计价法、先进先出法、加权平均法）以及成本与市价孰低法等内容，帮助MBA学员理解存货管理决策对企业财务报告和税务的深远影响。

## 核心知识点

### 6.1 存货概述

**定义：** 存货是指企业在日常生产经营过程中持有以备出售，或者仍然处在生产过程中，或者在生产或提供劳务过程中将消耗的材料或物料等。

**要点：**
- 存货按经济用途分为：制造业存货（原材料、在产品、产成品）、商品流通企业存货、其他行业存货
- 存货按存放地点分为：库存存货、在途存货、委托加工存货
- 存货确认的两个条件：①与该存货有关的经济利益很可能流入企业；②该存货的成本能够可靠地计量
- 存货范围界定的关键是**法定所有权**，而非实际存放地点

**示例：** 委托代销商品虽存放在受托方，但因委托方拥有法定所有权，应列为委托方的存货。

### 6.2 存货盘存制度

**定义：** 存货盘存制度是确定存货实物数量的方法，主要有定期盘存制和永续盘存制两种。

**要点：**

| 特征 | 定期盘存制 | 永续盘存制 |
|------|-----------|-----------|
| 记录方式 | 平时只记录购入，期末盘点确定结存 | 随时记录购入和发出 |
| 计算公式 | 发出量 = 期初 + 本期购入 - 期末 | 逐笔计算结存 |
| 优点 | 核算简单 | 实时掌握存货信息 |
| 缺点 | 无法及时发现短缺 | 核算工作量大 |
| 适用范围 | 价值低、收发频繁的存货 | 大多数企业 |

**实务提示：** 即使采用永续盘存制，也应定期进行实地盘点以保证账实相符。

### 6.3 存货计价方法

**定义：** 存货计价方法是确定发出存货成本和期末存货成本的方法，基于不同的成本流转假设。

**要点：**

1. **个别计价法**：按每批存货的实际进价确定成本。适用于体积大、价格高、数量少的存货（如古董、高档汽车）。

2. **先进先出法**：假设先入库的存货先发出。期末存货以最近购入的单价计价，接近当前市价；但在物价上涨时利润偏高，税收负担加重。

3. **加权平均法**：
   - 一次加权平均法：月末计算一次平均成本（适用于定期盘存制）
   - 移动加权平均法：每收入一批存货即重新计算平均成本（适用于永续盘存制）

**MBA实务要点：**
- 存货计价方法的选择会影响利润、税收和现金流
- 我国准则已剔除后进先出法（LIFO），允许使用先进先出法、加权平均法或个别计价法
- 计价方法一经确定，前后年度应保持一致，不得随意更改

### 6.4 成本与市价孰低法

**定义：** 成本与市价孰低法要求以存货的账面成本与可变现净值中较低者作为计价基础，体现谨慎性原则。

**要点：**
- **可变现净值** = 估计售价 - 至完工估计将要发生的成本 - 估计的销售费用 - 相关税费
- 比较方法：单项比较法（最保守）> 分类比较法 > 综合比较法
- 跌价损失确认采用**备抵法**：设置"存货跌价准备"科目
- 已计提跌价准备的存货，市价回升时可转回，但转回金额不得超过原已确认的跌价损失

**示例：** 甲公司存货成本80,000元，可变现净值69,000元，应计提存货跌价准备11,000元。若下期可变现净值回升至85,000元，可转回全部11,000元跌价准备。

## 关键术语表

| 术语 | 定义 |
|------|------|
| 存货（Inventory） | 企业在日常活动中持有以备出售的产品、处在生产过程中的在产品、在生产或提供劳务过程中耗用的材料和物料 |
| 定期盘存制（Periodic Inventory System） | 通过期末实地盘点确定存货结存数量和金额的方法 |
| 永续盘存制（Perpetual Inventory System） | 在明细账上序时登记存货购入、发出，随时计算结存数量的方法 |
| 先进先出法（FIFO） | 假设先入库的存货先发出，据此确定发出和期末存货成本 |
| 加权平均法（Weighted Average Method） | 以期初存货和本期各批收入数量为权数计算平均单位成本 |
| 成本与市价孰低法（Lower of Cost or Market） | 以存货成本与可变现净值中较低者计价的方法 |
| 存货跌价准备（Allowance to Reduce Inventory） | 存货成本高于可变现净值时计提的减值准备 |
| 可变现净值（Net Realizable Value） | 存货的估计售价减去至完工时估计将要发生的成本、销售费用及相关税费后的金额 |

## 实务应用要点

1. **存货范围判断**：以法定所有权为标准，而非实物存放地点。在途商品的归属取决于交货条件（目的地交货 vs 起运点交货）。

2. **计价方法选择**：企业应根据存货特点、税收影响、利润目标综合考虑。先进先出法使期末存货更接近市价，但物价上涨时利润偏高；加权平均法较为折中。

3. **跌价准备的利润管理**：管理者可通过高估或低估可变现净值来调节利润，报表使用者应关注存货跌价准备的计提与转回情况。

4. **存货信息披露**：企业应在附注中披露各类存货的账面价值、计价方法、跌价准备的计提与转回情况等。
