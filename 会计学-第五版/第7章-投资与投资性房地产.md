# 第7章 投资与投资性房地产

## 章节概述

本章讨论企业对外投资的会计处理，涵盖金融资产的分类与计量、长期股权投资的确认与后续计量，以及投资性房地产的会计处理。2017年修订的金融工具准则将金融资产由"四分类"改为"三分类"，引入预期信用损失法，这些变化对投资的会计处理产生了深远影响。本章帮助MBA学员理解投资决策的会计后果以及准则变革的逻辑。

## 核心知识点

### 7.1 投资概述

**定义：** 财务会计中的投资指对外投资，即将资金投入其他单位以获取收益或控制权的经济行为。

**要点：**
- 对外投资分为直接投资（取得股权或债权）和间接投资（购买有价证券）
- 按投资期限分为短期投资（不超过一年）和长期投资
- 按投资品种分为债权投资、股权投资和混合投资
- 按现行准则分为**金融资产**和**长期股权投资**两大类

**投资动机：** 利用闲置资金获取回报、获取长期稳定收益、获得对被投资单位的控制权或重大影响。

### 7.2 金融资产

**定义：** 金融资产是企业持有的现金、其他方的权益工具以及从其他方收取现金或其他金融资产的合同权利等。

**要点：**

**新准则"三分类"体系：**

| 分类 | 计量方式 | 适用条件 | 会计科目 |
|------|---------|---------|---------|
| 第一类 | 以摊余成本计量 | 业务模式为收取合同现金流量，且合同现金流量仅为本金和利息 | 债权投资、应收账款等 |
| 第二类 | 以公允价值计量且变动计入其他综合收益（FVOCI） | 业务模式为既收取合同现金流量又出售 | 其他债权投资、其他权益工具投资 |
| 第三类 | 以公允价值计量且变动计入当期损益（FVTPL） | 不满足上述两类条件 | 交易性金融资产 |

**金融资产减值：** 新准则采用**预期信用损失法**，要求基于过去、现在和前瞻性信息计提减值准备，比原准则的已发生损失法更加及时、充分。

**初始计量：** 按公允价值计量。FVTPL的交易费用直接计入当期损益；其他类别的交易费用计入初始确认金额。

**后续计量：**
- 以摊余成本计量的金融资产：按实际利率法确认利息收入，摊销溢折价
- 以公允价值计量的金融资产：按公允价值变动确认损益或其他综合收益

### 7.3 长期股权投资

**定义：** 长期股权投资是指投资方对被投资单位实施控制、重大影响的权益性投资，以及对其合营企业的权益性投资。

**要点：**
- **控制**：通常持股50%以上，采用成本法核算
- **重大影响**：通常持股20%-50%，采用权益法核算
- **合营企业**：共同控制，采用权益法核算
- 成本法下，投资按初始成本计量，被投资单位宣告分派利润时确认投资收益
- 权益法下，投资账面价值随被投资单位净资产变动而调整

### 7.4 投资性房地产

**定义：** 投资性房地产是指为赚取租金或资本增值（或两者兼有）而持有的房地产。

**要点：**
- 包括：已出租的土地使用权、已出租的建筑物、持有并准备增值后转让的土地使用权
- 不包括：自用房地产（固定资产）、作为存货的房地产
- 计量模式：**成本模式**（主流）或**公允价值模式**
- 成本模式下参照固定资产或无形资产进行折旧/摊销
- 公允价值模式下不计提折旧，公允价值变动计入当期损益

## 关键术语表

| 术语 | 定义 |
|------|------|
| 金融资产（Financial Assets） | 企业持有的现金、其他方的权益工具以及符合特定条件的合同权利 |
| 摊余成本（Amortized Cost） | 金融资产的初始确认金额经调整利息摊销、减值等后的金额 |
| 公允价值（Fair Value） | 市场参与者在计量日发生的有序交易中出售资产所能收到的价格 |
| FVTPL | 以公允价值计量且其变动计入当期损益的金融资产 |
| FVOCI | 以公允价值计量且其变动计入其他综合收益的金融资产 |
| 预期信用损失（Expected Credit Loss） | 以违约概率为权重的加权平均信用损失 |
| 长期股权投资（Long-term Equity Investment） | 对被投资单位实施控制、重大影响的权益性投资 |
| 投资性房地产（Investment Property） | 为赚取租金或资本增值而持有的房地产 |
| 权益法（Equity Method） | 根据被投资单位净资产变动调整投资账面价值的方法 |
| 成本法（Cost Method） | 投资按初始成本计量，收到利润分配时确认投资收益的方法 |

## 实务应用要点

1. **金融资产分类判断**：需要综合考虑"业务模式"和"合同现金流量特征"两个维度，具有一定的主观判断空间，管理层应审慎判断。

2. **减值准备的前瞻性**：预期信用损失法要求企业建立信用风险管理体系，基于前瞻性信息计提减值，对金融机构影响尤为重大。

3. **权益法 vs 成本法**：持股比例是判断标准之一，但实质重于形式——有证据表明具有重大影响的，即使持股低于20%也应采用权益法。

4. **投资性房地产计量选择**：公允价值模式可避免折旧导致的利润波动，但需要可靠的公允价值证据。我国多数企业采用成本模式。
