# 第12章 影响净利润的其他项目

## 章节概述

本章系统介绍了除销售成本之外影响企业净利润的主要项目，包括期间费用、营业外收支、所得税会计、会计政策变更与差错更正、汇兑损益与折算损益、综合收益与其他综合收益等。通过本章学习，读者可以全面了解影响企业利润的各类因素，掌握相关会计处理方法，提高分析企业盈利质量的能力。本章内容对理解企业利润表的构成和进行财务分析具有重要意义。

## 核心知识点

### 2.1 期间费用

**定义：** 期间费用是指企业本期发生的、不能直接或间接归入某种产品成本的、直接计入损益的各项费用。

**要点：**
- 期间费用包括：销售费用、管理费用、研发费用、财务费用
- 销售费用：企业在销售商品、提供劳务过程中发生的各种费用
- 管理费用：企业为组织和管理企业生产经营所发生的各种费用
- 研发费用：企业进行研究与开发过程中发生的费用化支出
- 财务费用：企业为筹集生产经营所需资金等而发生的应予费用化的筹资费用

**示例：** 销售费用包括保险费、包装费、展览费和广告费、商品维修费、运输费、装卸费等；管理费用包括开办费、公司经费、工会经费、待业保险费、董事会会费等。

### 2.2 营业外收支

**定义：** 营业外收支是指企业发生的与日常活动无直接关系的各项收支。

**要点：**
- 营业外收入：非流动资产毁损报废利得、债务重组利得、政府补助、盘盈利得、捐赠利得等
- 营业外支出：非流动资产毁损报废损失、债务重组损失、公益性捐赠支出、非常损失、盘亏损失等
- 营业外收入和营业外支出应当分别核算，不得相互冲减

**示例：** 企业出售毁损报废的固定资产的残料价值及变价收入等，扣除处置固定资产的账面价值、清理费用和相关税费后的净收益，计入营业外收入。

### 2.3 所得税与所得税会计

**定义：** 所得税是根据企业的所得额计征的税种，其计税基础（应纳税所得额）可能与企业会计的账面利润不同。

**要点：**
- 所得税会计方法：我国采用资产负债表债务法
- 暂时性差异：资产、负债的账面价值与其计税基础不同所产生的差额
- 应纳税暂时性差异：导致产生应税金额的暂时性差异，确认递延所得税负债
- 可抵扣暂时性差异：导致产生可抵扣金额的暂时性差异，确认递延所得税资产
- 所得税费用 = 当期所得税 + 递延所得税

**示例：** 资产的计税基础是企业收回资产账面价值过程中，计算应纳税所得额时按照税法规定可以自应税经济利益中抵扣的金额。固定资产在某一资产负债表日的计税基础是其成本扣除按照税法规定已在以前期间税前扣除的累计折旧额后的余额。

### 2.4 会计政策、会计估计变更和前期差错更正

**定义：** 会计政策是指企业在会计确认、计量和报告中所采用的原则、基础和会计处理方法。

**要点：**
- 会计政策变更：企业对相同的交易或者事项由原来采用的会计政策改用另一会计政策
- 会计估计变更：由于资产和负债的当前状况及预期经济利益和义务发生了变化，从而对资产或负债的账面价值或者资产的定期消耗金额进行调整
- 前期差错更正：由于没有运用或错误运用编报前期财务报表时预期能够取得的可靠信息，而对前期财务报表造成的遗漏或误报

**示例：** DH钢铁公司对各类固定资产重新核定了使用年限，决定从2020年1月1日起对公司部分固定资产折旧年限进行调整，预计使2020年所有者权益和净利润均增加9亿元。

### 2.5 汇兑损益与折算损益的影响

**定义：** 汇兑损益是企业发生的外币业务由于汇率变动而产生的损益；折算损益是外币财务报表折算过程中产生的损益。

**要点：**
- 外币业务及汇兑损益的产生：企业发生外币交易时，由于记账汇率与账面汇率不同而产生的损益
- 外币财务报表的折算与折算损益：将境外经营的财务报表折算为记账本位币报表时产生的损益

**示例：** 企业进口原材料时以美元结算，由于汇率变动，导致实际支付的人民币金额与账面金额不同，产生汇兑损益。

### 2.6 综合收益与其他综合收益

**定义：** 综合收益是指企业在某一期间除与所有者以其所有者身份进行的交易之外的其他交易或事项所引起的所有者权益变动。

**要点：**
- 综合收益 = 净利润 + 其他综合收益
- 其他综合收益：企业根据会计准则的规定未在当期损益中确认的各项利得和损失
- 其他综合收益包括以后会计期间不能重分类进损益的和以后会计期间满足规定条件时将重分类进损益的两类

**示例：** 企业持有的可供出售金融资产公允价值变动计入其他综合收益，在处置时将累计变动转入当期损益。

## 实务应用要点

**MBA实务视角：**

1. **期间费用控制**：期间费用是企业利润的重要影响因素，管理者需要通过预算管理和成本控制来优化期间费用结构。

2. **营业外收支分析**：营业外收支具有偶发性，在分析企业盈利能力时应予以剔除，关注企业核心业务的盈利质量。

3. **所得税筹划**：企业应合理利用暂时性差异进行税收筹划，但必须符合税法规定，避免税务风险。

4. **会计政策选择**：会计政策变更可能对财务报表产生重大影响，企业应谨慎选择并保持一致性。

5. **盈余管理识别**：通过分析会计估计变更、折旧年限调整等手段，识别企业可能存在的盈余管理行为。

6. **外币风险管理**：涉及外币业务的企业应建立完善的外币风险管理体系，合理控制汇兑损益。

## 与其他章节的关联

- **与第3章（资产负债表与利润表）的关联**：本章涉及的期间费用、营业外收支、所得税费用等都是利润表的重要组成部分。
- **与第5章（收入与货币性资产）的关联**：销售费用与收入确认密切相关，汇兑损益涉及货币性资产的核算。
- **与第6章（存货与销售成本）的关联**：管理费用中的存货跌价准备、研发费用中的材料消耗等与存货核算相关。
- **与第11章（所有者权益）的关联**：净利润的变动直接影响未分配利润，其他综合收益属于所有者权益的组成部分。
- **与第13章（现金流量表）的关联**：所得税费用、财务费用等对现金流量表的编制有重要影响。

1. **期间费用的构成**：掌握销售费用、管理费用、研发费用、财务费用的内容和核算方法。
2. **营业外收支的核算**：理解营业外收入和营业外支出的主要内容及其对利润的影响。
3. **所得税会计**：掌握资产负债表债务法、暂时性差异、递延所得税资产和负债的确认与计量。
4. **会计政策变更**：理解会计政策变更、会计估计变更和前期差错更正的区别及会计处理。

6. **综合收益**：理解综合收益和其他综合收益的概念及其在利润表中的列报。
