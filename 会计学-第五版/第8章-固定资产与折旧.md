# 第8章 固定资产与折旧

## 章节概述

本章系统介绍固定资产的确认、计量、折旧、后续支出及处置的会计处理。固定资产是企业重要的生产资料，尤其在制造业企业中占比较大。折旧作为将固定资产成本系统分摊到各使用期间的过程，直接影响企业的利润和税负。本章帮助MBA学员理解不同折旧方法的经济后果以及固定资产会计政策选择的实务考量。

## 核心知识点

### 8.1 固定资产概述

**定义：** 固定资产是指同时具备下列特征的有形资产：①为生产商品、提供劳务、出租或经营管理而持有的；②使用寿命超过一个会计年度。

**要点：**
- 固定资产三大特征：有形资产、为经营持有（非出售）、使用寿命超一年
- 区分固定资产与存货的关键在于持有目的：同一房屋在一般企业是固定资产，在房地产公司是存货
- 需遵循**资本性支出与收益性支出**的区分原则
- 我国企业不能拥有土地所有权，只能拥有土地使用权（属于无形资产）

**分类：**
- 按经济用途：生产经营用 vs 非生产经营用
- 按使用情况：使用中、未使用、不需用

### 8.2 固定资产取得成本

**定义：** 固定资产的取得成本是指使其达到预定可使用状态前所发生的一切合理、必要的支出。

**要点：**

| 取得方式 | 成本构成 |
|---------|---------|
| 外购 | 购买价款 + 相关税费 + 运输费 + 装卸费 + 安装费 + 专业人员服务费 |
| 自行建造 | 建造过程中发生的全部必要支出（含符合条件的借款费用资本化） |
| 接受投资 | 投资合同或协议约定的公允价值 |
| 接受捐赠 | 凭据金额或同类资产市场价格，计入营业外收入 |
| 非货币性资产交换 | 具有商业实质且公允价值可靠：按公允价值计量；否则按账面价值 |

**借款费用资本化条件（同时满足）：**
1. 资产支出已经发生
2. 借款费用已经发生
3. 为使资产达到预定可使用状态所必要的购建活动已经开始

**一揽子购入：** 以总价购入多项无单独标价的固定资产时，应按各项固定资产的公允价值比例分摊总成本。

### 8.3 固定资产折旧

**定义：** 折旧是指在固定资产使用寿命内，按照确定的方法对应计折旧总额进行系统分摊。

**影响折旧的三个因素：**
1. **应计折旧总额** = 固定资产原值 - 预计净残值
2. **估计使用年限**（考虑有形损耗和无形损耗）
3. **折旧方法**

**折旧方法对比：**

| 方法 | 特点 | 适用场景 |
|------|------|---------|
| 直线法（年限平均法） | 各期折旧额相等，计算简单 | 各期使用情况大致相同的资产（房屋建筑物） |
| 工作量法 | 按实际工作量计提，考虑使用程度 | 使用情况各期变动大的资产（运输车辆） |
| 年数总和法 | 变率加速折旧法，折旧率逐年递减 | 初期效能高、维修费低的资产 |
| 双倍余额递减法 | 定率加速折旧法，以双倍直线折旧率计算 | 技术更新快的资产 |

**实务提示：**
- 我国实务：当月新增固定资产从次月起计提折旧；当月减少的固定资产当月照提折旧
- 折旧方法一经选定不得随意更改（除非经济利益预期实现方式有重大改变）
- 折旧按用途计入相关成本费用科目（制造费用、管理费用、销售费用等）

**MBA实务要点——不同方法的影响：**

- **直线法**：各期利润平稳，计算简便
- **加速折旧法**：前期利润偏低、税负较轻（相当于获得无息贷款），但可能影响投资者信心；后期利润偏高
- 折旧费用是估计结果，报表使用者应关注折旧政策是否合理

### 8.4 固定资产后续支出与处置

**后续支出：**
- 日常维修：直接费用化，计入当期损益
- 大修理/改良：符合固定资产确认条件的，资本化计入固定资产成本
- 判断标准：是否延长使用年限、提高服务潜能

**固定资产减值：**
- 存在减值迹象时，应估计可收回金额
- 可收回金额 = max（公允价值 - 处置费用，预计未来现金流量现值）
- 减值损失 = 账面价值 - 可收回金额
- **减值损失一经确认，以后会计期间不得转回**

**固定资产处置：**
- 通过"固定资产清理"科目核算
- 出售净损益计入"资产处置损益"或"营业外收入/支出"
- 盘亏通过"待处理财产损溢"科目核算

## 关键术语表

| 术语 | 定义 |
|------|------|
| 固定资产（Fixed Assets） | 为生产商品、提供劳务、出租或经营管理而持有，使用寿命超过一个会计年度的有形资产 |
| 原始成本（Historical Cost） | 取得固定资产所支付的全部货币总额 |
| 折旧（Depreciation） | 在固定资产使用寿命内对应计折旧总额进行系统分摊 |
| 应计折旧额（Depreciable Cost） | 固定资产原值减去预计净残值后的金额 |
| 净残值（Residual Value） | 固定资产报废时残值收入扣除清理费用后的金额 |
| 累计折旧（Accumulated Depreciation） | 固定资产自取得以来已计提的折旧总额 |
| 直线法（Straight-line Method） | 将应计折旧额平均分摊到各使用年限的方法 |
| 加速折旧法（Accelerated Depreciation） | 在使用初期多提折旧、后期少提折旧的方法 |
| 利息资本化（Capitalization of Interest） | 将符合条件的借款费用计入相关资产成本 |
| 资产减值（Impairment） | 资产可收回金额低于账面价值时确认的损失 |
| 一揽子购入（Lump-sum Purchase） | 以一笔款项购入多项没有单独标价的固定资产 |

## 实务应用要点

1. **折旧年限的调整**：企业可在合理范围内调整折旧年限（如鞍钢股份案例），但应注意是否构成利润操纵。调整折旧年限属于会计估计变更，采用未来适用法。

2. **加速折旧法的税务优势**：前期折旧多→利润低→税负轻→获得更多营运资金，但需权衡对投资者信心的影响。

3. **借款费用资本化的判断**：对自建固定资产项目，正确区分资本化期间和费用化期间至关重要，直接影响资产价值和当期利润。

4. **减值测试的审慎性**：固定资产减值不得转回的规定体现了谨慎性原则，但也可能导致资产价值被低估。
