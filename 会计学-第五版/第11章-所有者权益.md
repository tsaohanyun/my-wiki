# 第11章 所有者权益

## 章节概述

本章系统介绍了所有者权益的定义、特点、构成及会计处理。所有者权益是企业资产扣除负债后由所有者享有的剩余权益，包括实收资本（股本）、其他权益工具、资本公积、其他综合收益、盈余公积和未分配利润等。本章分别阐述了非公司制企业（独资企业、合伙企业）和公司制企业的所有者权益核算特点，重点讲解了股份有限公司的股东权益核算，包括普通股、优先股、库存股的会计处理，以及权益工具与金融负债的区分和复合金融工具的分拆。

## 核心知识点

### 2.1 所有者权益概述

**定义：** 所有者权益是指企业资产扣除负债后由所有者享有的剩余权益，企业的所有者权益又称为股东权益。

**要点：**
- 所有者权益的分类：
  - 按构成分：投入资本、资本公积、留存收益
  - 按经济内容分：投入资本、其他权益工具、资本公积、其他综合收益、盈余公积、未分配利润
  - 按形成渠道分：原始投入的资本、经营中形成的资本
- 所有者权益的特点：
  - 在经营期内可供企业长期、持续使用，不必向投资者返还本金
  - 企业所有者凭其投入的资本，拥有享受税后分配利润的权利
  - 企业所有者享有行使企业经营管理的权利
  - 企业所有者对企业债务和亏损负有无限责任或有限责任

**示例：** 企业期末所有者权益金额并不一定等于企业净资产的市场价值。变卖企业资产时，如果当时的市价超出净资产的账面价值，其超出部分通常作为商誉处理。

### 2.2 非公司制企业的所有者权益

**定义：** 非公司制企业包括独资企业和合伙企业。

**要点：**
- 独资企业：由个人出资经营、归个人所有和控制、由个人承担经营风险和享有全部经营收益的企业
- 合伙企业：由两个或两个以上合伙人共同出资经营的企业组织形式
- 独资企业通常只设置两个科目：实收资本（或业主权益）和提款科目
- 合伙企业通常为无限责任公司，交纳个人所得税
- 有限责任合伙企业：合伙人对企业债务承担有限责任，但纳税时按照合伙企业处理

**示例：** 赵茗与李杰两人合伙开办了一家企业，合伙协议中明确写明赵茗每年固定收入为2000000元，李杰每年固定收入为4000000元。资本投资收益率都是8%，如果在支付了两个人的固定开支和其他成本后还有余额，则对剩余净收入进行平均分配。

### 2.3 公司制企业的所有者权益

**定义：** 公司制企业是指可以通过发行股票和债券筹集资金，对债务承担有限责任，而且可以拥有永久存续权的企业组织形式。

**要点：**
- 公司制企业与独资企业、合伙企业的最大不同：作为整体对外部承担责任和交纳税收，纳税主体是企业而不是投资者个人
- 股份有限公司的股东权益构成：
  - 普通股：拥有表决权、分红权、剩余资产要求权、优先认股权
  - 优先股：股利分派次序优先、剩余资产要求权优先，通常没有表决权
- 股本核算：公司发行股票应以其面值作为股本，超过面值的部分作为股本溢价计入资本公积
- 库存股：公司已经发行的股票重新购回的股份，通常没有表决权和股利分配权

**示例：** K公司经批准允许发行2000000股面值为1元的普通股。发行其中的100000股时，发行价格为每股7元。会计分录为：借：现金700000；贷：股本——普通股100000，资本公积600000。

### 2.4 所有者权益的披露与分析

**定义：** 所有者权益在资产负债表中的反映，以及每股收益的计算。

**要点：**
- 所有者权益在资产负债表中的列示
- 每股收益的基本概念和计算方法
- 权益工具与金融负债的区分原则
- 复合金融工具的分拆方法

**示例：** 对于可转换公司债券等复合金融工具，需要将负债成分和权益成分进行分拆，分别确认为应付债券和其他权益工具。

## 关键术语表

| 术语 | 定义 |
|------|------|
| 所有者权益 | 企业资产扣除负债后由所有者享有的剩余权益 |
| 实收资本（股本） | 投资者按照企业章程或合同、协议的约定实际投入企业的资本 |
| 资本公积 | 企业收到投资者超出其在企业注册资本中所占份额的投资以及某些特定情况下直接计入所有者权益的项目 |
| 其他综合收益 | 企业根据会计准则的规定未在当期损益中确认的各项利得和损失 |
| 盈余公积 | 企业按照规定从净利润中提取的各种积累资金 |
| 未分配利润 | 企业留待以后年度进行分配的结存利润 |
| 普通股 | 公司最基本的股份，拥有表决权、分红权、剩余资产要求权、优先认股权 |
| 优先股 | 在股利分派次序和剩余资产要求权方面优先于普通股，通常没有表决权 |
| 库存股 | 公司已经发行的股票重新购回的股份，通常没有表决权和股利分配权 |
| 权益工具 | 能证明拥有企业在扣除所有负债后的资产中的剩余权益的合同 |
| 金融负债 | 符合下列条件之一的负债：向其他方交付现金或其他金融资产的合同义务；在潜在不利条件下，与其他方交换金融资产或金融负债的合同义务 |

## 实务应用要点

**MBA实务视角：**

1. **所有者权益结构分析**：通过分析所有者权益的构成，可以了解企业的资本来源和利润积累情况，评估企业的财务稳定性和成长潜力。

2. **股票发行定价**：股票发行价格超过面值的部分计入资本公积，这对企业的资本结构和后续融资有重要影响。

3. **库存股管理**：企业回购股票的原因多样，包括调节股价、实施股权激励计划、防止恶意收购等，需要根据企业战略合理决策。

4. **优先股融资**：优先股兼具债券和股票的特征，是企业重要的融资工具，需要权衡其固定收益特征和税后支付特点。

5. **权益工具与金融负债的区分**：对于可转换债券等复合金融工具，正确的分类对财务报表的真实反映至关重要。

## 与其他章节的关联

- **与第3章（资产负债表与利润表）的关联**：所有者权益是资产负债表的重要组成部分，其变动直接影响企业的财务状况。
- **与第10章（负债）的关联**：所有者权益与负债共同构成企业的权益，两者的区分对财务分析至关重要。
- **与第12章（影响净利润的其他项目）的关联**：净利润的变动直接影响未分配利润，进而影响所有者权益总额。
- **与第14章（企业合并与合并财务报表）的关联**：企业合并中的权益变动和合并报表编制涉及所有者权益的复杂处理。

## 复习要点

1. **所有者权益的定义和特点**：理解所有者权益作为剩余权益的本质特征。
2. **所有者权益的构成**：掌握实收资本、资本公积、其他综合收益、盈余公积、未分配利润等组成部分。
3. **普通股与优先股的区别**：理解两者在表决权、股利分配、剩余资产要求权等方面的差异。
4. **股本和资本公积的核算**：掌握股票发行、回购、库存股等业务的会计处理。
5. **权益工具与金融负债的区分**：理解区分原则和复合金融工具的分拆方法。
6. **不同企业组织形式的特点**：比较独资企业、合伙企业和公司制企业在所有者权益核算上的差异。
