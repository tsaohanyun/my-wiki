# 第4章 会计循环

## 章节概述

本章介绍会计循环的完整过程——从经济业务发生到财务报表编制的全流程。会计循环是将企业经济业务转换为会计信息的过程，包括分析业务、编制分录、过账、试算平衡、调整分录、结账和编制报表七个步骤。本章重点讲解借贷记账法、调整分录的四种类型、结账程序以及工作底稿的编制方法，是掌握会计实务操作的核心章节。

## 核心知识点

### 4.1 会计系统与会计循环

**会计循环定义：** 将企业发生的经济业务通过一定程序和方法以财务报表形式表现出来的过程。

**七个步骤：**

| 步骤 | 内容 | 时机 |
|------|------|------|
| ① 分析经济业务 | 收集原始凭证 | 日常 |
| ② 编制会计分录 | 翻译为会计语言（记账凭证） | 日常 |
| ③ 登记账簿（过账） | 分类整理到总账和明细账 | 日常 |
| ④ 试算平衡 | 检查借贷是否平衡 | 期末 |
| ⑤ 编制调整分录 | 按权责发生制调整 | 期末 |
| ⑥ 编制结账分录 | 结清损益类账户 | 期末 |
| ⑦ 编制财务报表 | 对外报告 | 期末 |

**前三个步骤**是日常基础工作，**后四个步骤**一般发生在期末。

### 4.2 账户及其分类

**会计科目与会计账户：** 会计科目是账户的名称，账户是科目的具体运用。

**按经济内容分类：**

| 类别 | 记账规则（借方/贷方） | 示例 |
|------|----------------------|------|
| 资产类 | 增加/减少 | 银行存款、应收账款 |
| 负债类 | 减少/增加 | 应付账款、短期借款 |
| 所有者权益类 | 减少/增加 | 实收资本、未分配利润 |
| 收入类 | 减少/增加 | 主营业务收入 |
| 费用类 | 增加/减少 | 管理费用、销售费用 |
| 成本类 | 增加/减少 | 生产成本、制造费用 |

**口诀：** 资产费用借增贷减，负债权益收入贷增借减。

**按详细程度分类：**
- **总分类账（总账）：** 一级账户，总括核算
- **明细分类账（明细账）：** 二级/三级账户，详细核算

### 4.3 交易事项分析与复式记账

#### 4.3.1 借贷记账法

**编制会计分录的程序：**
1. 分析经济业务，找出涉及的会计科目（至少两个）
2. 确定科目类别
3. 确定增加还是减少
4. 根据记账规则确定借贷方向

**记账规则：有借必有贷，借贷必相等**

**典型业务分录示例：**

```
（1）所有者投资300,000元存入银行
借：银行存款        300,000
  贷：实收资本        300,000

（2）购入汽车100,000元，以支票支付
借：固定资产——汽车  100,000
  贷：银行存款        100,000

（3）提供运输服务，收入现金25,000元
借：现金            25,000
  贷：营业收入        25,000

（4）计提折旧30,000元
借：营业费用——折旧费 30,000
  贷：累计折旧         30,000
```

**累计折旧的特殊性：** 虽属资产类科目，但作为固定资产的抵减科目，增加记贷方，减少记借方。

### 4.4 调整分录

#### 4.4.1 权责发生制与收付实现制

**权责发生制下四类调整分录：**

| 类型 | 含义 | 示例 |
|------|------|------|
| 应付费用（应计费用） | 已发生但未付的费用 | 应付利息、应付工资 |
| 应收收入（应计收入） | 已赚取但未收的收入 | 应收利息、应收租金 |
| 预付费用（递延费用） | 已付但未消耗的费用 | 预付保险费、预付租金 |
| 预收收入（递延收入） | 已收但未赚取的收入 | 预收货款、预收租金 |

#### 4.4.2 应计项目

**应付费用示例：**
```
长期借款20,000元，年利率6%，月末应计利息
当期利息 = 20,000 × 6% × 1/12 = 100元

借：财务费用          100
  贷：应付利息          100
```

**应收收入示例：**
```
应收票据20,000元，年利率6%，月末应计利息
当期利息 = 20,000 × 6% × 1/12 = 100元

借：应收利息          100
  贷：利息收入          100
```

#### 4.4.3 递延项目

**预付费用示例：**
```
8月1日支付一年期保险费1,200元
8月31日确认当月费用 = 1,200/12 = 100元

借：保险费            100
  贷：预付保险费        100
```

**预收收入示例：**
```
8月1日收到一年租金6,000元
8月31日确认当月收入 = 6,000/12 = 500元

借：预收收入          500
  贷：租金收入          500
```

#### 4.4.4 估计项目

**坏账损失估计：**
```
月末估计应收账款约1,500元不能收回

借：信用减值损失——坏账损失  1,500
  贷：坏账准备              1,500
```

### 4.5 结账

#### 4.5.1 虚账户与实账户

| 类型 | 别名 | 包含科目 | 结账方式 |
|------|------|----------|----------|
| 虚账户 | 临时性账户 | 收入、费用类（利润表科目） | 期末结转为零 |
| 实账户 | 永久性账户 | 资产、负债、所有者权益（资产负债表科目） | 余额结转下期 |

**虚账户结账分录：**
```
借：本年利润
  贷：费用类账户

借：收入类账户
  贷：本年利润
```

#### 4.5.2 月度结账与年度结账

- **月度/季度：** 采用"表结法"，不编制正式结账分录，仅在工作底稿中结转
- **年度：** 采用"账结法"，编制正式结账分录，结清所有虚账户

### 4.6 工作底稿

**工作底稿的必备栏目：**
1. 会计科目栏
2. 试算平衡表栏（借贷）
3. 调整分录栏（借贷）
4. 利润表栏（借贷）
5. 资产负债表栏（借贷）

**编制流程：**
1. 填列会计科目和试算平衡表
2. 分析调整事项，填列调整分录
3. 编制调整后试算平衡表
4. 将损益项目转入利润表栏
5. 将资产负债项目转入资产负债表栏

## 实务应用要点

1. **创业企业会计：** 创业者应了解会计循环，能看懂财务报表，理解销售额增长但利润增长不成比例的原因（如费用同步增长、折旧增加等）。

2. **调整分录的必要性：** 权责发生制要求在期末进行账项调整，这是很多企业利润与现金流不一致的重要原因。管理者应理解调整分录对企业利润的影响。

3. **试算平衡的局限性：** 试算平衡只能发现借贷不等的错误，无法发现以下错误：借贷同时遗漏/重复、借贷等额错误、科目用错、日期不准等。

4. **年度 vs 月度结账：** 月度采用"表结法"简化处理，年度必须采用"账结法"正式结账，确保全年经营成果完整反映。

## 复习要点

1. **掌握借贷记账法：** 资产费用借增贷减，负债权益收入贷增借减；有借必有贷，借贷必相等

2. **四类调整分录：** 应付费用、应收收入、预付费用、预收收入——都源于权责发生制

3. **虚账户与实账户的区别：** 虚账户期末结零（收入费用），实账户余额结转（资产负债权益）

4. **工作底稿的作用：** 不是正式会计资料，但能减少错误、方便期末处理

5. **试算平衡的局限：** 平衡≠正确，不平衡一定有错
