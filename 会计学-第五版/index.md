## 核心概念速查

### 会计基本等式
```
资产 = 负债 + 所有者权益
```

### 利润计算
```
利润 = 收入 - 费用 + 利得 - 损失
```

### 现金流量分类
- **经营活动**：日常业务产生的现金流
- **投资活动**：长期资产购建和处置
- **筹资活动**：借款和权益融资
