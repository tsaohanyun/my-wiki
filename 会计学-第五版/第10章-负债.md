# 第10章 负债

## 章节概述

本章系统介绍企业负债的确认、计量与核算，涵盖流动负债（短期借款、应付账款、合同负债、应交税费等）、或有负债与预计负债、以及非流动负债（长期借款、应付债券、租赁负债等）。负债是企业重要的资金来源，合理的负债管理对企业持续发展至关重要。本章帮助MBA学员理解不同负债的会计处理方法及其对企业财务状况的影响。

## 核心知识点

### 10.1 负债概述

**定义：** 负债是指企业过去的交易或者事项形成的、预期会导致经济利益流出企业的现时义务。

**三大特点：**
1. 由过去经济业务产生的现时义务，具有法律约束力
2. 必须能以货币确认或合理计量
3. 必须有确切的或合理估计的债权人及到期日

**分类：** 按流动性分为流动负债（一年或一个营业周期内偿还）和非流动负债（超过一年或一个营业周期）。

### 10.2 流动负债

**定义：** 流动负债是指将在一年（含一年）或者超过一年的一个营业周期内偿还的债务。

**确认条件之一：**
- 预计在一个正常营业周期中清偿
- 主要为交易目的而持有
- 自资产负债表日起一年内到期应予以清偿
- 企业无权自主将其推迟至资产负债表日后一年以上

**计量原则：** 流动负债偿还期短，到期值与现值相差很小，一般按未来需偿付的金额（面值）计量。

**主要流动负债项目：**

#### 短期借款
- 向银行或金融机构借入的期限在一年以下的借款
- 利息作为财务费用计入当期损益

#### 应付账款
- 因购买材料、商品或接受劳务等应付给供应单位的款项
- 现金折扣的处理：**总价法**（我国常用）vs 净价法

#### 合同负债
- 企业已收或应收客户对价而应向客户转让商品的义务
- 与预收账款的区别：合同负债以履约义务确立为前提，不以是否收到款项为前提
- 合同一旦成立，预收账款应转入合同负债

#### 应交税费
- 企业应交未交的各种税费
- **应交增值税** = 当期销项税额 - 当期进项税额
- 未按规定取得增值税扣税凭证的，进项税额不能抵扣

#### 其他流动负债
- 预收账款、应付职工薪酬、应付股利、其他应付款等
- 一年内到期的非流动负债应重分类为流动负债

### 10.3 或有负债与预计负债

**或有负债定义：** 由过去交易或事项形成的潜在义务，或履行该义务不是很可能导致经济利益流出企业或金额不能可靠计量的现时义务。

**或有负债特点：**
- 由过去交易或事项形成
- 结果具有不确定性（是否发生不确定，或金额难以预计）

**常见或有负债：** 应收票据贴现、未决诉讼和仲裁、债务担保、产品质量保证。

**预计负债：** 与或有事项相关的义务同时满足三个条件时确认为预计负债：
1. 是企业承担的**现时义务**
2. 履行该义务**很可能**（>50%且≤95%）导致经济利益流出企业
3. 该义务的金额能够**可靠地计量**

**或有负债 vs 预计负债 vs 一般负债：**

| 项目 | 确认为负债 | 披露要求 |
|------|-----------|---------|
| 一般负债 | 确认 | 资产负债表列报 |
| 预计负债 | 确认 | 资产负债表单独列报 |
| 或有负债 | 不确认 | 附注披露（极小可能导致流出的除外） |

### 10.4 非流动负债

**定义：** 非流动负债是指偿还期在一年以上或超过一年的一个营业周期以上的债务。

**特点：** 偿还期限长、金额通常较大、需承担利息负担。

**借款费用处理：**
- 可直接归属于符合条件资产的购建或生产：**资本化**
- 其他借款费用：**费用化**，计入当期损益

#### 长期借款
- 向银行或金融机构借入的期限在一年以上的借款
- 设置"长期借款"科目，核算本金、利息调整等

#### 应付债券

**发行价格的确定：**
- 债券发行价格 = 面值的现值 + 各期利息的现值
- 折现率使用发行时的市场利率

| 情况 | 关系 | 发行方式 |
|------|------|---------|
| 票面利率 = 市场利率 | 按面值发行（平价） |
| 票面利率 < 市场利率 | 低于面值发行（折价） |
| 票面利率 > 市场利率 | 高于面值发行（溢价） |

**溢折价摊销方法：**
- **直线摊销法**：将溢折价在存续期内平均摊销
- **实际利息法**（准则要求）：每期利息费用 = 期初账面价值 × 市场利率

**可转换公司债券：**
- 初始确认时分拆为负债成分和权益成分
- 负债成分确认为应付债券，权益成分确认为其他权益工具
- 交易费用按各自相对公允价值分摊

#### 租赁负债
- 承租人在租赁期开始日尚未支付的租赁付款额的现值
- 折现率：租赁内含利率或承租人增量借款利率
- 与使用权资产对应确认

## 关键术语表

| 术语 | 定义 |
|------|------|
| 负债（Liabilities） | 企业过去的交易或事项形成的、预期会导致经济利益流出企业的现时义务 |
| 流动负债（Current Liabilities） | 将在一年或一个营业周期内偿还的债务 |
| 非流动负债（Non-current Liabilities） | 偿还期超过一年或一个营业周期的债务 |
| 合同负债（Contract Liabilities） | 企业已收或应收客户对价而应向客户转让商品的义务 |
| 或有负债（Contingent Liabilities） | 由过去事项形成的潜在义务，或金额/可能性不确定的现时义务 |
| 预计负债（Provision） | 因或有事项确认的、履行很可能导致经济利益流出且金额能可靠计量的负债 |
| 应付债券（Bonds Payable） | 企业为筹集长期资金而发行的债券及应付利息 |
| 债券折价（Bond Discount） | 债券发行价格低于面值的部分 |
| 债券溢价（Bond Premium） | 债券发行价格高于面值的部分 |
| 可转换债券（Convertible Bonds） | 可按约定条件转换为发行公司股票的债券 |
| 租赁负债（Lease Liabilities） | 承租人在租赁期开始日尚未支付的租赁付款额的现值 |
| 借款费用（Borrowing Costs） | 企业因借款而发生的利息及其他相关成本 |
| 金融负债（Financial Liabilities） | 以摊余成本或公允价值计量的负债性金融工具 |

## 实务应用要点

1. **合同负债的确认时点**：新收入准则下，企业收到预收款时，需判断合同是否已成立。合同成立前收到的款项作为预收账款，合同成立后转入合同负债。

2. **或有负债的披露**：企业对外提供担保、涉及诉讼等或有事项，即使可能性不大，也应在附注中充分披露，以便报表使用者全面了解企业风险。

3. **债券发行价格的理解**：债券折价并非损失，溢价并非收益，而是对票面利率与市场利率差异的调整。溢折价摊销使各期实际利息费用趋于合理。

4. **可转换债券的双重属性**：初始确认时分拆为负债和权益，对资本结构分析有重要影响。转股时负债减少、所有者权益增加。

5. **适度负债的重要性**：负债经营可以发挥财务杠杆效应，但过度负债可能导致财务危机。利息的税前扣除使债务融资具有税收优势。
