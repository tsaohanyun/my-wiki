# 第9章 无形资产及其他资产

## 章节概述

本章介绍无形资产及其他资产的会计处理，包括无形资产的特征与种类、取得与摊销、商誉、研究与开发支出的资本化与费用化、使用权资产、长期待摊费用和递耗资产等。在知识经济时代，无形资产在企业资产中的比重日益增大，其确认与计量问题也越发重要。本章帮助MBA学员理解无形资产会计处理的难点以及研发支出政策选择对企业财务报告的影响。

## 核心知识点

### 9.1 无形资产的特征与种类

**定义：** 无形资产是企业拥有或者控制的没有实物形态的可辨认非货币性资产。

**四大特征：**
1. 没有实物形态（区别于有形资产）
2. 属于非货币性资产（未来变现金额不确定）
3. 能为企业带来长期的未来经济利益
4. 经济利益具有很大的不确定性（受技术进步和竞争影响）

**分类：**

| 分类标准 | 类型 | 示例 |
|---------|------|------|
| 按期限 | 有限期 / 无限期 | 专利权 / 土地使用权 |
| 按是否可辨认 | 可辨认 / 不可辨认 | 专利权、商标权 / 商誉 |
| 按经济内容 | 多种 | 专利权、商标权、非专利技术、著作权、土地使用权、特许权 |

**注意：** 我国准则中的无形资产是狭义概念，仅指可辨认无形资产，不包括商誉。

### 9.2 无形资产的会计处理

**确认条件：**
1. 与该无形资产有关的经济利益很可能流入企业
2. 该无形资产的成本能够可靠地计量

**取得的会计处理：**

| 取得方式 | 入账价值 |
|---------|---------|
| 外购 | 购买价格 + 相关税费 + 直接归属于使该资产达到预定用途的支出 |
| 接受投资 | 投资合同或协议约定的公允价值 |
| 接受捐赠 | 发票金额或同类资产市价 |
| 自创 | 区分研究阶段（费用化）和开发阶段（符合条件可资本化） |

**摊销的会计处理：**
- 使用寿命有限的无形资产：在使用寿命内系统摊销
- 摊销方法：直线法或其他能合理反映经济利益预期实现方式的方法
- 一般不考虑残值（除非有可靠证据表明存在残值）
- 使用寿命不确定的无形资产：不摊销，但需每年进行减值测试
- 摊销一般计入管理费用

**处置的会计处理：**
- 转让无形资产：取得价款与账面价值的差额计入当期损益
- 预期不能带来经济利益时：将账面价值予以转销

### 9.3 商誉

**定义：** 商誉是企业赚取超额利润的能力，源于优越的地理位置、良好客户关系、优秀管理团队等因素。

**要点：**
- **自创商誉**：不予确认（无法可靠计量，且与企业不可分离）
- **外购商誉**：仅在企业合并时确认
- 商誉 = 购买价格 - 被购买方可辨认净资产公允价值
- 后续计量：不摊销，但每年年末进行减值测试
- 商誉不属于无形资产（不可辨认）

**MBA实务要点：** 在标准普尔500强企业中，品牌价值平均占总市值的30%以上，但这些价值大多未在资产负债表中反映，这是理解企业估值的重要背景。

### 9.4 研究与开发支出

**定义：**
- **研究阶段**：为获取新的科学或技术知识而进行的独创性有计划调查
- **开发阶段**：在商业性生产或使用前，将研究成果应用于生产新的或有实质性改进的材料、产品等

**会计处理原则：**
- 研究阶段支出：**全部费用化**，计入当期损益
- 开发阶段支出：**满足五个条件可资本化**：
  1. 技术上具有可行性
  2. 具有完成并使用或出售的意图
  3. 能证明存在市场或有用性
  4. 有足够的技术和财务资源支持
  5. 支出能够可靠地计量

**华为案例启示：** 华为2021年研发费用1,427亿元，全部计入费用，资产负债表中商誉及无形资产仅81亿元。这说明自创无形资产（如11万件授权专利）的价值大多未在报表中反映。

**软件开发成本：** 以"技术可行性"为分界点——之前费用化，之后符合条件的资本化。

### 9.5 其他资产

**使用权资产：**
- 承租人在租赁期内使用租赁资产的权利
- 成本包括：租赁负债初始计量金额 + 预付租金 + 初始直接费用 + 复原成本
- 后续计量：成本模式，计提折旧和减值准备
- 折旧年限：能确定取得所有权的按剩余使用寿命；否则按租赁期与剩余使用寿命孰短

**长期待摊费用：**
- 已支出但不能全部计入当期损益，应在以后年度分期摊销的费用
- 主要包括：租入固定资产的改良支出
- 通过"长期待摊费用"科目核算

**递耗资产：**
- 企业拥有的矿藏、森林等自然资源
- 具有不可再生性，随开采逐渐耗竭
- 折耗方法：生产数量法（按每期实际开采量占总蕴藏量的比例分摊成本）
- 勘探支出的处理：成果法（仅成功勘探资本化）vs 全部成本法（全部资本化）

## 关键术语表

| 术语 | 定义 |
|------|------|
| 无形资产（Intangible Assets） | 企业拥有或控制的没有实物形态的可辨认非货币性资产 |
| 专利权（Patents） | 政府给予发明者在一定期限内的制造、使用和出售的专门权利 |
| 商标权（Trademarks） | 在特定商品上使用特定名称或图案的权利 |
| 非专利技术（Trade Secrets） | 未申请专利的先进实用技术或资料 |
| 商誉（Goodwill） | 企业合并时购买价格超过被购买方可辨认净资产公允价值的差额 |
| 摊销（Amortization） | 将无形资产成本在有效期内系统分摊的过程 |
| 研究与开发支出（R&D Costs） | 企业为获取新知识或开发新产品而发生的支出 |
| 使用权资产（Right-of-use Asset） | 承租人在租赁期内使用租赁资产的权利 |
| 递耗资产（Depletable Assets） | 企业拥有的矿藏、森林等自然资源 |
| 折耗（Depletion） | 将递耗资产成本按开采量进行系统分摊的过程 |
| 成果法（Successful-efforts Method） | 仅将成功的勘探支出资本化的方法 |
| 全部成本法（Full-cost Method） | 将所有勘探支出均资本化的方法 |

## 实务应用要点

1. **研发支出资本化的判断**：区分研究阶段和开发阶段是关键。企业有一定的判断空间，可能被利用来调节利润。报表使用者应关注资本化比例的合理性。

2. **商誉减值测试**：商誉不摊销而进行减值测试的做法，可能导致减值损失集中确认（"商誉暴雷"），对利润产生巨大冲击。

3. **使用权资产的影响**：2018年新租赁准则要求承租人确认使用权资产和租赁负债，对零售、航空等大量使用租赁资产的行业影响重大。

4. **资产负债表的局限性**：许多高价值无形资产（如自创品牌、研发成果）未在报表中反映，分析企业价值时应充分关注表外无形资产。
