# 第3章 资产负债表与利润表

## 章节概述

本章介绍两张最基本的财务报表——资产负债表和利润表。资产负债表反映企业在特定时点的财务状况（资产、负债、所有者权益），是静态报表；利润表汇总企业在一定时期内的经营成果（收入、费用、利润），是动态报表。本章还介绍了利用这两张报表进行财务分析的基本指标，包括偿债能力、盈利能力和运营效率指标。

## 核心知识点

### 3.1 资产负债表

#### 3.1.1 基本结构

**定义：** 反映企业在某一特定时点的财务状况的会计报表。

**核心等式：** 资产 = 负债 + 所有者权益

**两种解释：**
1. **资金来源与运用平衡：** 来源于股东和债权人的资金，以各种资产形态存在
2. **经济资源与索取权平衡：** 资产是经济资源，负债和所有者权益是债权人和股东的索取权

**格式：**
- **账户式（常用）：** 左边列示资产，右边列示负债和所有者权益
- **报告式：** 上方列示资产，中间列示负债，下方列示所有者权益

#### 3.1.2 项目分类

**分类原则：** 按流动性分类，流动性项目列前，非流动性项目列后

**资产分类标准（满足任一条件即为流动资产）：**
- 预计在一个正常营业周期中变现、出售或耗用
- 主要为交易目的而持有
- 预计在资产负债表日起一年内变现
- 一年内交换其他资产或清偿负债不受限制的现金

#### 3.1.3 资产类主要项目

| 项目 | 定义 | 要点 |
|------|------|------|
| 货币资金 | 可立即用于支付的资金 | 现金+银行存款+其他货币资金；被冻结存款需剔除 |
| 金融资产 | 以公允价值计量且变动计入当期损益 | 2017年新准则后重新分类 |
| 应收款项 | 应收而未收的结算款项 | 应收账款+应收票据+其他应收款 |
| 存货 | 持有以备出售或耗用的资产 | 原材料+在产品+产成品 |
| 固定资产 | 使用寿命超一年的有形资产 | 按历史成本计量，分期计提折旧 |
| 无形资产 | 无实物形态的可辨认非货币性资产 | 专利权、商标权、著作权等 |

**资产确认条件：**
1. 与该资源有关的经济利益**很可能**流入企业
2. 该资源的成本或价值能够**可靠地计量**

#### 3.1.4 负债类主要项目

| 项目 | 定义 | 流动/非流动 |
|------|------|-------------|
| 短期借款 | 期限≤一年的银行借款 | 流动 |
| 应付款项 | 应付而未付的结算款项 | 流动 |
| 应付职工薪酬 | 应付未付的薪酬 | 流动 |
| 应交税费 | 应交未交的税款 | 流动 |
| 长期借款 | 期限>一年的借款 | 非流动 |
| 应付债券 | 一年期以上的债券本金及利息 | 非流动 |
| 预计负债 | 与或有事项相关的现时义务 | 非流动 |

**负债确认条件：**
1. 与该义务有关的经济利益**很可能**流出企业
2. 未来流出的经济利益金额能够**可靠地计量**

#### 3.1.5 所有者权益类项目

| 项目 | 定义 |
|------|------|
| 实收资本/股本 | 投资人投入的资本（股本=面值部分） |
| 资本公积 | 来源于盈利之外的公共性资本积累（如股票溢价） |
| 盈余公积 | 从税后净利润中提取的利润积累 |
| 未分配利润 | 历年已实现利润中留于以后年度分配的部分 |

**重要提示：** 盈余公积和未分配利润≠现金充裕。留存利润可能已被投资于长期资产或用于偿债。

### 3.2 利润表

#### 3.2.1 基本结构

**定义：** 汇总企业在一定时期内经营成果的动态报表。

**基本等式：** 收入 - 费用 = 利润

**格式：**
- **单步式：** 所有收入利得列上方，所有费用损失列下方，一次计算净利润
- **多步式（常用）：** 分步计算利润形成过程，便于分析利润来源结构

#### 3.2.2 利润表主要项目

| 项目 | 说明 |
|------|------|
| 营业收入 | 销售商品或提供劳务的净收入 |
| 营业成本 | 已销售产品/劳务的生产/采购成本 |
| 毛利 | 营业收入 - 营业成本 |
| 期间费用 | 管理费用 + 销售费用 + 财务费用 |
| 营业利润 | 毛利 - 期间费用 + 投资收益等 |
| 净利润 | 税前利润 - 所得税费用 |
| 综合收益 | 净利润 + 其他综合收益 |

**多步式利润表层次：**
```
营业收入
- 营业成本
= 毛利
- 管理费用 - 销售费用 - 财务费用
+ 投资收益
± 公允价值变动损益
- 资产减值损失
= 营业利润
± 营业外收支
= 利润总额
- 所得税费用
= 净利润
```

### 3.3 财务报表分析

#### 3.3.1 资产负债表分析（偿债能力）

| 指标 | 公式 | 经验值 | 含义 |
|------|------|--------|------|
| 流动比率 | 流动资产/流动负债 | 2:1 | 短期偿债能力 |
| 速动比率 | (流动资产-存货)/流动负债 | 1:1 | 扣除存货后的短期偿债能力 |
| 现金比率 | 现金及等价物/流动负债 | — | 最保守的流动性指标 |
| 资产负债率 | 负债总额/资产总额 | 30%-70% | 长期偿债能力/资本结构 |

#### 3.3.2 利润表分析（盈利能力）

| 指标 | 公式 | 含义 |
|------|------|------|
| 毛利率 | 毛利/营业收入 | 收入弥补基本费用后的剩余 |
| 营业利润率 | 营业利润/营业收入 | 经营过程的获利水平 |
| 销售净利率 | 净利润/营业收入 | 最终的销售获利水平 |

#### 3.3.3 联合分析（运营效率）

| 指标 | 公式 | 含义 |
|------|------|------|
| 应收账款周转率 | 营业收入/平均应收账款 | 应收账款回收效率 |
| 存货周转率 | 营业成本/平均存货 | 存货周转效率 |
| 资产周转率 | 营业收入/平均总资产 | 资产产生收入的能力 |
| 资产利润率(ROA) | 息税前利润/平均总资产 | 全部资产获利水平 |
| 净资产利润率(ROE) | 净利润/平均净资产 | 股东资本赚取利润的能力 |
| 利息保障倍数 | 息税前利润/利息费用 | 支付利息的能力 |

## 关键术语表

| 术语 | 英文 | 定义 |
|------|------|------|
| 资产负债表 | Balance Sheet | 反映特定时点财务状况的静态报表 |
| 利润表 | Income Statement | 汇总一定时期经营成果的动态报表 |
| 资产 | Asset | 过去交易形成、企业拥有或控制、预期带来经济利益的资源 |
| 负债 | Liability | 过去交易形成、预期导致经济利益流出的现时义务 |
| 所有者权益 | Owners' Equity | 资产扣除负债后的剩余权益 |
| 收入 | Revenue | 日常活动形成的经济利益总流入 |
| 费用 | Expense | 日常活动发生的经济利益总流出 |
| 流动比率 | Current Ratio | 流动资产/流动负债 |
| 资产负债率 | Debt Ratio | 负债总额/资产总额 |
| 净资产利润率 | ROE | 净利润/平均净资产 |

## 实务应用要点

1. **"保壳"利润操纵：** 上市公司面临退市风险时，可能通过政府补贴（计入营业外收入）、资产减值"洗大澡"等手段扭亏为盈。分析利润表时应区分**经常性损益**和**非经常性损益**。

2. **利润≠现金：** 未分配利润和盈余公积不等于现金充裕。利润可能已投资于固定资产、存货等，或用于偿债和分红。

3. **多步式利润表的价值：** 不同来源的利润可持续性不同。营业利润具有持续性，而营业外收入通常不具有持续性。投资者应重点关注**营业利润**。

4. **产权保护：** 资产负债表中负债与所有者权益的分类，有助于明晰企业产权关系，保护投资者利益。

## 复习要点

1. **理解会计等式：** 资产 = 负债 + 所有者权益，这是编制资产负债表的理论基础

2. **区分流动性与非流动性：** 流动资产/流动负债 vs 非流动资产/非流动负债的判断标准

3. **多步式利润表的层次：** 营业收入→毛利→营业利润→利润总额→净利润→综合收益

4. **财务分析指标体系：** 偿债能力（流动比率、速动比率、资产负债率）、盈利能力（毛利率、净利率）、运营效率（周转率、ROA、ROE）

5. **警惕利润操纵：** 关注非经常性损益、资产减值准备的异常变动、政府补贴等对利润的影响
