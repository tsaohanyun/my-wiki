# 第2章 财务会计概念

## 章节概述

本章系统阐述财务会计的概念框架，包括财务会计报告目标、四项会计假设、会计基础（权责发生制与收付实现制）、八项会计信息质量要求以及四项会计基本原则。这些概念构成了理解财务会计的理论基石，是后续学习各具体会计处理方法的前提。MBA学员应重点理解各概念的实务含义及其对利润操纵的约束作用。

## 核心知识点

### 2.1 财务会计报告目标

**定义：** 向财务会计报告使用者提供与企业财务状况、经营成果和现金流量等有关的会计信息，反映管理层受托责任履行情况，有助于使用者作出经济决策。

**三大特点：**

| 特点 | 说明 |
|------|------|
| 信息内容扩展 | 不仅包括财务状况和经营成果，还包括**现金流量**信息 |
| 双重目标并重 | 同时满足**受托责任观**和**决策有用观** |
| 使用者范围扩大 | 包括投资者、债权人、**政府及有关部门和社会公众** |

**受托责任观 vs 决策有用观：**

| 维度 | 受托责任观 | 决策有用观 |
|------|-----------|-----------|
| 背景 | 所有权与经营权直接分离 | 通过资本市场间接分离 |
| 目标 | 恰当反映受托经管责任 | 提供对决策有用的信息 |
| 侧重 | 实物资产保护、价值完整 | 未来现金流量预测 |

### 2.2 会计假设

#### 2.2.1 会计主体假设

**定义：** 核算和报告会计信息的特定单位或组织，界定会计确认、计量和报告的**空间范围**。

**要点：**
- 会计主体 ≠ 法律主体（个人独资企业、分公司、企业集团都是会计主体但非法律主体）
- 要求区分企业交易与所有者个人交易
- 关联交易操纵利润是对会计主体假设的背离

**实务警示：** 夫妻经营的养殖场，住房租金中多少算经营费用、多少算生活费用，需要合理划分。

#### 2.2.2 持续经营假设

**定义：** 企业将在未来无限期地经营下去，界定会计核算与报告的**时间范围**。

**实务影响：**
- 持续经营下：固定资产按**历史成本**计量，分期计提折旧
- 清算假设下：按**清算价值**计量，不计提折旧
- 在产品按已投入成本计量（非清算价值）

#### 2.2.3 会计分期假设

**定义：** 将企业持续经营过程划分为连续的、等长的期间。

**要点：**
- 我国会计年度：公历1月1日至12月31日
- 中期报告：季度报告、半年度报告
- 年度报告须经审计，中期报告通常不需审计

**实务问题：** 会计分期导致"跨期"业务处理的困难，为企业操纵"期间利润"提供了机会（如利润平滑、"洗大澡"）。

#### 2.2.4 货币计量假设

**定义：** 财务会计只记录能以货币度量的交易或事项。

**要点：**
- 记账本位币：人民币（业务收支以其他货币为主的可选定其他货币，但报表须折算为人民币）
- 深层含义：假定币值相对稳定
- 局限性：不能以货币计量的信息（如人力资源、品牌价值）无法在报表中反映

### 2.3 会计基础

| 基础 | 别名 | 确认标准 | 适用范围 |
|------|------|----------|----------|
| **权责发生制** | 应收应付制/应计制 | 以应收应付为标准 | 企业会计 |
| **收付实现制** | 现金制 | 以现金收付为标准 | 行政单位会计 |

**权责发生制示例：**
- 2022年12月收到预付款1500万，但开发工作2023年才开始 → 2022年**不确认收入**，记为负债
- 2022年应付房租1200万未付 → 2022年**仍应确认为费用**

**权责发生制的利弊：**
- 优点：恰当反映经营成果，有助业绩评价和分析预测
- 缺点：导致利润与现金流不一致，增加财务管理难度

### 2.4 会计信息质量要求

| 要求 | 核心含义 | 实务要点 |
|------|----------|----------|
| **可靠性** | 真实可靠、内容完整 | 可核实、中立性；审计提高中立性 |
| **相关性** | 与决策需要相关 | 反馈价值+预测价值 |
| **可理解性** | 清晰明了 | 不要求所有人都能理解 |
| **可比性** | 纵向和横向可比 | 同类业务采用一致的会计政策 |
| **实质重于形式** | 按经济实质处理 | 如融资租赁资产虽法律上不属承租方，但应确认为资产 |
| **重要性** | 反映所有重要事项 | 次要事项可简化处理（如低值易耗品一次性费用化） |
| **谨慎性** | 不高估资产/收益，不低估负债/费用 | 计提减值准备；但不可过度谨慎 |
| **及时性** | 及时确认、计量和报告 | 年报须在年度结束后四个月内发布 |

**相关性 vs 可靠性的冲突：**
- 历史成本计量：可靠但可能不相关
- 公允价值计量：相关但可能不够可靠
- 现代会计：坚持历史成本为主，特定条件下采用公允价值

### 2.5 会计原则

#### 2.5.1 历史成本原则

**定义：** 资产或负债按交易发生时的取得成本或承诺支付金额计量。

**五种计量属性：**

| 计量属性 | 定义 | 适用场景 |
|----------|------|----------|
| 历史成本 | 购置时支付的金额 | 一般资产初始确认 |
| 重置成本 | 现在购买相同资产所需金额 | 资产评估 |
| 可变现净值 | 正常销售收到的金额减相关成本 | 存货期末计量 |
| 现值 | 未来现金流入的折现金额 | 长期应收款/应付款 |
| 公允价值 | 有序交易中的市场价格 | 金融工具、投资性房地产 |

#### 2.5.2 实现原则（收入确认原则）

**定义：** 企业在已实现时确认收入——因销售商品或提供服务引起现金或现金索取权的流入。

**2017年修订后：** 企业在履行履约义务、客户取得商品控制权时确认收入。

#### 2.5.3 配比原则

**定义：** 收入确认与相关费用确认应相互匹配——同一交易的收入和费用在同一会计期间确认。

**要点：**
- 直接费用：与特定收入直接配比（如销售成本）
- 间接费用（期间费用）：在发生期间确认，不强调与特定收入配比

#### 2.5.4 划分收益性支出与资本性支出原则

| 类型 | 受益期 | 会计处理 | 示例 |
|------|--------|----------|------|
| 收益性支出 | 本期 | 作为本期费用 | 销售业务费用 |
| 资本性支出 | 本期及以后多期 | 资本化为资产，分期摊销 | 设备采购支出 |

## 关键术语表

| 术语 | 英文 | 定义 |
|------|------|------|
| 会计主体假设 | The Accounting Entity Assumption | 界定会计信息空间范围的假设 |
| 持续经营假设 | The Going Concern Assumption | 假定企业将持续经营下去 |
| 会计分期假设 | The Periodicity Assumption | 将经营过程划分为等长期间 |
| 货币计量假设 | The Monetary Assumption | 只记录能以货币度量的事项 |
| 权责发生制 | Accrual Basis | 以应收应付为标准确认收入费用 |
| 可靠性 | Reliance | 会计信息真实可靠、内容完整 |
| 相关性 | Relevance | 会计信息与决策需要相关 |
| 谨慎性 | Conservatism | 不高估资产收益、不低估负债费用 |
| 历史成本原则 | Historical Cost Principle | 按取得成本计量 |
| 实现原则 | Realization Principle | 收入在实现时确认 |
| 配比原则 | Matching Principle | 收入与相关费用在同一期间确认 |

## 实务应用要点

1. **利润操纵识别：** 理解会计假设和原则是识别利润操纵的基础。会计分期假设为利润操纵提供了空间，谨慎性原则要求不高估收益。

2. **实质重于形式：** 融资租赁、售后回购等业务，需要穿透法律形式看经济实质，这是MBA管理者必须具备的分析能力。

3. **重要性判断：** 金额较小的支出可简化处理（如低值易耗品一次性费用化），但重大事项必须严格按准则处理。重要性判断依赖职业经验。

4. **公允价值运用：** 现代会计在坚持历史成本原则的同时，在金融工具等领域引入公允价值计量，这增加了报表信息的相关性，但也带来了波动性和操纵风险。

## 复习要点

1. **四大假设的空间-时间维度：** 会计主体（空间）→ 持续经营（时间前提）→ 会计分期（时间划分）→ 货币计量（计量手段）

2. **权责发生制 vs 收付实现制：** 企业会计必须采用权责发生制，它使收入费用配比更合理，但导致利润与现金流不一致

3. **八项信息质量要求的层次：** 可靠性和相关性是最基本的质量特征；其他要求（可理解性、可比性、实质重于形式、重要性、谨慎性、及时性）是对基本特征的补充

4. **谨慎性的边界：** 不高估资产/收益、不低估负债/费用 ≠ 故意低估资产/收益、高估负债/费用

5. **思考题：** 相关性与可靠性的冲突如何缓解？——在坚持历史成本的基础上，特定条件下引入公允价值
