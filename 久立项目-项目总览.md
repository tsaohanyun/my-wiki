# 久立项目-项目总览

项目：**久立特材**

## 项目概况

| 项目 | 内容 |
|------|------|
| 客户 | 浙江久立特材科技股份有限公司 |
| 项目名 | 久立数字化制造平台（MES） |
| 实施方 | [实施方] |
| 启动时间 | 2019年7月 |
| 总体范围 | 车间执行（MES）+ 仓储管理（WMS）+ 质量管理（QMS） |
| 核心集成 | SAP ERP 双向接口 |

## 项目分期

### 一期（2020.07 - 2021.09）
- **覆盖区域**：冷轧一厂、JCO车间、辅料仓库、成品仓库
- **核心模块**：系统建模、工艺管理、工序流转、仓储收发存
- **接口**：MES↔SAP（物料、报工、库存）、OA→SAP（采购申请）
- **关键里程碑**：2020.09-10 蓝图设计 → 2020.11-2021.03 系统实现 → 2021.04 上线 → 2022.09 运维结束

### 二期（2021.05 - 2021.12）
- **覆盖区域**：中台系统升级、南浔厂区扩展
- **重点**：系统架构优化、新产线接入

### 三期（2022.01 - 2022.12）
- **覆盖区域**：冷轧二厂/三厂、精密分厂、核电分厂、换热分厂、冷轧五厂、穿孔分厂、挤压分厂、质检站
- **分阶段上线**：一阶段（冷轧二三厂+精密+核电+换热） → 二阶段（五厂+穿孔+挤压+质检）
- **新增模块**：质检站执行、安灯系统、外协管理扩展

### 四期（2023 - 2024）
- **覆盖区域**：管件分厂（板制车间+老车间）、合金材料、原材料仓库、废料仓库
- **新增模块**：成品&废料执行、原材料仓储

                    │  制造中台    │

                    └──┬───┬───┬──┘
                       │   │   │
              ┌────────┘   │   └────────┐

         ┌────────┐  ┌────────┐  ┌────────┐
         │IPC终端 │  │PC浏览器│  │PDA手持 │

## 功能模块体系

基于PPF编码的标准化功能模块：

| 编码域 | 模块类 | 包含功能 |
|--------|--------|----------|
| PPF-WM | 仓储管理 | 系统建模、采购接收、入库、出库、库位转移、盘点 |
| PPF-ME-2xx | 工艺管理 | 工艺库建立、工艺卡开卡/打印/查看、分卡方案 |
| PPF-ME-3xx | 人员班组 | 工作日历、工时工分、班组管理 |
| PPF-ME-4xx | 生产执行 | 订单投料/退料、三班协作、工序流转、酸洗/热处理、报废 |
| PPF-ME-5xx | 返工与质量 | 直接返工、不合格品处置、改制品入出库 |
| PPF-ME-6xx | 安灯系统 | 安灯维护、安灯执行 |
| PPF-ME-7xx | 协作管理 | 内协发出/接收/加工、外协发出/接收 |
