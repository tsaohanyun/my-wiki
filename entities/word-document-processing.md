# Word 文档处理

## 概述
使用 python-docx 库处理 Word 文档的完整指南。
## 核心功能
| 功能 | 方法 |
|------|------|
| 创建文档 | `Document()` |
| 添加标题 | `add_heading()` |
| 添加段落 | `add_paragraph()` |
| 添加表格 | `add_table()` |
| 添加图片 | `add_picture()` |
| 保存文档 | `save()` |
## 使用模式
- **报告生成** - 自动生成报告
- **模板填充** - 填充文档模板
- **批量处理** - 批量处理多个文档
- **格式转换** - 转换为其他格式
