# 饲料行业知识

## 行业概述

饲料工业是连接种植业与养殖业的关键环节，是为动物饲养提供营养全面、均衡的配合饲料及饲料添加剂的制造业。中国是全球最大的饲料生产国，饲料年产量超过3亿吨。

## 饲料产品分类

### 按形态分类
- **颗粒饲料**：经制粒机制成的颗粒状饲料，适口性好，避免动物挑食
- **粉状饲料**：粉碎混合后的粉状（适用于部分禽类和水产苗种）
- **膨化饲料**：经膨化处理，提高淀粉糊化度和消化率，多用于水产和宠物饲料
- **散装饲料**：无包装，通过散装车运输，降低成本
- **袋装饲料**：标准包装（通常40kg/袋），便于仓储和销售

### 按营养功能分类
- **配合饲料**：根据动物营养需求配制的全价饲料
- **浓缩饲料**：不含能量原料（玉米等）的高蛋白预混料
- **预混合饲料**：维生素、矿物质、氨基酸等微量成分预混
- **添加剂**：单一饲料添加剂（酶制剂、酸化剂、益生菌等）

### 按动物种类分类
- **猪饲料**：母猪料、仔猪料、生长育肥猪料
- **禽饲料**：蛋鸡料、肉鸡料、鸭料
- **水产饲料**：鱼料、虾料、蟹料等（多用膨化工艺）
- **反刍饲料**：牛料、羊料
- **宠物饲料**：猫粮、狗粮

## 饲料生产工艺流程

### 核心工段

通威农发智能工厂方案中定义了以下核心工段：

1. **粉碎工段**
   - 主要设备：粉碎机、超微粉碎机
   - 将原料粉碎至合格粒度（如40-60目）

2. **混合工段**
   - 主要设备：混合机
   - 将各种原料按配方比例混合均匀

3. **制粒工段**
   - 主要设备：制粒机
   - 将粉状饲料压制成颗粒，需要调节温度和水分

4. **膨化工段**
   - 主要设备：膨化机
   - 高温高压膨化处理，适用于水产饲料

5. **喷涂工段**
   - 主要设备：油脂喷涂机
   - 对颗粒表面喷涂油脂、酶制剂等液体添加剂

6. **打包工段**
   - 主要设备：打包秤、封口机
   - 完成成品计量、包装、码垛

### 特殊工段
- **预混稀释料**：微量添加成分的预处理工艺
- **膏体预处理**：膏状原料的预处理
- **浆体预处理**：液态/浆状原料的预处理
- **清洗工段**：换产时的设备清洗

## 配方管理

### 配方类型
| 类型 | 定义 | 管理权限 |
|------|------|---------|
| 营养配方 | 基于动物营养需求的完整配方 | 营养师（总部） |
| 大小料配方 | 大料（玉米/豆粕等大宗）+小料（添加剂等微量） | 品管部经理 |
| 半成品配方 | 大料中的部分原料分解为中间产品 | 中控技师 |
| 工艺配方 | 物料分解到各工段投料口的工艺参数 | 中控技师 |

### 大小料定义
- **大料**（大宗原料）：玉米、豆粕、小麦、鱼粉、油脂等占配方比例大的成分
- **小料**（微量添加剂）：维生素、矿物质、氨基酸、药物添加剂等

### 配方管理关键业务
- 配方版本管理（编号+版本号唯一标识）
- 配方临时调整（缺料/质量异常触发）
- 返工料添加（不合格品回收利用）
- 调料配方确认（基地间调拨的配方一致性）

## 计划管理

### 计划层级
| 层级 | 类型 | 周期 |
|------|------|------|
| 年度计划 | 生产总量规划 | 年 |
| 月度计划 | 分品种/生产线 | 月 |
| 日计划 | 按工单排产 | 日 |
| 换包计划 | 批次切换 | 按需 |

### 关键业务
- 产销平衡分析
- 物料齐套检查（配料可用性）
- 换产任务管理
- 返工料并名

## 仓储管理

### 原料入库方式
| 方式 | 说明 | 适用物料 |
|------|------|---------|
| 汽运入库 | 散装/袋装汽车运输入库 | 大宗原料 |
| 船运入库（流量秤） | 船舶卸货、流量秤计量 | 大宗原料（河运） |
| 船运入库（码头磅） | 船舶过磅计量 | 大宗原料 |
| 油车入库 | 液体油罐车 | 油脂类 |
| 集装箱甩柜入库 | 集装箱整体卸货 | 进口原料 |

### 仓储类型
- **原料仓**：大宗原料（筒仓/平房仓）
- **辅料/添加剂仓**：小料存储区
- **车间仓**：生产线旁的暂存区
- **成品仓**：完工成品存储区（袋装/散装）
- **油罐区**：液态油脂存储

### 关键业务
- 投料计量/均重出库
- 采购退货/销售退货
- 月度盘点/散料盘点/车间仓盘点
- 库内货位移动/破包处理/子库转移
- 内部调拨/磅差异常处理

## 质量管理

### 检验类型
| 检验类型 | 时机 | 重点 |
|---------|------|------|
| 来料检验（QC003） | 原料入库前 | 理化指标、感官指标、卫生指标 |
| 过程检验（QC005） | 生产过程中 | 工艺参数、半成品质量 |
| 在库检验（QC006） | 存储期间 | 变质/虫害/超期 |
| 完工检验（QC007） | 成品入库前 | 产品全项指标检测 |
| 退货检验（QC010） | 客户退货 | 确认退货原因和责任 |

### 质量标准体系
- **国标**：国家强制性标准
- **企业标准（企标）**：企业内控标准（通常严于国标）
- **内控标准**：更严格的内部质量标准

### 不合格管理
不合格管理流程（QC009）包含：
- 不合格品标识、隔离
- 评审（让步接收/降级/报废/返工）
- 纠正与预防措施

### 质量追溯
从原料批次到成品批次的**正反向追溯**：
- 正向：原料→生产批→成品→客户
- 反向：客户反映问题→成品批→生产批→原料批次

## 设备管理

### 四大主机（饲料行业核心设备）
1. **粉碎机** — 原料粉碎
2. **超微粉碎机** — 超细微粉碎（水产料等）
3. **制粒机** — 颗粒压制
4. **膨化机** — 膨化处理（水产料、宠物料）

### 维保类型
| 类型 | 周期 | 内容 |
|------|------|------|
| 点巡检 | 每日/每班 | 日常检查、润滑、清洁 |
| 月度维保 | 每月 | 常规保养、易损件更换 |
| 年度维保 | 每年 | 大修、精度校准 |
| 特种设备定检 | 法定周期 | 压力容器/锅炉/电梯法定检验 |
| 计量器具校验 | 按规程 | 地磅/打包秤/化验仪器校准 |

## 饲料行业数字化/智能化

### 核心系统架构
```
┌─────────────────────────────────────────┐
│           集团管理系统 (SAP)              │
├─────────────────────────────────────────┤
│    数智制造平台 (生产管理核心)            │
├─────────────────────────────────────────┤
│  MES (工厂执行系统)  │  SWMS (仓储管理)  │
├─────────────────────────────────────────┤
│       IOT平台 (设备数据采集)              │
├─────────────────────────────────────────┤
│         PLC/SCADA (自动化控制)            │
└─────────────────────────────────────────┘
```

### 数字化目标
- **标准化**：工艺标准、操作标准、质量标准统一化
- **自动化**：生产过程自动化控制
- **透明化**：生产过程数据实时采集与展现
- **智能化**：基于数据分析的决策优化
- **精益化**：减少浪费、提升效率

### 关键系统
- **配方系统（FBC）**：营养配方编制与管理
- **平台系统（SME/SWMS）**：总部级数智制造平台
- **工厂MES**：车间级生产执行系统
- **SAP**：企业资源计划系统

## 行业关键术语

| 术语 | 解释 |
|------|------|
| 营养配方 | 满足动物营养需求的饲料原料配比方案 |
| 大小料 | 大料为大宗能量/蛋白原料，小料为微量添加剂 |
| 返工料 | 生产过程中不合格品经处理后回用的物料 |
| 调料 | 分子公司间的物料调拨 |
| 齐套检查 | 生产前检查所有物料是否齐全可用 |
| 换包/换产 | 从生产一种产品切换到另一种产品时的产线调整 |
| 工段 | 饲料生产线上独立的功能单元（粉碎/混合/制粒等） |
| 生产批 | 同一配方/同一时间段生产的批次单元 |
| 成品批 | 完工后形成的最终产品批次 |
| 感官检验 | 通过视觉/嗅觉/触觉判断原料品质（颜色/气味/质地） |
| 理化指标 | 水分/蛋白/脂肪/灰分/纤维等化学成分指标 |
