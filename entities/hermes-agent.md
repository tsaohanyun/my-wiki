# Hermes Agent

## 概述
Hermes Agent 是一个功能丰富的 CLI AI Agent 平台，支持技能扩展、多平台集成和自主代理。
## 核心功能
| 功能 | 描述 |
|------|------|
| 技能系统 | 可扩展的技能框架 |
| 多平台网关 | Telegram/Discord/Slack/CLI |
| 子代理 | 任务委派和并行执行 |
| 语音交互 | TTS/STT 支持 |
| 持久记忆 | 跨会话记忆系统 |
## 使用模式
- **CLI 交互** - 命令行直接交互
- **网关模式** - 通过消息平台交互
- **自主代理** - 无人值守的自主执行
- **技能开发** - 创建自定义技能
