# PDF 文档处理

## 概述
处理 PDF 文档的完整指南，包括读取、创建、编辑和转换。
## 核心功能
| 功能 | 工具 |
|------|------|
| 读取文本 | pymupdf/marker-pdf |
| 创建 PDF | reportlab/fpdf |
| 合并拆分 | PyPDF2 |
| 表单填充 | pdfrw |
| OCR 识别 | marker-pdf/tesseract |
| 加密解密 | PyPDF2 |
## 使用模式
- **文档转换** - 转换为其他格式
- **批量处理** - 批量处理多个 PDF
- **OCR 识别** - 扫描件文字识别
- **表单处理** - 自动填充表单
