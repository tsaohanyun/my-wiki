# PowerPoint 演示文稿处理

## 概述
使用 python-pptx 库处理 PowerPoint 演示文稿的完整指南。
## 核心功能
| 功能 | 方法 |
|------|------|
| 创建演示文稿 | `Presentation()` |
| 添加幻灯片 | `slides.add_slide()` |
| 添加文本 | `add_textbox()` |
| 添加图片 | `add_picture()` |
| 添加图表 | `add_chart()` |
| 保存文件 | `save()` |
## 使用模式
- **演示文稿创建** - 自动生成演示文稿
- **模板使用** - 使用现有模板
- **批量生成** - 批量生成多个演示文稿
- **格式转换** - 转换为其他格式
