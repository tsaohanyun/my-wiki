# Excel 电子表格处理

## 概述
处理 Excel 电子表格的完整指南，支持 .xlsx/.xlsm/.csv 格式。
## 核心功能
| 功能 | 库 |
|------|-----|
| 读取数据 | pandas/openpyxl |
| 写入数据 | openpyxl |
| 公式计算 | openpyxl |
| 图表创建 | openpyxl |
| 数据清洗 | pandas |
## 使用模式
- **数据分析** - 读取和分析数据
- **报表生成** - 生成 Excel 报表
- **数据清洗** - 清洗和转换数据
- **批量处理** - 批量处理多个文件
