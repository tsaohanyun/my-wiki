# CDGA 考试概述

## 什么是CDGA

CDGA（Certified Data Governance Associate）是DAMA国际颁发的数据治理助理认证，是数据治理领域的入门级认证。

## 考试信息

| 项目 | 内容 |
|------|------|
| 考试形式 | 闭卷笔试 |
| 考试时间 | 100分钟 |
| 题目数量 | 100道单选题 |
| 通过分数 | 60分 |
| 考试语言 | 中文/英文 |
| 证书有效期 | 3年 |

## 考试大纲

### 第一部分：数据治理基础（30%）

1. **数据治理概念**（10%）
   - 数据治理定义
   - 数据治理vs数据管理
   - 数据治理的价值

2. **数据治理组织**（10%）
   - 数据治理委员会
   - 数据管理专员
   - CDO职责

3. **数据治理制度**（10%）
   - 数据政策
   - 数据标准
   - 数据流程

### 第二部分：数据管理知识（50%）

4. **数据架构**（8%）
   - 企业数据架构
   - 数据建模
   - 数据流设计

5. **数据建模**（8%）
   - 概念模型
   - 逻辑模型
   - 物理模型

6. **数据存储**（6%）
   - 数据库设计
   - 数据仓库
   - 数据湖

7. **数据安全**（8%）
   - 数据分类
   - 访问控制
   - 隐私保护

8. **数据集成**（6%）
   - ETL/ELT
   - 数据同步
   - 互操作性

9. **元数据管理**（6%）
   - 技术元数据
   - 业务元数据
   - 数据目录

10. **数据质量**（8%）
    - 质量维度
    - 质量评估
    - 质量改进

### 第三部分：高级主题（20%）

11. **大数据**（6%）
    - 大数据概念
    - Hadoop生态
    - 数据科学

12. **成熟度评估**（6%）
    - 成熟度模型
    - 评估方法
    - 改进路径

13. **组织变革**（8%）
    - 变革管理
    - 数据文化
    - 培训体系

## 备考建议

### 学习路径

```
第1-2周：通读DAMA-DMBOK2，建立知识框架
第3-4周：精读重点章节，做笔记
第5-6周：刷题+模拟考试
第7周：查漏补缺，重点复习
```

### 学习资源

1. **官方教材**：DAMA-DMBOK2（数据管理知识体系指南）
2. **精读笔记**：本Wiki的18章精读笔记
3. **练习题**：CDGA练习题集
4. **案例分析**：本Wiki的实践案例

### 考试技巧

1. **理解为主**：CDGA考察理解能力，不是死记硬背
2. **排除法**：遇到不确定的题目，先排除明显错误选项
3. **时间管理**：平均每题1分钟，不要在难题上花费太多时间
4. **审题仔细**：注意题目中的关键词（最、首先、不包括等）
