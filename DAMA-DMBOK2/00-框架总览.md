# DAMA-DMBOK2 框架总览

## 概述

DAMA-DMBOK2（Data Management Body of Knowledge, 2nd Edition Revised）是国际数据管理协会（DAMA International）发布的数据管理知识体系指南，是全球数据管理领域最权威的参考框架。修订版于2024年发布，中文版于2025年由机械工业出版社出版。

本wiki基于DAMA-DMBOK2（原书第2版修订版）编订，涵盖全部17个知识领域的核心概念、方法论、组织角色和最佳实践。

## DMBOK金字塔

DMBOK金字塔描述了数据管理从基础到高阶的层次：

```
        ┌─────────────┐
        │    智 慧     │  ← 战略决策
        ├─────────────┤
        │    知 识     │  ← 洞察和模式
        ├─────────────┤
        │    信 息     │  ← 有上下文的数据
        ├─────────────┤
        │    数 据     │  ← 原始事实
        └─────────────┘
```

- **数据**：原始事实和统计数据，需要被捕获、存储和保护
- **信息**：有上下文的数据，能够回答"谁、什么、何时、哪里"等问题
- **知识**：经过分析和理解的信息，能够回答"如何"和"为什么"
- **智慧**：能够做出正确判断和决策的能力

## 战略一致性模型（SAM）

DAMA采用战略一致性模型（Strategic Alignment Model）来确保数据管理与业务战略对齐：

```
┌──────────────────┬──────────────────┐
│   业务战略        │   IT战略         │
├──────────────────┼──────────────────┤
│   业务基础设施    │   IT基础设施      │
└──────────────────┴──────────────────┘
```

四个象限之间需要持续的对齐和调整：
- **业务战略 ↔ IT战略**：IT如何支撑业务目标
- **业务战略 ↔ 业务基础设施**：组织架构如何支撑战略执行
- **IT战略 ↔ IT基础设施**：技术架构如何支撑IT战略
- **业务基础设施 ↔ IT基础设施**：系统如何支撑业务流程

## 数据管理10项原则

DAMA-DMBOK2定义了10项数据管理核心原则：

1. **数据是有价值的企业资产**：数据应被视为战略资产进行管理
2. **数据管理需要企业级视角**：跨部门、跨系统的全局管理
3. **数据管理需要利益相关方参与**：业务和技术共同承担数据责任
4. **数据管理是跨职能的**：需要多种角色和技能的协作
5. **数据生命周期管理**：从创建到处置的全程管理
6. **数据管理需要单一/可信数据源**：主数据和参考数据的统一
7. **数据管理需要元数据**：数据的数据是理解数据的基础
8. **数据管理需要数据质量**：低质量数据无法支撑业务价值
9. **数据管理需要数据安全**：保护数据免受未授权访问和泄露
10. **数据管理是持续改进的过程**：需要定期评估和优化

## 知识领域之间的关系

### 数据治理的中心地位
数据治理为所有其他知识领域提供：
- 方针和制度
- 决策权和职责分配
- 合规性监督
- 质量标准

### 元数据的贯穿作用
元数据管理（[[DAMA-DMBOK2/12-元数据管理]]）为其他领域提供：
- 数据目录和数据字典
- 数据血缘追踪
- 影响分析基础

### 数据质量的基础作用
数据质量管理（[[DAMA-DMBOK2/13-数据质量管理]]）支撑：
- 数据仓库和BI的可信度
- 主数据管理的数据一致性
- 数据集成的数据标准化

### 数据安全的保障作用
数据安全（[[DAMA-DMBOK2/07-数据安全]]）贯穿：
- 所有数据访问和使用场景
- 数据生命周期的每个阶段
- 合规性要求（GDPR等）

## CDMP认证体系

DAMA国际提供数据管理专业人员认证（CDMP），分为三个级别：

| 级别 | 名称 | 要求 | 适用人群 |
|------|------|------|---------|
| 基础级 | CDGA（数据治理工程师） | 通过基础考试 | 入门从业者 |
| 专家级 | CDGP（数据治理专家） | 通过专家考试+经验 | 资深从业者 |
| 大师级 | CDMP（数据管理大师） | 考试+论文+推荐 | 行业领袖 |
