## 2. 数据标准模板

### 数据命名规范

```
[系统前缀]_[业务域]_[实体]_[属性]

示例：
- MES_PROD_ORDER_STATUS（MES_生产_工单_状态）
- WMS_INV_QTY_CURRENT（WMS_库存_数量_当前）
- QMS_INSP_RESULT_FINAL（QMS_检验_结果_最终）
```

### 数据字典模板

| 字段名 | 中文名 | 数据类型 | 长度 | 必填 | 默认值 | 说明 |
|--------|--------|----------|------|------|--------|------|
| order_id | 工单号 | VARCHAR | 32 | 是 | - | 主键 |
| order_status | 工单状态 | VARCHAR | 20 | 是 | DRAFT | DRAFT/IN_PROGRESS/COMPLETED |
| create_time | 创建时间 | DATETIME | - | 是 | NOW() | 记录创建时间 |
| validflag | 有效标志 | CHAR | 1 | 是 | 1 | 1有效/0删除 |

## 3. 数据质量检查清单

### 每日检查项

```sql
-- 1. 数据完整性检查
SELECT 
    'table_name' as table_name,
    COUNT(*) as total_count,
    SUM(CASE WHEN required_field IS NULL THEN 1 ELSE 0 END) as null_count,
    ROUND(SUM(CASE WHEN required_field IS NULL THEN 1 ELSE 0 END) / COUNT(*) * 100, 2) as null_rate
FROM target_table
WHERE validflag = '1';

-- 2. 数据唯一性检查
SELECT 
    unique_field,
    COUNT(*) as duplicate_count
FROM target_table
WHERE validflag = '1'
GROUP BY unique_field
HAVING COUNT(*) > 1;

-- 3. 数据一致性检查
SELECT 
    a.id,
    a.field_a,
    b.field_b,
    CASE WHEN a.field_a != b.field_b THEN 'INCONSISTENT' ELSE 'OK' END as status
FROM table_a a
JOIN table_b b ON a.id = b.id
WHERE a.field_a != b.field_b;
```
