# 数据治理：方法论与实践

> **所属知识体系**：[[DAMA-DMBOK2/03-数据治理|DAMA-DMBOK2 第3章：数据治理]]

---

## 方法论

### 数据治理运营模型

数据治理运营模型分为三种类型：

| 模型 | 特征 | 适用场景 |
|------|------|----------|
| **集中式（Centralized）** | 一个数据治理机构监督所有主题域的全部活动 | 组织规模较小或治理初期 |
| **复制式（Replicated）** | 每个业务单元采用同样的数据治理运营模型和标准 | 多业务单元但流程一致的组织 |
| **联邦式（Federated）** | 一个数据治理组织与多个业务单元协调以维持一致的定义和标准 | 大型复杂企业 |

数据治理可借用政治治理来理解，拥有：
- **立法职能**：定义制度、标准、企业数据架构
- **司法职能**：问题管理和问题修复
- **行政职能**：保护和服务、行政责任

### 数据治理实施流程

数据治理的实施采用渐进式策略，核心流程包括四大阶段：

#### 阶段一：定义组织的数据治理

**1. 执行就绪评估**

| 评估类型 | 说明 |
|----------|------|
| 数据管理成熟度 | 了解组织使用数据做什么，衡量其当前数据管理能力和负荷 |
| 变革能力 | 评估组织对数据治理所需变革的能力，识别潜在阻力 |
| 协作准备度 | 评估组织在管理和使用数据时的协作能力 |
| 业务一致性 | 检验组织在多大程度上将数据的使用和业务战略对齐 |

**2. 对齐业务执行探索**：识别和评估现行制度有效性、数据质量分析、数据管理实践评估

**3. 创建组织触点**：采购与合同、预算与资金、制度合规、软件开发生命周期

#### 阶段二：制定数据治理战略

交付成果一般应包括：章程、运营框架和责任、实施路线图、规划运营目标

**构建运营模式须考虑**：数据对组织的价值、商业模式、文化因素、监管影响

#### 阶段三：实施数据治理

早期优先活动：
1. 定义数据治理流程以满足高优先级目标
2. 建立业务术语表，记录术语和标准
3. 与企业架构和数据架构协调
4. 为数据资产分配经济价值

#### 阶段四：嵌入数据治理

将数据治理活动嵌入将数据作为资产的一系列相关流程。创建**数据治理兴趣社区**，促进本地化和员工相互学习。

### 问题管理流程

问题管理是识别、量化、划分优先级及解决数据治理相关问题的过程。

问题类别包括：权力、变革管理升级、合规、冲突、一致性、合同、数据安全和身份识别、数据质量。

问题升级路径：数据管理团队 → 业务单元数据治理委员会 → 数据治理委员会 → 数据治理指导委员会

### 变革管理

变革管理团队职责：规划、培训、影响软件系统的开发、制度实施、沟通。

沟通应关注：推广数据资产价值、对反馈开展监测和行动、开展数据管理培训、衡量变革管理的效果（意识、愿望、知识、能力、力度）、执行新指标和KPI。

### 法规遵从

全球性法规对数据管理实践有重大影响，包括：会计标准、BCBS 239、CPG 235、PCI-DSS、Solvency Ⅱ、隐私法案等。

## 工具与方法

### 核心工具

| 工具类型 | 说明 |
|----------|------|
| **在线展示/网站** | 提供文档库、搜索功能、简单工作流程管理 |
| **业务术语表** | 包含已达成统一定义的业务术语并与数据相关联的核心工具 |
| **工作流工具** | 管理流程，将流程与文档相连接 |
| **文档管理工具** | 协助管理制度和程序 |
| **数据治理计分卡** | 收集指标跟踪数据治理活动和制度合规情况 |

### 业务术语表

业务术语表的目标：
1. 形成对核心业务概念和术语的共同理解
2. 降低由于业务概念理解不一致而造成的数据滥用风险
3. 提升技术资产与业务组织之间的协调性
4. 最大限度提升搜索能力使得组织知识得以访问

## 度量指标

### 价值指标
对业务目标的贡献、降低风险、提升运营效率

### 有效性指标
目标达成情况、数据管理人员工具使用度、沟通有效性、教育/培训有效性、变革速度

### 可持续性指标
制度和流程执行情况、标准和规程符合情况

## 实施指南

### 渐进式实施策略

在大多数情况下，数据治理工作的实施会采取渐进式策略：
- 将其纳入大型项目（如主数据管理）一并实施
- 按区域或部门逐步推进

### 关键成功因素

1. **高层支持**：需要"首席"高管的支持
2. **文化变革**：需要强有力的领导支持
3. **可持续资金**：确保流程和资金到位
4. **持续沟通**：持续利用现有沟通渠道以一致的方式传递关键信息
5. **行为变革**：离不开正式的变革管理
