# 阿里云服务指南

## 服务概览

阿里云（Alibaba Cloud）是中国最大的云服务提供商，全球市场份额第三。本文档介绍最常用的核心服务及其配置。

## 1. ECS（Elastic Compute Service）

### 实例规格族

| 规格族 | 用途 | 示例规格 |
|--------|------|----------|
| ecs.t6 | 突发性能型 | ecs.t6-c1m2.large |
| ecs.g7 | 通用型 | ecs.g7.2xlarge |
| ecs.c7 | 计算型 | ecs.c7.4xlarge |
| ecs.r7 | 内存型 | ecs.r7.2xlarge |
| ecs.gn7 | GPU计算型 | ecs.gn7i-c8g1.2xlarge |

### Terraform配置ECS

```hcl
provider "alicloud" {
  region = "cn-hangzhou"
}

resource "alicloud_instance" "web_server" {
  instance_name        = "web-server-01"
  instance_type        = "ecs.g7.2xlarge"
  image_id             = "ubuntu_22_04_x64_20G_alibase_20240101.vhd"
  system_disk_category = "cloud_essd"
  system_disk_size     = 40

  vswitch_id                 = alicloud_vswitch.main.id
  security_groups            = [alicloud_security_group.web_sg.id]
  internet_max_bandwidth_out = 10

  tags = {
    Environment = "production"
    Team        = "platform"
  }
}

resource "alicloud_security_group" "web_sg" {
  name   = "web-server-sg"
  vpc_id = alicloud_vpc.main.id
}

resource "alicloud_security_group_rule" "allow_https" {
  type              = "ingress"
  ip_protocol       = "tcp"
  port_range        = "443/443"
  security_group_id = alicloud_security_group.web_sg.id
  cidr_ip           = "0.0.0.0/0"
}
```

### ECS最佳实践

- ✅ 使用抢占式实例降低70-90%成本
- ✅ 开启自动快照策略
- ✅ 使用专有宿主机满足合规要求
- ✅ 使用弹性伸缩组（ESS）自动扩缩容
- ✅ 使用镜像标准化部署流程

## 2. OSS（Object Storage Service）

### 存储类型

| 存储类型 | 最低存储时间 | 检索费用 | 适用场景 |
|----------|-------------|----------|----------|
| 标准存储 | 无 | 无 | 频繁访问 |
| 低频访问 | 30天 | 按GB计费 | 低频访问 |
| 归档存储 | 60天 | 按GB计费 | 归档数据 |
| 冷归档 | 180天 | 按GB计费 | 冷数据 |
| 深度冷归档 | 180天 | 按GB计费 | 极冷数据 |

### OSS配置示例

```hcl
resource "alicloud_oss_bucket" "data_bucket" {
  bucket = "my-app-data-2024"

  server_side_encryption_rule {
    sse_algorithm = "AES256"
  }

  versioning {
    status = "Enabled"
  }

  lifecycle_rule {
    id      = "archive-old-data"
    enabled = true

    transitions {
      days          = 30
      storage_class = "IA"
    }

    transitions {
      days          = 90
      storage_class = "Archive"
    }

    transitions {
      days          = 365
      storage_class = "ColdArchive"
    }
  }

  logging {
    target_bucket = alicloud_oss_bucket.log_bucket.id
    target_prefix = "oss-logs/"
  }
}
```

### OSS SDK使用（Python）

```python
import oss2
from oss2.models import LifecycleRule, LifecycleExpiration

# 配置认证
auth = oss2.Auth('access_key_id', 'access_key_secret')
bucket = oss2.Bucket(auth, 'https://oss-cn-hangzhou.aliyuncs.com', 'my-bucket')

# 上传文件
def upload_file(local_file, oss_key):
    """上传文件到OSS"""
    result = bucket.put_object_from_file(oss_key, local_file)
    print(f"Upload status: {result.status}")
    return result

# 生成签名URL
def get_signed_url(oss_key, expires=3600):
    """生成临时访问URL"""
    url = bucket.sign_url('GET', oss_key, expires)
    return url

# 分片上传大文件
def multipart_upload(local_file, oss_key):
    """分片上传大文件"""
    total_size = os.path.getsize(local_file)
    part_size = 10 * 1024 * 1024  # 10MB

    # 初始化分片
    upload_id = bucket.init_multipart_upload(oss_key).upload_id
    parts = []

    # 上传分片
    with open(local_file, 'rb') as f:
        part_number = 1
        while True:
            data = f.read(part_size)
            if not data:
                break
            result = bucket.upload_part(oss_key, upload_id, part_number, data)
            parts.append(oss2.models.PartInfo(part_number, result.etag))
            part_number += 1

    # 完成分片上传
    result = bucket.complete_multipart_upload(oss_key, upload_id, parts)
    return result
```

## 3. RDS（Relational Database Service）

### 支持的数据库引擎

- MySQL
- PostgreSQL
- SQL Server
- MariaDB
- PPAS（PostgreSQL兼容Oracle）

### RDS配置示例

```hcl
resource "alicloud_db_instance" "main" {
  engine               = "MySQL"
  engine_version       = "8.0"
  instance_type        = "rds.mysql.s3.large"
  instance_storage     = 100
  instance_charge_type = "Postpaid"

  vswitch_id       = alicloud_vswitch.main.id
  security_ips     = ["10.0.0.0/8"]

  security_group_ids = [alicloud_security_group.db_sg.id]

  # 高可用配置
  zone_id              = "cn-hangzhou-h"
  zone_id_slave_a      = "cn-hangzhou-g"

  # 备份配置
  backup_period = ["Monday", "Wednesday", "Friday"]
  backup_time   = "03:00Z-04:00Z"

  # 性能监控
  sql_collector_status = "Enable"
  sql_collector_config_value = 30  # 保留30天

  # 参数优化
  parameters {
    name  = "innodb_buffer_pool_size"
    value = "6442450944"  # 6GB
  }

  tags = {
    Environment = "production"
  }
}

resource "alicloud_db_readonly_instance" "read_replica" {
  master_db_instance_id = alicloud_db_instance.main.id
  engine_version        = alicloud_db_instance.main.engine_version
  instance_type         = alicloud_db_instance.main.instance_type
  instance_storage      = alicloud_db_instance.main.instance_storage
  zone_id               = "cn-hangzhou-g"
  vswitch_id            = alicloud_vswitch.main.id
}
```

## 4. 函数计算（Function Compute）

### 函数计算配置

```hcl
resource "alicloud_fc_service" "api_service" {
  name        = "api-service"
  description = "API Service using Function Compute"
  internet_access = true

  log_config {
    project  = alicloud_log_project.fc_logs.name
    logstore = alicloud_log_store.fc_logs.name
  }
}

resource "alicloud_fc_function" "api_handler" {
  service     = alicloud_fc_service.api_service.name
  name        = "api-handler"
  description = "API Request Handler"
  filename    = "code.zip"
  runtime     = "python3.10"
  handler     = "index.handler"
  memory_size = 256
  timeout     = 30

  environment_variables = {
    TABLE_NAME = "my-table"
    REGION     = "cn-hangzhou"
  }
}

resource "alicloud_fc_trigger" "http_trigger" {
  service  = alicloud_fc_service.api_service.name
  function = alicloud_fc_function.api_handler.name
  name     = "http-trigger"
  type     = "http"

  config = jsonencode({
    authType = "function"
    methods  = ["GET", "POST", "PUT", "DELETE"]
  })
}
```

### 函数代码示例（Python）

```python
import json
import logging

logger = logging.getLogger()

def handler(event, context):
    """
    函数计算HTTP触发器处理函数
    """
    logger.info(f"Event: {event}")

    # 解析请求
    request = json.loads(event)
    http_method = request.get('httpMethod', 'GET')
    path = request.get('path', '/')

    # 路由处理
    if http_method == 'GET' and path == '/health':
        return {
            'statusCode': 200,
            'body': json.dumps({'status': 'healthy'})
        }

    return {
        'statusCode': 404,
        'body': json.dumps({'error': 'Not found'})
    }
```

## 6. RAM（Resource Access Management）

### RAM策略配置

```hcl
# RAM用户
resource "alicloud_ram_user" "developer" {
  name         = "developer-01"
  display_name = "Developer 01"
  comments     = "Development team member"
}

# RAM策略
resource "alicloud_ram_policy" "ecs_readonly" {
  name        = "ECSReadOnly"
  document    = <<EOF
{
  "Version": "1",
  "Statement": [
    {
      "Action": [
        "ecs:Describe*",
        "ecs:List*",
        "ecs:Get*"
      ],
      "Resource": "*",
      "Effect": "Allow"
    }
  ]
}
EOF
  description = "ECS read-only access policy"
}

# 用户策略关联
resource "alicloud_ram_user_policy_attachment" "developer_ecs" {
  policy_name = alicloud_ram_policy.ecs_readonly.name
  policy_type = "Custom"
  user_name   = alicloud_ram_user.developer.name
}
```

### RAM最佳实践

- ✅ 使用RAM角色替代长期AccessKey
- ✅ 启用MFA多因素认证
- ✅ 遵循最小权限原则
- ✅ 定期审计RAM用户和策略
- ✅ 使用STS临时凭证
