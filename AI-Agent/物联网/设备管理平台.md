## 设备注册与认证

### 设备身份模型

```
Product (产品/设备类型)
├── ProductKey: 产品唯一标识
├── 设备模型 (Thing Model)
│   ├── 属性: temperature, humidity
│   ├── 事件: overheat_alarm
│   └── 服务: set_threshold
│
└── Device (设备实例)
    ├── DeviceName: 设备名称/序列号
    ├── DeviceSecret: 设备密钥
    └── 状态: 未激活 / 在线 / 离线
```

### 一机一密 vs 一型一密

| 模式 | 说明 | 安全性 | 适用场景 |
|------|------|--------|----------|
| **一机一密** | 每个设备有独立密钥 | ⭐⭐⭐⭐⭐ | 量产设备 |
| **一型一密** | 同产品共享密钥+动态注册 | ⭐⭐⭐ | 开发/测试 |
| **TLS证书** | X.509 证书认证 | ⭐⭐⭐⭐⭐ | 高安全要求 |

### 设备注册服务实现

```python
"""
设备注册与管理服务
"""
import uuid
import hashlib
import json
from datetime import datetime, timezone
from dataclasses import dataclass, field
from enum import Enum
from typing import Optional

class DeviceStatus(Enum):
    UNACTIVATED = "unactivated"
    ONLINE = "online"
    OFFLINE = "offline"
    DISABLED = "disabled"

@dataclass
class Device:
    device_id: str
    device_name: str
    product_key: str
    device_secret: str
    status: DeviceStatus = DeviceStatus.UNACTIVATED
    metadata: dict = field(default_factory=dict)
    created_at: str = ""
    last_active: Optional[str] = None
    firmware_version: str = ""
    tags: list = field(default_factory=list)

@dataclass
class Product:
    product_key: str
    name: str
    description: str
    thing_model: dict  # 设备模型
    created_at: str = ""


class DeviceRegistry:
    def __init__(self, db_client):
        self.db = db_client  # Redis / PostgreSQL / MongoDB

    def _generate_secret(self, device_name: str, product_key: str) -> str:
        """生成设备密钥"""
        raw = f"{product_key}:{device_name}:{uuid.uuid4()}"
        return hashlib.sha256(raw.encode()).hexdigest()[:32]

    def create_product(self, name: str, description: str, thing_model: dict) -> Product:
        """创建产品（设备类型）"""
        product = Product(
            product_key=f"PK_{uuid.uuid4().hex[:8].upper()}",
            name=name,
            description=description,
            thing_model=thing_model,
            created_at=datetime.now(timezone.utc).isoformat()
        )
        self.db.save_product(product)
        return product

    def register_device(self, product_key: str, device_name: str,
                        metadata: dict = None) -> Device:
        """注册新设备（一机一密模式）"""
        # 检查产品是否存在
        product = self.db.get_product(product_key)
        if not product:
            raise ValueError(f"Product {product_key} not found")

        # 检查设备名是否已存在
        if self.db.get_device(product_key, device_name):
            raise ValueError(f"Device {device_name} already exists")

        # 创建设备
        device = Device(
            device_id=f"{product_key}.{device_name}",
            device_name=device_name,
            product_key=product_key,
            device_secret=self._generate_secret(device_name, product_key),
            status=DeviceStatus.UNACTIVATED,
            metadata=metadata or {},
            created_at=datetime.now(timezone.utc).isoformat()
        )
        self.db.save_device(device)
        return device

    def authenticate_device(self, product_key: str, device_name: str,
                            secret: str) -> bool:
        """设备认证"""
        device = self.db.get_device(product_key, device_name)
        if not device:
            return False
        if device.status == DeviceStatus.DISABLED:
            return False
        # 安全比较（防止时序攻击）
        import hmac
        return hmac.compare_digest(device.device_secret, secret)

    def activate_device(self, device_id: str, firmware_version: str):
        """设备首次连接时激活"""
        device = self.db.get_device_by_id(device_id)
        if device and device.status == DeviceStatus.UNACTIVATED:
            device.status = DeviceStatus.ONLINE
            device.firmware_version = firmware_version
            device.last_active = datetime.now(timezone.utc).isoformat()
            self.db.save_device(device)
```

## 设备影子（Device Shadow）

设备影子是一份存储在云端的 JSON 文档，包含设备的 **期望状态** 和 **上报状态**。解决设备离线时的状态同步问题。

### 影子文档结构

```json
{
    "state": {
        "reported": {
            "temperature": 25.3,
            "fan_speed": "low",
            "firmware_version": "v1.2.0"
        },
        "desired": {
            "fan_speed": "high",
            "threshold": 30.0
        }
    },
    "metadata": {
        "reported": {
            "temperature": { "timestamp": "2026-06-28T10:00:00Z" },
            "fan_speed": { "timestamp": "2026-06-28T09:55:00Z" }
        },
        "desired": {
            "fan_speed": { "timestamp": "2026-06-28T10:01:00Z" }
        }
    },
    "timestamp": "2026-06-28T10:01:00Z",
    "version": 42
}
```

### 影子服务实现

```python
"""
设备影子管理服务
"""
import json
from typing import Any, Optional
from datetime import datetime, timezone

class DeviceShadow:
    def __init__(self, device_id: str, store=None):
        self.device_id = device_id
        self.store = store  # Redis / MongoDB
        self._shadow_key = f"shadow:{device_id}"

    def _default_shadow(self) -> dict:
        return {
            "state": {"reported": {}, "desired": {}},
            "metadata": {"reported": {}, "desired": {}},
            "version": 0
        }

    def get(self) -> dict:
        """获取当前影子状态"""
        if self.store:
            data = self.store.get(self._shadow_key)
            return json.loads(data) if data else self._default_shadow()
        return self._default_shadow()

    def update_reported(self, state: dict) -> dict:
        """设备上报状态更新"""
        shadow = self.get()
        now = datetime.now(timezone.utc).isoformat()

        for key, value in state.items():
            shadow["state"]["reported"][key] = value
            shadow["metadata"]["reported"][key] = {"timestamp": now}

        # 检查 desired 和 reported 的差异（delta）
        delta = self._compute_delta(shadow)

        shadow["version"] += 1
        shadow["timestamp"] = now
        self._save(shadow)

        return {"shadow": shadow, "delta": delta}

    def update_desired(self, state: dict) -> dict:
        """应用端下发期望状态"""
        shadow = self.get()
        now = datetime.now(timezone.utc).isoformat()

        for key, value in state.items():
            shadow["state"]["desired"][key] = value
            shadow["metadata"]["desired"][key] = {"timestamp": now}

        delta = self._compute_delta(shadow)
        shadow["version"] += 1
        shadow["timestamp"] = now
        self._save(shadow)

        return {"shadow": shadow, "delta": delta}

    def delete_desired(self, keys: list = None) -> dict:
        """清除期望状态（设备已同步）"""
        shadow = self.get()

        if keys is None:
            shadow["state"]["desired"] = {}
            shadow["metadata"]["desired"] = {}
        else:
            for key in keys:
                shadow["state"]["desired"].pop(key, None)
                shadow["metadata"]["desired"].pop(key, None)

        shadow["version"] += 1
        self._save(shadow)
        return shadow

    def _compute_delta(self, shadow: dict) -> dict:
        """计算 desired 和 reported 的差异"""
        delta = {}
        desired = shadow["state"]["desired"]
        reported = shadow["state"]["reported"]

        for key, value in desired.items():
            if key not in reported or reported[key] != value:
                delta[key] = value

        return delta

    def _save(self, shadow: dict):
        if self.store:
            self.store.set(self._shadow_key, json.dumps(shadow))


# ─── 使用示例 ───
class MockStore:
    def __init__(self): self.data = {}
    def get(self, k): return self.data.get(k)
    def set(self, k, v): self.data[k] = v

store = MockStore()
shadow = DeviceShadow("PK_ABC123.TH001", store)

# 设备上报当前状态
result = shadow.update_reported({"temperature": 25.3, "fan_speed": "low"})
print(f"Reported: {result['shadow']['state']['reported']}")

# 应用端下发期望状态
result = shadow.update_desired({"fan_speed": "high", "threshold": 30.0})
print(f"Delta (待同步): {result['delta']}")
# → {'fan_speed': 'high', 'threshold': 30.0}

# 设备同步后上报
result = shadow.update_reported({"fan_speed": "high", "threshold": 30.0})
print(f"Delta after sync: {result['delta']}")
# → {} (无差异)
```

### OTA 服务实现

```python
"""
OTA 固件升级管理服务
"""
import uuid
import json
import hashlib
from datetime import datetime, timezone
from dataclasses import dataclass, field
from enum import Enum
from typing import List, Optional
import paho.mqtt.client as mqtt

class FirmwareStatus(Enum):
    DRAFT = "draft"
    PUBLISHED = "published"
    DEPRECATED = "deprecated"

class TaskStatus(Enum):
    PENDING = "pending"
    DOWNLOADING = "downloading"
    INSTALLING = "installing"
    SUCCESS = "success"
    FAILED = "failed"
    ROLLED_BACK = "rolled_back"

@dataclass
class Firmware:
    firmware_id: str
    version: str
    product_key: str
    file_url: str
    file_size: int
    file_hash: str  # SHA256
    description: str = ""
    status: FirmwareStatus = FirmwareStatus.DRAFT
    created_at: str = ""

@dataclass
class OTATask:
    task_id: str
    firmware_id: str
    device_list: List[str]
    target_version: str
    status: dict = field(default_factory=dict)  # device_id -> TaskStatus
    strategy: str = "immediate"  # immediate / scheduled / staged
    created_at: str = ""


class OTAService:
    def __init__(self, db_client, mqtt_client=None):
        self.db = db_client
        self.mqtt = mqtt_client

    def upload_firmware(self, version: str, product_key: str,
                        file_url: str, file_size: int,
                        file_hash: str, description: str = "") -> Firmware:
        """上传新固件"""
        fw = Firmware(
            firmware_id=f"FW_{uuid.uuid4().hex[:8].upper()}",
            version=version,
            product_key=product_key,
            file_url=file_url,
            file_size=file_size,
            file_hash=file_hash,
            description=description,
            created_at=datetime.now(timezone.utc).isoformat()
        )
        self.db.save_firmware(fw)
        return fw

    def publish_firmware(self, firmware_id: str):
        """发布固件"""
        fw = self.db.get_firmware(firmware_id)
        if fw and fw.status == FirmwareStatus.DRAFT:
            fw.status = FirmwareStatus.PUBLISHED
            self.db.save_firmware(fw)

    def create_task(self, firmware_id: str, device_list: List[str],
                    strategy: str = "immediate") -> OTATask:
        """创建 OTA 升级任务"""
        fw = self.db.get_firmware(firmware_id)
        if not fw or fw.status != FirmwareStatus.PUBLISHED:
            raise ValueError("Firmware not available")

        task = OTATask(
            task_id=f"OTA_{uuid.uuid4().hex[:8].upper()}",
            firmware_id=firmware_id,
            device_list=device_list,
            target_version=fw.version,
            strategy=strategy,
            created_at=datetime.now(timezone.utc).isoformat()
        )
        for dev in device_list:
            task.status[dev] = TaskStatus.PENDING
        self.db.save_task(task)

        # 通过 MQTT 通知设备升级
        if strategy == "immediate":
            for device_id in device_list:
                self._notify_device(device_id, fw, task.task_id)

        return task

    def _notify_device(self, device_id: str, fw: Firmware, task_id: str):
        """通过 MQTT 下发升级通知"""
        if not self.mqtt:
            return

        notification = {
            "task_id": task_id,
            "version": fw.version,
            "url": fw.file_url,
            "size": fw.file_size,
            "hash": fw.file_hash,
            "timestamp": datetime.now(timezone.utc).isoformat()
        }
        topic = f"device/{device_id}/ota/notify"
        self.mqtt.publish(topic, json.dumps(notification), qos=1)

    def update_device_status(self, task_id: str, device_id: str,
                             status: TaskStatus, message: str = ""):
        """设备上报升级状态"""
        task = self.db.get_task(task_id)
        if task:
            task.status[device_id] = status
            self.db.save_task(task)
            print(f"[OTA] {device_id}: {status.value} - {message}")


# ─── 设备端 OTA Agent ───

class DeviceOTAAgent:
    """设备端 OTA 代理（嵌入式设备上运行）"""

    def __init__(self, device_id: str, current_version: str):
        self.device_id = device_id
        self.current_version = current_version

    def handle_ota_notification(self, payload: str) -> dict:
        """处理 OTA 通知"""
        info = json.loads(payload)
        target_version = info["version"]

        # 版本检查
        if self._compare_version(target_version, self.current_version) <= 0:
            return {"action": "skip", "reason": "already up to date"}

        # 完整性校验
        downloaded_hash = self._download_and_verify(
            info["url"], info["hash"], info["size"]
        )
        if not downloaded_hash:
            return {"action": "failed", "reason": "hash mismatch"}

        # 安装（设备重启）
        self._install_firmware(target_version)

        return {"action": "success", "version": target_version}

    def _compare_version(self, v1: str, v2: str) -> int:
        parts1 = [int(x) for x in v1.lstrip('v').split('.')]
        parts2 = [int(x) for x in v2.lstrip('v').split('.')]
        for a, b in zip(parts1, parts2):
            if a != b:
                return a - b
        return 0

    def _download_and_verify(self, url: str, expected_hash: str,
                             expected_size: int) -> bool:
        """模拟固件下载与校验"""
        import requests
        resp = requests.get(url)
        data = resp.content

        if len(data) != expected_size:
            return False

        actual_hash = hashlib.sha256(data).hexdigest()
        return actual_hash == expected_hash

    def _install_firmware(self, version: str):
```

```python
"""
设备分组管理服务
"""
from dataclasses import dataclass, field
from typing import List, Set, Optional
import uuid

@dataclass
class DeviceGroup:
    group_id: str
    name: str
    parent_id: Optional[str] = None  # 支持层级分组
    description: str = ""
    tags: dict = field(default_factory=dict)  # 用于动态匹配
    device_ids: Set[str] = field(default_factory=set)
    dynamic_rules: Optional[dict] = None  # 动态分组规则
    created_at: str = ""


class GroupManager:
    def __init__(self, db_client):
        self.db = db_client
        self.groups = {}  # group_id -> DeviceGroup

    def create_group(self, name: str, parent_id: str = None,
                     description: str = "", tags: dict = None) -> DeviceGroup:
        """创建设备分组"""
        group = DeviceGroup(
            group_id=f"GRP_{uuid.uuid4().hex[:8].upper()}",
            name=name,
            parent_id=parent_id,
            description=description,
            tags=tags or {},
            created_at=datetime.now(timezone.utc).isoformat()
        )
        self.groups[group.group_id] = group
        return group

    def add_device(self, group_id: str, device_id: str):
        """将设备加入分组"""
        group = self.groups.get(group_id)
        if group:
            group.device_ids.add(device_id)

    def remove_device(self, group_id: str, device_id: str):
        """将设备移出分组"""
        group = self.groups.get(group_id)
        if group:
            group.device_ids.discard(device_id)

    def get_child_groups(self, group_id: str) -> List[DeviceGroup]:
        """获取子分组"""
        return [g for g in self.groups.values() if g.parent_id == group_id]

    def get_all_devices(self, group_id: str, recursive: bool = True) -> Set[str]:
        """获取分组下所有设备（支持递归）"""
        group = self.groups.get(group_id)
        if not group:
            return set()

        devices = set(group.device_ids)

        if recursive:
            for child in self.get_child_groups(group_id):
                devices.update(self.get_all_devices(child.group_id, recursive))

        return devices

    def create_dynamic_group(self, name: str, rules: dict) -> DeviceGroup:
        """
        创建动态分组（基于标签自动匹配设备）
        rules: {"product_key": "PK_ABC123", "tags.location": "上海"}
        """
        group = DeviceGroup(
            group_id=f"DGRP_{uuid.uuid4().hex[:8].upper()}",
            name=name,
            dynamic_rules=rules,
            created_at=datetime.now(timezone.utc).isoformat()
        )
        # 查询匹配的设备
        matched = self.db.query_devices_by_rules(rules)
        group.device_ids = set(matched)
        self.groups[group.group_id] = group
        return group

    def broadcast_command(self, group_id: str, command: dict,
                          mqtt_client=None):
        """向分组内所有设备广播控制命令"""
        devices = self.get_all_devices(group_id)
        results = {}
        for device_id in devices:
            topic = f"device/{device_id}/cmd"
            if mqtt_client:
                mqtt_client.publish(topic, json.dumps(command), qos=1)
            results[device_id] = "sent"
        return results


# ─── 使用示例 ───
from datetime import datetime, timezone

gm = GroupManager(db=None)

# 创建层级分组
root = gm.create_group("华东区域")
sh = gm.create_group("上海机房", parent_id=root.group_id)
f1 = gm.create_group("楼层F1", parent_id=sh.group_id)

# 添加设备到分组
gm.add_device(f1.group_id, "PK_ABC123.TH001")
gm.add_device(f1.group_id, "PK_ABC123.TH002")
gm.add_device(sh.group_id, "PK_ABC123.GW001")

# 查询上海机房所有设备（含子分组）
all_devices = gm.get_all_devices(sh.group_id)
print(f"上海机房设备: {all_devices}")
# → {'PK_ABC123.TH001', 'PK_ABC123.TH002', 'PK_ABC123.GW001'}
```

```python
"""
设备管理平台 REST API
pip install flask flask-restful

from flask import Flask, request, jsonify
from flask_restful import Resource, Api

app = Flask(__name__)
api = Api(app)

# 初始化服务
registry = DeviceRegistry(MockDB())
shadow_service = DeviceShadow  # 类引用
ota = OTAService(MockDB())
groups = GroupManager(MockDB())


class DeviceRegisterAPI(Resource):
    def post(self):
        """注册设备: POST /api/devices"""
        data = request.json
        try:
            device = registry.register_device(
                product_key=data["product_key"],
                device_name=data["device_name"],
                metadata=data.get("metadata")
            )
            return jsonify({
                "device_id": device.device_id,
                "device_secret": device.device_secret,
                "status": device.status.value
            })
        except ValueError as e:
            return {"error": str(e)}, 400

    def get(self):
        """查询设备列表"""
        return jsonify({"devices": []})


class DeviceShadowAPI(Resource):
    def get(self, device_id):
        """获取设备影子: GET /api/devices/<id>/shadow"""
        s = shadow_service(device_id, MockStore())
        return jsonify(s.get())

    def put(self, device_id):
        """更新期望状态: PUT /api/devices/<id>/shadow/desired"""
        data = request.json
        s = shadow_service(device_id, MockStore())
        result = s.update_desired(data.get("state", {}))
        return jsonify(result)


class OTAFirmwareAPI(Resource):
    def post(self):
        """上传固件: POST /api/ota/firmware"""
        data = request.json
        fw = ota.upload_firmware(
            version=data["version"],
            product_key=data["product_key"],
            file_url=data["file_url"],
            file_size=data["file_size"],
            file_hash=data["file_hash"],
            description=data.get("description", "")
        )
        return jsonify({"firmware_id": fw.firmware_id, "version": fw.version})


class OTATaskAPI(Resource):
    def post(self):
        """创建升级任务: POST /api/ota/tasks"""
        data = request.json
        task = ota.create_task(
            firmware_id=data["firmware_id"],
            device_list=data["device_list"],
            strategy=data.get("strategy", "immediate")
        )
        return jsonify({"task_id": task.task_id, "devices": len(task.device_list)})


class GroupAPI(Resource):
    def post(self):
        """创建分组: POST /api/groups"""
        data = request.json
        group = groups.create_group(
            name=data["name"],
            parent_id=data.get("parent_id"),
            tags=data.get("tags")
        )
        return jsonify({"group_id": group.group_id})

    def get(self, group_id):
        """获取分组设备: GET /api/groups/<id>/devices"""
        devices = groups.get_all_devices(group_id)
        return jsonify({"device_ids": list(devices)})

api.add_resource(OTATaskAPI, "/api/ota/tasks")
api.add_resource(GroupAPI, "/api/groups", "/api/groups/<string:group_id>")

if __name__ == "__main__":
    app.run(debug=True, port=5000)
```

## 最佳实践

### 1. 设备安全
- **一机一密** + TLS 加密：生产环境标配
- **密钥安全存储**：使用安全芯片 (TPM/SE) 或 OTP 区域
- **令牌轮换**：定期更新设备凭据
- **ACL 控制**：设备只能访问自己的 Topic

### 2. 设备影子设计
- 影子不宜过大（<32KB），大文件走文件服务
- 使用 `version` 字段实现乐观并发控制
- 合理区分 `reported` 和 `desired`

### 3. OTA 升级策略
```
升级流程：
1. 灰度测试（1%~5% 设备）
2. 监控成功率 > 95%
3. 批量推送（10% ~ 50%）
4. 全量升级
5. 回滚机制准备

失败处理：
- 最多重试 3 次
- 超时 30 分钟自动标记失败
- 失败率 > 10% 暂停升级
```

### 4. 设备分组策略
- **静态分组**：按区域/产线/项目划分
- **动态分组**：按标签自动归组（设备标签变化时自动调整）
- **分组广播**：利用分组实现批量命令下发

### 5. 监控与告警
- 设备在线率监控
- 消息到达率/延迟
- OTA 升级成功率
- 异常设备自动隔离

### 6. 扩展性设计
- Broker 集群水平扩展
- 使用 Redis 缓存设备状态
- 时序数据分库分表
- API 服务无状态化
