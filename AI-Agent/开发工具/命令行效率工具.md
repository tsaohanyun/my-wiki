# 命令行效率工具

## 1. Zsh + Oh My Zsh

### 安装 Zsh
```bash
# Ubuntu/Debian
sudo apt install zsh

# macOS（默认已安装）
brew install zsh

# 设为默认shell
chsh -s $(which zsh)
```

### 安装 Oh My Zsh
```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

### 推荐插件
```bash
# ~/.zshrc
plugins=(
  git                    # git别名和提示
  zsh-autosuggestions    # 历史命令建议（需安装）
  zsh-syntax-highlighting # 语法高亮（需安装）
  z                     # 智能目录跳转
  docker                # docker补全
  kubectl               # k8s补全
  fzf                   # fzf集成
  zsh-completions       # 更多补全（需安装）
)

# 安装社区插件
git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions
git clone https://github.com/zsh-users/zsh-syntax-highlighting ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting
```

### 推荐主题
```bash
# 内置主题推荐
ZSH_THEME="agnoster"      # 经典powerline风格
ZSH_THEME="powerlevel10k/powerlevel10k"  # 最流行（需安装）
ZSH_THEME="robbyrussell"   # 简洁默认

# 安装 Powerlevel10k
git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k
```

### 实用别名
```bash
# ~/.zshrc
alias ll='ls -lah'
alias gs='git status'
alias gc='git commit'
alias gp='git push'
alias k='kubectl'
alias d='docker'
alias dc='docker compose'
alias py='python3'
alias v='nvim'
alias cls='clear'

# 快速编辑配置
alias zshrc='vim ~/.zshrc && source ~/.zshrc'
alias reload='source ~/.zshrc'
```

## 2. fzf（模糊搜索）

### 安装
```bash
# Ubuntu/Debian
sudo apt install fzf

# macOS
brew install fzf

# 启用快捷键和补全
$(brew --prefix)/opt/fzf/install
```

### 核心用法
```bash
# 基本文件搜索
fzf

# 搜索并用vim打开
vim $(fzf)

# 搜索文件内容
grep -r "pattern" . | fzf

# 搜索目录并cd
cd $(find . -type d | fzf)

# 历史命令搜索（绑定Ctrl+R，安装时自动配置）
# 按 Ctrl+R 触发
```

### 环境变量配置
```bash
# ~/.zshrc
export FZF_DEFAULT_OPTS='
  --height 40%
  --layout=reverse
  --border
  --preview "head -100 {}"
  --preview-window=right:50%
'

# 使用fd作为默认搜索命令（更快）
export FZF_DEFAULT_COMMAND='fd --type f --hidden --follow --exclude .git'
export FZF_CTRL_T_COMMAND="$FZF_DEFAULT_COMMAND"
```

### 高级用法
```bash
# Git分支切换
git branch | fzf | xargs git checkout

# 搜索并杀进程
ps aux | fzf | awk '{print $2}' | xargs kill

# 搜索npm scripts
cat package.json | jq -r '.scripts | keys[]' | fzf | xargs npm run

# 与rg结合搜索内容
rg --color=always --line-number '' | fzf --ansi
```

## 3. ripgrep（rg，快速搜索）

### 安装
```bash
sudo apt install ripgrep   # Ubuntu
brew install ripgrep       # macOS
```

### 核心用法
```bash
# 基本搜索
rg "pattern"

# 指定文件类型
rg "function" -t py
rg "import" -t js

# 正则搜索
rg "\d{3}-\d{4}"

# 只显示文件名
rg -l "TODO"

# 忽略大小写
rg -i "error"

# 显示上下文
rg -C 3 "error"     # 前后3行
rg -B 2 -A 2 "error" # 前2后2

# 替换（预览）
rg "old_name" --replace "new_name"

# 搜索压缩文件
rg -z "pattern"
```

### 实用组合
```bash
# 统计每个文件匹配数
rg -c "TODO" | sort -t: -k2 -rn

# 搜索并排除目录
rg "secret" --glob '!node_modules' --glob '!.git'

# 搜索特定扩展名
rg "config" -g '*.yaml' -g '*.yml'

# JSON输出（配合jq处理）
rg --json "pattern" | jq 'select(.type=="match")'
```

## 4. fd（快速查找文件）

### 安装
```bash
sudo apt install fd-find  # Ubuntu（命令为fdfind）
brew install fd           # macOS
```

### 核心用法
```bash
# 基本查找
fd "pattern"

# 指定类型
fd -e py              # 所有.py文件
fd -t f "test"        # 文件
fd -t d "src"         # 目录

# 指定深度
fd -d 2 "config"

# 排除目录
fd -E node_modules -E .git "js"

# 执行命令
fd -e py -x wc -l {} \;

# 与xargs结合
fd -e py -0 | xargs -0 grep "import"
```

### 配置
```bash
# ~/.config/fd/ignore
node_modules
.git
dist
build
__pycache__
*.pyc
```

### fd vs find
```bash
# find等价写法对比
fd "test" -e py          # find . -name "*.py" -path "*test*"
fd -H ".env"             # find . -name ".env"（包含隐藏文件）
fd -t f -s "Camel"       # find . -type f -name "*Camel*"（大小写敏感）
```

## 5. bat（增强版cat）

### 安装
```bash
sudo apt install bat     # Ubuntu（命令为batcat）
brew install bat         # macOS
```

### 核心用法
```bash
# 查看文件（带语法高亮和行号）
bat README.md

# 查看多个文件
bat src/*.py

# 指定语言
bat --language json data.txt

# 显示不可见字符
bat -A file.txt

# 只显示特定行
bat --line-range 10:20 file.py

# 作为git的diff工具
bat --diff file1.txt file2.txt
```

### 配置
```bash
# ~/.config/bat/config
--theme="OneHalfDark"
--style="numbers,changes,header"
--italic-text=always
--map-syntax "*.conf:INI"
--map-syntax ".ignore:Git Ignore"
```

### 替换cat
```bash
# 在.zshrc中设置别名
alias cat='bat --paging=never'

# 作为man的pager
export MANPAGER="sh -c 'col -bx | bat -l man -p'"
```

## 最佳实践

1. **组合使用** — `fd`找文件 + `rg`搜内容 + `fzf`交互选择
2. **用 `--help`** — 每个工具都有 `--help` 和 man page
3. **配置dotfiles** — 用git管理配置文件，多设备同步
4. **管道思维** — 小工具组合，通过管道 `|` 串联
5. **alias优先** — 常用命令组合写成别名
6. **善用环境变量** — `FZF_DEFAULT_COMMAND`、`BAT_THEME` 等
