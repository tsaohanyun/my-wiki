# 元编程指南

> 元编程（Meta-programming）是"编写编写程序的程序"的技术，即在运行时或编译时**操作程序本身的结构**。涵盖反射、注解处理、代码生成、宏系统和 AST 操作等领域。

---

## 一、反射（Reflection）

反射允许程序在**运行时**检查和修改自身的结构和行为。

### Java 反射

```java
import java.lang.reflect.*;

public class ReflectionDemo {

    // 1. 获取 Class 对象
    public void getClassObject() throws Exception {
        Class<?> clazz1 = String.class;                          // 类字面量
        Class<?> clazz2 = "hello".getClass();                    // 实例获取
        Class<?> clazz3 = Class.forName("java.lang.String");     // 全限定名
    }

    // 2. 检查类结构
    public void inspectClass(Class<?> clazz) {
        // 字段
        for (Field field : clazz.getDeclaredFields()) {
            System.out.printf("Field: %s %s%n",
                field.getType().getSimpleName(), field.getName());
        }

        // 方法
        for (Method method : clazz.getDeclaredMethods()) {
            System.out.printf("Method: %s %s(%s)%n",
                method.getReturnType().getSimpleName(),
                method.getName(),
                String.join(", ", Arrays.stream(method.getParameterTypes())
                    .map(Class::getSimpleName).toArray(String[]::new)));
        }

        // 构造器
        for (Constructor<?> constructor : clazz.getConstructors()) {
            System.out.printf("Constructor: %s(%s)%n",
                clazz.getSimpleName(),
                constructor.getParameterCount());
        }
    }

    // 3. 动态调用
    public void dynamicInvocation() throws Exception {
        Class<?> clazz = Class.forName("com.example.UserService");
        Object instance = clazz.getDeclaredConstructor().newInstance();

        // 调用方法
        Method method = clazz.getMethod("findById", String.class);
        Object result = method.invoke(instance, "user-123");

        // 修改私有字段
        Field field = clazz.getDeclaredField("cache");
        field.setAccessible(true);  // 突破访问控制
        field.set(instance, new HashMap<>());
    }

    // 4. 动态代理
    public void dynamicProxy() {
        // JDK 动态代理（需要接口）
        UserService proxy = (UserService) Proxy.newProxyInstance(
            getClass().getClassLoader(),
            new Class[]{UserService.class},
            (proxyObj, method, args) -> {
                // 前置处理
                System.out.println("Before: " + method.getName());

                // 执行真实逻辑
                Object result = method.invoke(realService, args);

                // 后置处理
                System.out.println("After: " + method.getName());
                return result;
            }
        );
        proxy.findById("123");  // 会触发代理逻辑
    }
}
```

### Python 反射

```python
import inspect
from typing import Any, get_type_hints

class ReflectionDemo:
    """Python 反射示例"""

    def inspect_object(self, obj: Any) -> None:
        # 获取所有属性和方法
        members = inspect.getmembers(obj)
        for name, value in members:
            if not name.startswith('_'):
                print(f"{name}: {type(value).__name__} = {value}")

    def get_class_info(self, cls: type) -> None:
        # 类信息
        print(f"Class: {cls.__name__}")
        print(f"Module: {cls.__module__}")
        print(f"Bases: {[b.__name__ for b in cls.__bases__]}")
        print(f"MRO: {[c.__name__ for c in cls.__mro__]}")

        # 获取类型注解
        hints = get_type_hints(cls)
        print(f"Type hints: {hints}")

        # 获取源代码
        try:
            source = inspect.getsource(cls)
            print(f"Source:\n{source[:200]}...")
        except (OSError, TypeError):
            pass

    def dynamic_attribute_access(self, obj: Any) -> None:
        # getattr / setattr / hasattr
        if hasattr(obj, 'name'):
            name = getattr(obj, 'name')
            print(f"name = {name}")

        setattr(obj, 'dynamic_attr', 42)      # 动态设置属性
        delattr(obj, 'dynamic_attr')           # 动态删除属性

    def dynamic_method_call(self, obj: Any, method_name: str, *args) -> Any:
        method = getattr(obj, method_name, None)
        if callable(method):
            return method(*args)
        raise AttributeError(f"No method '{method_name}'")

# 使用
class User:
    def __init__(self, name: str, age: int):
        self.name = name
        self.age = age

    def greet(self) -> str:
        return f"Hello, I'm {self.name}"

demo = ReflectionDemo()
user = User("Alice", 30)
demo.get_class_info(User)
demo.inspect_object(user)
print(demo.dynamic_method_call(user, 'greet'))  # Hello, I'm Alice
```

### C# 反射与特性

```csharp
using System;
using System.Reflection;

// 自定义特性
[AttributeUsage(AttributeTargets.Class | AttributeTargets.Method)]
public class RouteAttribute : Attribute {
    public string Path { get; }
    public string Method { get; set; } = "GET";
    public RouteAttribute(string path) { Path = path; }
}

[Route("/api/users")]
public class UserController {
    [Route("/{id}", Method = "GET")]
    public User GetUser(string id) => new User(id);

    [Route("", Method = "POST")]
    public User CreateUser(User user) => user;
}

// 反射扫描路由
public class RouteScanner {
    public void ScanRoutes(Assembly assembly) {
        foreach (var type in assembly.GetTypes()) {
            var classRoute = type.GetCustomAttribute<RouteAttribute>();
            if (classRoute == null) continue;

            foreach (var method in type.GetMethods()) {
                var methodRoute = method.GetCustomAttribute<RouteAttribute>();
                if (methodRoute == null) continue;

                var fullPath = classRoute.Path + methodRoute.Path;
                Console.WriteLine($"{methodRoute.Method} {fullPath} -> {type.Name}.{method.Name}");
            }
        }
    }
}
// 输出:
// GET /api/users/{id} -> UserController.GetUser
// POST /api/users -> UserController.CreateUser
```

## 二、注解处理（Annotation Processing）

编译时注解处理在编译期间读取注解，生成新的源代码或字节码。

### Java 注解处理器（APT）

```java
// 1. 定义注解
@Target(ElementType.TYPE)
@Retention(RetentionPolicy.SOURCE)  // 仅源码级别
public @interface Builder {
}

// 2. 定义注解处理器
@SupportedAnnotationTypes("com.example.Builder")
@SupportedSourceVersion(SourceVersion.RELEASE_17)
public class BuilderProcessor extends AbstractProcessor {

    @Override
    public boolean process(Set<? extends TypeElement> annotations,
                           RoundEnvironment roundEnv) {
        for (Element element : roundEnv.getElementsAnnotatedWith(Builder.class)) {
            if (element.getKind() == ElementKind.CLASS) {
                generateBuilderClass((TypeElement) element);
            }
        }
        return true;
    }

    private void generateBuilderClass(TypeElement classElement) {
        String className = classElement.getSimpleName().toString();
        String builderClassName = className + "Builder";
        String packageName = processingEnv.getElementUtils()
            .getPackageOf(classElement).toString();

        // 收集字段
        List<? extends Element> fields = classElement.getEnclosedElements().stream()
            .filter(e -> e.getKind() == ElementKind.FIELD)
            .collect(Collectors.toList());

        // 生成代码
        String code = generateBuilderCode(packageName, builderClassName, className, fields);

        // 写入文件
        try {
            JavaFileObject file = processingEnv.getFiler()
                .createSourceFile(packageName + "." + builderClassName);
            try (Writer writer = file.openWriter()) {
                writer.write(code);
            }
        } catch (IOException e) {
            processingEnv.getMessager().printMessage(
                Diagnostic.Kind.ERROR, "Failed to generate builder: " + e);
        }
    }

    private String generateBuilderCode(String pkg, String builderName,
                                        String className, List<? extends Element> fields) {
        StringBuilder sb = new StringBuilder();
        sb.append("package ").append(pkg).append(";\n\n");
        sb.append("public class ").append(builderName).append(" {\n");

        // 字段
        for (Element field : fields) {
            sb.append("    private ")
              .append(field.asType().toString())
              .append(" ").append(field.getSimpleName()).append(";\n");
        }

        // setter 方法
        for (Element field : fields) {
            String name = field.getSimpleName().toString();
            String type = field.asType().toString();
            sb.append("\n    public ").append(builderName)
              .append(" ").append(name).append("(").append(type).append(" val) {\n")
              .append("        this.").append(name).append(" = val;\n")
              .append("        return this;\n")
              .append("    }\n");
        }

        // build 方法
        sb.append("\n    public ").append(className).append(" build() {\n")
          .append("        return new ").append(className).append("(");
        sb.append(fields.stream()
            .map(Element::getSimpleName)
            .collect(Collectors.joining(", ")));
        sb.append(");\n    }\n}\n");

        return sb.toString();
    }
}
```

## 三、代码生成

### Python：动态创建类

```python
from types import new_class
from typing import Any

def create_dto_class(class_name: str, fields: dict[str, type]) -> type:
    """动态创建 DTO 类"""

    # __init__ 方法
    def __init__(self, **kwargs):
        for name, type_ in fields.items():
            value = kwargs.get(name)
            if value is not None and not isinstance(value, type_):
                raise TypeError(f"{name} must be {type_.__name__}, got {type(value).__name__}")
            setattr(self, name, value)

    # __repr__ 方法
    def __repr__(self):
        attrs = ", ".join(f"{k}={getattr(self, k)!r}" for k in fields)
        return f"{class_name}({attrs})"

    # __eq__ 方法
    def __eq__(self, other):
        if not isinstance(other, self.__class__):
            return False
        return all(getattr(self, k) == getattr(other, k) for k in fields)

    # 使用 type() 动态创建类
    namespace = {
        '__init__': __init__,
        '__repr__': __repr__,
        '__eq__': __eq__,
        '__annotations__': fields,
    }

    return new_class(class_name, bases=(object,), exec_body=lambda ns: ns.update(namespace))

# 使用
UserDTO = create_dto_class("UserDTO", {"name": str, "age": int, "email": str})
user = UserDTO(name="Alice", age=30, email="alice@example.com")
print(user)        # UserDTO(name='Alice', age=30, email='alice@example.com')
print(user.name)   # Alice
```

### Java：使用 JavaPoet 生成代码

```java
import com.squareup.javapoet.*;

// 使用 JavaPoet 生成 Java 源代码
public class CodeGenerator {

    public void generateDTO() throws Exception {
        // 构建字段
        FieldSpec nameField = FieldSpec.builder(String.class, "name")
            .addModifiers(Modifier.PRIVATE)
            .build();

        // 构建 getter
        MethodSpec getName = MethodSpec.methodBuilder("getName")
            .addModifiers(Modifier.PUBLIC)
            .returns(String.class)
            .addStatement("return this.name")
            .build();

        // 构建 setter
        MethodSpec setName = MethodSpec.methodBuilder("setName")
            .addModifiers(Modifier.PUBLIC)
            .addParameter(String.class, "name")
            .addStatement("this.name = name")
            .build();

        // 构建类
        TypeSpec userDTO = TypeSpec.classBuilder("UserDTO")
            .addModifiers(Modifier.PUBLIC)
            .addField(nameField)
            .addMethod(getName)
            .addMethod(setName)
            .build();

        // 写入文件
        JavaFile javaFile = JavaFile.builder("com.example.generated", userDTO)
            .indent("    ")
            .build();

        javaFile.writeTo(new File("src/generated/java"));
    }
}
```

### Rust：derive 宏（过程宏）

```rust
use proc_macro::TokenStream;
use quote::quote;
use syn::{parse_macro_input, DeriveInput, Data, Fields};

// 自定义 derive 宏
#[proc_macro_derive(Builder)]
pub fn derive_builder(input: TokenStream) -> TokenStream {
    let ast = parse_macro_input!(input as DeriveInput);
    let name = &ast.ident;
    let builder_name = format!("{}Builder", name);
    let builder_ident = syn::Ident::new(&builder_name, name.span());

    // 提取字段
    let fields = match &ast.data {
        Data::Struct(data) => match &data.fields {
            Fields::Named(named) => &named.named,
            _ => panic!("Builder only supports named fields"),
        },
        _ => panic!("Builder only supports structs"),
    };

    let field_names: Vec<_> = fields.iter().map(|f| &f.ident).collect();
    let field_types: Vec<_> = fields.iter().map(|f| &f.ty).collect();

    let expanded = quote! {
        pub struct #builder_ident {
            #(#field_names: Option<#field_types>),*
        }

        impl #builder_ident {
            pub fn new() -> Self {
                Self {
                    #(#field_names: None),*
                }
            }

            #(
                pub fn #field_names(mut self, val: #field_types) -> Self {
                    self.#field_names = Some(val);
                    self
                }
            )*

            pub fn build(self) -> Result<#name, String> {
                Ok(#name {
                    #(#field_names: self.#field_names.ok_or(
                        format!("Field {} is not set", stringify!(#field_names))
                    )?),*
                })
            }
        }

        impl #name {
            pub fn builder() -> #builder_ident {
                #builder_ident::new()
            }
        }
    };

    expanded.into()
}
```

## 四、宏

### Lisp/Scheme 宏（卫生宏的鼻祖）

```lisp
;; Common Lisp: defmacro
(defmacro when (condition &body body)
  `(if ,condition
       (progn ,@body)
       nil))

;; 展开: (when (> x 0) (print x))
;; =>   (if (> x 0) (progn (print x)) nil)

;; 更复杂的宏：unless
(defmacro unless (condition &body body)
  `(if (not ,condition)
       (progn ,@body)
       nil))

;; anaphoric 宏（引入 it 变量）
(defmacro aif (condition then &optional else)
  `(let ((it ,condition))
     (if it ,then ,else)))

;; 使用
(aif (find-user id)
     (format t "Found: ~a" it)
     (format t "Not found"))
;; it 自动绑定为 (find-user id) 的结果

;; 宏展开检查
(macroexpand-1 '(when (> x 0) (print x)))
;; => (IF (> X 0) (PROGN (PRINT X)) NIL)
```

## 五、AST（抽象语法树）操作

### Python AST 操作

```python
import ast
import inspect
from typing import Any

# 1. 解析代码为 AST
code = """
def calculate(x, y):
    result = x + y
    return result * 2
"""

tree = ast.parse(code)
print(ast.dump(tree, indent=2))

# 2. 遍历 AST
class FunctionVisitor(ast.NodeVisitor):
    def visit_FunctionDef(self, node: ast.FunctionDef):
        print(f"Function: {node.name}")
        print(f"  Args: {[a.arg for a in node.args.args]}")
        print(f"  Body: {len(node.body)} statements")
        self.generic_visit(node)

    def visit_Assign(self, node: ast.Assign):
        targets = [ast.dump(t) for t in node.targets]
        value = ast.dump(node.value)
        print(f"  Assign: {targets} = {value}")
        self.generic_visit(node)

    def visit_Return(self, node: ast.Return):
        print(f"  Return: {ast.dump(node.value)}")
        self.generic_visit(node)

visitor = FunctionVisitor()
visitor.visit(tree)

# 3. AST 转换：自动添加日志
class LoggingTransformer(ast.NodeTransformer):
    """为每个函数调用添加日志"""

    def visit_FunctionDef(self, node: ast.FunctionDef) -> ast.FunctionDef:
        # 在函数体开头插入日志语句
        log_stmt = ast.Expr(
            value=ast.Call(
                func=ast.Attribute(
                    value=ast.Name(id='print', ctx=ast.Load()),
                    attr='__call__',
                    ctx=ast.Load(),
                ),
                args=[ast.Constant(value=f"Entering {node.name}")],
                keywords=[],
            )
        )
        # 插入到函数体开头
        node.body.insert(0, ast.parse(f'print("Entering {node.name}")').body[0])
        return node

# 4. 编译和执行修改后的 AST
source = """
def greet(name):
    return f"Hello, {name}!"
"""

tree = ast.parse(source)
transformed = LoggingTransformer().visit(tree)
ast.fix_missing_locations(transformed)

# 编译执行
code_obj = compile(transformed, "<string>", "exec")
namespace = {}
exec(code_obj, namespace)
# namespace['greet']("World") 会先打印日志
```

### Rust syn/quote（AST 解析与生成）

```rust
use syn::{parse_file, Item, ItemFn};
use quote::quote;

fn analyze_and_transform(source: &str) -> proc_macro2::TokenStream {
    // 解析为 AST
    let syntax_tree = parse_file(source).expect("Failed to parse");

    let mut functions = Vec::new();

    // 遍历 AST
    for item in &syntax_tree.items {
        if let Item::Fn(ItemFn { sig, .. }) = item {
            functions.push(sig.ident.clone());
        }
    }

    // 生成新代码
    let instrumentation: Vec<_> = functions.iter().map(|name| {
        quote! {
            println!("Function defined: {}", stringify!(#name));
        }
    }).collect();

    quote! {
        #(#instrumentation)*
    }
}
```

### JavaScript Babel 插件（AST 转换）

```javascript
// Babel 插件：自动将 console.log 转换为带函数名的日志
module.exports = function consoleLogEnhancer(babel) {
    const { types: t } = babel;

    return {
        name: "enhance-console-log",
        visitor: {
            CallExpression(path) {
                // 匹配 console.log(...)
                if (
                    t.isMemberExpression(path.node.callee) &&
                    t.isIdentifier(path.node.callee.object, { name: "console" }) &&
                    t.isIdentifier(path.node.callee.property, { name: "log" })
                ) {
                    // 找到最近的函数名
                    const functionName = findEnclosingFunctionName(path);

                    // 在参数前插入函数名标签
                    const label = t.stringLiteral(`[${functionName}]`);
                    path.node.arguments.unshift(label);
                }
            }
        }
    };
};

function findEnclosingFunctionName(path) {
    let current = path;
    while (current) {
        if (
            current.isFunctionDeclaration() ||
            current.isFunctionExpression() ||
            current.isArrowFunctionExpression()
        ) {
            if (current.node.id) {
                return current.node.id.name;
            }
            return "<anonymous>";
        }
        current = current.parentPath;
    }
    return "<global>";
}

// 输入代码:
// function calculate(x, y) {
//     console.log(x + y);
// }
//
// 输出代码:
// function calculate(x, y) {
//     console.log("[calculate]", x + y);
// }
```

## 最佳实践

### ✅ 推荐做法

1. **优先编译时元编程**：错误在编译时暴露比运行时好
2. **保持可读性**：元编程代码必须注释充分
3. **限制作用域**：元编程的"魔法"应该有明确的边界
4. **提供逃生舱**：允许手动编写替代自动生成的代码
5. **测试生成代码**：对生成的代码进行集成测试

### ❌ 避免

1. **过度使用反射**：性能开销大，类型安全丢失
2. **黑魔法宏**：过于复杂的宏让调试变得困难
3. **运行时 eval**：安全风险（代码注入）
4. **忽略 IDE 支持**：元编程代码往往缺少 IDE 补全和跳转
5. **在热路径使用反射**：缓存反射结果或使用代码生成替代

### 性能建议

```java
// ❌ 每次调用都反射
public Object getProperty(Object obj, String name) throws Exception {
    Field f = obj.getClass().getDeclaredField(name);  // 慢
    f.setAccessible(true);
    return f.get(obj);
}

// ✅ 缓存反射结果
private static final Map<String, Field> FIELD_CACHE = new ConcurrentHashMap<>();

public Object getProperty(Object obj, String name) throws Exception {
    String key = obj.getClass().getName() + "#" + name;
    Field f = FIELD_CACHE.computeIfAbsent(key, k -> {
        try {
            Field field = obj.getClass().getDeclaredField(name);
            field.setAccessible(true);
            return field;
        } catch (NoSuchFieldException e) {
            throw new RuntimeException(e);
        }
    });
    return f.get(obj);
}
```
