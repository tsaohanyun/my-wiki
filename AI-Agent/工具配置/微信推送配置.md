# 微信推送配置

## 概述

通过iLink接口实现微信消息推送。

## 配置方法

### 1. 使用hermes send命令

```bash
# 发送消息到微信
hermes send weixin "消息内容"
```

### 2. 在cron任务中使用

```yaml
# config.yaml
cron:
  jobs:
    - name: daily-report
      schedule: "0 9 * * *"
      deliver: "weixin"
```

## 限制

### iLink限流

- 发送频率有限制
- 过于频繁会触发限流
- 建议间隔至少30秒

### QQ不支持文件附件

- QQ chat_id: `431492A9B43DB65AD5F22284974A29E1`
- 仅支持纯文字消息
- 文件需通过其他方式传输

## 最佳实践

1. **批量消息合并**：减少发送次数
2. **错峰发送**：避免高峰期
3. **备用通道**：配置多个推送渠道
