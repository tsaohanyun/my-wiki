# AI Engineer 核心概念

## 概述

AI Engineer 是构建基于LLM的应用程序的工程师角色。

## 核心技能

### 1. Prompt Engineering
- 系统提示词设计
- Few-shot学习
- Chain-of-Thought

### 2. Agent架构
- ReAct模式
- Tool Use
- Memory系统

### 3. RAG系统
- 文档处理
- 向量检索
- 上下文管理

### 4. Fine-tuning
- 数据准备
- 模型训练
- 评估优化

## 工具链

### LLM Provider
- OpenAI (GPT-4)
- Anthropic (Claude)
- 智谱 (GLM)
- 百度 (文心)

### Agent框架
- LangChain
- AutoGPT
- Hermes Agent

### 向量数据库
- Pinecone
- Weaviate
- Chroma

## 应用场景

1. **智能客服**：自动回答用户问题
2. **代码助手**：代码生成和审查
3. **内容创作**：文章、报告生成
4. **数据分析**：自然语言查询
