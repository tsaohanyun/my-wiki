# 分布式系统设计

## 概述

分布式系统是由多台通过网络互联的计算机协同工作的系统。设计分布式系统需要面对**一致性**、**可用性**、**分区容错**等核心挑战。

## 1. CAP 定理

### 核心概念

```
              Consistency（一致性）
                    /\
                   /  \
                  /    \
                 / CAP  \
                /  定理  \
               /          \
              /____________\
   Availability          Partition
    （可用性）           Tolerance（分区容错）

在分布式系统中，三者最多只能同时满足两个：
  - CP 系统：牺牲可用性，保证一致性和分区容错（如 ZooKeeper、etcd）
  - AP 系统：牺牲强一致性，保证可用性和分区容错（如 Cassandra、DynamoDB）
  - CA 系统：理论上不存在（单机系统可以做到）
```

### CAP 对比

| 特性 | CP 系统 | AP 系统 |
|------|---------|---------|
| 一致性 | 强一致 | 最终一致 |
| 可用性 | 分区时不可用 | 始终可用 |
| 典型代表 | ZooKeeper, etcd, HBase | Cassandra, DynamoDB, CouchDB |
| 适用场景 | 配置管理、分布式锁 | 社交媒体、内容分发 |

## 2. 一致性模型

### 一致性级别

```
强一致性（Linearizability）
  │  所有读操作都能看到最近一次写入的值
  │  代价：高延迟，低可用
  │
  ▼
顺序一致性（Sequential Consistency）
  │  所有操作存在一个全局顺序，每个进程的操作按程序顺序
  │
  ▼
因果一致性（Causal Consistency）
  │  有因果关系的操作保持顺序，无因果关系的操作可以乱序
  │
  ▼
最终一致性（Eventual Consistency）
  │  如果没有新的写入，所有副本最终会收敛到相同的值
  │  代价：可能读到旧数据
  ▼
```

### 共识算法

#### Raft 共识算法

```
┌─────────────────────────────────────────────────────┐
│                  Raft 状态机                          │
├─────────────────────────────────────────────────────┤
│                                                     │
│  ┌─────────┐    选举超时     ┌──────────┐           │
│  │ Follower │ ──────────────→│ Candidate │           │
│  └────┬────┘                └─────┬────┘           │
│       │                           │                 │
│       │ 收到心跳                    │ 获得多数投票     │
│       │                           │                 │
│       ▼                           ▼                 │
│  ┌─────────┐    发现更高任期   ┌──────────┐         │
│  │ Follower │ ←────────────── │  Leader   │         │
│  └─────────┘                 └──────────┘          │
│                                                     │
│  Leader 职责：                                       │
│  - 接收客户端请求                                      │
│  - 复制日志到 Followers                               │
│  - 发送心跳维持权威                                    │
│                                                     │
│  日志复制流程：                                        │
│  Client → Leader → AppendEntry → Followers           │
│  多数确认 → Leader 提交 → 通知 Followers 提交          │
└─────────────────────────────────────────────────────┘
```

#### Raft 实现（简化版）

```python
import random
import time
from enum import Enum
from dataclasses import dataclass, field
from typing import List, Optional, Dict

class NodeState(Enum):
    FOLLOWER = "follower"
    CANDIDATE = "candidate"
    LEADER = "leader"

@dataclass
class LogEntry:
    term: int
    index: int
    command: str

class RaftNode:
    """Raft 共识节点"""

    def __init__(self, node_id: str, peers: List[str]):
        self.node_id = node_id
        self.peers = peers
        self.state = NodeState.FOLLOWER
        self.current_term = 0
        self.voted_for: Optional[str] = None
        self.log: List[LogEntry] = []
        self.commit_index = -1
        self.last_applied = -1
        self.leader_id: Optional[str] = None
        self.votes_received: set = set()
        self._reset_election_timer()

    def _reset_election_timer(self):
        self.election_timeout = random.uniform(0.15, 0.3)
        self.last_heartbeat = time.time()

    def _start_election(self):
        """发起选举"""
        self.state = NodeState.CANDIDATE
        self.current_term += 1
        self.voted_for = self.node_id
        self.votes_received = {self.node_id}
        self._reset_election_timer()

        # 向所有 peer 发送 RequestVote RPC
        for peer in self.peers:
            self._send_request_vote(peer)

    def _send_request_vote(self, peer: str):
        last_log_index = len(self.log) - 1
        last_log_term = self.log[-1].term if self.log else 0
        # RPC call to peer with: term, candidateId, lastLogIndex, lastLogTerm
        print(f"[{self.node_id}] RequestVote to {peer}, term={self.current_term}")

    def handle_request_vote(self, term: int, candidate_id: str,
                            last_log_index: int, last_log_term: int) -> bool:
        """处理投票请求"""
        if term < self.current_term:
            return False

        if term > self.current_term:
            self.current_term = term
            self.state = NodeState.FOLLOWER
            self.voted_for = None

        log_ok = (last_log_term > (self.log[-1].term if self.log else 0)) or \
                 (last_log_term == (self.log[-1].term if self.log else 0) and
                  last_log_index >= len(self.log) - 1)

        if self.voted_for is None or self.voted_for == candidate_id:
            if log_ok:
                self.voted_for = candidate_id
                self._reset_election_timer()
                return True
        return False

    def handle_vote_response(self, term: int, vote_granted: bool, voter_id: str):
        """处理投票响应"""
        if self.state != NodeState.CANDIDATE:
            return
        if term > self.current_term:
            self.current_term = term
            self.state = NodeState.FOLLOWER
            return
        if vote_granted:
            self.votes_received.add(voter_id)
            if len(self.votes_received) > (len(self.peers) + 1) / 2:
                self._become_leader()

    def _become_leader(self):
        self.state = NodeState.LEADER
        self.leader_id = self.node_id
        print(f"[{self.node_id}] Became leader for term {self.current_term}")
        # 开始发送心跳
        self._send_heartbeats()

    def _send_heartbeats(self):
        """发送心跳（AppendEntries RPC）"""
        for peer in self.peers:
            print(f"[{self.node_id}] Heartbeat to {peer}")

    def append_entries(self, command: str) -> LogEntry:
        """Leader 追加日志条目"""
        if self.state != NodeState.LEADER:
            raise ValueError("Only leader can append entries")
        entry = LogEntry(
            term=self.current_term,
            index=len(self.log),
            command=command,
        )
        self.log.append(entry)
        # 复制到 followers
        self._replicate_to_followers(entry)
        return entry

    def _replicate_to_followers(self, entry: LogEntry):
        for peer in self.peers:
            print(f"[{self.node_id}] Replicating entry {entry.index} to {peer}")

    def tick(self):
        """定期调用：检查选举超时"""
        if self.state == NodeState.LEADER:
            self._send_heartbeats()
        elif time.time() - self.last_heartbeat > self.election_timeout:
            self._start_election()
```

### Redis 分布式锁实现

```python
import redis
import uuid
import time
from typing import Optional

class RedisDistributedLock:
    """基于 Redis 的分布式锁"""

    UNLOCK_SCRIPT = """
    if redis.call("GET", KEYS[1]) == ARGV[1] then
        return redis.call("DEL", KEYS[1])
    else
        return 0
    end
    """

    EXTEND_SCRIPT = """
    if redis.call("GET", KEYS[1]) == ARGV[1] then
        return redis.call("PEXPIRE", KEYS[1], ARGV[2])
    else
        return 0
    end
    """

    def __init__(self, redis_client: redis.Redis, key: str,
                 ttl_ms: int = 30000, retry_count: int = 3,
                 retry_delay_ms: int = 200):
        self.redis = redis_client
        self.key = f"lock:{key}"
        self.ttl_ms = ttl_ms
        self.retry_count = retry_count
        self.retry_delay_ms = retry_delay_ms
        self.token: Optional[str] = None
        self._unlock_script = self.redis.register_script(self.UNLOCK_SCRIPT)
        self._extend_script = self.redis.register_script(self.EXTEND_SCRIPT)

    def acquire(self) -> bool:
        """获取锁"""
        self.token = str(uuid.uuid4())
        for attempt in range(self.retry_count):
            acquired = self.redis.set(
                self.key, self.token,
                nx=True, px=self.ttl_ms
            )
            if acquired:
                return True
            if attempt < self.retry_count - 1:
                time.sleep(self.retry_delay_ms / 1000)
        return False

    def release(self) -> bool:
        """释放锁（原子操作，只有持有者能释放）"""
        if not self.token:
            return False
        result = self._unlock_script(keys=[self.key], args=[self.token])
        self.token = None
        return result == 1

    def extend(self, ttl_ms: Optional[int] = None) -> bool:
        """延长锁的过期时间"""
        ttl = ttl_ms or self.ttl_ms
        result = self._extend_script(keys=[self.key], args=[self.token, ttl])
        return result == 1

    def __enter__(self):
        if not self.acquire():
            raise TimeoutError(f"Failed to acquire lock: {self.key}")
        return self

    def __exit__(self, exc_type, exc_val, exc_tb):
        self.release()
        return False


# RedLock 算法
class RedLock:
    """RedLock - 多节点分布式锁"""

    def __init__(self, redis_nodes: List[redis.Redis], key: str,
                 ttl_ms: int = 30000):
        self.nodes = redis_nodes
        self.key = key
        self.ttl_ms = ttl_ms
        self.token: Optional[str] = None
        self.acquired_nodes: List[redis.Redis] = []

    def acquire(self) -> bool:
        self.token = str(uuid.uuid4())
        start_time = time.time() * 1000
        self.acquired_nodes = []

        # 尝试在所有节点上获取锁
        for node in self.nodes:
            try:
                if node.set(self.key, self.token, nx=True, px=self.ttl_ms):
                    self.acquired_nodes.append(node)
            except redis.RedisError:
                pass

        elapsed = time.time() * 1000 - start_time
        validity = self.ttl_ms - elapsed - 2  # 减去时钟漂移

        # 需要在多数节点上成功，且剩余有效时间 > 0
        if len(self.acquired_nodes) >= len(self.nodes) // 2 + 1 and validity > 0:
            return True
        else:
            self._release_all()
            return False

    def release(self):
        self._release_all()

    def _release_all(self):
        for node in self.acquired_nodes:
            try:
                # Lua 脚本保证原子释放
                script = """
                if redis.call("GET", KEYS[1]) == ARGV[1] then
                    return redis.call("DEL", KEYS[1])
                end
                """
                node.eval(script, 1, self.key, self.token)
            except redis.RedisError:
                pass
        self.acquired_nodes = []
```

### ZooKeeper 分布式锁

```python
# ZooKeeper 临时顺序节点实现分布式锁
class ZooKeeperLock:
    """
    ZooKeeper 分布式锁原理：
    1. 在锁节点下创建临时顺序节点
    2. 获取所有子节点，判断自己是否最小
    3. 如果不是最小，监听前一个节点
    4. 前一个节点删除后，再次检查自己是否最小
    5. 获得锁
    """

    def __init__(self, zk_client, lock_path: str):
        self.zk = zk_client
        self.lock_path = lock_path
        self.my_node: Optional[str] = None

    def acquire(self, timeout: float = 30.0) -> bool:
        # 1. 创建临时顺序节点
        self.my_node = self.zk.create(
            f"{self.lock_path}/lock-",
            ephemeral=True,
            sequence=True,
        )
        start = time.time()

        while time.time() - start < timeout:
            # 2. 获取所有子节点并排序
            children = sorted(self.zk.get_children(self.lock_path))
            my_name = self.my_node.split("/")[-1]

            # 3. 如果自己是最小节点，获得锁
            if children[0] == my_name:
                return True

            # 4. 监听前一个节点
            my_index = children.index(my_name)
            prev_node = children[my_index - 1]
            event = self.zk.exists(f"{self.lock_path}/{prev_node}", watch=True)

            if event is None:
                # 前一个节点已删除，重新检查
                continue

            # 5. 等待事件触发
            time.sleep(0.01)

        # 超时，删除节点
        self.zk.delete(self.my_node)
        return False

    def release(self):
        if self.my_node:
            self.zk.delete(self.my_node)
            self.my_node = None
```

### Saga 模式

```python
from dataclasses import dataclass
from typing import Callable, List, Optional
import asyncio

@dataclass
class SagaStep:
    name: str
    action: Callable       # 正向操作
    compensation: Callable  # 补偿操作

class SagaOrchestrator:
    """Saga 编排器 - 分布式事务"""

    def __init__(self):
        self.steps: List[SagaStep] = []
        self.completed_steps: List[SagaStep] = []

    def add_step(self, name: str, action: Callable, compensation: Callable):
        self.steps.append(SagaStep(name, action, compensation))
        return self

    async def execute(self, context: dict) -> bool:
        for step in self.steps:
            try:
                print(f"[Saga] Executing step: {step.name}")
                result = await step.action(context)
                context[f"{step.name}_result"] = result
                self.completed_steps.append(step)
            except Exception as e:
                print(f"[Saga] Step {step.name} failed: {e}")
                await self._compensate(context)
                return False
        return True

    async def _compensate(self, context: dict):
        """逆序执行补偿操作"""
        print("[Saga] Starting compensation...")
        for step in reversed(self.completed_steps):
            try:
                print(f"[Saga] Compensating: {step.name}")
                await step.compensation(context)
            except Exception as e:
                print(f"[Saga] Compensation failed for {step.name}: {e}")
                # 补偿失败需要人工介入


# 使用示例
async def create_order_saga():
    saga = SagaOrchestrator()

    saga.add_step(
        name="create_order",
        action=lambda ctx: {"order_id": "ORD-001"},
        compensation=lambda ctx: print("Rolling back order"),
    )
    saga.add_step(
        name="reserve_inventory",
        action=lambda ctx: {"reserved": True},
        compensation=lambda ctx: print("Releasing inventory"),
    )
    saga.add_step(
        name="process_payment",
        action=lambda ctx: {"payment_id": "PAY-001"},
        compensation=lambda ctx: print("Refunding payment"),
    )
    saga.add_step(
        name="send_notification",
        action=lambda ctx: print("Notification sent"),
        compensation=lambda ctx: print("Send cancellation notice"),
    )

    context = {}
    success = await saga.execute(context)
    print(f"Saga {'succeeded' if success else 'failed'}")
```

## 5. 一致性哈希

```python
import hashlib
from bisect import bisect_right
from typing import List, Optional, Any

class ConsistentHash:
    """一致性哈希环"""

    def __init__(self, nodes: List[str], virtual_nodes: int = 150):
        self.virtual_nodes = virtual_nodes
        self.ring: dict = {}        # hash_value -> node_name
        self.sorted_keys: list = []  # 排序的哈希值
        for node in nodes:
            self.add_node(node)

    def _hash(self, key: str) -> int:
        return int(hashlib.md5(key.encode()).hexdigest(), 16)

    def add_node(self, node: str) -> None:
        for i in range(self.virtual_nodes):
            virtual_key = f"{node}:vn{i}"
            h = self._hash(virtual_key)
            self.ring[h] = node
            self.sorted_keys.append(h)
        self.sorted_keys.sort()

    def remove_node(self, node: str) -> None:
        for i in range(self.virtual_nodes):
            virtual_key = f"{node}:vn{i}"
            h = self._hash(virtual_key)
            self.ring.pop(h, None)
            self.sorted_keys.remove(h)

    def get_node(self, key: str) -> Optional[str]:
        if not self.ring:
            return None
        h = self._hash(key)
        idx = bisect_right(self.sorted_keys, h) % len(self.sorted_keys)
        return self.ring[self.sorted_keys[idx]]


# 使用示例
hash_ring = ConsistentHash(["node-1", "node-2", "node-3"])
for i in range(10):
    key = f"user:{i}"
    node = hash_ring.get_node(key)
    print(f"{key} -> {node}")
```

## 最佳实践

### ✅ 推荐

1. **拥抱最终一致性** - 除非业务强需求，否则优先选择最终一致性
2. **幂等设计** - 所有操作必须支持重复执行
3. **超时与重试** - 所有远程调用设置合理超时和指数退避重试
4. **优雅降级** - 依赖不可用时提供降级方案
5. **分布式追踪** - 使用 OpenTelemetry 进行全链路追踪
6. **幂等键** - 关键操作使用唯一幂等键防止重复执行

### ❌ 避免

1. **避免分布式单体** - 不要把所有服务强依赖在一起
2. **避免同步调用链过长** - 超过 3 层同步调用应考虑异步
3. **忽略网络分区** - 网络不可靠是常态，不是异常
4. **过度追求强一致性** - 强一致性的代价往往被低估
