# IIoT 设备接入指南

## 概述

本指南介绍工业设备接入工业互联网平台的方法和最佳实践。

## 1. 通信协议对比

| 协议 | 用途 | 特点 | 应用场景 |
|------|------|------|----------|
| Modbus | 设备通信 | 简单、可靠 | PLC、仪表 |
| OPC UA | 数据交换 | 统一架构、安全 | SCADA、MES |
| MQTT | 消息传输 | 轻量、发布订阅 | 物联网 |
| HTTP | 数据传输 | 通用、易集成 | Web应用 |
| CoAP | 受限设备 | 低功耗 | 传感器 |

## 2. 设备接入流程

1. **设备评估**：确定设备类型、通信协议、接口类型
2. **网关配置**：配置边缘网关，实现协议转换
3. **数据采集**：设置采集频率、数据格式
4. **数据上云**：通过MQTT/HTTP上传数据
5. **数据处理**：云端数据存储、分析、展示

## 3. 边缘网关功能

- 协议转换：支持多种工业协议
- 数据采集：定时采集设备数据
- 数据预处理：过滤、聚合、压缩
- 本地存储：断网时本地缓存
- 安全防护：防火墙、加密
