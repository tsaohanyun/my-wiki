## 1. ECS 架构

### 1.1 什么是 ECS

ECS（Entity-Component-System）是一种**数据导向设计**（Data-Oriented Design）架构，通过将数据（Component）与逻辑分离，实现高性能的批量处理。

```
传统 OOP:                         ECS:
┌──────────────────┐              ┌──────────┐     ┌─────────────┐     ┌────────┐
│   Enemy (Class)   │              │  Entity   │ ←── │  Component   │ ──→ │ System  │
│                    │              │  (ID only)│     │  (Pure Data) │     │(Logic)  │
│  int health        │              └──────────┘     └─────────────┘     └────────┘
│  Vector3 position  │              ┌──────────┐     ┌─────────────┐
│  void Update()     │              │ Entity 1  │ ←── │ Position     │ ──→ MoveSystem
│  void TakeDamage() │              │ Entity 2  │ ←── │ Health       │ ──→ HealthSystem
│  void Move()       │              │ Entity 3  │ ←── │ Velocity     │ ──→ PhysicsSystem
└──────────────────┘              └──────────┘     └─────────────┘

优势:
  - 数据连续排列 → CPU Cache 友好
  - 逻辑并行化 → 多线程友好
  - 组合优于继承 → 灵活扩展
```

### 1.2 ECS 三要素

| 概念 | 角色 | 类比 |
|------|------|------|
| **Entity** | 唯一 ID（无数据、无逻辑） | 数据库主键 |
| **Component** | 纯数据结构（无逻辑） | 数据库列 |
| **System** | 纯逻辑（操作特定 Component 组合） | SQL 查询 |

### 1.3 ECS 核心实现（C#）

```csharp
using System;
using System.Collections.Generic;

namespace ECS
{
    // ==================== Component ====================

    /// <summary>组件基接口（仅标记）</summary>
    public interface IComponent { }

    // 具体组件：纯数据，无逻辑
    public struct Position : IComponent
    {
        public float X, Y, Z;
    }

    public struct Velocity : IComponent
    {
        public float X, Y, Z;
    }

    public struct Health : IComponent
    {
        public float Current;
        public float Max;
    }

    public struct Renderable : IComponent
    {
        public int MeshId;
        public int MaterialId;
    }

    public struct Tag : IComponent
    {
        public string Name;
    }

    // ==================== Entity ====================

    /// <summary>Entity 仅为一个唯一 ID</summary>
    public readonly struct Entity : IEquatable<Entity>
    {
        public readonly int Id;
        public Entity(int id) => Id = id;
        public bool Equals(Entity other) => Id == other.Id;
        public override bool Equals(object obj) => obj is Entity e && Equals(e);
        public override int GetHashCode() => Id;
        public static bool operator ==(Entity a, Entity b) => a.Id == b.Id;
        public static bool operator !=(Entity a, Entity b) => a.Id != b.Id;
        public override string ToString() => $"Entity({Id})";
    }

    // ==================== System ====================

    /// <summary>System 基接口</summary>
    public interface ISystem
    {
        void OnUpdate(float deltaTime);
    }

    // ==================== World（ECS 管理器） ====================

    public class World
    {
        private int nextEntityId = 0;
        private readonly Dictionary<Type, object> componentPools = new();
        private readonly Dictionary<Entity, HashSet<Type>> entityComponents = new();
        private readonly List<ISystem> systems = new();
        private readonly Queue<Entity> entityDestroyQueue = new();

        // ===== Entity 管理 =====

        public Entity CreateEntity()
        {
            var entity = new Entity(nextEntityId++);
            entityComponents[entity] = new HashSet<Type>();
            return entity;
        }

        public void DestroyEntity(Entity entity)
        {
            entityDestroyQueue.Enqueue(entity);
        }

        // ===== Component 管理 =====

        public ref T AddComponent<T>(Entity entity) where T : struct, IComponent
        {
            var pool = GetOrCreatePool<T>();
            ref T component = ref pool.GetOrAdd(entity.Id);
            entityComponents[entity].Add(typeof(T));
            return ref component;
        }

        public ref T GetComponent<T>(Entity entity) where T : struct, IComponent
        {
            var pool = GetOrCreatePool<T>();
            return ref pool.Get(entity.Id);
        }

        public bool HasComponent<T>(Entity entity) where T : struct, IComponent
        {
            return entityComponents.TryGetValue(entity, out var types) && types.Contains(typeof(T));
        }

        public void RemoveComponent<T>(Entity entity) where T : struct, IComponent
        {
            var pool = GetOrCreatePool<T>();
            pool.Remove(entity.Id);
            entityComponents[entity].Remove(typeof(T));
        }

        /// <summary>查询拥有所有指定组件类型的实体</summary>
        public IEnumerable<Entity> Query<T1>() where T1 : struct, IComponent
        {
            foreach (var kvp in entityComponents)
            {
                if (kvp.Value.Contains(typeof(T1)))
                    yield return kvp.Key;
            }
        }

        public IEnumerable<Entity> Query<T1, T2>()
            where T1 : struct, IComponent
            where T2 : struct, IComponent
        {
            foreach (var kvp in entityComponents)
            {
                if (kvp.Value.Contains(typeof(T1)) && kvp.Value.Contains(typeof(T2)))
                    yield return kvp.Key;
            }
        }

        // ===== System 管理 =====

        public void AddSystem(ISystem system)
        {
            systems.Add(system);
        }

        public void Update(float deltaTime)
        {
            foreach (var system in systems)
            {
                system.OnUpdate(deltaTime);
            }
            ProcessDestroyQueue();
        }

        // ===== 内部方法 =====

        private ComponentPool<T> GetOrCreatePool<T>() where T : struct, IComponent
        {
            var type = typeof(T);
            if (!componentPools.TryGetValue(type, out var pool))
            {
                pool = new ComponentPool<T>();
                componentPools[type] = pool;
            }
            return (ComponentPool<T>)pool;
        }

        private void ProcessDestroyQueue()
        {
            while (entityDestroyQueue.Count > 0)
            {
                var entity = entityDestroyQueue.Dequeue();
                if (entityComponents.TryGetValue(entity, out var types))
                {
                    foreach (var type in types)
                    {
                        ((IComponentPoolInternal)componentPools[type]).Remove(entity.Id);
                    }
                }
                entityComponents.Remove(entity);
            }
        }
    }

    // ==================== Component Pool（底层存储） ====================

    internal interface IComponentPoolInternal
    {
        void Remove(int entityId);
    }

    /// <summary>
    /// 组件池：使用稀疏数组存储，保证内存连续性
    /// </summary>
    internal class ComponentPool<T> : IComponentPoolInternal where T : struct, IComponent
    {
        private T[] data = new T[256];
        private readonly HashSet<int> hasEntity = new();

        public ref T Get(int entityId)
        {
            if (entityId >= data.Length)
            {
                Array.Resize(ref data, entityId * 2);
            }
            return ref data[entityId];
        }

        public ref T GetOrAdd(int entityId)
        {
            hasEntity.Add(entityId);
            return ref Get(entityId);
        }

        public void Remove(int entityId)
        {
            hasEntity.Remove(entityId);
            data[entityId] = default;
        }

        public bool Has(int entityId) => hasEntity.Contains(entityId);
    }
}
```

### 1.4 具体系统实现

```csharp
using UnityEngine;

namespace ECS
{
    /// <summary>移动系统：处理所有有 Position + Velocity 的实体</summary>
    public class MovementSystem : ISystem
    {
        private readonly World world;

        public MovementSystem(World world)
        {
            this.world = world;
        }

        public void OnUpdate(float deltaTime)
        {
            foreach (var entity in world.Query<Position, Velocity>())
            {
                ref var pos = ref world.GetComponent<Position>(entity);
                ref var vel = ref world.GetComponent<Velocity>(entity);

                pos.X += vel.X * deltaTime;
                pos.Y += vel.Y * deltaTime;
                pos.Z += vel.Z * deltaTime;
            }
        }
    }

    /// <summary>生命系统：处理死亡逻辑</summary>
    public class HealthSystem : ISystem
    {
        private readonly World world;

        public HealthSystem(World world)
        {
            this.world = world;
        }

        public void OnUpdate(float deltaTime)
        {
            foreach (var entity in world.Query<Health>())
            {
                ref var health = ref world.GetComponent<Health>(entity);

                if (health.Current <= 0)
                {
                    world.DestroyEntity(entity);
                    Debug.Log($"Entity {entity} died");
                }
            }
        }
    }

    /// <summary>渲染同步系统：将 ECS 数据同步到 Unity Transform</summary>
    public class RenderSyncSystem : ISystem
    {
        private readonly World world;
        private readonly Dictionary<int, GameObject> viewMap = new();

        public RenderSyncSystem(World world)
        {
            this.world = world;
        }

        public void OnUpdate(float deltaTime)
        {
            foreach (var entity in world.Query<Position, Renderable>())
            {
                ref var pos = ref world.GetComponent<Position>(entity);

                if (!viewMap.TryGetValue(entity.Id, out var go))
                {
                    go = GameObject.CreatePrimitive(PrimitiveType.Cube);
                    viewMap[entity.Id] = go;
                }

                go.transform.position = new Vector3(pos.X, pos.Y, pos.Z);
            }
        }
    }
}
```

### 1.5 使用示例

```csharp
using UnityEngine;

public class ECSDemo : MonoBehaviour
{
    private World world;

    private void Start()
    {
        world = new World();

        // 注册系统
        world.AddSystem(new MovementSystem(world));
        world.AddSystem(new HealthSystem(world));
        world.AddSystem(new RenderSyncSystem(world));

        // 创建实体并添加组件
        var enemy1 = world.CreateEntity();
        world.AddComponent<Position>(enemy1) = new Position { X = 0, Y = 1, Z = 0 };
        world.AddComponent<Velocity>(enemy1) = new Velocity { X = 1, Z = 0 };
        world.AddComponent<Health>(enemy1) = new Health { Current = 100, Max = 100 };
        world.AddComponent<Renderable>(enemy1) = new Renderable { MeshId = 0, MaterialId = 0 };

        var enemy2 = world.CreateEntity();
        world.AddComponent<Position>(enemy2) = new Position { X = 5, Y = 1, Z = 0 };
        world.AddComponent<Velocity>(enemy2) = new Velocity { X = -1, Z = 1 };
        world.AddComponent<Health>(enemy2) = new Health { Current = 50, Max = 100 };
        world.AddComponent<Renderable>(enemy2) = new Renderable { MeshId = 0, MaterialId = 0 };
    }

    private void Update()
    {
        world.Update(Time.deltaTime);
    }
}
```

## 2. 有限状态机（FSM）

### 2.1 状态机概念

```
状态机: 实体在有限个状态之间切换，每个状态定义独立的行为。

示例 - 角色状态:

     ┌──────────┐  受伤     ┌──────────┐
     │   Idle   │──────────→│   Hurt   │
     │          │←──────────│          │
     └────┬─────┘  恢复      └──────────┘
          │ 按方向键
          ▼
     ┌──────────┐  松开方向   ┌──────────┐
     │   Walk   │──────────→│   Idle   │
     │          │            │          │
     └────┬─────┘            └──────────┘
          │ 按跳跃键
          ▼
     ┌──────────┐  落地       ┌──────────┐
     │   Jump   │──────────→│   Idle   │
     │          │            │          │
     └──────────┘            └──────────┘
```

### 2.2 状态机框架实现

```csharp
using System;
using System.Collections.Generic;

namespace FSM
{
    /// <summary>状态接口</summary>
    public interface IState
    {
        void OnEnter();
        void OnUpdate(float deltaTime);
        void OnExit();
    }

    /// <summary>状态机</summary>
    public class StateMachine
    {
        private IState currentState;
        private readonly Dictionary<Type, IState> states = new();
        private readonly Dictionary<(Type, string), Type> transitions = new();

        public IState CurrentState => currentState;
        public Type CurrentStateType => currentState?.GetType();

        /// <summary>注册状态</summary>
        public void AddState<T>(T state) where T : IState
        {
            states[typeof(T)] = state;
        }

        /// <summary>添加转换规则</summary>
        public void AddTransition<TFrom, TTo>(string trigger = "") where TFrom : IState where TTo : IState
        {
            transitions[(typeof(TFrom), trigger)] = typeof(TTo);
        }

        /// <summary>设置初始状态</summary>
        public void SetInitialState<T>() where T : IState
        {
            if (states.TryGetValue(typeof(T), out var state))
            {
                currentState = state;
                currentState.OnEnter();
            }
        }

        /// <summary>触发状态转换</summary>
        public void Trigger(string trigger)
        {
            if (currentState == null) return;

            var key = (currentState.GetType(), trigger);
            if (transitions.TryGetValue(key, out var targetType))
            {
                ChangeState(targetType);
            }
        }

        /// <summary>直接切换状态</summary>
        public void ChangeState<T>() where T : IState
        {
            ChangeState(typeof(T));
        }

        private void ChangeState(Type targetType)
        {
            if (!states.TryGetValue(targetType, out var newState))
            {
                UnityEngine.Debug.LogError($"State {targetType.Name} not registered!");
                return;
            }

            currentState?.OnExit();
            currentState = newState;
            currentState.OnEnter();
        }

        /// <summary>每帧更新当前状态</summary>
        public void Update(float deltaTime)
        {
            currentState?.OnUpdate(deltaTime);
        }
    }
}
```

### 2.3 角色状态机示例

```csharp
using UnityEngine;

namespace FSM
{
    // ===== 具体状态实现 =====

    public class IdleState : IState
    {
        private readonly PlayerController controller;

        public IdleState(PlayerController controller)
        {
            this.controller = controller;
        }

        public void OnEnter()
        {
            controller.Animator.SetBool("IsMoving", false);
            Debug.Log("进入待机状态");
        }

        public void OnUpdate(float deltaTime)
        {
            if (Input.GetAxisRaw("Horizontal") != 0 || Input.GetAxisRaw("Vertical") != 0)
            {
                controller.StateMachine.Trigger("Move");
            }

            if (Input.GetButtonDown("Jump"))
            {
                controller.StateMachine.Trigger("Jump");
            }
        }

        public void OnExit()
        {
            Debug.Log("离开待机状态");
        }
    }

    public class WalkState : IState
    {
        private readonly PlayerController controller;

        public WalkState(PlayerController controller)
        {
            this.controller = controller;
        }

        public void OnEnter()
        {
            controller.Animator.SetBool("IsMoving", true);
            Debug.Log("进入行走状态");
        }

        public void OnUpdate(float deltaTime)
        {
            float x = Input.GetAxis("Horizontal");
            float z = Input.GetAxis("Vertical");

            Vector3 move = new Vector3(x, 0, z).normalized;
            controller.CharacterController.Move(move * controller.MoveSpeed * deltaTime);

            if (move != Vector3.zero)
            {
                controller.transform.rotation = Quaternion.LookRotation(move);
            }

            if (move == Vector3.zero)
            {
                controller.StateMachine.Trigger("Stop");
            }

            if (Input.GetButtonDown("Jump"))
            {
                controller.StateMachine.Trigger("Jump");
            }
        }

        public void OnExit() { }
    }

    public class JumpState : IState
    {
        private readonly PlayerController controller;

        public JumpState(PlayerController controller)
        {
            this.controller = controller;
        }

        public void OnEnter()
        {
            controller.Animator.SetTrigger("Jump");
            controller.VerticalVelocity = controller.JumpForce;
            Debug.Log("进入跳跃状态");
        }

        public void OnUpdate(float deltaTime)
        {
            // 应用重力
            controller.VerticalVelocity += Physics.gravity.y * deltaTime;

            Vector3 move = new Vector3(
                Input.GetAxis("Horizontal") * controller.MoveSpeed,
                controller.VerticalVelocity,
                Input.GetAxis("Vertical") * controller.MoveSpeed
            ) * deltaTime;

            controller.CharacterController.Move(move);

            if (controller.CharacterController.isGrounded && controller.VerticalVelocity < 0)
            {
                controller.StateMachine.Trigger("Land");
            }
        }

        public void OnExit() { }
    }

    public class HurtState : IState
    {
        private readonly PlayerController controller;
        private float timer;

        public HurtState(PlayerController controller)
        {
            this.controller = controller;
        }

        public void OnEnter()
        {
            timer = 0;
            controller.Animator.SetTrigger("Hurt");
            Debug.Log("进入受伤状态");
        }

        public void OnUpdate(float deltaTime)
        {
            timer += deltaTime;
            if (timer >= 0.5f) // 受伤硬直 0.5 秒
            {
                controller.StateMachine.Trigger("Recover");
            }
        }

        public void OnExit() { }
    }

    // ===== 角色控制器 =====

    public class PlayerController : MonoBehaviour
    {
        [Header("组件")]
        public CharacterController CharacterController;
        public Animator Animator;

        [Header("参数")]
        public float MoveSpeed = 5f;
        public float JumpForce = 8f;

        [HideInInspector] public float VerticalVelocity;
        [HideInInspector] public StateMachine StateMachine;

        private void Awake()
        {
            StateMachine = new StateMachine();

            // 注册状态
            StateMachine.AddState(new IdleState(this));
            StateMachine.AddState(new WalkState(this));
            StateMachine.AddState(new JumpState(this));
            StateMachine.AddState(new HurtState(this));

            // 定义转换
            StateMachine.AddTransition<IdleState, WalkState>("Move");
            StateMachine.AddTransition<WalkState, IdleState>("Stop");
            StateMachine.AddTransition<IdleState, JumpState>("Jump");
            StateMachine.AddTransition<WalkState, JumpState>("Jump");
            StateMachine.AddTransition<JumpState, IdleState>("Land");
            StateMachine.AddTransition<IdleState, HurtState>("Hit");
            StateMachine.AddTransition<WalkState, HurtState>("Hit");
            StateMachine.AddTransition<HurtState, IdleState>("Recover");

            // 设置初始状态
            StateMachine.SetInitialState<IdleState>();
        }

        private void Update()
        {
            StateMachine.Update(Time.deltaTime);
        }

        public void TakeDamage()
        {
            StateMachine.Trigger("Hit");
        }
    }
}
```

## 3. 行为树

### 3.1 行为树概念

行为树是 AI 决策的层次化结构，通过节点组合实现复杂行为。

```
行为树节点类型:

  ┌─────────────────────────────────────────────┐
  │              Selector（选择节点）               │
  │  依次尝试子节点，第一个成功则返回成功            │
  │  类似: if-else if-else                       │
  └─────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────┐
  │              Sequence（序列节点）               │
  │  依次执行子节点，全部成功才返回成功              │
  │  类似: && (逻辑与)                            │
  └─────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────┐
  │           Decorator（装饰节点）                │
  │  修饰子节点的行为（如: 反转结果、重复执行）      │
  └─────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────┐
  │              Action（动作节点）                │
  │  执行具体行为（如: 攻击、移动、播放动画）       │
  └─────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────┐
  │           Condition（条件节点）                │
  │  判断条件是否满足（如: 血量<30%、看到敌人）     │
  └─────────────────────────────────────────────┘

状态返回值:
  Success → 成功
  Failure → 失败
  Running → 执行中（需要多帧）
```

### 3.2 行为树框架实现

```csharp
using System;
using System.Collections.Generic;

namespace BehaviorTree
{
    /// <summary>节点状态</summary>
    public enum NodeState
    {
        Success,
        Failure,
        Running
    }

    /// <summary>行为树节点基类</summary>
    public abstract class Node
    {
        protected List<Node> children = new();
        public Node Parent { get; private set; }

        private Dictionary<string, object> dataContext = new();

        public Node() { }

        public Node(List<Node> children)
        {
            foreach (var child in children)
            {
                Attach(child);
            }
        }

        private void Attach(Node child)
        {
            child.Parent = this;
            children.Add(child);
        }

        /// <summary>执行节点逻辑</summary>
        public abstract NodeState Evaluate();

        // ===== 共享数据（黑板模式）=====

        public void SetData(string key, object value)
        {
            dataContext[key] = value;
        }

        public object GetData(string key)
        {
            if (dataContext.TryGetValue(key, out var value))
                return value;

            // 向上查找父节点
            return Parent?.GetData(key);
        }

        public bool ClearData(string key)
        {
            if (dataContext.ContainsKey(key))
            {
                dataContext.Remove(key);
                return true;
            }
            return Parent?.ClearData(key) ?? false;
        }
    }

    // ==================== 复合节点 ====================

    /// <summary>选择节点：依次尝试，第一个成功就返回</summary>
    public class Selector : Node
    {
        public Selector() : base() { }
        public Selector(List<Node> children) : base(children) { }

        public override NodeState Evaluate()
        {
            foreach (var child in children)
            {
                switch (child.Evaluate())
                {
                    case NodeState.Success:
                        return NodeState.Success;
                    case NodeState.Running:
                        return NodeState.Running;
                    case NodeState.Failure:
                        continue;
                }
            }
            return NodeState.Failure;
        }
    }

    /// <summary>序列节点：依次执行，全部成功才返回</summary>
    public class Sequence : Node
    {
        public Sequence() : base() { }
        public Sequence(List<Node> children) : base(children) { }

        public override NodeState Evaluate()
        {
            foreach (var child in children)
            {
                switch (child.Evaluate())
                {
                    case NodeState.Success:
                        continue;
                    case NodeState.Failure:
                        return NodeState.Failure;
                    case NodeState.Running:
                        return NodeState.Running;
                }
            }
            return NodeState.Success;
        }
    }

    // ==================== 装饰节点 ====================

    /// <summary>反转节点：子节点结果取反</summary>
    public class Inverter : Node
    {
        public Inverter(Node child)
        {
            Attach(child);
        }

        public override NodeState Evaluate()
        {
            switch (children[0].Evaluate())
            {
                case NodeState.Success:
                    return NodeState.Failure;
                case NodeState.Failure:
                    return NodeState.Success;
                case NodeState.Running:
                    return NodeState.Running;
            }
            return NodeState.Failure;
        }
    }

    /// <summary>重复节点：无论结果如何，重复执行子节点</summary>
    public class Repeater : Node
    {
        private readonly int repeatCount;

        public Repeater(Node child, int count = -1) // -1 = 无限
        {
            repeatCount = count;
            Attach(child);
        }

        public override NodeState Evaluate()
        {
            if (repeatCount < 0)
            {
                children[0].Evaluate();
                return NodeState.Running;
            }

            for (int i = 0; i < repeatCount; i++)
            {
                children[0].Evaluate();
            }
            return NodeState.Success;
        }
    }

    /// <summary>成功节点：无论子节点返回什么，都返回成功</summary>
    public class Succeeder : Node
    {
        public Succeeder(Node child)
        {
            Attach(child);
        }

        public override NodeState Evaluate()
        {
            children[0].Evaluate();
            return NodeState.Success;
        }
    }

    // ==================== 叶节点 ====================

    /// <summary>条件节点：评估一个谓词函数</summary>
    public class Condition : Node
    {
        private readonly Func<bool> condition;

        public Condition(Func<bool> condition)
        {
            this.condition = condition;
        }

        public override NodeState Evaluate()
        {
            return condition() ? NodeState.Success : NodeState.Failure;
        }
    }

    /// <summary>动作节点：执行一个动作</summary>
    public class Action : Node
    {
        private readonly Func<NodeState> action;

        public Action(Func<NodeState> action)
        {
            this.action = action;
        }

        public override NodeState Evaluate()
        {
            return action();
        }
    }
}
```

### 3.3 敌人 AI 行为树示例

```csharp
using UnityEngine;

namespace BehaviorTree
{
    /// <summary>敌人 AI 行为树</summary>
    /*
     * 行为树结构:
     *
     *     Selector (根)
     *     ├── Sequence (战斗)
     *     │   ├── Condition: 玩家在攻击范围内?
     *     │   └── Action: 攻击玩家
     *     ├── Sequence (追击)
     *     │   ├── Condition: 发现玩家?
     *     │   └── Action: 追向玩家
     *     └── Sequence (巡逻)
     *         ├── Action: 移动到巡逻点
     *         └── Action: 等待
     */
    public class EnemyAI : MonoBehaviour
    {
        [Header("配置")]
        [SerializeField] private float detectionRange = 15f;
        [SerializeField] private float attackRange = 2f;
        [SerializeField] private float moveSpeed = 3f;
        [SerializeField] private Transform[] patrolPoints;

        [Header("引用")]
        [SerializeField] private Transform player;

        private Node root;
        private Vector3 currentPatrolTarget;
        private int patrolIndex = 0;
        private float waitTimer = 0;

        private void Start()
        {
            BuildBehaviorTree();
            if (patrolPoints.Length > 0)
                currentPatrolTarget = patrolPoints[0].position;
        }

        private void Update()
        {
            root?.Evaluate();
        }

        private void BuildBehaviorTree()
        {
            root = new Selector(new List<Node>
            {
                // 战斗序列
                new Sequence(new List<Node>
                {
                    new Condition(() => DistanceToPlayer() <= attackRange),
                    new Action(() =>
                    {
                        AttackPlayer();
                        return NodeState.Success;
                    })
                }),

                // 追击序列
                new Sequence(new List<Node>
                {
                    new Condition(() => CanSeePlayer()),
                    new Action(() =>
                    {
                        ChasePlayer();
                        return NodeState.Running;
                    })
                }),

                // 巡逻序列
                new Sequence(new List<Node>
                {
                    new Action(() => Patrol()),
                    new Action(() => WaitAtPatrolPoint())
                })
            });
        }

        // ===== AI 行为方法 =====

        private float DistanceToPlayer()
        {
            return Vector3.Distance(transform.position, player.position);
        }

        private bool CanSeePlayer()
        {
            float dist = DistanceToPlayer();
            if (dist > detectionRange) return false;

            // 视线检测
            Vector3 dirToPlayer = (player.position - transform.position).normalized;
            float dot = Vector3.Dot(transform.forward, dirToPlayer);
            return dot > 0.5f; // 视野锥角约 120 度
        }

        private void AttackPlayer()
        {
            // 面向玩家
            transform.LookAt(player.position);
            Debug.Log("攻击玩家!");
            // 触发攻击动画/伤害逻辑...
        }

        private void ChasePlayer()
        {
            Vector3 dir = (player.position - transform.position).normalized;
            transform.position += dir * moveSpeed * Time.deltaTime;
            transform.LookAt(player.position);
        }

        private NodeState Patrol()
        {
            if (Vector3.Distance(transform.position, currentPatrolTarget) < 0.5f)
            {
                return NodeState.Success; // 到达巡逻点
            }

            Vector3 dir = (currentPatrolTarget - transform.position).normalized;
            transform.position += dir * moveSpeed * Time.deltaTime;
            transform.LookAt(currentPatrolTarget);
            return NodeState.Running;
        }

        private NodeState WaitAtPatrolPoint()
        {
            waitTimer += Time.deltaTime;
            if (waitTimer >= 2f)
            {
                waitTimer = 0;
                patrolIndex = (patrolIndex + 1) % patrolPoints.Length;
                currentPatrolTarget = patrolPoints[patrolIndex].position;
                return NodeState.Success;
            }
            return NodeState.Running;
        }
    }
}
```

## 4. 事件系统

### 4.1 为什么需要事件系统

```
传统方式（紧耦合）:
  玩家击杀 → 直接调用 UI.ShowScore()、AudioManager.Play()、Achievement.Check()、Analytics.Send()

事件驱动（松耦合）:
  玩家击杀 → 发布 "OnEnemyKilled" 事件
  UI、AudioManager、Achievement、Analytics 各自订阅该事件

优势: 发布者不需要知道谁在监听，新增监听者无需修改发布者代码。
```

### 4.2 类型安全的事件系统

```csharp
using System;
using System.Collections.Generic;

namespace EventSystem
{
    /// <summary>游戏事件基接口</summary>
    public interface IGameEvent { }

    // 具体事件定义
    public struct EnemyKilledEvent : IGameEvent
    {
        public string EnemyId;
        public int Points;
        public UnityEngine.Vector3 Position;
    }

    public struct PlayerDamagedEvent : IGameEvent
    {
        public float Damage;
        public float CurrentHealth;
        public string Source;
    }

    public struct LevelCompletedEvent : IGameEvent
    {
        public int LevelId;
        public float TimeTaken;
        public int Score;
    }

    public struct ItemPickupEvent : IGameEvent
    {
        public string ItemId;
        public int Quantity;
    }

    /// <summary>事件处理器委托</summary>
    /// <typeparam name="T">事件类型</typeparam>
    public delegate void EventHandler<T>(T eventData) where T : IGameEvent;

    /// <summary>
    /// 类型安全的事件总线
    /// </summary>
    public static class EventBus
    {
        private static readonly Dictionary<Type, Delegate> handlers = new();

        /// <summary>订阅事件</summary>
        public static void Subscribe<T>(EventHandler<T> handler) where T : IGameEvent
        {
            var type = typeof(T);
            if (handlers.TryGetValue(type, out var existing))
            {
                handlers[type] = Delegate.Combine(existing, handler);
            }
            else
            {
                handlers[type] = handler;
            }
        }

        /// <summary>取消订阅</summary>
        public static void Unsubscribe<T>(EventHandler<T> handler) where T : IGameEvent
        {
            var type = typeof(T);
            if (handlers.TryGetValue(type, out var existing))
            {
                var newDelegate = Delegate.Remove(existing, handler);
                if (newDelegate == null)
                {
                    handlers.Remove(type);
                }
                else
                {
                    handlers[type] = newDelegate;
                }
            }
        }

        /// <summary>发布事件</summary>
        public static void Publish<T>(T eventData) where T : IGameEvent
        {
            if (handlers.TryGetValue(typeof(T), out var existing))
            {
                ((EventHandler<T>)existing)?.Invoke(eventData);
            }
        }

        /// <summary>清空所有订阅（场景切换时使用）</summary>
        public static void Clear()
        {
            handlers.Clear();
        }
    }
}
```

### 4.3 使用示例

```csharp
using UnityEngine;
using EventSystem;

// ===== 事件发布者 =====
public class CombatSystem : MonoBehaviour
{
    private void KillEnemy(string enemyId, int points, Vector3 position)
    {
        // 执行击杀逻辑...

        // 发布事件（不需要知道谁在监听）
        EventBus.Publish(new EnemyKilledEvent
        {
            EnemyId = enemyId,
            Points = points,
            Position = position
        });
    }
}

// ===== 事件订阅者：UI =====
public class ScoreUI : MonoBehaviour
{
    private int totalScore = 0;

    private void OnEnable()
    {
        EventBus.Subscribe<EnemyKilledEvent>(OnEnemyKilled);
    }

    private void OnDisable()
    {
        EventBus.Unsubscribe<EnemyKilledEvent>(OnEnemyKilled);
    }

    private void OnEnemyKilled(EnemyKilledEvent e)
    {
        totalScore += e.Points;
        UpdateScoreDisplay();
    }

    private void UpdateScoreDisplay()
    {
        Debug.Log($"Total Score: {totalScore}");
    }
}

// ===== 事件订阅者：成就系统 =====
public class AchievementSystem : MonoBehaviour
{
    private int killCount = 0;

    private void OnEnable()
    {
        EventBus.Subscribe<EnemyKilledEvent>(OnEnemyKilled);
    }

    private void OnDisable()
    {
        EventBus.Unsubscribe<EnemyKilledEvent>(OnEnemyKilled);
    }

    private void OnEnemyKilled(EnemyKilledEvent e)
    {
        killCount++;
        if (killCount == 100)
        {
            UnlockAchievement("Centurion");
        }
        if (killCount == 1000)
        {
            UnlockAchievement("Exterminator");
        }
    }

    private void UnlockAchievement(string name)
    {
        Debug.Log($"Achievement Unlocked: {name}!");
    }
}

// ===== 事件订阅者：音效系统 =====
public class AudioSystem : MonoBehaviour
{
    private void OnEnable()
    {
        EventBus.Subscribe<EnemyKilledEvent>(OnEnemyKilled);
        EventBus.Subscribe<PlayerDamagedEvent>(OnPlayerDamaged);
    }

    private void OnDisable()
    {
        EventBus.Unsubscribe<EnemyKilledEvent>(OnEnemyKilled);
        EventBus.Unsubscribe<PlayerDamagedEvent>(OnPlayerDamaged);
    }

    private void OnEnemyKilled(EnemyKilledEvent e)
    {
        // 在敌人位置播放死亡音效
        AudioSource.PlayClipAtPoint(deathClip, e.Position);
    }

    private void OnPlayerDamaged(PlayerDamagedEvent e)
    {
        AudioSource.PlayClipAtPoint(hurtClip, transform.position);
    }

    [SerializeField] private AudioClip deathClip;
    [SerializeField] private AudioClip hurtClip;
}
```

### 4.4 事件系统最佳实践

- ✅ 使用 `struct` 定义事件（避免 GC 分配）
- ✅ 在 `OnEnable` 订阅，`OnDisable` 取消订阅（配对使用）
- ✅ 场景切换时调用 `EventBus.Clear()` 防止跨场景泄漏
- ✅ 事件数据应该是只读的（发布后不应被修改）
- ❌ 避免在事件处理中再次发布事件（可能引发无限循环）
- ❌ 避免在事件处理中做重计算（保持响应快速）

## 5. 对象池模式

### 5.1 对象池 vs 直接实例化

```
直接实例化:
  Create → Instantiate → [使用] → Destroy
  每次都有 GC 开销 + 内存碎片 + CPU 开销

对象池:
  Create → Get (从池中取出) → [使用] → Return (归还到池中)
  初始化时预创建，运行时零分配
```

### 5.2 接口驱动的对象池

```csharp
using System;
using System.Collections.Generic;
using UnityEngine;

namespace ObjectPoolPattern
{
    /// <summary>可池化对象接口</summary>
    public interface IPoolable
    {
        void OnGetFromPool();
        void OnReturnToPool();
    }

    /// <summary>
    /// 通用对象池（非 Unity 专用，可用于纯 C# 对象）
    /// </summary>
    public class GenericObjectPool<T> where T : class, IPoolable, new()
    {
        private readonly Stack<T> pool = new();
        private readonly int maxSize;
        private int totalCreated;

        public int PooledCount => pool.Count;
        public int TotalCreated => totalCreated;
        public int ActiveCount => totalCreated - pool.Count;

        public GenericObjectPool(int initialSize = 0, int maxSize = int.MaxValue)
        {
            this.maxSize = maxSize;
            for (int i = 0; i < initialSize; i++)
            {
                var obj = new T();
                totalCreated++;
                obj.OnReturnToPool();
                pool.Push(obj);
            }
        }

        public T Get()
        {
            T obj;
            if (pool.Count > 0)
            {
                obj = pool.Pop();
            }
            else
            {
                obj = new T();
                totalCreated++;
            }
            obj.OnGetFromPool();
            return obj;
        }

        public void Return(T obj)
        {
            if (obj == null) return;

            obj.OnReturnToPool();

            if (pool.Count < maxSize)
            {
                pool.Push(obj);
            }
            // 超过 maxSize 则丢弃，等待 GC 回收
        }
    }

    // ===== 示例：可池化的伤害信息 =====

    public class DamageInfo : IPoolable
    {
        public float Amount;
        public string Source;
        public string Target;
        public bool IsCritical;

        public void OnGetFromPool()
        {
            // 重置状态
            Amount = 0;
            Source = null;
            Target = null;
            IsCritical = false;
        }

        public void OnReturnToPool()
        {
            // 清理引用
            Source = null;
            Target = null;
        }

        public override string ToString()
        {
            return $"{Target} takes {Amount} damage from {Source}" +
                   (IsCritical ? " (CRITICAL!)" : "");
        }
    }

    // 使用示例
    public class CombatManager
    {
        private readonly GenericObjectPool<DamageInfo> damagePool = new(initialSize: 20);

        public void DealDamage(string target, string source, float amount, bool critical)
        {
            DamageInfo info = damagePool.Get();
            info.Target = target;
            info.Source = source;
            info.Amount = amount;
            info.IsCritical = critical;

            // 处理伤害逻辑
            UnityEngine.Debug.Log(info);

            // 归还到池
            damagePool.Return(info);
        }
    }
}
```

## 6. 模式对比与选择

### 6.1 何时使用哪种模式

| 模式 | 适用场景 | 不适用场景 |
|------|----------|------------|
| **ECS** | 大量同类实体（万级单位）、需要高性能批量处理、DOD 优化 | 小型项目、原型开发（过于复杂） |
| **FSM** | 简单状态切换（角色控制、UI 流程、游戏状态） | 状态爆炸（>20 个状态）、复杂嵌套 |
| **行为树** | 复杂 AI 决策、需要层次化组合行为 | 简单的线性逻辑（杀鸡用牛刀） |
| **事件系统** | 松耦合的跨系统通信、全局事件广播 | 高频每帧通信（性能开销）、紧密关联的逻辑 |
| **对象池** | 高频创建/销毁的对象（子弹、粒子、敌人） | 长期存在的对象（角色、关卡） |

### 6.2 模式组合使用

```
真实项目中的典型组合:

游戏框架:
  ├── ECS: 管理游戏世界中的所有实体
  ├── FSM: 管理游戏全局状态（菜单→加载→游戏中→暂停→结束）
  ├── 行为树: 敌人 AI 决策
  ├── 事件系统: 跨系统通信（UI ↔ 战斗 ↔ 成就 ↔ 音效）
  └── 对象池: 子弹、粒子、伤害信息等高频对象
```
