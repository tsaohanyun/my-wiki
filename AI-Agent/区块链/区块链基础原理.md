## 1. 区块结构

每个区块由 **区块头 (Block Header)** 和 **区块体 (Block Body)** 组成。

### 区块头字段

| 字段 | 大小 | 说明 |
|------|------|------|
| `Version` | 4 bytes | 区块版本号 |
| `Previous Hash` | 32 bytes | 前一区块的哈希值 |
| `Merkle Root` | 32 bytes | 区块中所有交易的默克尔根 |
| `Timestamp` | 4 bytes | 区块生成时间戳 |
| `Difficulty Target` | 4 bytes | 工作量证明难度目标 |
| `Nonce` | 4 bytes | 随机数，用于PoW |

### Python 实现区块结构

```python
import hashlib
import json
import time

class Block:
    def __init__(self, index, transactions, previous_hash, timestamp=None):
        self.index = index
        self.transactions = transactions
        self.previous_hash = previous_hash
        self.timestamp = timestamp or time.time()
        self.nonce = 0
        self.hash = self.calculate_hash()

    def calculate_hash(self):
        block_data = json.dumps({
            "index": self.index,
            "transactions": self.transactions,
            "previous_hash": self.previous_hash,
            "timestamp": self.timestamp,
            "nonce": self.nonce,
        }, sort_keys=True)
        return hashlib.sha256(block_data.encode()).hexdigest()

    def mine_block(self, difficulty):
        target = "0" * difficulty
        while self.hash[:difficulty] != target:
            self.nonce += 1
            self.hash = self.calculate_hash()
        print(f"Block mined: {self.hash} (nonce={self.nonce})")


class Blockchain:
    def __init__(self):
        self.chain = [self.create_genesis_block()]
        self.difficulty = 4

    def create_genesis_block(self):
        return Block(0, "Genesis Block", "0")

    def get_latest_block(self):
        return self.chain[-1]

    def add_block(self, new_block):
        new_block.previous_hash = self.get_latest_block().hash
        new_block.mine_block(self.difficulty)
        self.chain.append(new_block)

    def is_chain_valid(self):
        for i in range(1, len(self.chain)):
            current = self.chain[i]
            previous = self.chain[i - 1]
            if current.hash != current.calculate_hash():
                return False
            if current.previous_hash != previous.hash:
                return False
        return True


# === 运行示例 ===
blockchain = Blockchain()
blockchain.add_block(Block(1, {"amount": 50}, ""))
blockchain.add_block(Block(2, {"amount": 120}, ""))
print(f"Blockchain valid? {blockchain.is_chain_valid()}")
```

## 2. 共识机制

共识机制是区块链网络中各节点就账本状态达成一致的算法。

### 2.1 工作量证明 (Proof of Work, PoW)

- **代表链**: Bitcoin, Litecoin, Dogecoin
- **原理**: 矿工通过消耗计算资源寻找满足难度的哈希值
- **优点**: 安全性高，经过长期验证
- **缺点**: 能耗高，吞吐量低（~7 TPS for Bitcoin）

```python
# 简化版 PoW 算法
import hashlib

def proof_of_work(block_data, difficulty=4):
    nonce = 0
    target = "0" * difficulty
    while True:
        data = f"{block_data}{nonce}".encode()
        hash_result = hashlib.sha256(data).hexdigest()
        if hash_result[:difficulty] == target:
            return nonce, hash_result
        nonce += 1

nonce, hash_result = proof_of_work("block_42_data")
print(f"Found nonce: {nonce}, hash: {hash_result}")
```

### 2.2 权益证明 (Proof of Stake, PoS)

- **代表链**: Ethereum 2.0 (The Merge), Cardano, Algorand
- **原理**: 验证者根据持有的代币数量和持币时间被选中出块
- **优点**: 能耗低（降低 99.95%），可扩展性更好
- **缺点**: 富者愈富问题，需要质押锁仓

```python
import random

class PoSValidator:
    def __init__(self, address, stake):
        self.address = address
        self.stake = stake

class PoSBlockchain:
    def __init__(self):
        self.validators: list[PoSValidator] = []

    def add_validator(self, address, stake):
        self.validators.append(PoSValidator(address, stake))

    def select_validator(self):
        """按质押比例加权随机选择出块节点"""
        total_stake = sum(v.stake for v in self.validators)
        # 按质押量加权随机
        weights = [v.stake / total_stake for v in self.validators]
        selected = random.choices(self.validators, weights=weights, k=1)[0]
        return selected

    def slash(self, validator_address, penalty):
        """惩罚恶意验证者（Slashing）"""
        for v in self.validators:
            if v.address == validator_address:
                v.stake = max(0, v.stake - penalty)
                print(f"Slashed {validator_address}, stake now: {v.stake}")


# === 运行示例 ===
pos = PoSBlockchain()
pos.add_validator("0xAlice", 1000)
pos.add_validator("0xBob", 500)
pos.add_validator("0xCharlie", 200)
validator = pos.select_validator()
print(f"Selected validator: {validator.address}")
```

### 2.3 委托权益证明 (DPoS)

- **代表链**: EOS, TRON, Cosmos
- **原理**: 代币持有者投票选举出少量委托人（超级节点）来出块
- **优点**: 高吞吐量（数千 TPS），快速确认
- **缺点**: 中心化风险，超级节点可能合谋

```python
class DPoSDelegate:
    def __init__(self, name, votes):
        self.name = name
        self.votes = votes

class DPoSSystem:
    def __init__(self, max_delegates=21):
        self.max_delegates = max_delegates
        self.candidates: list[DPoSDelegate] = []
        self.active_delegates: list[DPoSDelegate] = []

    def register_candidate(self, name):
        self.candidates.append(DPoSDelegate(name, 0))

    def vote(self, delegate_name, vote_power):
        for c in self.candidates:
            if c.name == delegate_name:
                c.votes += vote_power
                break

    def elect_delegates(self):
        """选出票数最高的 max_delegates 个节点"""
        self.candidates.sort(key=lambda x: x.votes, reverse=True)
        self.active_delegates = self.candidates[:self.max_delegates]
        return [d.name for d in self.active_delegates]

    def produce_block(self, round_number):
        """轮流出块"""
        if not self.active_delegates:
            self.elect_delegates()
        producer = self.active_delegates[round_number % len(self.active_delegates)]
        print(f"[Round {round_number}] Block produced by: {producer.name}")
        return producer


# === 运行示例 ===
dpos = DPoSSystem(max_delegates=5)
dpos.register_candidate("Node-A")
dpos.register_candidate("Node-B")
dpos.register_candidate("Node-C")
dpos.register_candidate("Node-D")
dpos.vote("Node-A", 5000)
dpos.vote("Node-B", 3000)
dpos.vote("Node-C", 2000)
delegates = dpos.elect_delegates()
print(f"Active delegates: {delegates}")
dpos.produce_block(1)
dpos.produce_block(2)
```

## 3. 默克尔树

默克尔树（Merkle Tree）是一种二叉哈希树，用于高效验证大规模数据完整性。

```
                Root Hash: H(H12 + H34)
                /                      \
        H12 = H(H1+H2)         H34 = H(H3+H4)
           /       \              /       \
      H1=H(Tx1) H2=H(Tx2)  H3=H(Tx3) H4=H(Tx4)
```

### Python 实现

```python
import hashlib

def hash_data(data: str) -> str:
    return hashlib.sha256(data.encode()).hexdigest()

class MerkleTree:
    def __init__(self, leaves: list[str]):
        if not leaves:
            raise ValueError("Cannot create Merkle tree with no leaves")
        self.leaves = [hash_data(leaf) for leaf in leaves]
        self.tree = self.build_tree(self.leaves)

    def build_tree(self, leaves: list[str]) -> list[list[str]]:
        if len(leaves) == 1:
            return [leaves]
        # 补齐奇数叶子节点（复制最后一个）
        if len(leaves) % 2 != 0:
            leaves.append(leaves[-1])
        next_level = []
        for i in range(0, len(leaves), 2):
            combined = hash_data(leaves[i] + leaves[i + 1])
            next_level.append(combined)
        return [leaves] + self.build_tree(next_level)

    @property
    def root(self) -> str:
        return self.tree[-1][0]

    def get_proof(self, index: int) -> list[dict]:
        """生成默克尔证明路径 (Merkle Proof)"""
        proof = []
        for level in self.tree[:-1]:  # 排除根层
            if index % 2 == 0:
                # 当前节点是左子节点，需要右兄弟节点
                sibling_index = index + 1
                direction = "right"
            else:
                sibling_index = index - 1
                direction = "left"
            if sibling_index < len(level):
                proof.append({"hash": level[sibling_index], "direction": direction})
            index = index // 2
        return proof

    @staticmethod
    def verify_proof(leaf: str, proof: list[dict], root: str) -> bool:
        """验证默克尔证明"""
        computed = hash_data(leaf)
        for step in proof:
            if step["direction"] == "right":
                computed = hash_data(computed + step["hash"])
            else:
                computed = hash_data(step["hash"] + computed)
        return computed == root


# === 运行示例 ===
transactions = ["tx1", "tx2", "tx3", "tx4"]
tree = MerkleTree(transactions)
print(f"Merkle Root: {tree.root}")

# 验证 tx2 是否在树中
proof = tree.get_proof(1)  # tx2 的索引是 1
is_valid = MerkleTree.verify_proof("tx2", proof, tree.root)
print(f"tx2 in tree? {is_valid}")

# 篡改验证
is_fake = MerkleTree.verify_proof("tx_fake", proof, tree.root)
print(f"tx_fake in tree? {is_fake}")
```

### 默克尔树在区块链中的应用

1. **SPV (Simple Payment Verification)**: 轻节点只需下载区块头和默克尔证明即可验证交易
2. **交易存在性证明**: O(log n) 时间复杂度验证交易是否在区块中
3. **以太坊状态树**: 使用 Merkle Patricia Trie 存储全局状态

## 4. 智能合约

智能合约是部署在区块链上的自动执行程序，当预定条件满足时自动执行条款。

### Solidity 示例：简单投票合约

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.24;

/// @title 去中心化投票合约
/// @author Wiki Example
contract Voting {
    struct Candidate {
        uint256 id;
        string name;
        uint256 voteCount;
    }

    mapping(uint256 => Candidate) public candidates;
    mapping(address => bool) public hasVoted;
    uint256 public candidatesCount;
    address public owner;

    event Voted(address indexed voter, uint256 indexed candidateId);

    modifier onlyOwner() {
        require(msg.sender == owner, "Only owner can call this");
        _;
    }

    constructor() {
        owner = msg.sender;
    }

    function addCandidate(string memory _name) public onlyOwner {
        candidatesCount++;
        candidates[candidatesCount] = Candidate(candidatesCount, _name, 0);
    }

    function vote(uint256 _candidateId) public {
        require(!hasVoted[msg.sender], "You have already voted");
        require(_candidateId > 0 && _candidateId <= candidatesCount, "Invalid candidate");
        hasVoted[msg.sender] = true;
        candidates[_candidateId].voteCount++;
        emit Voted(msg.sender, _candidateId);
    }

    function getWinner() public view returns (string memory name, uint256 votes) {
        uint256 maxVotes = 0;
        uint256 winnerId = 0;
        for (uint256 i = 1; i <= candidatesCount; i++) {
            if (candidates[i].voteCount > maxVotes) {
                maxVotes = candidates[i].voteCount;
                winnerId = i;
            }
        }
        return (candidates[winnerId].name, maxVotes);
    }
}
```

### 智能合约生命周期

```
编写合约 (Solidity) → 编译 → 部署到链上 → 用户交互 → (可选) 升级
```

### Web3.py 调用智能合约

```python
from web3 import Web3

# 连接到以太坊节点
w3 = Web3(Web3.HTTPProvider("http://localhost:8545"))
assert w3.is_connected(), "Failed to connect"

# 合约 ABI 和地址
contract_abi = '''[{"inputs":[],"name":"candidatesCount","outputs":[{"type":"uint256"}],"stateMutability":"view","type":"function"}, ...]'''
contract_address = "0x1234...abcd"

contract = w3.eth.contract(address=contract_address, abi=contract_abi)

# 读取数据
count = contract.functions.candidatesCount().call()
print(f"Total candidates: {count}")

# 发送交易
tx = contract.functions.vote(1).build_transaction({
    'from': w3.eth.accounts[0],
    'nonce': w3.eth.get_transaction_count(w3.eth.accounts[0]),
    'gas': 200000,
    'gasPrice': w3.eth.gas_price,
})
signed_tx = w3.eth.account.sign_transaction(tx, private_key="0x...")
tx_hash = w3.eth.send_raw_transaction(signed_tx.raw_transaction)
receipt = w3.eth.wait_for_transaction_receipt(tx_hash)
print(f"Vote tx confirmed: {tx_hash.hex()}")
```

## 5. 最佳实践

### 安全最佳实践

1. **使用 `SafeMath`**（Solidity 0.8+ 已内置溢出保护）
2. **重入锁防重入攻击**:

```solidity
bool private locked;
modifier noReentrancy() {
    require(!locked, "Reentrant call");
    locked = true;
    _;
    locked = false;
}
```

3. **使用 `Checks-Effects-Interactions` 模式**: 先检查条件 → 再更新状态 → 最后外部调用
4. **权限控制**: 使用 `Ownable` / `AccessControl`，关键操作加 `onlyOwner`
5. **整数溢出**: Solidity 0.8+ 默认检查溢出；低版本用 `SafeMath`
6. **审计**: 主网上线前必须经过专业安全审计（CertiK, ConsenSys Diligence, OpenZeppelin）
7. **升级模式**: 使用代理合约模式实现可升级性

### 开发最佳实践

1. **测试覆盖**: 单元测试 + 集成测试 + 模糊测试
2. **Gas 优化**: 减少 SLOAD/SSTORE，使用 `calldata` 代替 `memory`，批处理操作
3. **事件日志**: 所有状态变更都应发出事件，便于链下索引
4. **NatSpec 注释**: 为所有 public/external 函数添加文档
5. **版本锁定**: 使用确定性的合约版本和依赖
