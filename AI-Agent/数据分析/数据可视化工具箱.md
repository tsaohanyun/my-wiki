# 数据可视化工具箱

## 概述

本工具箱提供数据可视化的常用工具和代码模板。

## 1. Matplotlib基础

### 折线图

```python
import matplotlib.pyplot as plt

# 数据
x = [1, 2, 3, 4, 5]
y = [2, 4, 6, 8, 10]

# 绘图
plt.figure(figsize=(10, 6))
plt.plot(x, y, marker='o', linewidth=2)
plt.title('折线图示例')
plt.xlabel('X轴')
plt.ylabel('Y轴')
plt.grid(True)
plt.show()
```

### 柱状图

```python
# 数据
categories = ['A', 'B', 'C', 'D']
values = [10, 20, 15, 25]

# 绘图
plt.figure(figsize=(10, 6))
plt.bar(categories, values, color=['#FF6B6B', '#4ECDC4', '#45B7D1', '#96CEB4'])
plt.title('柱状图示例')
plt.xlabel('类别')
plt.ylabel('值')
plt.show()
```

### 饼图

```python
# 数据
labels = ['A', 'B', 'C', 'D']
sizes = [30, 25, 20, 25]

# 绘图
plt.figure(figsize=(8, 8))
plt.pie(sizes, labels=labels, autopct='%1.1f%%', startangle=90)
plt.title('饼图示例')
plt.show()
```

### 散点图

```python
import numpy as np

# 数据
x = np.random.randn(100)
y = np.random.randn(100)

# 绘图
plt.figure(figsize=(10, 6))
plt.scatter(x, y, alpha=0.5, c='blue', s=50)
plt.title('散点图示例')
plt.xlabel('X轴')
plt.ylabel('Y轴')
plt.show()
```

## 2. Plotly交互式图表

### 折线图

```python
import plotly.express as px
import pandas as pd

# 数据
df = pd.DataFrame({
    'x': [1, 2, 3, 4, 5],
    'y': [2, 4, 6, 8, 10]
})

# 绘图
fig = px.line(df, x='x', y='y', title='交互式折线图')
fig.show()
```

### 柱状图

```python
# 数据
df = pd.DataFrame({
    'category': ['A', 'B', 'C', 'D'],
    'value': [10, 20, 15, 25]
})

# 绘图
fig = px.bar(df, x='category', y='value', title='交互式柱状图')
fig.show()
```

### 散点图

```python
# 数据
df = pd.DataFrame({
    'x': np.random.randn(100),
    'y': np.random.randn(100),
    'size': np.random.rand(100) * 100
})

# 绘图
fig = px.scatter(df, x='x', y='y', size='size', title='交互式散点图')
fig.show()
```

## 3. Seaborn统计图表

### 热力图

```python
import seaborn as sns

# 数据
data = np.random.rand(10, 10)

# 绘图
plt.figure(figsize=(10, 8))
sns.heatmap(data, annot=True, cmap='YlOrRd')
plt.title('热力图示例')
plt.show()
```

### 箱线图

```python
# 数据
df = pd.DataFrame({
    'category': ['A'] * 50 + ['B'] * 50,
    'value': np.concatenate([np.random.randn(50), np.random.randn(50) + 2])
})

# 绘图
plt.figure(figsize=(10, 6))
sns.boxplot(x='category', y='value', data=df)
plt.title('箱线图示例')
plt.show()
```

## 4. 图表选择指南

### 根据数据类型选择

| 数据类型 | 推荐图表 | 适用场景 |
|----------|----------|----------|
| 时间序列 | 折线图 | 趋势变化 |
| 分类对比 | 柱状图 | 大小比较 |
| 占比关系 | 饼图 | 比例展示 |
| 相关性 | 散点图 | 关系分析 |
| 分布 | 直方图 | 频率分布 |

### 根据分析目的选择

| 分析目的 | 推荐图表 | 示例 |
|----------|----------|------|
| 趋势分析 | 折线图 | 销售趋势 |
| 对比分析 | 柱状图 | 产品对比 |
| 组成分析 | 饼图 | 市场份额 |
| 相关分析 | 散点图 | 价格vs销量 |
| 分布分析 | 直方图 | 用户年龄分布 |

## 5. 图表美化技巧

### 颜色方案

```python
# 使用预设颜色方案
colors = ['#FF6B6B', '#4ECDC4', '#45B7D1', '#96CEB4', '#FFEAA7']

# 使用渐变色
cmap = plt.cm.get_cmap('viridis')
```

### 字体设置

```python
# 设置中文字体
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False
```

### 图表布局

```python
# 多子图布局
fig, axes = plt.subplots(2, 2, figsize=(12, 10))

axes[0, 0].plot(x, y)
axes[0, 0].set_title('子图1')

axes[0, 1].bar(categories, values)
axes[0, 1].set_title('子图2')

axes[1, 0].pie(sizes, labels=labels)
axes[1, 0].set_title('子图3')

axes[1, 1].scatter(x, y)
axes[1, 1].set_title('子图4')

plt.tight_layout()
plt.show()
```
