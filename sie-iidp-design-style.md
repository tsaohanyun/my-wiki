# SIE-IIDP 网页设计风格

SIE-IIDP（工业互联网数据平台）是中国制造业 MES 系统的前端设计风格代表，以**#AC2A48**为品牌主色、**极高信息密度**、**功能主义美学**为核心特征的技术后台设计系统。

技术栈：Vue.js 2.x + Element UI + 自研 SMI Design 组件库 + Tech 微前端框架。

## 排版与尺寸

### 根字号缩放

**`html { font-size: 100px }`** — 这是最独特的设计决策。所有尺寸以 rem 为单位，1rem=100px，形成直观映射：
- `0.14rem = 14px`（正文）
- `0.32rem = 32px`（按钮高度）
- `0.12rem = 12px`（表单项间距）

### 按钮圆角

**2px** — 极其微小的圆角，近乎直角，这是该系统区别于 Apple Design（大圆角）和 Material Design（4-8px）的关键视觉标识。

## 布局架构

```
┌──────────────────────────────────────────┐
│ 顶部 Header (#3D4547, 60px)               │
│ [DIM] [☰]    [🏭租户] [中文▼] [🔔³] [👤▼] │
├────────┬─────────────────────────────────┤
│ 侧边栏 │ 面包屑导航 (红色文字)             │
│ (红色) ├─────────────────────────────────┤
│ 220px  │ 主表 — 核心列表                   │
│ SVG    │  [搜索栏] [查询] [重置]           │
│ 图标   │  [工具栏] 新增|批量操作|导出       │
│ (白色) │  ☑ 序号 编号 名称 ... 操作        │
│        │  分页                             │
│        ├─────────────────────────────────┤
│        │ 子表 — 关联详情（标签页切换）       │
│        │  [生产进度] [异常记录]             │

```

**鲜明特征：** 顶部深色Header (#3D4547, 60px) + 红色侧边栏 (#AC2A48, 220px) 双层布局；内容区采用**主子表上下结构**，主表列表在上、子表详情在下，点击主表行联动子表数据。

## 主子表上下布局（核心模式）

当页面包含**多个相关的核心信息区域**时（如订单列表 + 生产进度 + 异常记录），SIE-IIDP 采用**主子表上下布局**而非平级标签页切换。

### 布局规则

- **上方主表**：核心列表数据（搜索栏 + 工具栏 + 数据表格 + 分页），是页面主体
- **下方子表**：用标签页组织多个关联信息区域，标签页样式同通用标签页组件
- **行点击联动**：点击主表某行 → 该行 `.row-active` 高亮 → 子表自动更新该行对应数据
- **默认标签页**：子表默认显示第一个标签页
- **信息栏**：子表标签页下方显示当前选中记录的关键标识（如"当前订单：PO-001 - 法兰盘"）
- **空状态**：未选择主表行时，子表显示提示"请在上方列表中点击选择一条记录"

### 交互区分

主表有两种独立交互，互不干扰：
- **行点击**（非复选框/操作列区域）→ 联动子表，`.row-active` 高亮
- **复选框选中** → 批量操作，`.row-selected` 高亮
- **操作列按钮** → 执行对应操作

### 适用场景判断

| 场景 | 布局方式 |
|------|---------|
| 1个主列表 + N个关联详情 | **主子表上下布局** |
| 多个独立平级视图 | 顶部平级标签页切换 |

## 图标系统

- **格式：** 内联 SVG
- **尺寸：** 20×20 viewBox，17.6×17.6 裁剪区
- **风格：** 圆形徽章 + 工业主题内部图形
- **颜色：** 纯白填充（`fill="white"`），配合红色侧边栏形成高对比
- **主题：** 组件、质量、物资、生产、设备、计划、接口、工具等

| 组件 | 关键特征 |

## 设计哲学

- **企业优先：** 一切为数据展示和操作效率服务
- **品牌驱动：** #AC2A48深度嵌入每个组件
- **功能主义：** 极低圆角、最少阴影、无渐变装饰
- **高信息密度：** 表格多列、表单多字段、紧凑留白
- **结构化层次：** 侧边栏 → 面包屑 → 标签页 → 内容区
- **主子表联动：** 多核心信息时用上下布局，主表选中行联动子表
