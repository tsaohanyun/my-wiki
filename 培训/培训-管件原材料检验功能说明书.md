## 功能概述

本开发项实现管件原材料检验的信息化管理，覆盖从物料到货、检验申请、检验执行、结果判定到合格入库/不合格处置的全流程。系统通过结构化表单与工作流引擎，规范检验操作，实现检验数据的电子化记录与追溯。

### 功能目标

- 规范原材料检验流程，减少人为操作差异
- 实现检验数据电子化管理，支持历史追溯
- 自动关联采购订单与供应商信息，提升检验效率
- 不合格品处置流程标准化，降低质量风险
- 生成检验报告，为供应商评价提供数据支撑

## 业务流程

### 整体业务流程图

```
物料到货 → 登记检验申请 → 分配检验员 → 执行检验 → 结果录入
    → 判定合格 → 合格入库 → 流程结束
    → 判定不合格 → 不合格处置 → 让步接收 / 退货 / 报废 → 流程结束
```

### 流程节点说明

1. **物料到货**：供应商送货，仓库收货登记，生成待检验记录
2. **检验申请**：系统自动生成或手动创建检验申请单，关联采购订单号、物料信息、供应商信息
3. **分配检验员**：根据物料类别与检验员排班，自动或手动分配检验任务
4. **执行检验**：检验员按检验标准执行外观检查、尺寸测量、性能测试等
5. **结果录入**：将各项检验指标数据录入系统，支持附件上传（照片、检测报告等）
6. **结果判定**：系统根据检验标准自动判定或由质检人员手动判定
7. **合格入库**：合格物料转为可用库存，更新库存状态
8. **不合格处置**：不合格物料进入处置流程，根据处置意见执行相应操作

## 检验流程

### 检验类型

| 检验类型 | 说明 | 触发条件 |
|----------|------|----------|
| 来料检验 | 对到货原材料进行质量检验 | 采购到货登记 |
| 让步检验 | 对不合格品进行让步接收评估 | 不合格处置环节 |
| 复检 | 对上次检验结果有异议时重新检验 | 手动发起 |

### 检验项目

管件原材料检验通常包含以下项目：

#### 外观检查

- 表面是否有裂纹、气孔、夹渣等缺陷
- 表面锈蚀、油污情况
- 管端切割面平整度
- 标识标记清晰度与完整性

#### 尺寸测量

- 外径（OD）测量
- 内径（ID）测量
- 壁厚测量
- 管长测量
- 弯曲度测量

#### 性能测试

- 化学成分分析（C、Si、Mn、P、S 等）
- 机械性能测试（抗拉强度、屈服强度、延伸率）
- 硬度测试
- 水压试验（如适用）

### 检验标准与判定规则

- 每个检验项目设定标准值与容差范围
- 系统支持配置判定规则：关键项不合格则整批判不合格；一般项按不合格率判定
- 检验结果分为：合格、不合格、让步接收

### 检验结果处理

| 判定结果 | 后续操作 | 责任人 |
|----------|----------|--------|
| 合格 | 转入可用库存 | 仓库管理员 |
| 不合格-退货 | 通知采购，安排退货 | 采购员 |
| 不合格-让步接收 | 提交让步申请，审批后放行 | 质量经理 |
| 不合格-报废 | 走报废流程，系统扣减库存 | 仓库管理员 |

## 权限与角色

| 角色 | 权限范围 |
|------|----------|
| 检验员 | 创建检验记录、录入检验数据、上传附件 |
| 质量主管 | 审核检验结果、判定不合格品处置方式 |
| 质量经理 | 审批让步接收、查看检验统计报表 |
| 仓库管理员 | 查看检验结论、执行入库或退货操作 |
| 采购员 | 查看供应商来料检验结果 |

---

## 报表与统计

系统支持以下统计报表：

- **来料检验日报/月报**：按日期汇总检验批次数量、合格率
- **供应商质量统计**：按供应商汇总不合格率，支持排名与趋势分析
- **物料质量趋势**：按物料类别统计质量波动，辅助供应商评价
- **检验员工作量统计**：按检验员汇总检验任务完成情况

## 注意事项

1. 检验标准变更时需同步更新系统配置，确保判定规则与最新标准一致
2. 关键检验项的判定结果不可修改，如需修正需走变更审批流程
3. 不合格品的让步接收需经质量经理审批，系统自动记录审批链路
4. 检验附件支持图片、PDF、Excel 等格式，单文件大小限制 10MB
5. 历史检验数据保留不少于 3 年，支持按批次号、供应商、日期等多维度查询
