# 飞书 Aily 智能体平台 — 培训指南

> 官方帮助文档：https://aily.feishu.cn/doc

## 1. 平台概述

飞书 Aily（原名"飞书智能伙伴创建平台"）是飞书旗下企业级 AI 智能体平台，围绕大语言模型（LLM）构建。

### 核心能力

| 能力维度 | 说明 |
|---------|------|
| AI 技能编排 | 可视化配置智能体的技能、工具调用链 |
| 知识数据处理 | 接入企业文档、数据库、业务系统等多源知识 |
| 效果调优 | 持续优化提示词、模型选择、知识检索策略 |
| 持续运营 | 数据看板、使用分析、版本管理 |

### 信息来源

- 飞书云文档、聊天记录、妙记
- 图片、PDF、Excel 等多种格式文件
- 数据库、业务系统等第三方系统数据
- 互联网公开信息

### 覆盖场景

分析报告、产品原型图、数据图表与洞察、云文档/多维表格、思维导图/流程图、轻量级网页应用

## 2. 产品矩阵与选型

飞书 Aily 平台包含三款产品，面向不同使用场景：

| 产品 | 定位 | 面向人群 |
|------|------|---------|
| **aily 智能伙伴 / 专业版** | 个人 AI 助手，侧重长期陪跑与代办执行 | 员工个人 |
| **aily 自定义智能体** | 一个 Agent 服务一群人的企业级团队平台（1:N） | 团队、部门、组织 |
| **飞书 OpenClaw（妙搭）** | 极致个性化的开源智能体框架，自由度最高 | AI 极客、深度定制需求 |

### 选型口诀

- 🎁 "AI 给我个人用" → 选 **aily 智能伙伴 / 专业版**
- 👥 "一个 Agent 给团队用" → 选 **aily 自定义智能体**
- 🔧 "完全按我的方式定制" → 选 **飞书 OpenClaw**

详见 [[concepts/飞书Aily-自定义智能体配置|自定义智能体配置指南]]

## 3. 智能伙伴（开箱即用）

智能伙伴常驻飞书联系人，零配置门槛，核心功能：

- **技能（Skill）**：知识检索、内容生成、数据分析、工具调用、代码辅助
- **定时任务**：定时执行任务，如每日待办推送、周报生成
- **知识检索范围**：配置检索哪些文档/知识库
- **指令模板**：预设常用指令一键触发
- **模型管理**：速度优先型/效果优先型/多模态型可选

## 4. 自定义智能体（企业级定制）

详见 [[concepts/飞书Aily-自定义智能体配置|自定义智能体配置指南]]

核心配置项：人设与 Prompt → 知识空间 → 技能与工具（MCP） → 发布渠道

## 5. 工作流应用

适用于流程逻辑复杂、节点控制严格的业务需求（AI 审批、AI 质检等）：

- 多步骤判断与条件执行
- 多系统数据集成与响应
- 严格的流程治理（审批节点、人工介入、异常处理）
- AI Agent 节点嵌入

### 与多维表格自动化的关系

| 特性 | 多维表格自动化 | 工作流应用 |
|------|-------------|-----------|
| 复杂度 | 简单触发+动作 | 完整应用逻辑 |
| AI 能力 | 有限 | 深度 AI 编排 |
| 适用场景 | 数据变更触发 | 企业级应用 |

## 6. MCP 工具集成

MCP（Model Context Protocol）是标准化协议，让 AI 智能体调用外部业务系统 API。

### 飞书官方 MCP 工具

| 工具 | 功能 |
|------|------|
| 飞书云文档 MCP | 创建、编辑、搜索文档 |
| 多维表格 MCP | 读写表格数据、触发自动化 |
| 飞书消息 MCP | 发送消息、创建群聊 |
| 日历 MCP | 查询/创建日程 |
| 审批 MCP | 发起/查询审批流程 |

### MCP 最佳实践

- 接口描述要详细（AI 靠描述决定何时调用）
- 参数校验要严格（防止 AI 传入非法参数）
- 返回格式要规范（JSON，含错误码）
- 权限最小化（只暴露必要的读写权限）
