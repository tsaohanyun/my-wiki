# 效率与效率改善方法

精益生产领域的效率提升方法论，涵盖效率定义、人力主导型改善、设备主导型改善和OPE指标体系。

## 精益改善十大精神

1. 打破固有观念
2. 寻找可行的办法
3. 不要分辨，要否定现状
4. 不要力求完美，50分也好，立即实施
5. 错了马上改善
6. 先从不花钱的改善做起
7. 穷则变，变则通
8. 追求根源，反复5次"为什么"
9. 十个人的智慧比一个人的智慧高明
10. 改善是无限的

## 什么是效率

效率 = 产出 / 投入，即将投入转化为产出的能力。

常见误解：
- 计划完成率 ≠ 效率
- 没有标准工时的产能统计可信度不高
- 产能 ≠ 效率

## 效率损失结构

| 损失类型 | 说明 |
|---------|------|
| 管理损失 | 计划停机、故障、换模线、换刀具、暖机 |
| 线平衡损失 | 工序间负荷不均导致的等待 |
| 动作损失 | 违反动作经济原则、可自动化替换的动作 |
| 小停机空转损失 | 短暂停机、空转 |
| 速度降低损失 | 未达到设计速度运行 |
| 品质不良损失 | 不良品返工、整修 |

## OPE（Overall Plant Efficiency）

总体工厂效能，是衡量以人为主的工厂效率及质量的综合指标。

**OPE架构**：
投入工时 = 稼动率 × 工时平衡率 × 操作效率

- **稼动率** = 稼动工时 / 负荷工时 × 100%
  - 稼动损失：支援其他部门、休息、早会、5S等
- **工时平衡率** = 有效工时 / (瓶颈工站C/T × 人数) × 100%
  - 平衡损失：工序间等待
- **操作效率** = (良品数 × 瓶颈工站C/T) / 稼动时间 × 100%
  - 操作损失：不良返工、操作速度慢

## 动作经济原则

四项基本原则（详见培训材料），旨在减少动作浪费，提升操作效率。

## 效率提升步骤

1. 现状把握 — 收集数据、绘制流程图
2. 问题分析 — 识别效率损失、找出瓶颈
3. 改善方案制定 — 应用精益工具和方法
4. 方案实施 — 小步快跑、PDCA循环
5. 效果确认与标准化 — 持续改善
