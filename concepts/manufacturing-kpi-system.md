# 制造业数据指标体系

制造业企业的数据指标体系，涵盖指标体系搭建方法论与六大类业务KPI参考。

## 为什么要搭建指标体系

单一指标无法全面衡量企业的经营状况。成体系化的指标能够结合问题背景、串联各个指标、通过各种维度进行分析，使解决方案更有针对性，也能在出现异动时快速定位问题环节。

指标体系是企业对如何实现战略目标、提升创造价值的思考，也是连接战略目标与实际工作的纽带。

## 指标体系搭建四大原则

1. **用户第一原则** — 指标核心是为实际业务服务，指标不是越多越好，不需要"虚荣指标"
2. **典型性原则** — 选择能够反映业务真实情况的代表性指标
3. **系统性原则** — 找到核心原子指标并延伸，形成树状结构指标体系，让每个指标有根可循
4. **动态性原则** — 随业务发展和数据分析需求变化持续迭代维护

## 指标金字塔

指标体系采用三层分层结构：

- **核心指标**（北极星指标，1-3个）— 与整体业务年度目标一致，反映整个业务走向，如公司战略目标
- **业务指标** — 支撑核心目标完成的关键动作，从业务角度拆解出过程性关键指标
- **操作指标** — 由业务指标进一步拆解，落实到基础岗位和具体执行人员

## 搭建指标体系四大步骤

1. **确定指引方向的核心指标** — 选用北极星指标，不超过3个
2. **基于业务拆解核心指标** — 从业务角度拆出过程性关键指标，每个业务方向1个
3. **拆解业务落实到操作指标** — 落实到具体的业务执行人员，避免重复，相互独立
4. **进行系统性梳理和整合** — 梳理、排查、演练，输出完整的指标体系思维导图

## 指标拆解方法

指标拆解四步流程：
1. **明确分析目标** — 确定要分析的对象和目标
2. **确定问题** — 用思维导图列出围绕目标需要解决的问题
3. **拆解问题（确定计算公式）** — 将问题转化为可计算的指标公式
4. **拆解指标 & 拓展维度布局** — 分析组成公式的指标，按时间、地区、品类等维度展开

## 指标规范化管理

### 指标命名公式

```
限定词 + 业务主题 + 指标名称 + 量化词
```

- **限定词**：当天、本周、当月、平均、累计
- **业务主题**：描述业务过程阶段（打开页面、下单、支付等）
- **指标名称**：统计的对象实体名称（订单、用户等）
- **量化词**：金额、份额、次数、率

### 指标字典

指标字典是公司常用指标的有组织整理，明确指标口径、维度、取数逻辑，形成统一认同的标准化数据体系。

指标字典五要素：**指标归类** → **指标名称** → **计算方法** → **可用维度** → **指标映射**（应用场景）

建立要求：规范命名、统一口径、覆盖核心维度量度、注入元数据中心。

## 制造业KPI参考

### 库存类指标
- **设施空间利用度**：储区面积率、可供保管面积率、储位容积使用率、平均每品项所占储位数、库存周转率
- **存货效益**：库存周转率、库存掌握程度、物资入库差错率、物资领用差错率、物资缺货率、货物损耗率、库存结构分析
- **呆废料**：呆废料率（数量/金额）

### 生产类指标
- **生产计划**：产量、计划量、计划完成率、计划批次
- **生产周期**：制造周期、在制品天数
- **交付能力**：按时交货率、缺货率
- **生产能力**：产能利用率、劳动生产率、投入产出率、产品合格率、各工序返工率
- **生产成本**：生产总成本、车间变动/固定费用、产品变动/固定成本、设备折旧率、设备故障率、工具消耗率、研发费用、隐形成本
- **生产效益**：增产效益、节降效益、生产边际贡献、原粉收率、单耗

### 设备分析指标
- 设备利用率、设备完好率、设备定期保养计划完成率
- 设备故障停机率、万元设备固定资产创产值率、设备维修费用率、故障次数

### 质量分析指标
- 产品一次合格率、产品抽检合格率、订单合格率、产品出货检验合格率
- 来料检验合格率、来料在线质量问题发生率
- 不良原因统计分析、不良TOPN分析、异常防呆

### 出入库与库存分析指标
- 入库差错率、入库良品率、出货达成率、出货差错率
- 库存金额/库存量（按物料组/库龄/状态统计）
- 入库产值完成率、配套率、下料率、库存周转天数、库存结构
- 呆滞库存量、包装回收率/速度/完成率
- 物流成本、运输费用达成率、配送及时率、仓库利用率（储区面积率）

### 采购类指标
- **采购订单**：订单下达情况、订单完成情况、采购物资入库情况
- **采购执行**：采购物资挂账情况、发票挂账率
- **供应商管理**：Top10供应商、供应商名录、供应商承诺交期达成率
- **进销存**：采购入库、采购量、采购入库动态
- **价格与合同**：价格分析、合同审批历史、固定资产采购
- **付款报销**：资金计划、发票金额、已支付金额、应付款余额
- **采购成本**：原材料采购金额/量、采购降本额、预算完成率
- **原材料交付**：交付周期、入场检验合格率

### 销售类指标
销售目标达成率、销售增长率、货款回收完成率、合同执行比率、市场份额

### 财务分析指标
- **责任令指标**：营业收入、营业收入增速、净利润、EVA、经营性净现金流、存货增速、应收账款增速、销售回款、成本费用率、毛利率
- **盈利能力**：主营业务利润率、总资产收益率、销售毛利率、销售净利率、成本费用利润率、净资产收益率
- **资产质量**：总资产周转率、流动资产周转率、存货周转率、应收账款周转率
- **债务风险**：资产负债率、流动比率、速动比率
- **资金分析**：资金存量、现金流、负债、投融资
- **费用分析**：费用金额、费用预算达成率
- **三大报表**：资产负债表、损益表、现金流量表

### 人力资源类指标
员工增加率、员工结构比例、关键人才流失率、工资增加率、人力资源培训完成率、人才引进完成率、考核通过率、部门员工出勤情况
