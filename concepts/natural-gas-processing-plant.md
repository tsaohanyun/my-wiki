# 天然气处理厂

## 概述

天然气处理厂（Natural Gas Processing Plant）是将从气井采出的原料天然气（Raw Natural Gas）加工为符合管道输送标准或液化标准的商品天然气的工业设施。

原料天然气除主要成分甲烷外，还含有水蒸气、硫化氢（H₂S）、二氧化碳（CO₂）、氮气、氦气，以及乙烷、丙烷、丁烷等天然气凝液（NGL）。这些杂质和重组分若不脱除，会在管道中造成腐蚀、冰堵，降低热值，且硫化氢具有剧毒。

## 核心功能

天然气处理厂的主要功能包括：

| 工艺环节 | 目的 | 典型技术 |
|---------|------|---------|
| 脱水 | 脱除水蒸气，防止水合物冰堵 | 三甘醇（TEG）脱水、分子筛脱水 |
| 脱硫 | 脱除H₂S，满足产品气硫含量标准 | 胺法（MEA/DEA/MDEA）、LO-CAT |
| 脱碳 | 脱除CO₂，提高热值 | 胺法、膜分离 |
| 凝液回收 | 回收乙烷/丙烷/丁烷等NGL | 低温分离、膨胀机制冷 |
| 硫磺回收 | 将H₂S转化为元素硫 | Claus工艺 + SCOT尾气处理 |

## 分类

### 按处理能力

- **大型处理厂**：日处理量 > 500万立方米，通常建在大型气田附近
- **中型处理厂**：日处理量 100-500万立方米
- **小型/撬装处理厂**：日处理量 < 100万立方米，适用于边远气井或页岩气

### 按产品方向

- **管道气处理厂**：产出符合管输标准的干气（C₁为主 + 少量C₂）
- **NGL回收型处理厂**：深度回收乙烷及以上组分，供乙烯裂解或燃料
- **LNG液化型处理厂**：将净化后天然气液化为LNG（-162°C），便于远洋运输

## 工艺流程总览

```
原料天然气 → 分离器(气液分离) → 脱硫/脱碳 → 脱水 → NGL回收 → 产品气
                                   ↓            ↓        ↓
                              硫磺回收      废水处理   LPG/轻油
```

## 关键指标

- **产品气水露点**：≤ -5°C（管输标准）或 ≤ -40°C（LNG原料）
- **H₂S含量**：≤ 6 mg/m³（一类气）或 ≤ 20 mg/m³（二类气）
- **总硫含量**：≤ 60 mg/m³（一类气）
- **CO₂含量**：≤ 3%（管输标准）
- **乙烷回收率**：深冷工艺可达 85-95%
