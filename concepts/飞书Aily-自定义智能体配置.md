# 飞书 Aily 自定义智能体配置指南

> 前置阅读：[[培训/培训-飞书Aily智能体平台|飞书Aily智能体平台培训指南]]

## 产品定位

自定义智能体是 Aily 平台的企业级定制功能，允许为每个业务场景打造专属 AI 数字员工。典型场景：报销专员、项目助手、HR 答疑机器人、IT 运维助手、客户服务智能体。

核心区别：智能伙伴是 1:1 个人专属助理，自定义智能体是 1:N 团队共享平台（用户间记忆隔离、完整权限管控）。

## 创建流程（6步）

### 第1步：进入开发后台

访问 https://aily.feishu.cn → 进入「开发后台」→ 新建智能体

选择进化方式：
- **标准模式**：仅开发者可修改配置，发布生效
- **无感发布模式**：支持在线实时调整，用户无感知

### 第2步：编写 System Prompt

Prompt 是智能体的灵魂，决定其角色、技能和行为边界。

**模板结构**：

```markdown
# 角色
你是[角色描述]，负责[核心职责]。

## 技能
### 技能 1：[名称]
[触发条件] + [执行行为] + [输出格式]

### 技能 2：[名称]
[触发条件] + [执行行为] + [输出格式]

## 限制
- [行为边界1]
- [行为边界2]
- [不得做的操作]
```

**设计要点**：
- 角色定义要具体，避免泛泛而谈
- 技能描述要有明确的触发条件和执行步骤
- 限制条件要清晰，防止越权操作
- 可让其他 LLM 辅助生成 Prompt，再微调

### 第3步：配置知识空间

| 知识类型 | 接入方式 | 适用场景 |
|---------|---------|---------|
| 云文档 | 选择飞书文档/知识库 | 产品文档、操作手册 |
| 文件上传 | 上传 PDF/Word/Excel | 规章制度、技术规范 |
| FAQ 对 | 手动录入或批量导入 | 常见问题解答 |
| 数据库 | 通过 MCP 连接 | 实时业务数据 |
| 网页 | 输入 URL 爬取 | 公开信息参考 |

### 第4步：配置技能与工具（Skill/MCP）

- **内置技能**：平台提供的开箱即用技能
- **自定义 MCP**：连接企业内部 API、数据库
- **子智能体调用**：一个智能体可调用其他智能体完成子任务

详见 [[培训/培训-飞书Aily智能体平台#6. MCP 工具集成|MCP 工具集成]]

### 第5步：配置发布渠道

| 渠道 | 说明 | 接入方式 |
|------|------|---------|
| 飞书渠道 | 在飞书群/私聊中使用 | 配置飞书机器人 |
| Web SDK | 嵌入企业网站/应用 | 前端 JS SDK 集成 |
| OpenAPI | 程序化调用 | HTTP API 接口 |

### 第6步：发布与测试

1. 在预览窗口测试对话
2. 确认回复质量达标
3. 选择发布渠道 → 发布
4. 灰度测试 → 全量开放

## 智能体进化模式

| 模式 | 特点 | 适用场景 |
|------|------|---------|
| 标准模式 | 开发者修改配置 → 发布生效 | 严格管控、正式业务场景 |
| 无感发布模式 | 在线实时调整，用户无感知 | 快速迭代、测试场景 |

## 数据看板

自定义智能体提供完整的使用数据分析：
- 对话量统计（日/周/月）
- 用户满意度评分
- 热门问题排行
- 技能调用频率
- 响应时间分析
- 错误率监控

## 数据表结构模板

自定义智能体通常需要对接飞书多维表格作为数据源。通用表结构设计原则：

| 字段类型 | 用途 | 示例 |
|---------|------|------|
| 文本 | 名称、编号、描述 | 客户名称、订单号 |
| 数字 | 金额、数量、指标 | 交易额、完成率 |
| 日期 | 时间节点 | 创建时间、到期日 |
| 单选 | 状态、分类 | 启用/停用、优先级 |
| 人员 | 负责人、审批人 | 销售、主管 |
| 关联 | 跨表引用 | 回款关联合同 |
| 公式 | 自动计算 | 未回款余额 = 应收 - 已收 |

## 最佳实践

1. Prompt 角色明确 + 技能具体 + 限制清晰
2. 知识空间分层组织 + 定期更新 + 质量审核
3. MCP 接口描述详细 + 权限最小 + 错误处理完善
4. 数据看板驱动效果调优 + A/B 测试 + 用户反馈
5. 敏感数据脱敏 + 审计日志 + 权限管控
