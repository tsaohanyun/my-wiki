# 价值流图 VSM（Value Stream Mapping）

价值流图析（VSM），又称物流情报图，是精益生产中的核心分析工具，用于可视化产品从原材料到成品交付给客户的全过程。

## 课程目标

1. **转换思维** — 望远镜思维、站在山顶、总经理的思维
2. **绘制现状图** — 绘制现状价值流图、布局图、制程山积图，将八大浪费具体化
3. **绘制未来图** — 基于公司规划目标和精益原则，绘制未来价值流图、山积图、布局图
4. **制定改善计划** — 根据现状与未来的差异（GAP）进行方针策略展开，制定一年详细计划

## 核心概念

### 增值比
增值时间 / 总时间 = 增值比

案例：去医院买药，总耗时156.7分钟，增值时间仅3.2分钟，增值比仅2%。

### 八大浪费
等待、搬运、不良、加工、库存、动作、过量生产、未被利用的人才

### 三大系统
- **物流改善** — 物料流动优化
- **情报流改善** — 信息传递优化
- **效率改善** — 直接通过率提升

## VSM 绘制框架

```
现状价值流图 → 现状山积图 → 现状布局图
                    ↓
        公司3-5年规划目标
                    ↓
未来山积图 → 未来价值流图 → 未来布局图
                    ↓
        GAP差异分析 → 方针策略展开 → 年份计划
```

## 改善目标设定

改善通常设定3年规划，分年推进物流、情报流、效率三大方向的改善，设定直达率、效率等KPI目标。
