# 数据可视化选型

## 定义

根据数据关系类型（比较、联系、分布、构成）选择最合适的图表类型，使数据表达准确、直观、高效。

## 选型矩阵 ^[raw/articles/data-visualization-selection.md]

### 比较

| 数据特征 | 分类方式 | 推荐图表 |
|----------|----------|----------|
| 每个项目两个变量 | 基于分类 | 不等宽柱形图 |
| 每个项目一个变量，多种分类 | 基于分类 | 表格 |
| 多个项目，少数分类 | 基于分类 | 条形图 |
| 少数项目，少数分类 | 基于分类 | 柱形图 |
| 多周期，循环数据 | 基于时间 | 雷达图 |
| 多周期，非循环数据 | 基于时间 | 折线图 |
| 少数周期，单个/少数分类 | 基于时间 | 柱形图 |
| 少数周期，多数分类 | 基于时间 | 曲线图 |

### 联系

| 变量数 | 推荐图表 |
|--------|----------|
| 两个变量 | 散点图 |
| 三个变量 | 气泡图 |

### 分布

| 数据特征 | 推荐图表 |
|----------|----------|
| 单变量，数据量少 | 直方图 |
| 单变量，数据量多 | 正态分布图 |
| 两个变量 | 散点图 |
| 三个变量 | 曲面图 |

### 构成

| 时间维度 | 差异关注点 | 推荐图表 |
|----------|------------|----------|
| 少数周期 | 仅相对差异 | 百分比堆积柱形图 |
| 少数周期 | 相对和绝对差异 | 堆积柱形图 |
| 多数周期 | 仅相对差异 | 堆积百分比面积图 |
| 多数周期 | 相对和绝对差异 | 堆积面积图 |
| 静态 | 占总体比例 | 饼图 |
| 静态 | 累计或扣减与总计 | 瀑布图 |
| 静态 | 构成的构成 | 复合堆积百分比柱形图 |

## 选型决策流程

```
数据关系类型？
├── 比较 → 分类 or 时间？→ 分类数/周期数？→ 选图表
├── 联系 → 变量数？→ 散点/气泡
├── 分布 → 变量数 + 数据量？→ 直方/正态/散点/曲面
└── 构成 → 时间变化 or 静态？→ 差异关注点？→ 选图表
```
