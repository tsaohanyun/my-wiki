# B栋MES接口集成方案

本文档梳理B栋MES系统与WMS、SAP等外围系统的接口集成方案，涵盖接口清单、SAP集成方向、单机上线风险及系统联调测试方案。

## MES-WMS接口清单

MES与WMS之间共计12个核心接口，采用HTTP协议实时触发，覆盖备料、退料、外包装工单及返工等生产执行业务场景。

### 接口总览

| 序号 | 接口名称 | 传递方向 | 说明 |
|------|----------|----------|------|
| 1 | 备料需求下达接口 | MES→WMS | MES备料需求生效时调用WMS接口下达备料信息，支持新增、删除检查、删除三种操作类型 |
| 2 | 备料信息反馈接口（罐区） | WMS→MES | WMS反馈罐区备料详情（批号、数量、仓库等），MES生成本地备料记录 |
| 3 | 备料信息反馈接口 | WMS→MES | WMS反馈非罐区备料详情，MES提供独立服务与罐区接口分开 |
| 4 | 备料信息签收接口 | MES→WMS | MES对实物进行签收或拒收，签收后WMS同步移库数据给SAP（311移动类型） |
| 5 | 备料需求关闭接口 | WMS→MES | WMS通知MES对已完成或异常的备料需求进行整单关单操作 |
| 6 | 车间物料退料信息下达接口 | MES→WMS | MES触发退料时推送退料信息给WMS，MES同步给SAP（303移动类型） |
| 7 | 车间物料退料确认接口 | WMS→MES | WMS确认退料结果（接收/拒收），MES同步给SAP（接收：305，拒收：304） |
| 8 | 外包装工单下达接口 | MES→WMS | 外包装工单生效时推送工单信息给WMS，用于成品入库调度 |
| 9 | 外包装工单报工接口（产成品入库） | WMS→MES | WMS调用MES进行包装工单报工，WMS给SAP传入库，MES给SAP进行工时报工 |
| 10 | 取消外包装工单报工接口 | WMS→MES | WMS调用MES取消已报工的包装工单，WMS给SAP传取消入库，MES给SAP取消工时报工 |
| 11 | 返工出库反馈接口 | WMS→MES | WMS将立体库返工品出库信息同步给MES，一步签收，返工品从成品仓返回车间 |
| 12 | 返工出库签收接口 | MES→WMS | MES对返工实物进行签收或拒收，签收后WMS同步移库数据给SAP |

### 关键接口字段说明

**备料需求下达接口**（MES→WMS）核心字段：

| 字段 | 说明 |
|------|------|
| `inWarehouse` | 目的仓库（SAP库存地点） |
| `externorderkey` | 备料指令单号 |
| `planDate` | 计划送货时间（yyyy-MM-dd hh:mm:ss） |
| `type` | 类型：1-删除检查，2-新增，3-删除 |
| `sku` | 物料号 |
| `istank` | 是否罐区：1-是，0-否 |
| `quantity` | 数量 |
| `factoryCode` | 工厂编码 |
| `outWarehouse` | 出库仓库（SAP库存地点） |

**外包装工单下达接口**（MES→WMS）核心字段：

| 字段 | 说明 |
|------|------|
| `productCode` | 产品编码 |
| `batch` | 生产批次（10位） |
| `outputNum` | 计划数量 |
| `factory` | 工厂代码（区分家园立库和双流立库） |
| `sapbillno` | SAP生产订单号（12位） |

**退料下达接口**特殊说明：`source` 字段用于标识退料来源，家园固定传递"F"，双流固定传递"A"。

### 通用接口规范

- 传输协议：HTTP
- 触发方式：实时触发
- 返回格式统一为 `{ code, data, message/msg }` 或 `{ code, data, message, success }`
- 认证方式：通过 `token` 字段进行接口鉴权

## MES-SAP集成（PCE集成方向）

MES与新SAP通过PCE中间件集成，共计28个接口，其中B栋项目涉及26个（备件接口不涉及B栋项目）。

### 接口分类

| 方向 | 数量 | 说明 |
|------|------|------|
| MES调用SAP接口 | 21个 | 开发计划启动时间为8月1日 |
| SAP调用MES接口 | 5个 | 开发计划启动时间为8月15日，8月20日开始交付第一个接口 |

### 主要集成业务

- **MES推送方向**：订单创建与执行、物料消耗提交、返工业务、盘点业务等在MES端发生的业务数据
- **MES拉取方向**：[[bom-management|BOM]]主数据、工艺主数据、客商主数据、工艺工时、跨车间物流、预留信息
- **SAP推送方向**：个别成本、跨工厂业务等由SAP主动调用MES

### 旧SAP对接

旧SAP对接共计17个接口，均为MES调用旧SAP接口，同样涵盖订单创建与执行、物料消耗提交以及工艺工时、跨车间物流、预留信息的拉取。

### 一物一码系统对接

MES向一物一码系统推送工单数据，共计1个接口。

## 单机上线风险说明

MES系统在B栋单机上线期间，由于部分接口尚未联调完成，存在以下风险项。详见[[tianwei-mes-go-live-strategy|上线策略]]。

### 风险清单

| 序号 | 风险项 | 风险说明 | 影响范围 |
|------|--------|----------|----------|
| 1 | 基础数据变更 | 系统初始数据导入后，后续数据变化时MES数据更新不及时 | 生产工单、任务单建立失败 |
| 2 | 车间退料 | 无接口时需在SAP二次制单，录错导致账务不符 | MES与SAP账务差异 |
| 3 | 称重接收 | B栋新增场景，称重差异成本需发SAP成本中心 | SAP库存虚高、工作量增加 |
| 4 | 生产工单创建 | 单机模式下工单在MES内创建，不经过SAP | SAP无对应工单记录 |
| 5 | 物料消耗 | 无接口时MES投退料数据无法实时回传SAP | SAP库存账务不准 |
| 6 | 成品入库 | 外包装报工后无法自动触发WMS入库 | 成品入库延迟 |
| 7 | 返工品管理 | 返工品出库与投料无接口支撑 | 返工品账务差异 |

## 系统联调测试方案

### 联调范围

联调测试覆盖MES与WMS、SAP的全部已开发接口，验证数据传输的准确性和业务流程的完整性。

### 测试环境

- MES测试环境与生产环境VLAN隔离
- SAP测试环境由SAP团队提供
- WMS测试环境由WMS团队提供

### 测试策略

1. **单接口测试**：逐个验证接口的请求/响应格式和数据正确性
2. **业务流程联调**：按业务场景串联多个接口，验证端到端流程
3. **异常场景测试**：模拟网络中断、数据异常等场景的容错处理
