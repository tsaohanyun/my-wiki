# 技能创建与优化

## 概述
创建和优化 Hermes Agent 技能的完整指南，从创建到性能测试。
## 技能生命周期
1. **创建** - 编写 SKILL.md 和相关文件
2. **测试** - 测试技能触发和执行
3. **优化** - 优化描述和触发准确率
4. **维护** - 持续更新和改进
## 核心原则
- **明确触发** - 清晰定义何时使用技能
- **完整文档** - 包含使用示例和陷阱
- **可测试** - 技能可被测试和验证
- **可维护** - 易于更新和改进
