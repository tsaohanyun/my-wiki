# 数据中台解决方案

数据中台是整合企业内外部数据资源，提供统一数据采集、存储、计算、治理、服务的综合平台方案，是企业数字化转型的基础设施层。

## 数据中台建设蓝图

数据中台架构从下到上分为：

- **后台数据源** — 核心业务系统、网银系统、交易平台、风险管理系统、CRM、支付清算等
- **开发中心** — 批处理、流式处理、异构数据处理、手工填报、手工采集
- **存储中心** — MPP数仓，节点动态扩展、平台监控、数据计算、资源管理
- **数据治理中心** — 数据标准、数据模型、数据质量、数据地图、血缘分析、工作调度
- **应用平台** — 风险管理（信贷风险、供应链金融、骗保识别、交易欺诈）、市场决策（智能投顾、资产管理）、数据服务（风险定价、尽职调查、监管科技）
- **统一门户** — 驾驶舱体系、上报监控体系、登录认证、权限管理

## 基于大数据的数据中台整体解决方案

### 技术栈分层

| 层次 | 技术组件 |
|------|---------|
| 数据源 | 结构化(RDBMS)、半结构化(HDFS/S3/Blob/Kafka)、非结构化(音视频) |
| 数据集成 | 离线(Sqoop/DataX/FTP)、流式(Flume/FlinkCDC)、手工 |
| 数据存储 | HDFS/S3、HBase/ES/Redis、GreenPlum/OceanBase/ClickHouse |
| 资源管理 | YARN、DolphinScheduler任务调度 |
| 数据计算 | 离线(Hive/Spark)、流(Flink)、交互式(Impala)、隐私计算(Fate)、ML(Spark/TensorFlow) |
| 数据服务 | API网关(KONG)、数据API服务 |
| 数据应用 | 用户洞察、智能营销、高管驾驶舱、数据大屏、BI分析 |

### 数据架构

数据生产者 → 数据融合平台（清洗、转换、加载）→ 大数据中心（标准化、规范化、目录资源、元数据）→ 数据消费者

## 产品功能架构

一站式数据融合平台的核心能力：

- **数据分析与应用** — 自助查询、BI分析、AI SQL、Office插件、数据研究室
- **数据服务** — 低代码API构建、接口服务编排、API网关
- **数据治理** — 数据质量（规则模板、智能规则推荐）、数据资产地图（元数据采集、数据目录、血缘、影响分析）、数据安全（权限管理、分级分类、隐私保护）
- **数据开发** — 数仓规划、数据标准、数据指标、离线/实时/交互式/AI一体化IDE
- **数据集成** — 全量同步、增量同步、实时同步、数据迁移

## 数据中台 vs 指标中台

- **数据中台**：范围更广，涵盖数据全生命周期（采集→存储→计算→治理→服务→应用），是企业数据基础设施
- **指标中台**：数据中台的上层应用，专注于指标的定义、管理、计算和服务
