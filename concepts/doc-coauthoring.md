# 文档协作流程

## 概述
文档协作的结构化工作流，帮助用户高效创建和优化文档。
## 协作流程
1. **上下文转移** - 传递项目背景和需求
2. **内容创建** - 创建初稿
3. **迭代优化** - 多轮反馈和修改
4. **验证可用性** - 确保文档对读者有用
## 核心原则
- **读者导向** - 为读者写作
- **迭代改进** - 多轮优化
- **清晰结构** - 逻辑清晰的结构
- **可验证** - 文档内容可验证
