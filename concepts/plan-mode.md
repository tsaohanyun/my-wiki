# 计划模式

## 概述
Hermes Agent 的计划模式，用于检查上下文、创建计划文档但不执行实际工作。
## 使用场景
- **复杂任务规划** - 任务开始前的规划阶段
- **需求分析** - 分析需求和约束
- **方案设计** - 设计实现方案
- **风险评估** - 识别潜在风险
## 核心原则
- **只规划不执行** - 计划模式不执行代码
- **文档化** - 计划保存为 markdown 文件
- **可审查** - 计划可被审查和修改
