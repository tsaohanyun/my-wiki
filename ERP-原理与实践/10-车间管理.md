# 车间管理

## 概述

车间管理是企业生产计划实施的主体，处于ERP的**计划执行与控制层**。其管理目标是按物料需求计划的要求，按时、按质、按量完成加工制造任务，并且花费最低的劳动成本。

> 📎 作业排序规则、派工单、调度方法等详细内容请参见 → [[10a-作业排序与调度]]

### 车间的定义与特点

车间是企业内直接从事生产活动的场所，是企业组织生产的基本经济单位。其特点：

- 必须具备一定规模
- 具有一定的管理职能
- 具有相对明确的生产对象和生产条件
- 是企业生产活动的第一线

### 管理层次定位

```
企业管理（最高层）──→ 车间管理（中间层/执行型+指令型）──→ 班组管理（作业层）
```

## 核心内容

### 一、车间管理的主要内容

| 内容 | 说明 |
|------|------|
| 根据MRP生成车间任务 | 核实工作中心、工具、物料及提前期的有效性 |
| 生成加工任务与作业排序 | 编排工作中心的任务计划，控制任务流动和优先级 |
| 下达生产指令 | 生产工单/工票，进行生产调度与进度控制 |
| 能力投入产出控制 | 平衡发挥各工序能力，控制在制品库存量 |
| 在制品管理 | 定期盘点，盘盈盘亏及时调整 |
| 登记加工信息 | 记录工票信息，收集车间数据 |
| 统计分析 | 进度分析、在制物流分析、投入产出分析、成本分析等 |

### 二、生产任务管理单据体系

#### 单据层次关系

```
生产任务单（对象：车间）
  └── 加工单（对象：物料/工序计划）
        └── 派工单（对象：工作中心/工序）
              └── 领料单（物料领用）
                    └── 完工入库单（产品入库）
```

#### 各单据说明

| 单据 | 对象 | 作用 |
|------|------|------|
| **生产任务单** | 车间 | 生产系统唯一的需求来源，可手工输入或MRP自动生成 |
| **生产投料单** | 领料计划 | 根据BOM生成，控制物料领用数量 |
| **加工单** | 物料 | 说明加工工序、工作中心、工进度及工装设备 |
| **派工单** | 工作中心 | 说明某时段加工任务与优先级别，指导生产作业 |
| **领料单** | 物料出库 | 生产领料的书面证明 |
| **完工入库单** | 产品入库 | 生产完工后合格物料入库的单据 |

#### 领料方式

| 方式 | 特点 | 适用场景 |
|------|------|----------|
| **一次全部领料** | 生产准备时把所有物料全部领到车间 | 管理水平一般的企业 |
| **按工序领料** | 按生产进度在需要时领料 | 管理水平较高的企业 |
| **倒冲法领料（反冲领料）** | 完工入库时按BOM定额自动生成领料单 | 低值易耗品（螺丝钉、胶水等） |

### 五、作业信息收集

车间数据分为四类：

| 数据类型 | 包含内容 |
|----------|----------|
| **人工数据** | 考勤、工时、工资、班组信息 |
| **生产数据** | 进度、产量、机器运转信息 |
| **质量控制数据** | 质检、废品、返工信息 |
| **物料移动数据** | 领料、转移、入库信息 |

数据采集方式：

| 方式 | 特点 |
|------|------|
| 完全手工采集 | 表格、卡片、台账记录，再汇总分析 |
| 完全计算机采集 | 联机终端实时采集，扫描器、传感器等 |
| 混合采集 | 手工+自动混合采集，数据处理自动化 |

## 关键概念

| 术语 | 英文 | 说明 |
|------|------|------|
| 车间管理 | Shop Floor Management | ERP计划执行与控制层的管理 |
| 生产任务单 | Production Order | 生产系统的需求来源 |

### 核心公式

| 公式 | 表达式 |
|------|--------|
| 工序在制品数量 | `期初结存 + 移入数量 - 报废数量 - 移出数量 + 盘盈 - 盘亏` |

## 实际应用

### 案例：某企业ERP系统生产管理

**企业特点**：
- 电子类及汽车行业金属冲压件，约100人
- 订单驱动生产，产品种类多而杂
- 工人计件制，工序包括冲压、折弯、电镀等

**五大关注点及解决方案**：

| 关注点 | ERP/MES解决方案 |
|--------|-----------------|
| **生产进度** | 工序报工→自动汇总→生产进度表 |
| **材料控制** | 材料运算→领用统计→耗用差异分析 |
| **人工效率** | 定义工序单价→工艺报工→工人生产报表→计算工资 |
| **节省人工** | 30+种自动生成报表，条码枪/手机app报工 |
| **追溯** | 追溯报表：生产过程、领料记录、出入库记录 |

**关键经验**：
- 一线员工操作要简单（避免键盘，用条码+鼠标点击）
- ERP作为数据中心统一管理文件，减少"人为失误"
- 追溯内容包括：加工人、确认人、时间、数量、机床、加工要求等

### 模块间关系

```
MRP ──→ 车间管理 ──→ 库存管理（完工入库）
  │         │
  │         ├──→ 质量管理（质量检验）
  │         ├──→ 成本管理（加工成本核算）
  │         └──→ 人力资源（考勤、工资）
  │
  └── 能力需求计划 ←── 车间管理（实际能力反馈）
```
