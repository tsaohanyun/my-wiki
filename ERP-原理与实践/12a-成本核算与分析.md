# 第12章 成本核算与分析

> 项目：**ERP知识库**

## 概述

本文件详细介绍成本核算的具体方法、作业成本法（ABC）以及成本差异分析。这些内容是成本管理的技术核心，帮助企业精确计算产品成本、识别成本动因、分析成本偏差，为成本控制和决策提供依据。

> 本文件是 [[12-成本管理]] 的补充，主要聚焦于成本核算的技术方法和分析工具。

## 核心内容

### 产品成本的计算方法

#### 三种基本成本计算方法

| 方法 | 计算对象 | 适用场景 |
|------|----------|----------|
| **品种法** | 产品品种 | 单步骤大量生产（发电、采掘等），最基本的成本计算方法 |
| **分批成本法** | 产品批别（订单） | 单件小批量生产（飞机制造等），各批次差异较大 |
| **分步成本法** | 生产步骤 | 大批量生产（纺织厂等），单位成本相同 |

#### 材料价格的计价方法

| 计价方法 | 核心原理 | 优点 | 缺点 |
|----------|----------|------|------|
| **个别计价法** | 逐一辨认各批次实际成本 | 计算合理准确 | 工作量繁重 |
| **先进先出法** | 先入库的先出库 | 不能随意挑选计价调整利润 | 物价上涨时高估利润 |
| **加权平均法** | 期末计算加权平均单价 | 计算简单，成本平均化 | 不利于核算及时性 |
| **移动加权平均法** | 每次入库后重新计算平均价 | 及时了解存货结存情况 | 计算量大 |
| **计划成本法** | 按预先制定的计划成本计价 | 简化会计工作，强化成本管理 | 综合差异率可能偏差较大 |

> **计划成本法关键公式**：
> - 材料成本差异率 =（月初差异+本月收入差异）÷（月初计划成本+本月收入计划成本）×100%
> - 发出材料应负担差异 = 发出材料计划成本 × 成本差异率
> - 发出材料实际成本 = 计划成本 + 成本差异

#### 直接人工费计算

各层直接人工费 = 人工费率 × 工作小时数

#### 间接费用分配

产品间接费用 = 工作中心费用分配率 × 占用工作中心工时

#### 在制品成本分配方法

最常用的是**约当产量比例法**：

- 约当产量 = 在制品数量 × 完工程度
- 费用分配率 =（月初在制品成本+本月生产费用）÷（完工产品产量+在制品约当产量）
- 完工产品成本 = 完工产品产量 × 费用分配率
- 在制品成本 = 在制品约当产量 × 费用分配率

### 作业成本法（ABC）

#### 核心概念

| 概念 | 定义 |
|------|------|
| **资源** | 进入作业系统的人力、物力、财力（货币、材料、人力、动力资源） |
| **作业** | 企业为生产产品或提供劳务而进行的某项生产经营活动或某道工序 |
| **作业中心** | 一系列相互联系且能实现某种特定功能的作业集合 |
| **成本动因** | 引起成本发生的因素（分配基础） |
| **资源动因** | 资源被各种作业消耗的方式和原因，资源→作业的分配标准 |
| **作业动因** | 作业中心成本分配到产品的标准，作业→产品的分配标准 |
| **作业链** | 前后有序、由此及彼相连接的一系列作业 |
| **价值链** | 价值沿作业链在各作业之间转移的链条 |

#### 作业成本法核算原理

核心原则：**作业消耗资源，产品消耗作业**

核算步骤：
1. 确定成本计算对象
2. 确定直接生产成本类别
3. 确认作业类别，建立作业中心
4. 将资源分配到各作业中心（按资源动因）
5. 分配作业成本到产品（按作业动因）
6. 计算各产品总成本或单位成本

#### 作业成本法与传统成本法的区别

| 比较维度 | 传统成本法 | 作业成本法 |
|----------|------------|------------|
| 计算对象 | 以"产品"为中心 | 以"作业"为中心 |
| 计算程序 | 按部门归集，单一分配率 | 按作业归集，多成本动因分配 |
| 核算范围 | 产品成本 | 产品成本+作业成本 |
| 分配标准 | 财务变量（人工工时、机器工时） | 财务变量+非财务变量 |
| 成本信息 | 单一标准，可能扭曲 | 多动因标准，更精确 |

### 成本差异分析

成本差异 = 实际成本 - 标准成本（正数为不利差异/超支，负数为有利差异/节约）

| 差异类型 | 计算公式 |
|----------|----------|
| **直接材料成本差异** | 实际价格×实际数量 - 标准价格×标准数量 |
| 材料价格差异 | （实际价格-标准价格）× 实际数量 |
| 材料数量差异 | （实际耗用量-标准耗用量）× 标准价格 |
| **直接人工成本差异** | 实际工资价格×实际工时 - 标准工资价格×标准工时 |
| 人工工资价格差异 | （实际工资价格-标准工资价格）× 实际工时 |
| 人工效率差异 | （实际工时-标准工时）× 标准工资价格 |
| **变动制造费用差异** | 实际工时×实际分配率 - 标准工时×标准分配率 |
| **固定制造费用差异**（两差异法） | 预算差异 + 产量差异 |
| **固定制造费用差异**（三差异法） | 预算差异 + 能力差异 + 效率差异 |

## 关键概念

- **作业成本法（Activity-Based Costing, ABC）**：以作业为基础分配间接费用的成本计算方法
- **成本动因（Cost Driver）**：引起成本发生的因素，分为资源动因和作业动因
- **标准成本（Standard Cost）**：正常条件下经充分调查分析确定的目标成本（详见 [[12-成本管理]]）
- **成本差异（Cost Variance）**：实际成本与标准成本之间的差异
- **约当产量**：在制品数量按完工程度折算为相当于完工产品的数量
- **品种法**：以产品品种为成本计算对象的基本方法
- **分批成本法**：以产品批别（订单）为成本计算对象的方法
- **分步成本法**：以生产步骤为成本计算对象的方法

## 实际应用

### 案例：A地铁公司作业成本法应用

**背景**：A地铁公司运营8条线路（236km），年客流收入近30亿元，但运营成本急剧增大，票价优惠额度高达9亿元/年。

**存在的问题**：
- 成本目标分解逐层衰减，压力未有效传导至责任单位
- 未从作业源头制定有效成本控制策略
- 成本核算颗粒度不够，信息难以追踪至业务层面

**实施方法**：
1. 将运营成本分为六大类，聚焦前四类（占总成本75%）
2. 按价值活动进行作业分类：前端客运服务、中端维修服务、后端支持性服务
3. 建立"成本资源→作业活动→成本对象→责任中心"四维映射关系
4. 分析作业动因，建立成本指标体系
5. 落实成本责任，按作业环节映射责任单位

**效果**：通过作业标准成本实现了从预算值与实际值对比到作业标准值、预算值、实际值两两对比的转变，从业务源头发掘成本差异原因。
