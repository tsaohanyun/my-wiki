# 库存管理

> 项目：**ERP知识库**

---

## 概述

库存管理不能以狭义的仓库保管物料的观点来理解，必须从企业ERP系统的全局角度来理解。库存管理与ERP系统中的其他管理模块紧密联系在一起，以支持生产、维护、操作和客户服务为目的而存储各种物料。

库存资产一般占用企业总资产的**5%～40%**，库存管理在企业生产经营管理中起着非常重要的作用，是企业物料管理的核心。

### 库存的作用

| 作用 | 说明 |
|------|------|
| 维持销售稳定 | 应付市场销售变化 |
| 维持生产稳定 | 降低采购提前期不确定带来的风险 |
| 平衡企业物流 | 协调采购、生产、销售各环节 |
| 平衡流通资金占用 | 寻找订货费用与库存费用的最佳控制点 |
| 应对不确定因素 | 需求、交货期、质量等不确定性 |

### 高库存的弊端

| 弊端 | 说明 |
|------|------|
| 占用大量资金 | 增加产品成本与管理成本 |
| 市场价格波动风险 | 原材料价格下降导致库存贬值 |
| 技术进步风险 | 产品更新换代导致库存积压 |
| 掩盖管理问题 | 计划不周、采购不力、生产不均衡等问题被掩盖 |

## 核心内容

### 一、库存管理模式

#### 四种主要库存管理模式

| 模式 | 核心特征 | 优势 | 局限 |
|------|----------|------|------|
| **传统模式** | 各节点企业各自管理自有库存 | 单一库存角度有一定适用性 | 从供应链整体看不够 |
| **VMI（供应商管理库存）** | 供应商代理行使库存决策权 | 提高服务水平，降低"牛鞭"效应 | 供应商承担风险，协商不足 |
| **JMI（联合库存管理）** | 基于协调中心的协作模式 | 消除需求变异放大，简化管理 | 建立和协调成本较高 |
| **CPFR（协同式供应链库存管理）** | 协同计划、预测与补货 | 准确预测销售波动，多赢原则 | 缺乏系统集成，信任要求高 |

#### VMI模式特点
- 供应商拥有和管理库存
- 下游企业实现低库存或零库存
- 需要高度信任和信息共享
- 实际是对传统库存控制策略的"责任倒置"

#### JMI模式的两种形式
1. **集中库存**：供应商零部件直接存入核心企业原材料库
2. **无库存模式**：供应商直接向核心企业生产线连续补货

### 二、库存事务管理

| 事务类型 | 说明 |
|----------|------|
| **物料入库** | 采购收货入库、生产完工入库、剩余物料入库、销售退货入库 |
| **物料出库** | 生产计划领料、非生产领料、销售提货 |
| **物料移动** | 库存之间、分厂之间、分公司之间的物料调拨 |
| **库存盘点** | 数量清点、质量检查、账实核对 |
| **信息分析** | 入出库数据分析、资金占用分析、来源去向分析 |

### 三、库存成本管理

#### 库存成本构成

| 成本类型 | 说明 |
|----------|------|
| 物料本身费用 | 单位标准成本、计划价格或实际采购价格 |
| 订购费用 | 外部采购价格+运费，或内部生产成本 |
| 保管费用 | 报废、损坏、税金、保险金、管理费用、资金占用等 |
| 缺货费用 | 延期交货、取消订货、利润损失、客户罚款等 |
| 补货费用 | 欠账经营、进货后补货给客户的费用 |

#### 降低库存成本的方法

1. **正确确定库存物料**：区分库存与非库存物料
2. **减少不可用库存**：降低在途库存、淤滞库存、预留库存等
3. **适时处理积压库存**：退货、代销、清仓或报废

### 四、库存管理策略

#### ABC分类法

基于20/80原则，按库存资金占用分类管理：

| 类别 | 品种占比 | 资金占比 | 管理策略 |
|------|----------|----------|----------|
| **A类** | 10%～20% | 70%～80% | 重点监控，经常盘点，严格控制库存 |
| **B类** | 20%～30% | 约20% | 一般控制，定期订货，批量供应 |
| **C类** | 60%～70% | <10% | 简便管理，固定订货量，减少盘点次数 |

#### 安全库存

安全库存的制定方法：

**方法一：基于平均测量误差**
```
MAD = 各周期预测的绝对误差总和 / 预测周期数
安全库存 = MAD × 安全因子
```

**方法二：基于提前期内需求量的分布**
```
安全库存 = z × σ × √L
```
其中 z = 安全因子，σ = 提前期内需求量标准差，L = 提前期

#### 安全提前期与安全库存的区别

| 比较项 | 安全库存 | 安全提前期 |
|--------|----------|-----------|
| 缓冲对象 | 数量不确定性 | 时间不确定性 |
| 资金占用 | 较多 | 较少 |
| 系统处理 | 较简单 | 较复杂 |
| 适用场景 | 备品备件、公用件 | 运输因素影响的采购件 |

### 五、库存盘点

#### 盘点方法

| 方法 | 特点 | 适用场景 |
|------|------|----------|
| **账面盘点（永续盘点）** | 每日记录出入库，随时查询存量 | 存量少、单价高的物料 |
| **期末盘点** | 期末一次清点所有物料 | 全面盘点 |
| **循环盘点** | 按周期分批盘点少量物料 | 日常管理 |

#### 循环盘点的频率建议

- A类主要物料：每周盘点一次
- B类物料：每月盘点一次
- C类物料：半年盘点一次

### 六、库存补充方法

| 方法 | 特点 |
|------|------|
| **定量库存控制模型** | 库存降到某值时立即补充，连续检查 |
| **定期库存控制模型** | 按固定周期检查，补充到规定库存S，补充量Q=S-I |
| **MRP物料计划管理方法** | 基于生产计划的库存控制 |

## 关键概念

### 核心术语表

| 术语 | 英文 | 说明 |
|------|------|------|
| 经常性库存 | Cycle Stock | 为满足日常需要而建立的库存 |
| 安全库存 | Safety Stock | 应对不确定因素的缓冲库存 |
| 季节性库存 | Seasonal Stock | 满足特定季节需求的库存 |
| 战略性库存 | Strategic Stock | 预测涨价时的大批量储备 |
| 在途库存 | In-Transit Stock | 运输中的库存 |
| VMI | Vendor Managed Inventory | 供应商管理库存 |
| JMI | Joint Managed Inventory | 联合库存管理 |
| CPFR | Collaborative Planning Forecasting and Replenishment | 协同式供应链库存管理 |
| ABC分类法 | — | 按资金占用分级管理物料 |
| MAD | Mean Absolute Deviation | 平均绝对偏差 |

### 核心公式

| 公式 | 表达式 |
|------|--------|
| 安全库存（MAD法） | `安全库存 = MAD × 安全因子` |
| MAD | `各周期预测的绝对误差总和 / 预测周期数` |
| 安全库存（正态分布法） | `安全库存 = z × σ × √L` |
| 定期补货量 | `Q = S - I` （S为规定库存，I为当前库存） |

## 实际应用

### 案例一：戴尔公司的库存管理

**核心策略**：
- 推行**零库存管理**，平均物料库存时间缩短为**5天**
- 采用VMI模式：供应商直接根据生产进度管理库存
- 采用JMI模式：与供应商共同制订库存计划，利益共享、风险共担
- 95%的物料来自30家核心供应商组成的严密网络
- 90%的采购过程通过互联网完成

**效果**：过期零部件比例仅0.05%～0.1%，竞争对手为2%～3%。

### 案例二：某食品公司库存管理

**问题**：物料种类繁多、形式各异（箱式+流体），三班倒无法准确盘点。

**解决方案**：
- 实施基于条码的无线仓储管理系统
- 统一条码和格式，建立批次管理，先进先出
- 物料与仓位严格对应，规范管理
- 设计**动态盘点**算法，不停工盘点
- 与ERP系统集成，建立库存预警机制

**效果**：库存准确率提高到99.8%。

### 模块间关系

```
采购管理 ──→ 库存管理 ──→ 生产/销售
                │
                ├──→ 成本核算（存货成本）
                ├──→ 总账管理（资金核算）
                └──→ MPS/MRP（库存信息反馈）
```
