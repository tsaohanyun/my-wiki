# BOM 物料清单

## 概述

物料清单（Bill of Material，BOM）是以数据格式描述产品结构的文件，体现产品所需零部件的明细及其结构。BOM是MRP运算的基础数据，也是联系[[05-主生产计划|MPS]]与[[06-物料需求计划|MRP]]的桥梁。本页详细介绍BOM的结构要素、类型分类、查询方式及虚拟件等关键内容。

## 核心内容

### BOM的定义

物料清单是以数据格式描述产品结构的文件，体现产品所需零部件的明细及其结构，是产品→部件→组件→零件→原材料之间的**结构关系**和**数量关系**。

### BOM的结构要素

| 数据项 | 说明 |
|--------|------|
| 物料标志（编码） | 唯一标识每种物料 |
| 需求量 | 每个父项所需该子项的数量 |
| 层次码 | 物料在结构表中相对最终产品的位置（0～N） |
| 损耗率 | 生产过程中的损耗比例 |
| 有效日期 | BOM的有效生效/失效日期 |

### BOM的七大作用

1. 生成MRP的基本信息，联系MPS和MRP的桥梁
2. 生成产品工艺路线
3. 加工过程跟踪
4. 生产配套物料依据
5. 物料追溯依据
6. 成本计算
7. 销售定价依据

### BOM的查询形式

| 查询方式 | 说明 | 用途 |
|----------|------|------|
| 单层展开 | 查询某生产件的下一层物料 | BOM维护的基本方式 |
| 缩排法 | 以缩行形式列出各层物料 | 展示产品制造过程 |
| 汇总展开 | 列出所有物料的总数量 | 快速估算成本 |
| 单层追溯 | 直接使用某物料的上层物料 | 查找物料用途 |
| 缩行追溯 | 使用某物料的所有高层物料 | 追溯计划来源 |

### BOM的种类

| 种类 | 说明 | 典型应用 |
|------|------|----------|
| **基本BOM** | 产品实际结构组成 | 最常见、最常用 |
| **计划BOM** | 用于产品预测，简化MPS | 产品系列预测 |
| **模块化BOM** | 通用型产品组件的模块化管理 | 减少数据重复 |
| **成本BOM** | 描述每种物料的成本构成 | 成本核算 |
| **EBOM（设计BOM）** | 工程设计阶段的产品结构 | 纯技术文件 |
| **PBOM（计划BOM）** | 工艺计划阶段的BOM | 工艺计划 |
| **MBOM（制造BOM）** | 制造部门的装配步骤设计 | 生产计划关键数据 |
| **可选BOM** | 包含通用件+基本组件+可选件 | 汽车、电子、家具 |
| **CBOM（客户BOM）** | 客户订购产品结构筛选 | 按客户组织图纸 |
| **销售BOM** | 按用户要求配置的产品结构 | 客户订单跟踪 |
| **维修BOM** | 按维修要求产生的BOM | 消耗件、备用件清单 |

### 独立需求与相关需求

| 需求类型 | 定义 | 需求决定因素 | 示例 |
|----------|------|-------------|------|
| **独立需求** | 需求量不依赖于企业内其他物料 | 预测、客户订单（外在因素） | 最终产品、备品备件、维修件 |
| **相关需求** | 需求量可由其他物料需求量确定 | MRP根据BOM展开计算 | 原材料、零件、组件 |

> **BOM与需求的关系**：BOM是区分独立需求与相关需求的关键——独立需求由MPS层面管理，相关需求则通过BOM展开由MRP自动计算。

## 关键概念

### 虚拟件（Phantom）

虚拟件是BOM中一种特殊的物料类型，用于简化BOM结构管理。

| 作用 | 说明 |
|------|------|
| 简化BOM管理 | 多个BOM中相同子件重复出现时，用虚拟件封装减少定义工作量 |
| 简化工程变更 | 虚拟件子件变更只影响虚拟件层，不影响上层 |
| 一般性业务管理 | 组合采购、组合存储、组合发料 |

> **虚拟件关键特性**：不存在任何提前期，在MRP展开时直接跳过计算下层计划需求量。

### BOM层次结构

BOM采用层次码表示物料在产品结构中的位置：
- **第0层**：最终产品（成品）
- **第1层**：直接组成成品的部件
- **第2层**：组成部件的组件
- **第N层**：原材料或采购件

## 实际应用

### BOM在ERP流程中的位置

```
销售订单/预测
     ↓
  MPS（主生产计划）  ←── [[05-主生产计划]]
     ↓
  BOM（物料清单展开）  ←── 本页
     ↓
  MRP（物料需求计划）  ←── [[06-物料需求计划]]
     ↓
  采购计划 / 生产作业计划
```

### BOM维护要点

1. **准确性**：BOM必须与实际产品结构一致，是MRP正确运算的前提
2. **及时性**：工程变更后需同步更新BOM
3. **完整性**：所有物料（含损耗率、替代件）都应体现在BOM中
4. **版本管理**：不同产品版本需要对应的BOM版本
