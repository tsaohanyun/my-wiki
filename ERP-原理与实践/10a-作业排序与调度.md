# 作业排序与调度

## 概述

作业排序是将生产计划转变为每台设备、每个人员每天工作任务的过程，是车间管理的核心技术环节。生产作业控制（PAC）则是对整个产品生产过程的控制和管理工作。本页聚焦作业排序规则、排序方法、派工单以及生产调度与控制。

> 📎 车间任务管理、单据体系、领料方式等基础内容请参见 → [[10-车间管理]]

## 核心内容

### 一、作业排序

#### 作业排序分类

| 分类维度 | 类型 |
|----------|------|
| 机器种类 | 单台机器排序 / 多台机器排序 |
| 加工路线 | 单件车间排序（路线不同） / 流水车间排序（路线相同） |
| 工件到达 | 静态排序（全部到达） / 动态排序（陆续到达） |
| 目标函数 | 单目标排序 / 多目标排序 |

#### 三种主要排序方法

##### 1. 约翰逊法（n/2/P/F_max）

适用于两台机器的流水车间排序，目标是使全组零件具有最短生产周期。

**操作步骤**：
```
1. 列出零件组的工序矩阵
2. 选出加工时间最短的工序
   - 若属于第一工序 → 排在前面
   - 若属于第二工序 → 排在后面
3. 将已排序的工件从矩阵中消去
4. 重复以上步骤，直至所有工件排序结束
```

##### 2. 关键工序法

**操作步骤**：
```
1. 按工序汇总各零件加工工作量，定义工作量最大的工序为关键工序
2. 比较首尾两道工序大小，分三组：首<尾（第一组）、首=尾（第二组）、首>尾（第三组）
3. 各组内分别排序
   - 第一组：关键工序前各工序相加，按递增排序
   - 第二组：根据第一组和第三组的人数决定规则
   - 第三组：关键工序后各工序相加，按递减排序
4. 总排序：第一组 → 第二组 → 第三组
```

##### 3. 优先规则法

| 规则 | 缩写 | 说明 |
|------|------|------|
| 先到先服务 | **FCFS** | 优先安排最先到达的工件 |
| 最短加工时间 | **SPT** | 优先选择加工时间最短的工件 |
| 最长加工时间 | LPT | 优先选择加工时间最长的工件 |
| 最早交货期 | **EDD** | 优先选择交货期最近的工件 |
| 最小紧迫系数 | SCR | 紧迫系数=允许停留时间/剩余加工时间 |
| 最多剩余加工 | MWKR | 优先选择剩余加工时间最长的工件 |
| 最少剩余加工 | LWKR | 优先选择剩余加工时间最短的工件 |
| 最多剩余工序 | MOPNR | 优先选择剩余工序最多的工件 |
| 随机选择 | RANDOM | 随机挑选下一个工件 |
| 后到先服务 | LCFS | 优先安排后到达的工件 |

**FCFS/SPT/EDD 三大核心规则对比：**

| 规则 | 优化目标 | 优点 | 缺点 |
|------|----------|------|------|
| FCFS | 公平性 | 简单公平，易于实施 | 整体效率低，平均流程时间长 |
| SPT | 平均流程时间最短 | 平均等待时间最小，吞吐量高 | 长工件可能长期等待（饥饿效应） |
| EDD | 最大延迟最小化 | 保证交期，延迟最少 | 不考虑加工时间，效率不一定最优 |

### 二、生产作业控制（PAC）

生产作业控制是对整个产品生产过程的控制和管理工作，处于计划执行与控制层。

#### 控制三要素

| 要素 | 说明 |
|------|------|
| **控制标准** | 生产计划、MRP零部件投入产出计划、车间作业计划 |
| **信息** | 实际生产进度与计划偏离的信息 |
| **措施** | 对偏差做出的纠偏措施（通过调度实行） |

#### 投入/产出进度控制

| 指标 | 说明 |
|------|------|
| 计划投入 | 工作中心的计划订单与已下达订单所需工时 |
| 实际投入 | 工作中心实际接收任务的工时 |
| 累计投入偏差 | 实际投入 - 计划投入 |
| 计划产出 | 计划要求完成任务的工时 |
| 实际产出 | 实际完成的工时 |
| 累计产出偏差 | 实际产出 - 计划产出 |

## 关键概念

| 术语 | 英文 | 说明 |
|------|------|------|
| 作业排序 | Job Sequencing | 确定工件在机器上的加工顺序 |
| FCFS | First Come First Served | 先到先服务规则 |
| SPT | Shortest Processing Time | 最短加工时间规则 |
| EDD | Earliest Due Date | 最早交货期规则 |
| 约翰逊法 | Johnson Algorithm | 两台机器流水车间最优排序方法 |
| 生产作业控制 | PAC | 对整个产品生产过程的控制管理 |
| 投入/产出控制 | I/O Control | 衡量执行情况的方法 |
| 派工单 | Dispatch List | 面向工作中心的任务说明文件 |
| 流程时间 | Flow Time | 从可以开始加工至完工的时间 |

### 作业排序评价标准

| 标准 | 说明 |
|------|------|
| 工件流程时间 | 从可以开始加工至完工的时间 |
| 全部完工时间 | 完成一组工作所需的全部时间 |
| 延迟 | 比预定完工时间延迟的时间 |
| 在制品库存（WIP） | 正在加工或等待加工的物料 |
| 总库存量 | 计划入库量+现有库存量 |
| 利用率 | 有效生产时间占总工作时间的百分比 |

### 核心公式

| 公式 | 表达式 |
|------|--------|
| 该工序结束时的流程时间 | `该工序开始时间 + 该工序加工时间` |
| 平均流程时间 | `总流程时间 / 工作数` |
| 使用率 | `总加工时间 / 总流程时间` |
| 平均在制品库存 | `总流程时间 / 总加工时间` |
| 平均拖期量 | `总拖期量 / 工作数` |

## 实际应用

### 优先规则选择指南

| 场景 | 推荐规则 | 理由 |
|------|----------|------|
| 客户公平性要求高 | FCFS | 先到先服务，公平透明 |
| 产能紧张、追求高吞吐 | SPT | 平均等待时间最短 |
| 交期承诺严格 | EDD | 最大延迟最小化 |
| 混合目标 | SPT + EDD 组合 | 先按EDD分组，组内按SPT排序 |

### 调度方法应用

实际生产中，调度通常采用以下策略组合：

1. **前向调度**：从当前时间开始，按工序顺序逐一向后排程
2. **后向调度**：从交货期开始，按工序逆序向前排程
3. **双向调度**：从瓶颈工序两端同时排程，在瓶颈处汇合
4. **有限能力调度**：考虑工作中心实际产能约束
