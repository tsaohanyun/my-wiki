# 第11章 财务管理

> 项目：**ERP知识库**

## 概述

会计和财务管理的对象是企业资金流，是企业运营效果和效率的衡量与表现。在ERP系统中，财务管理始终是核心的应用模块之一，它与系统中其他模块有相应的接口，能够相互集成。国内外主要ERP供应商（SAP、Oracle、用友网络、金蝶国际等）都提供了功能强大、集成性好的财务应用系统。

ERP系统中的财务管理与一般财务软件不同，能将生产活动、采购活动输入的信息自动计入财务模块生成总账和会计报表，取消了手工凭证输入的繁琐过程，实现了物流、资金流、信息流的统一。

## 核心内容

### 财务会计与管理会计

| 维度 | 财务会计 | 管理会计 |
|------|----------|----------|
| 服务对象 | 企业外部（股东、债权人等） | 企业内部各级管理人员 |
| 信息类型 | 货币性数据 | 货币与非货币性数据 |
| 时间导向 | 过去已发生的交易 | 未来将发生的作业与决策 |
| 依据标准 | 一般公认会计原则 | 企业计划、控制与决策需要 |
| 报表形式 | 法定形式（资产负债表、利润表、现金流量表） | 企业自定义形式 |
| 法律约束 | 受法律限制 | 不受法律限制 |

### 财务管理的三大功能

1. **财务计划**：根据前期财务分析做出下期的财务计划、预算等。预算管理提供预算编制、控制和执行分析功能。
2. **财务分析**：运用各种专门分析方法（指标分析、报表分析、计划分析、现金收支分析、因素分析等），对财务数据做进一步加工。
3. **财务决策**：财务管理的核心部分，包括资金筹集、投放及资金管理。

### 总账管理

总账管理是整个会计核算的核心，以凭证处理为主线，提供凭证处理、预提摊销处理、自动转账、调汇、结转损益等功能。关键业务流程包括：

- **制作凭证**：收集原始凭证（发票、入库单、出库单、收款单等），编制记账凭证
- **凭证记账**：登记总账、明细账和日记账，处理往来账和银行账
- **银行对账**：企业银行存款日记账与银行对账单核对，编制银行余额调节表

### 应收账管理

应收账是企业因对外销售商品而应向购货方收取的款项。ERP应收账款管理的核心功能：

- 改善发票和付款处理精度
- 及时提供客户对账单
- 改善客户查询响应
- 与销售完全集成，有效进行成本控制与现金预测
- 账龄分析、回款分析、信用管理

### 应付账管理

应付账是企业因购买材料、商品和接受劳务供应等应支付的款项。ERP应付账管理的核心优势：

- 与采购模块、库存模块完全集成
- 减少处理应付账款的时间
- 改进现金支付的控制
- 提高商业信用，及时获取折扣

### 工资管理

面向企业人力资源人员，支持多种模式的薪酬核算管理。主要功能：

- 支持多种工资核算模式（计时、计件、绩效浮动工资等）
- 工资分次发放和银行代发业务
- 个人所得税计算、申报、代扣代缴
- 与人力资源管理子系统集成

### 固定资产管理

以固定资产卡片管理为基础，实现固定资产的全面管理。四种主要折旧方法：

| 折旧方法 | 核心公式 | 适用场景 |
|----------|----------|----------|
| 平均年限法（直线法） | 年折旧额 =（原值-净残值）÷ 折旧年限 | 各期使用程度均衡的资产 |
| 工作量法 | 每单位折旧额 =（原值-净残值）÷ 可工作总量 | 使用程度不均衡的资产 |
| 双倍余额递减法 | 年折旧率 =（2÷预计使用年限）×100% | 早期效益高的资产 |
| 年数总和法 | 年折旧率 = 尚可使用年限 ÷ 年限合计 | 加速折旧需求的资产 |

> **会计准则要点**：当月增加的固定资产当月不提折旧；当月减少的固定资产当月应提折旧。

### 现金管理

处理企业日常出纳业务，包括现金业务、银行业务、票据管理等。核心功能：

- 银行存款及现金对账
- 原币、本位币金额同时统计
- 支票管理（购置、领用、报销、作废、审核、核销）
- 自动生成凭证并传递到总账

### 财务分析方法

| 分析方法 | 说明 |
|----------|------|
| 结构分析 | 各组成部分占总体的比重分析 |
| 比较分析 | 同一口径指标在不同会计期间的增减比较 |
| 趋势分析 | 同一指标在时间上的变化趋势（绝对数/相对数） |
| 定基分析 | 各期与指定基期相比的变动趋势 |
| 环比分析 | 各会计期间指标与上期相比的发展趋势 |
| 因素分析 | 对选定因素（收入、利润、成本构成等）进行分析 |

## 关键概念

- **财务会计（Financial Accounting）**：主要为企业外部提供财务信息的会计事务
- **管理会计（Management Accounting）**：主要为企业内部提供管理决策信息的会计事务
- **总账（General Ledger）**：会计核算的核心，统率其他各部分功能
- **应收账款（Accounts Receivable）**：企业因销售应收取的款项
- **应付账款（Accounts Payable）**：企业因采购应支付的款项
- **银行余额调节表**：核对企业与银行未达账项后编制的调节表
- **账龄分析**：按应收账款挂账时间长短进行分类分析

## 实际应用

### 案例：联想集团ERP财务管理实践

**1. 准确的成本核算**：ERP实施后，财务能监控销售、采购、库房、生产的全部过程。采购订单发送时财务同步获取信息，库房收货后自动产生应付账并按订单价格计入成本。BOM的严格维护确保了成本的准确、实时核算。

**2. 有效的风险防范**：实施"计提两金"政策——对库存按存货期长短计提销价准备金，对应收账款按账龄长短计提坏账准备金。实施ERP后积压损失从总收入的2%以上降至0.19%。

**3. 严格的资金管理**：系统自动执行信用额度审核，支持不同的还款期和奖励比例，减少了人为控制的难度和随意性。

**4. 精准的预算管理**：财务能实时对照、检查各科目费用与预算的差异，业务部门负责人可在任何一天获得截至当时的收入、毛利、净利粗报表。

**5. 快速的财务报表**：月报表在月末结账后0.5个工作日完成（此前需30天），集团合并报表在结算日后第5天完成。

**6. 财务信息化的四个特点**：集成的、共享的、实时的、精准的。
