# 第3章 需求管理

## 概述

需求管理是ERP计划体系的起点，一切生产活动都围绕需求展开。本页涵盖需求管理的基础概念（物料编码、时间定义、安全库存）以及销售预测方法。有效的需求管理能够提高预测准确性、提升及时交付率、降低库存投资并提高库存周转率。

> 📎 销售业务流程、订单管理和信用管理等内容请参见 → [[03a-销售与订单管理]]

## 核心内容

### 3.1 相关基础概念

#### 3.1.1 物料编码

物料编码是计算机系统对物料的**唯一识别代码**，用于识别和检索企业生产经营过程中所需的各种物料。

**物料编码体系设计原则：**

- 信息分类编码体现科学化、标准化、规范化、合理化
- 参照国家标准中有关分类标准体系
- 参照企业原有编码体系，考虑企业习惯
- 保证编码的**唯一性**、可扩展性和方便性

**物料分类：**

| 分类 | 编码方式 | 示例 |
|------|----------|------|
| 采购件 | 大类+中类+小类+顺序号 | A0101-00007（大型型钢） |
| 自制件 | 图号属性+顺序码 | BZJ-02-0005（包装机部件零件） |

**物料主文件信息类别：**

| 信息类别 | 包含内容 |
|----------|----------|
| 技术资料信息 | 物料名称、规格、图号/配方、计量单位、工艺路线等 |
| 库存信息 | 物料类型、安全库存量、ABC码、批量规则、盘点周期等 |
| 计划管理信息 | 计划属性、需求标志、需求时界、计划时界、提前期等 |
| 采购管理信息 | 订货点数量、主供应商、次供应商等 |
| 销售管理信息 | 信用周期、信用额度、销售类型等 |
| 财务有关信息 | 标准成本、计划价、成本核算方法等 |
| 质量管理信息 | 检查标志、检查方式、批号、最长保存期等 |

#### 3.1.2 时间定义

**时段（Time Bucket）：**

- 定义：计划的时间粒度单位（天、周、旬、月、季、年）
- 原则：时段粒度越小，计划描述、执行和控制越准确
- 应用：远期用年、中期用月、短期用天或周

**提前期（Lead Time）：**

| 类型 | 定义 |
|------|------|
| 生产准备提前期 | 生产计划开始到生产准备完成 |
| 采购提前期 | 采购订单下达到物料完工入库 |
| 生产加工提前期 | 生产加工投入到生产完工入库 |
| 装配提前期 | 装配投入到装配完工 |
| 累计提前期 | 采购+加工+装配提前期之和 |
| 总提前期 | 产品整个生产周期（含设计、试车、检测、发运等） |

> **关键点**：累计提前期由系统根据BOM结构层次自动逐层滚动累加生成。

**计划展望期（Planning Horizon）：**

- 定义：计划的时间跨度，必须大于产品的累计提前期
- 特点：按不同产品设置不同的计划展望期（区别于传统管理的统一时间长度）

#### 3.1.3 安全库存

安全库存是为了应对意外事故、保证生产平稳运行而设置的物料库存水平。

- **安全库存过高**：订单按时供应有保障，但占用大量流动资金
- **安全库存过低**：降低库存持有量，但缺货概率大增
- **确定方法**：根据销售/生产数据及库存状态数据，选择适当预测模型

### 3.2 销售预测

#### 影响销售预测的因素

| 类别 | 因素 |
|------|------|
| **不可控因素** | 需求动向（流行趋势、人口迁移）、经济形势变动、同业竞争动向、政府/消费者团体动向 |
| **可控因素** | 营销活动策略（产品、价格、渠道、广告）、生产状况 |

#### 预测分类

**按预测范围：**
- 宏观市场需求预测 → 行业需求预测 → 企业需求预测

**按期间长短：**

| 预测类型 | 用途 | 时间范围 |
|----------|------|----------|
| 长期预测 | 工厂扩展与设备添置 | 5年或更早 |
| 中期预测 | 设备、资金等资源准备 | 提前1～2年 |
| 短期预测 | 确定订货量与订货时机 | 提前3～6个月 |
| 近期预测 | 每日/每周采购、生产进度 | 周/日 |

**按定性/定量：**

| 类型 | 方法 | 优点 | 缺点 |
|------|------|------|------|
| 定性预测 | 德尔菲法、头脑风暴法、经验估计法 | 灵活，省时省费 | 主观性强，缺乏数量精确描述 |
| 定量预测 | 时间序列分析、因果回归法 | 数量精确，客观依据 | 机械，对数据质量要求高 |

#### 预测模型

**移动平均法公式：**

```
销售量预测值(Q) = 最后n期销售量之和 / n
```

**【例3-1】** 已知1～9月销售量，n=3，求10月预测值：
```
10月预测 = (620 + 610 + 630) / 3 = 620(kg)
```

**指数平滑法公式：**

```
F(t+1) = α × A(t) + (1-α) × F(t)
```

其中：α为平滑指数（0<α<1），A(t)为t期实际值，F(t)为t期预测值

**α值选取原则：**

| 市场特征 | α取值范围 | 说明 |
|----------|----------|------|
| 趋势稳定 | 0.4～0.6 | 观察值权数接近 |
| 季节性变动明显 | 0.6～0.9 | 近期数据权重大 |
| 趋势变动缓慢 | 0.1～0.4 | 远期特征也要反映 |

**【例3-2】** α=0.4，9月实际630kg，9月预测608kg：
```
10月预测 = 0.4 × 630 + (1-0.4) × 608 = 616.8(kg)
```

## 关键概念

| 术语 | 英文/缩写 | 定义 |
|------|-----------|------|
| 物料编码 | Material Code | 计算机系统对物料的唯一识别代码 |
| 时段 | Time Bucket | 计划的时间粒度单位 |
| 提前期 | Lead Time | 从工作开始到结束的时间周期 |
| 计划展望期 | Planning Horizon | 计划的时间跨度，须大于累计提前期 |
| 安全库存 | Safety Stock | 应对意外事故的库存水平 |
| 移动平均法 | Moving Average | 用最近n期实际数据的算术平均进行预测 |
| 指数平滑法 | Exponential Smoothing | 加权移动平均，近期权重大、远期权重小 |
| ATP | Available to Promise | 可承诺量 |

## 实际应用

### 预测方法选择建议

- **数据量少、缺乏历史统计**：选用定性预测（德尔菲法、经验估计法）
- **数据量充足、趋势稳定**：选用移动平均法
- **数据量充足、趋势变动明显**：选用指数平滑法（较大α值）
- **需要长期规划**：结合定量与定性方法
