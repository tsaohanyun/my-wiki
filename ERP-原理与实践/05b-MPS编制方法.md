# MPS编制方法

## 概述

MPS编制是将生产规划转化为具体产品在具体时段生产计划的过程。本页详细介绍MPS编制的基础概念（时区与时界、工作日历）、相关术语（毛需求、净需求、ATP等）、完整的计算流程以及实际案例。这些内容是[[06-物料需求计划|MRP]]运算的核心前置知识。

## 核心内容

### 时区与时界

#### 两个时界点

| 时界 | 英文 | 对应提前期 | 含义 |
|------|------|------------|------|
| **需求时界（DTF）** | Demand Time Fence | 总装提前期 | 此界限内计划已在进行最后总装，一般不能改变 |
| **计划时界（PTF）** | Planned Time Fence | 累计提前期 | 此界限内采购/生产计划已开始执行，只能由计划员控制修改 |

#### 三个时区

| 时区 | 时间范围 | 计划修改权限 | 需求确定方式 |
|------|----------|-------------|-------------|
| **需求时区** | 当前 → DTF | 不能由系统自动改变 | 主要依据客户订单 |
| **计划时区** | DTF → PTF | 只能由计划人员/管理人员改变 | 订单量和预测量取较大值 |
| **预测时区** | PTF → 展望期末 | 系统可自动调整 | 主要依据预测量 |

#### 时区与计划展望期示意

```
|←——————————————— 计划展望期 ————————————————→|
|  预测时区  |    计划时区    |  需求时区  |
         PTF              DTF         当前
    (采购提前期)    (总装提前期)    (当前时刻)
```

#### 时区与多订单计划的关系
- **需求时区内**：订单已进入生产总装阶段
- **计划时区内**：订单已在采购过程中
- **预测时区内**：订单尚在计划期内

### 工作日历

工作日历（工厂生产日历）在自然日历基础上考虑生产安排，标明生产日期、休息日期和利用率。

**工作日历四大作用：**
1. 确定MPS/MRP中提前期计算的开工日期和完工日期
2. 计算工作中心产能负荷的日期基础
3. 车间排产的日期依据
4. 采购接收日期安排

> **注意**：采购计划不受工作日历限制（采购周期与企业工作日历无关），但到货日期一般确定在工作日。

### MPS相关概念

| 概念 | 定义 | 计算方法 |
|------|------|----------|
| **预测量** | 销售预测确定的市场需求量 | 来自销售预测单 |
| **订单量** | 客户下达订单确定的需求量 | 总订单量 - 已出库/已分配量 |
| **毛需求量** | 未扣除库存和预计入库的需求 | 按时区规则确定（见下） |
| **计划接收量** | 前期已下达执行中的订单产出 | 已执行订单将在计划日期后到达的数量 |
| **预计可用库存量** | 扣除需求后的库存量 | 前期末可用库存 + 本期计划接收量 - 本期毛需求 + 计划产出量 |
| **净需求量** | 扣除库存和预计入库后的需求差 | 本期毛需求 - 前期末可用库存 - 本期计划接收 + 安全库存 |
| **计划产出量** | 根据批量策略计算的供应数量 | 净需求量按批量规则取整 |
| **计划投入量** | 根据提前期和成品率计算的投入 | 由计划产出量倒推时间和数量 |
| **可供销售量（ATP）** | 产出超出订单的可自由支配数量 | 计划产出 + 计划接收 - 下次产出前合同量之和 |

#### 毛需求量按时区确定规则

| 时区 | 毛需求量计算 |
|------|-------------|
| 预测时区 | 毛需求量 = 预测量 |
| 计划时区 | 毛需求量 = max(预测量, 订单量) |
| 需求时区 | 毛需求量 = 订单量 |

#### 批量规则

| 方法 | 说明 | 适用场景 |
|------|------|----------|
| 直接批量法 | 批量 = 净需求量 | 价值高、不允许多产/保存的物料 |
| 固定批量法 | 批量 = 固定值的倍数 | 单次订货费用较大的物料 |
| 经济批量法 | 订购费+保管费最低时的批量 | 需求均衡、成本已知且常量 |

## 关键概念

### MPS计算流程

**完整计算流程：**

```
1. 设置计划展望期，划分时段、时区
2. 填写各时段的预测量和订单量
3. 计算毛需求量（按时区规则）
4. 确定计划接收量和期初库存量
5. 计算预计可用库存量
6. 计算净需求量和计划产出量
7. 根据提前期和成品率计算计划投入量
8. 计算可供销售量（ATP）
```

### 详细计算逻辑

**步骤3-6的计算逻辑：**

```
预计库存初值 = 前一期末库存 + 本期计划接收量 - 本期毛需求量

如果 预计库存初值 ≥ 安全库存:
    净需求量 = 0
    本期无计划产出量
    预计（期末）库存 = 预计库存初值
否则:
    净需求量 = 安全库存 - 预计库存初值
    计划产出量 = 按批量规则对净需求量取整
    预计（期末）库存 = 计划产出量 + 预计库存初值
```

**步骤7：计划投入量**
- 根据提前期前移：提前期为1个时段，则投入时间比产出时间提前1个时段
- 考虑成品率：计划投入量 = 计划产出量 / 成品率

**步骤8：可供销售量（ATP）**

```
某时段ATP = 该时段计划产出量 + 该时段计划接收量
           - 下次出现计划产出量之前各时段订单量之和
```

> **注意**：ATP可能为负数（因部分订单由前期库存满足），销售人员关注的是**累计ATP**。

## 实际应用

### 计算案例

**假设条件：**
- 期初库存量：16
- 安全库存量：5
- 固定批量：50
- 提前期：1周（=1个时段）
- 成品率：1

**应用步骤：**
1. 设置计划展望期，划分时段（如6周）和时区（DTF=2周，PTF=4周）
2. 填入各周预测量和订单量
3. 按时区规则计算毛需求量（需求时区取订单量，计划时区取max，预测时区取预测量）
4. 从第1周开始逐周计算：
   - 预计库存初值 = 前期末库存 + 计划接收 - 毛需求
   - 若初值 < 安全库存 → 触发净需求 → 按固定批量50产出
   - 若初值 ≥ 安全库存 → 本期无需产出
5. 计划投入量前移1个时段（提前期=1周）
6. 计算各周ATP，供销售部门承诺交货

### MPS编制常见问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 库存积压 | 预测量偏高或批量过大 | 调整预测方法，优化批量策略 |
| 频繁缺料 | 时界设置不合理或提前期不准 | 修正提前期数据，合理设置DTF/PTF |
| 计划频繁变动 | 需求波动大且时界过短 | 适当延长计划时界，增加计划稳定性 |
| 产能不匹配 | MPS未考虑资源约束 | 在MPS层增加粗能力计划(RCCP) |
